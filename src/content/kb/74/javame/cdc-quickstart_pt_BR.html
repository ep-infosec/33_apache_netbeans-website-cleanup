Guia de Início Rápido do Desenvolvimento de CDC em Java ME do NetBeans

Introdução

As aplicações CDC (Configuração do Dispositivo Conectado) são projetados para serem executados em dispositivos móveis e incorporados com pelo menos 2MB de memória. Esta configuração suporta um JVM com mais funcionalidades do que os telefones celulares com base em MIDP, que geralmente possuem muito menos memória e utilizam Configuração de Dispositivos Limitados Conectados (CLDC). Este tutorial mostra como utilizar o NetBeans IDE para criar uma aplicação CDC da Plataforma Java, Micro Edition (plataforma Java ME). Mostramos como criar um projeto CDC Java ME que exibe um form simples em um emulador de dispositivo para os três perfis CDC disponíveis. Este documento destina-se a ajudá-lo a começar a criar aplicações o mais rápido possível.

Observação: Se você estiver usando o NetBeans IDE 7.3, consulte o Guia de Início Rápido ao Desenvolvimento no NetBeans 7.3 Java ME CDC.

Requisitos

Para concluir este tutorial, são necessários os seguintes recursos e softwares:

Software ou Recurso Versão Necessária
NetBeans IDE com Java ME 7.4
Java Development Kit (JDK) 7
Oracle Java ME SDK 3.2 ou uma versão mais recente

Plataforma do Emulador de CDC Suportada

O NetBeans IDE suporta Oracle Java ME SDK para a plataforma Windows. O Oracle Java ME SDK permite que você desenvolva as aplicações CDC. Observe que as diferentes plataformas de emulador CDC suportam perfis de desenvolvimento diferentes. O principal perfil de desenvolvimento e emulador que os suporta é o Perfil Pessoal Base (Java ME SDK) (mais informações)

início

Instalando o Oracle Java ME SDK (no Windows)

  1. Vá para a página Download do Java ME SDK.
  2. Clique na opção Aceitar Contrato de Licença.
  3. Faça download de um arquivo do instalador do Oracle Java ME SDK e salve-o em qualquer local de seu computador.
  4. Clique duas vezes no arquivo instalador para executá-lo, e siga as instruções.
    Quando a instalação for concluída, o ícone do Gerenciador de Dispositivo do Java (TM) ME Platform SDK ( ) será exibido na bandeja de sistema do Windows.
início

Instalando a Plataforma CDC do Oracle Java ME

  1. Selecione Ferramentas > Plataformas Java.
    A caixa de diálogo Gerenciador da Plataforma Java será exibida.

  2. Clique em Adicionar Plataforma.
  3. Na etapa Selecionar Tipo de Plataforma do assistente Adicionar Plataforma Java, selecione Java ME CDC Platform Emulator e clique em Próximo.
    O IDE exibe a etapa Escolher Pasta da Plataforma.
  4. Selecione o diretório em que você instalou a Plataforma Java ME SDK (por exemplo, C:\Java_ME_Platform_SDK_3.4) e clique em Próximo.
  5. No painel Nome da Plataforma do assistente Adicionar Plataforma Java, verifique se a plataforma selecionada na etapa anterior foi escolhida e clique em Finalizar.
    O Oracle Java ME CDC Platform agora está registrado no NetBeans IDE.

  6. Na caixa de diálogo Gerenciador de Plataformas Java, clique em Fechar.
início

Ativando o Java ME

  1. Selecione Ferramentas > Plug-ins na barra de ferramentas principal do IDE.
  2. Na caixa de diálogo Plug-ins, clique na guia Instalado e selecione Java ME na lista de funcionalidades disponíveis.

  3. Clique em Ativar.
    No painel de Boas-vindas da caixa de diálogo do Instalador, clique em Ativar.

  4. Quando a ativação foi concluída com êxito, clique em Finalizar.
  5. Clique em Fechar para fechar a caixa de diálogo Plug-ins.

Agora, seu NetBeans IDE está configurado para o desenvolvimento móvel.

Nas seções seguintes, criamos um projeto CDC usando cada perfil disponível e também usamos o GUI builder do IDE nas aplicações de perfil Pessoal e AGUI para aprimorar a Interface do Usuário.

Criando uma Nova Aplicação CDC: Perfil Pessoal Base

Aqui criaremos uma aplicação CDC para o Perfil Pessoal Base (PBP) 1.1 usando o Oracle Java ME SDK 3.0.

  1. Selecione Arquivo > Novo Projeto (Ctrl+Shift+N) na janela principal.
  2. Em Categorias, selecione Java ME. Em Projetos, selecione Aplicação CDC. Clique em Próximo.
  3. Na página Nome e Local, nomeie o projeto como newcdc e selecione a caixa Criar Classe Principal. Clique em Próximo.
  4. Na página Selecionar Plataforma, deixe as definições default - CDC Java(TM) Platform Micro Edition SDK 3.2 (Plataforma Java), DefaultCdcPbpPhone1 (Dispositivo) e PBP-1.1 (Perfil) - conforme mostrado na seguinte figura.

  5. Clique em Finalizar. O IDE cria uma nova aplicação e abre o form de classe principal.
  6. Na exibição do Código-fonte, localize g.drawString("Hello, World!", 80, 50); e adicione a palavra Java para que leia Hello, Java World!

  7. Para construir o projeto, escolha Executar > Construir Projeto na barra de ferramentas principal.
  8. Escolha Executar > Executar Projeto na barra de ferramentas principal para executar seu projeto.
    O projeto é acionado no emulador e exibe a mensagem "Hello World!" modificada.

Também é possível depurar o projeto, além de criar testes para os projetos CDC usando o framework JUnit.
