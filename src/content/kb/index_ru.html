Учебные карты

Следующие учебные карты обеспечивают для пользователей все ресурсы, необходимые для разработки приложений с помощью IDE NetBeans.

  • Общие сведения о разработке на Java
    Включая основы настройки проектов, советы по редактированию, построению, тестированию и отладке приложений Java и работы с Maven
  • Интеграция со внешними средствами и службами
    Включает в себя интеграцию с базами данных и использование систем управления исходным кодом
  • Приложения на Java с графическим интерфейсом
    Включает в себя конструктор GUI для Java и интернационализацию графических интерфейсов Java
  • Приложения Java EE и веб-приложения Java
    Включает в себя веб-технологии Java, Enterprise JavaBeans, сохранение состояний Java и веб-платформы, например JavaServer Faces, Struts, GWT и Spring
  • Веб-службы
    Включает в себя программное обеспечение как службы (SaaS) и веб-службы с поддержкой принципа REST
  • Платформа NetBeans (RCP) и разработка модулей
    Включает в себя подключаемый модуль NetBeans и разработку приложений рабочей среды на основе Swing
  • Приложения на C/C++
    Включает в себя создание проектов, помощь при работе с кодом и отладку.
  • Приложения PHP и HTML5
    Включая редактирование PHP, JavaScript, CSS и приложения отладки
  • Мобильные и встроенные приложения
    Включает в себя поддержку приложений Java ME MIDP/CLDC и CDC, а также Java-карты.
