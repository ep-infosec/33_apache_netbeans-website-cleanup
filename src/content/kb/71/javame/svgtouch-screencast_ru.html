Видео создания сенсорного пользовательского интерфейса SVG для устройств Java ME

В этой экранной демонстрации описывается разработка приложения, работающего на большинстве сенсорных устройств на основе Java ME, использующих виджеты пользовательских интерфейсов масштабируемой векторной графики (SVG) в IDE NetBeans.

Эта демонстрация основана на учебном курсе Создание сенсорного пользовательского интерфейса SVG для устройств Java ME, в котором описываются все предварительные условия, необходимые для создания данного приложения.
