Camins d'aprenentatge

Els següents camins d'aprenentatge us proporcionen qualsevol cosa que necessiteu conèixer per a desenvolupar utilitzant l'IDE NetBeans.

  • Desenvolupament general Java
    Incloent la base de la configuració d'un projecte, trucs per l'edició, construcció, testeig, i la depuració d'aplicacions Java i el treball amb Maven
  • Integració amb eines i serveis externs
    Incloent l'integració amb el projecte Kenai, la utilització del sistema de gestió del codi i bases de dades
  • Interfícies gràfiques d'usuari de Java i JavaFX
    Incloent l'internacionalització de les interfícies gràfiques d'usuari de Java i el lligament a bases de dades
  • Aplicacions Web de Java
    Incloent els frameworks web com Visual Web JSF, Spring, i Struts
  • Aplicacions de serveis web i EJB
    Incloent els components Java EE que implementen la tecnologia EJB, serveis web, i persistència Java
  • Aplicacions C/C++
    Incloent la creació de projectes, l'assistent de codi, i la depuració
  • Aplicacions PHP
    Incloent l'edició PHP, la depuració, i l'accés a bases de dades
  • Aplicacions per a dispositius mòbils
    Incloent aplicacions per a dispositius de mà i telèfons mòbils
  • Plataforma Netbeans i desenvolupament de mòduls
    Incloent la construcció de mòduls NetBeans i RIA (Rich Internet Applications)
