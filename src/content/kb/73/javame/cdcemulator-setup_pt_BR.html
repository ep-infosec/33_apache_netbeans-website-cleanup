Guia de Configuração da Plataforma do Emulador CDC do NetBeans

Embora o NetBeans ofereça suporte ao desenvolvimento de Configuração de Dispositivo Conectado (CDC) em Java Micro Edition (ME), o IDE não tem a plataforma CDC incluída no instalador que permite a criação de aplicações para dispositivos CDC usando os Perfis Pessoal, Base ou AGUI. Este documento fornece informações sobre download, instalação e registro do Sun Java Toolkit para CDC e outros Software Development Kits (SDK) populares para o desenvolvimento em CDC.

Software Necessário

Para concluir este tutorial, são necessários os seguintes recursos e softwares:

Software ou Recurso Versão Necessária
NetBeans IDE com Java ME Versão 6.9 ou posterior
Java Development Kit (JDK) Versão 6 ou versão 7

Instalando as Plataformas do Emulador

Plataforma do Nokia 80 SDK para SO Symbian, para Java, Perfil Pessoal

Os desenvolvedores dos dispositivos Nokia 80 podem instalar a Plataforma Series 80 SDK para Symbian OS, para Java, Perfil Pessoal, seguindo estas instruções:

  1. Faça o download do SDK da Plataforma Nokia Series 80 para Symbian OS, Java, Perfil Pessoal do Fórum do Desenvolvedor da Nokia (download). O nome do arquivo é S80_DP_2_0_PP_SDK.zip.
  2. Descompacte o arquivo.
  3. Navegue para o local no sistema onde você descompactou o arquivo. Clique duas vezes no ícone do arquivo setup.exe. O assistente do instalador do Nokia abre.
  4. Siga as instruções do assistente para instalar o SDK. O assistente orientará você para selecionar o dispositivo que deseja tornar o default. Selecione o Series80_DP2_0_PP_SDK.
  5. No Menu Principal do IDE, selecione Ferramentas > Plataformas Java.
  6. Clique em Adicionar Plataforma. A caixa de diálogo Adicionar Plataforma Java abre.
  7. Selecione Emulador da Plataforma Java ME CDC e clique em Próximo.
  8. Selecione a pasta que contém a plataforma do SDK. O diretório default é C:\Symbian\S80_DP2_0_PP_SDK. Clique em Próximo.
  9. Se desejar, use a caixa de diálogo Nome da Plataforma para adicionar ou remover bibliotecas de recursos adicionais.
  10. Clique em Finalizar. O IDE registra a plataforma CDC.

CrEme VM para Windows CE

Os desenvolvedores de dispositivos Windows CE podem instalar o NSIcom CrEme VM para Windows CE, seguindo estas instruções:

  1. Faça o download da plataforma CrEme 4.12 no NSIcom (download).
  2. Navegue para o local no sistema onde você fez o download do arquivo CrEmeDevSup410.exe. Clique duas vezes no arquivo.
  3. Siga as instruções do assistente InstallShield NSI.com para instalar o Developer Kit.
  4. No Menu Principal do IDE, selecione Ferramentas > Plataformas Java.
  5. Clique em Adicionar Plataforma. A caixa de diálogo Adicionar Plataforma Java abre.
  6. Selecione Emulador da Plataforma Java ME CDC e clique em Próximo.
  7. Selecione a pasta que contém a plataforma do SDK. O diretório default é C:\Program Files\NSIcom\CrE-ME V4.12.. Clique em Próximo.
  8. Se desejar, use a caixa de diálogo Nome da Plataforma para adicionar ou remover bibliotecas de recursos adicionais.
  9. Clique em Finalizar. O IDE registra a plataforma CDC.

Observação: o JFrame e o GroupLayout não são compatíveis com a licença de avaliação do CrEme, mas o NSI.com oferece um download para compatibilidade com gráficos avançados.
