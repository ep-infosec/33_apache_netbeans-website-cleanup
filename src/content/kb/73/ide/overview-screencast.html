NetBeans IDE 7.3 Overview

Find out what is new and cool in the latest NetBeans IDE release. The NetBeans IDE is an integrated development environment for software developers. It provides the tools you need to create professional desktop, enterprise, web, and mobile applications with the Java language, as well as PHP, HTML5, JavaScript, Groovy and Grails, and C/C++.

NetBeans IDE 7.3 empowers developers to create and debug rich Web and mobile applications using the latest HTML5, JavaScript, and CSS standards. Developers can expect state of the art rich web development experience with a page inspector and CSS style editor, completely revamped JavaScript editor, new JavaScript debugger, and more. Additional highlights available in 7.3 include continued enhancements to the IDE's support for Java, Groovy, PHP, JavaFX, and C/C++.
