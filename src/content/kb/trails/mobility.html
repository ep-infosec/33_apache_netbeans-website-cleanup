Java ME Embedded and Java Card Application Technology Trail

What is a Mobile Application?

Most mobile applications use the Java Micro Edition (Java ME) platform, which was developed for small devices like mobile phones, but is now used on a wide variety of devices. Java ME uses scaled down subsets of Java SE components, virtual machines and APIs. It also defines APIs that are specifically targeted at consumer mobile and embedded devices. NetBeans IDE supports Oracle Java ME Embedded and Java Card platform.
