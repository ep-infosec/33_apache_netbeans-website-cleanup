Handling Eclipse Project Discrepancies

If NetBeans identifies Eclipse project discrepancies while importing an Eclipse project, a warning dialog will be displayed explaining the Eclipse project discrepancies discovered along with the action NetBeans will take in dealing with the Eclipse project discrepancies found. . The warning dialog is informative. You do not need to take any additional action when this warning dialog appears.
