NetBeans Support Resources

When you run into a problem, it's good to know you can turn to experts for help. NetBeans products come with different levels of support that fit any developer environment, from individual developers to large enterprise development teams.

Professional Support for NetBeans Software

The Oracle Java Development Tools Support offering covers NetBeans, Oracle JDeveloper, and Oracle Enterprise Pack for Eclipse.

 

This offering includes:

  • Assistance with service requests 24 hours per day, 7 days a week
  • Access to My Oracle Support and Sun Connection Service, SunSpectrum Member Support Center, SunSolve Online Program (24 x 7 web-based technical support system), including the ability to log service requests online
  • Unlimited incident support
  • Non-technical customer service during normal business hours (e.g., assistance with support identification numbers, assistance with logging into My Oracle Support)
  • (Note: "Upgrades to new releases" is not listed as a component of this offering since these products are freely available.)



$1,200 per user per year

Community Support for NetBeans Software

The NetBeans community offers numerous other, free support resources:

  • Get free tutorials, video demos, and sample projects.
  • Check the NetBeans User FAQs or the NetBeans Module Developer FAQs.
  • Read the extensive built-in documentation in the IDE itself, available under the Help menu.
  • Search the nbusers mailing list archives â€” it is quite possible that your question has already been asked and answered on the list. If not, ask the nbusers list for help (you must subscribe to post).
  • Check the bug-tracking system to see if your problem is caused by a reported issue; if not, enter a new issue for this problem (you must be logged in to enter a new issue).
  • Review the patch information for the current NetBeans release.
  • Check out community training options.
