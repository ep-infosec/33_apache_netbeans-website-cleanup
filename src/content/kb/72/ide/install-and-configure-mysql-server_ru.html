Подготовка

Сервер баз данных MySQL является одним из наиболее популярных среди серверов баз данных с открытым исходным кодом, используемых при разработке веб-приложений.

Рекомендуемая версия продукта: MySQL Server 5.1. Загрузку можно осуществить отсюда. Приведенные в данном документе указания применимы только к операционной системе Windows. Здесь описывается только набор обязательных действий – подробности настройки MySQL не рассматриваются.

Сведения об установке и настройке сервера баз данных MySQL для другой операционной системы приведены здесь.

Начало установки

После завершения загрузки:
  1. На панели "Загрузки" выберите загруженный установочный файл MySQL и нажмите кнопку "Открыть". При этом запустится мастер настройки сервера MySQL. На панели "Приветствие" нажмите кнопку "Далее".
  2. На панели "Тип настройки" выберите параметр "Пользовательская", а затем нажмите кнопку "Далее".
  3. На панели "Пользовательская настройка" нажмите кнопку "Далее".
  4. На панели "Все готово для установки программы" нажмите кнопку "Установить".
  5. На панели "MySQL Enterprise" нажмите кнопку "Далее".

Настройка сервера MySQL

  1. На панели "Мастер завершен" установите флажок в поле "Настроить MySQL Server сейчас" и нажмите кнопку "Готово". При этом запустится мастер настройки экземпляра сервера MySQL.
  2. На панели "Приветствие" нажмите кнопку "Далее".

Определение типа настройки

  1. На панели "Тип настройки" выберите параметр "Подробная настройка". Нажмите кнопку "Далее".
    Примечание. При выборе этого параметра необходимо перейти к панели для установки параметров сети, где будет отключен брандмауэр для порта, используемого сервером MySQL.
  2. Выберите параметр "Компьютер разработчика" и нажмите кнопку "Далее".
  3. Выберите параметр "Многофункциональная база данных" и нажмите кнопку "Далее".
  4. На панели "Параметры табличного пространства InnoDB" выберите диск и файлы данных MySQL. Нажмите кнопку "Далее".
  5. Укажите примерное число параллельных подключений к серверу – выберите пункт "Поддержка принятия решений (DSS)/OLAP". Нажмите кнопку "Далее".

Настройка параметров сети

  1. Установите флажок в поле "Включить использование TCP/IP".
  2. В поле редактирования "Номер порта:" укажите порт подключения. По умолчанию установлено значение 3306; не следует изменять его без необходимости.
  3. Включите для этого порта функцию "Добавить исключение для брандмауэра".
  4. Установите флажок в поле "Включить строгий режим". Нажмите кнопку "Далее".

Настройка набора символов и параметров операционной системы

  1. Укажите набор символов по умолчанию и нажмите кнопку "Далее".
  2. На панели "Установить параметры Windows" установите флажок в "Установить в качестве службы Windows". Для автоматического запуска сервера MySQL установите флажок в поле "Запускать MySQL Server автоматически". Нажмите кнопку "Далее".

Определение параметров безопасности

  1. Выберите параметр "Изменить параметры безопасности".
  2. В поле "Новый пароль root" введите пароль пользователя root (администратора).
  3. В поле редактирования "Подтверждение" повторно введите пароль пользователя root (администратора).
  4. Нажмите кнопку "Далее".

    5. Примечание. Пользователь root - это пользователь, который имеет полный доступ к серверу баз данных MySQL - создание, обновление и удаление пользователей и так далее. Запомните пароль пользователя root (администратора) – он понадобится вам при создании примера базы данных.

Завершение настройки

  1. Нажмите кнопку "Выполнить" для запуска настройки.
  2. При успешном завершении настройки на экран выводится информационное сообщение. Нажмите кнопку "Завершить".
Примечание. Для проверки успешности настройки запустите диспетчер задач. Если в списке процессов присутствует MySQLd-nt.exe, то сервер базы данных функционирует.
