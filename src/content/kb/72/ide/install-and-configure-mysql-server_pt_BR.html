Preparando-se

O servidor de banco de dados MySQL é um dos mais populares servidores de banco de dados de código-fonte aberto geralmente utilizado no desenvolvimento de aplicações Web.

A versão recomendada do produto é MySQL Server 5.1. Os downloads estão disponíveis aqui. As instruções fornecidas no documento atual aplicam-se somente ao sistema operacional Windows. Este documento não aborda os detalhes da configuração do MySQL, descreve somente a sequência de etapas necessária.

Para obter mais informações sobre instalação e configuração do servidor de banco de dados MySQL em outros sistemas operacionais, consulte a Documentação do MySQL.

Iniciando a Instalação

Depois que o download estiver concluído:
  1. No painel Downloads, selecione o arquivo de instalação do MySQL submetido a download e clique em Abrir. O Assistente para Configuração do Servidor MySQL é iniciado. No painel Bem-vindo, clique em Próximo.
  2. No painel Tipo de Configuração, escolha a opção Personalizada e clique em Próximo.
  3. No painel Configuração Personalizada, clique em Próximo.
  4. No painel Pronto para Instalar o Programa, clique em Instalar.
  5. No painel MySQL Enterprise, clique em Próximo.

Configurando o Servidor MySQL

  1. No painel Assistente Concluído, ative a caixa de seleção Configurar Servidor MySQL agora e clique em Finalizar. O Assistente para Configuração da Instância do servidor MySQL é iniciado.
  2. No painel Bem-vindo, clique em Próximo.

Especificando o Tipo de Configuração

  1. No painel Tipo de Configuração, escolha a opção de Configuração Detalhada. Clique em Próximo.
    Observação: a escolha desta opção é necessária para se chegar ao painel para definir as opções de rede onde você desativará o firewall da porta usada pelo servidor MySQL.
  2. Escolha a opção Máquina do Desenvolvedor e clique em Próximo.
  3. Escolha a opção Banco de Dados Multifuncional e clique em Próximo.
  4. No painel Definições do Tablespace InnoDB, escolha a unidade \Arquivos de Dados MySQL\ e clique em Próximo.
  5. Especifique o número aproximado de conexões concorrentes com o servidor - escolha a opção Apoio à Decisão (DSS)/OLAP Clique em Próximo.

Definindo as Opções de Rede

  1. Ative a caixa de seleção Ativar Rede TCP/IP.
  2. Na caixa de edição Número da Porta: especifique a porta de conexão. A definição default é 3306 - não altere-a se não houver um motivo especial para isso.
  3. Ative Adicionar exceção de firewall para esta porta.
  4. Ative a caixa de seleção Ativar Modo Estrito. Clique em Próximo.

Configurando o Conjunto de Caracteres e as Opções do Sistema Operacional

  1. Especifique o conjunto de caracteres default e clique em Próximo.
  2. No painel Definir as opções do Windows, ative a caixa de seleção Instalar como Serviço do Windows. Para que o servidor MySQL seja iniciado automaticamente, ative a caixa de seleção Iniciar o Servidor MySQL automaticamente. Clique em Próximo.

Especificando as Opções de Segurança

  1. Escolha a opção Modificar as Definições de Segurança. !phpuser
  2. Na Nova senha da raiz, digite a senha do usuário-raiz.
  3. Na caixa de edição Confirmar, redigite a senha do usuário-raiz.
  4. Clique em Próximo.

    5. Observação: o usuário-raiz é um usuário que possui acesso total ao servidor de banco de dados MySQL - criação, atualização e remoção de usuários e assim por diante. Guarde a senha-raiz - você precisará dela posteriormente ao criar um banco de dados de amostra.

Concluindo a Configuração

  1. Clique em Executar para iniciar a configuração.
  2. Quando a configuração for concluída com êxito, a mensagem de informação será exibida. Clique em Finalizar.
Observação: para verificar se a instalação foi concluída com êxito, execute o Gerenciador de Tarefas. Se o MySQLd-nt.exe estiver na lista de Processos, o servidor de banco de dados está sendo executado.
