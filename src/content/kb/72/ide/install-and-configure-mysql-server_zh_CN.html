准备工作

MySQL 数据库服务器是最常用的一个开源数据库服务器，通常用于 Web 应用程序开发。

该产品的推荐版本为 MySQL Server 5.1。可以从此处进行下载。当前文档中提供的说明仅适用于 Windows 操作系统。本文档未介绍 MySQL 配置详细信息，它仅介绍了一系列必需的步骤。

有关为其他操作系统安装和配置 MySQL 数据库服务器的信息，请参见 MySQL 文档。

开始安装

完成下载后，请执行以下操作：
  1. 在“下载”面板中，选择已下载的 MySQL 安装文件，然后单击“打开”。将启动 MySQL 服务器安装向导。在“欢迎”面板上，单击“下一步”。
  2. 在“安装类型”面板上，选择“定制”选项，然后单击“下一步”。
  3. 在“定制安装”面板上，单击“下一步”。
  4. 在“准备安装程序”面板上，单击“安装”。
  5. 在 "MySQL Enterprise" 面板上，单击“下一步”。

配置 MySQL 服务器

  1. 在“向导已完成”面板上，选中“立即配置 MySQL 服务器”复选框，然后单击“完成”。将启动 MySQL 服务器实例配置向导。
  2. 在“欢迎”面板上，单击“下一步”。

指定配置类型

  1. 在“配置类型”面板上，选择“详细配置”选项。单击 "Next"（下一步）。
    注：必须选择此选项，才能到达设置网络选项的面板，您将在其中对 MySQL 服务器使用的端口关闭防火墙。
  2. 选择“开发者计算机”选项，然后单击“下一步”。
  3. 选择“多功能数据库”选项，然后单击“下一步”。
  4. 在“InnoDB 表空间设置”面板上，选择相应的驱动器和 MySQL 数据文件，然后单击“下一步”。
  5. 指定与服务器的并发连接的近似数目 - 选择“决策支持 (DSS)/OLAP”选项。单击 "Next"（下一步）。

设置联网选项

  1. 选中“启用 TCP/IP 联网”复选框。
  2. 在“端口号：”编辑框中，指定连接端口。默认设置为 3306 - 如果没有特殊理由更改此设置，请保留其不变。
  3. 选中“为此端口添加防火墙例外”。
  4. 选中“启用严格模式”复选框。单击 "Next"（下一步）。

设置字符集和操作系统选项

  1. 指定默认字符集，然后单击“下一步”。
  2. 在“设置 Windows 选项”面板上，选中“作为 Windows 服务安装”复选框。要自动启动 MySQL 服务器，请选中“自动启动 MySQL 服务器”复选框。单击 "Next"（下一步）。

指定安全选项

  1. 选择“修改安全设置”选项。
  2. 在“新的超级用户口令”中，输入超级用户的口令。
  3. 在“确认”编辑框中，重新键入超级用户的口令。
  4. 单击 "Next"（下一步）。

    5. 注：超级用户是对 MySQL 数据库服务器具有完全访问权限（如创建、更新和删除用户等）的用户。请记住超级用户口令 - 以后在创建样例数据库时，您将需要此口令。

完成配置

  1. 单击“执行”启动配置。
  2. 成功完成配置后，出现相应的信息消息。单击 "Finish"（完成）。
注：要检查是否已成功完成安装，请运行“任务管理器”。如果 MySQLd-nt.exe 位于“进程”列表中，则表示数据库服务器正在运行。
