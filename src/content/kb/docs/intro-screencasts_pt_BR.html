Visão Geral do NetBeans IDE 8.0


 O NetBeans IDE 8.0 oferece suporte total para as últimas tecnologias Java 8 -- Java SE 8, Java SE Embedded 8 e Java ME Embedded 8. O IDE oferece também diversos aprimoramentos novos para Maven e Java EE com PrimeFaces, novas ferramentas para HTML5, especialmente para AngularJS, bem como aperfeiçoamentos para PHP e suporte a C/C++.

Assista ao Vídeo da Visão Geral do NetBeans IDE 8.0 para descobrir as novidades e os destaques desta última release.
