NetBeans IDE 8.0 概述


 NetBeans IDE 8.0 完全支持最新的 Java 8 技术 -- Java SE 8、Java SE 嵌入式 8 和 Java ME 嵌入式 8。IDE 还为 Maven 以及带 PrimeFaces 的 Java EE 提供了一系列新的增强功能；提供了面向 HTML5 的全新工具，尤其适用于 AngularJS；还有针对 PHP 和 C/C++ 支持的改进。
