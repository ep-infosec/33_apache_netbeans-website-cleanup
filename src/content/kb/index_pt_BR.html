Trilhas do Aprendizado

As trilhas do aprendizado a seguir fornecem tudo o que você precisa saber para desenvolver aplicações utilizando o NetBeans IDE.

  • Desenvolvimento Geral em Java
    Incluindo noções básicas de configuração de projeto, dicas de edição, construção, teste e depuração de aplicações Java e trabalhos com Maven
  • Integração com Ferramentas e Serviços Externos
    Incluindo a integração com bancos de dados e uso de sistemas de gerenciamento de códigos-fonte
  • Aplicações de GUI do Java
    Incluindo o Java GUI Builder e GUIs do Java de internacionalização
  • Aplicações Java EE e Java Web
    Incluindo tecnologias Java Web, Enterprise JavaBeans, Java Persistence e frameworks da Web, como JavaServer Faces, Struts, GWT e Spring
  • Web Services
    Incluindo software como um Serviço e Web services RESTful
  • Plataforma NetBeans (RCP) e Desenvolvimento de Módulos
    Incluindo o plug-in NetBeans e o desenvolvimento de aplicações de Desktop do Swing
  • Aplicações C/C++
    Incluindo a criação, assistência de código e depuração do projeto
  • Aplicações em PHP e HTML5
    Incluindo edições de PHP, JavaScript e CSS e a depuração de aplicações
  • Aplicações Móveis e Incorporadas
    Incluindo Java ME MIDP/CLDC e CDC e Suporte a aplicação Java Card.
