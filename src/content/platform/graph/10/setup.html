NetBeans Graph Library - Setup Building Environment

Get the Sources

First you have to check out sources.

  1. Create a directory for NetBeans CVS e.g. called netbeans.
     
  2. Login to NetBeans CVS: 
    cvs -d :pserver:[login-name]@cvs.netbeans.org:/cvs login
    Login-name is your account name on www.netbeans.org. You can also use anonymous access using anoncvs as a login-name.
     
  3. Check out required modules: 
    cvs -d :pserver:[login-name]@cvs.netbeans.org:/cvs co core/external nbbuild openide/util
    cvs -d :pserver:[login-name]@cvs.netbeans.org:/cvs co -r graph1 graph

Compilation

Now you can compile the sources.

  1. Initialize NetBeans environment:
    cd down to nbbuild directory and run ant init
     
  2. Compile openide/util: cd down to openide/util directory and run ant
     
  3. Compile Graph Library:
    cd down to graph/lib directory and run ant
     
  4. Compile Graph Library VMD extension:
    cd down to graph/vmd directory and run ant
     
  5. Compile Graph Library Examples:
    cd down to graph/examples directory and run ant
    Note: Examples will automatically start, so close the dialog.
     
  6. Compile Graph Library WebStart Demo:
    cd down to graph/webstart directory and run ant
Now you have everything compiled.

Java Documentation

  1. Check out required module: 
    cvs -d :pserver:[login-name]@cvs.netbeans.org:/cvs co libs/external
  2. Optionally build openide-util javadoc:
    cd down to openide/util directory and run ant javadoc
     
  3. Build Graph Library javadoc:
    cd down to graph/lib directory and run ant javadoc
Javadocs are build into:
  1. nbbuild/build/javadoc/org-openide-util/ - contains openide utilities javadoc.
  2. nbbuild/build/javadoc/org-netbeans-graph/ - contains the library javadoc.

Distribution Packages

When you compile the library, you have following packages which you can use outside of NetBeans CVS.

Packages:

  1. nbbuild/netbeans/platform6/lib/org-openide-util.jar - contains openide utitities.
  2. nbbuild/netbeans/extra/modules/org-netbeans-graph.jar - contains the library.
  3. nbbuild/netbeans/extra/modules/org-netbeans-graph-vmd.jar - contains the VMD extension.
  4. nbbuild/netbeans/extra/modules/org-netbeans-graph-examples.jar - contains the examples.
  5. nbbuild/netbeans/extra/modules/org-netbeans-graph-webstart.jar - contains the WebStart demo.
  6. graph/www/graph.jnlp - the WebStart demo descriptor.

Compilation of Integration Module

In the CVS there is a graph/integration module. It shows graph library integrated into the NetBeans Platform.

For building the source, you have to check out source of the NetBeans Platform. Then you can compile this module using:
cd down to nbbuild directory and run ant init all-graph/integration

For running the NetBeans, you have to compile it first using:
cd down nbbuild directory and run ant

Then you can run the NetBeans using nbbuild/netbeans/bin/netbeans program. The module adds "Window | Open Graph Window" action into the main menu.
