netbeans.org에 기여

알려진 바와 같이 NetBeans는 오픈 소스 프로젝트입니다. 이 곳의 정보로 상업적 제품을 만드는 회사도 있지만, 일차적으로는 커뮤니티가 중심이 되는 곳입니다. 다시 말하면, 다른 사람에게 도움이 될 만한 지식을 주면 그 자신에게도 혜택이 돌아온다는 것이 핵심입니다. 누군가 버그 보고서 또는 개선 요청을 해서 이것이 이루어지면 해당 툴을 가진 모든 사람에게 혜택이 돌아갑니다. 한 마디로 NetBeans에서는 모두가 주인공입니다. 소프트웨어를 사용하는 사람이 없다면 이 곳의 존재 이유는 없습니다.

여기서 NetBeans 커뮤니티에 기여하는 방법을 소개합니다. 물론, 기여는 보답으로 돌아옵니다. 코드를 기여하는 것 외에도 이 프로젝트에 도움이 되는 방법은 많습니다!

커뮤니티 지원, 홍보

  • 메일링 리스트 참여
    netbeans.org에는 소프트웨어에 관해 전문 지식을 나누는 메일링 리스트 및 뉴스그룹이 있습니다. 이 커뮤니티 리스트는 여러분이 만들어 가는 곳입니다! 다른 사람에게 도움이 되는 지식을 알고 있다면 주저하지 말고 게시하십시오!

  • NetBeans 홍보!
    NetBeans 사용자라면 모두에게 이야기해 주십시오! 치열한 경쟁 속에서 NetBeans를 아예 들어본 적 없거나, 아니면 들어는 봤지만 잘못 알고 있거나 최근의 발전 내용은 모르는 개발자들이 허다합니다. 블로그, 업계 게시판, 잡지, 포럼 등에서 NetBeans의 장단점에 대해 소문을 내십시오. NetBeans에 대한 글이나 리뷰를 보거나 직접 쓰신 적이 있습니까? 꼭 알려 주시기 바랍니다!

컨텐츠 보내기, 번역

  • 웹 사이트 컨텐츠 기여
    까다로운 문제의 해결 방법을 알고 계십니까? 새로운 컨텐츠나 기존 컨텐츠의 업데이트는 언제나 환영입니다! FAQ, HOWTO 같은 글은 상당히 유용하며, 스크린샷과 플래시 동영상도 마찬가지입니다. 자세한 내용은 자습서 기여 지�. 데모 기여 시 이 웹 공간(무료)을 사용하십시오.

  • 번역, 현지화
    웹 사이트 컨텐츠 또는 IDE 자체의 모국어 번역에 흥미를 가지고 계십니까? 저희는 물론이고 현지 국민에게 큰 도움이 될 것입니다! 자세한 내용은 현지화 빠른 시작 페이지를 참조하십시오. 이미 다양한 언어 포럼을 만들어 Java를 처음 시작하고 NetBeans 이용을 시작한 초보자들에게 도움을 주고 있는 커뮤니티 구성원들도 있습니다.

  • 뉴스레터 편집, Edge 사연 보내기
    주간 발행되는 NetBeans Newsletter는 NetBeans 구성원들이 자발적으로 만드는 뉴스레터입니다. 메일링 리스트의 현재 행사에 참여하면 뉴스레터 편집자가 될 수 있습니다! 월간 Edge 뉴스레터에서는 인터뷰, 기사, 파트너 새 소식을 쓸 사람을 수시 모집합니다. 사연을 보내실 분은 연락 바랍니다. 또한 현지에 NetBeans에 관련된 행사나 회의 소식이 있으면 저희에게 알려 주십시오. 저희 행사 일정에 실어 드리겠습니다.

버그 보고서 제출, 코드 작성

  • 버그 보고서 제공
    버그 보고서는 개발자들에게 상당히 도움이 될 뿐 아니라 훌륭한 프로젝트 기여 방법이기도 합니다! 버그 보고서 제공 시 시간이 조금 걸리더라도 문제를 재현하는 방법과 명확한 설명을 제시해 주셔야 합니다.

  • 개선 요청 보고
    IDE에 있었으면 하는 기능, 개선 가능하다고 생각되는 부분이 있다면 언제든지 알려 주십시오! 개선 요청은 버그와 마찬가지로 문제 추적 시스템에 보고해야 합니다.

  • 코드 및 플러그인 모듈 기여
    위에 언급한 바와 같이 NetBeans는 오픈 소스 프로젝트입니다. 모든 소스 코드가 공개되어 있으며, 어떤 기능이 추가되거나 어떤 버그가 수정되기를 바란다면 다른 사람에게 부탁할 것 없이 직접 하실 수 있습니다. 플러그인 모듈 개발, 패치 기여, 플러그인 모듈 기여에 대해 자세히 알아 보십시오. 직접 만든 플러그인을 Plugin Portal에 게시하고 커뮤니티 동료 구성원들의 사용 의견을 들어 볼 수 있습니다!
