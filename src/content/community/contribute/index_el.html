Συνεισφέροντας στο netbeans.org

Όπως πιθανότατα γνωρίζετε, το NetBeans είναι ένα έργο ανοικτής πηγής. Αυτό σημαίνει ότι παρόλο που εταιρείες μπορεί να δημιουργήσουν εμπορικά προϊόντα από ότι υπάρχει εδώ, είναι πρωταρχικά καθοδηγούμενο από την κοινότητα. Αυτό σημαίνει, για παράδειγμα, άμα μπορείτε να συνεισφέρετε γνώση που βοηθά κάποιον, τότε κερδίζετε εσείς. Είτε είναι αναφορά σφάλματος, είτε είναι αίτημα βελτίωσης, όλοι θα κερδίζουν αφού προκύπτουν καλύτερα εργαλεία όταν οι διορθώσεις ή οι βελτιώσεις εφαρμοστούν. Στην ουσία, όταν πρόκειται για το NetBeans, είσαστε στην θέση του οδηγού - χωρίς ανθρώπους να χρησιμοποιούν το λογισμικό, δεν θα υπήρχε λόγος για τίποτα από αυτά.

Ακολουθούν τρόποι που μπορείτε να συνεισφέρετε στην κοινότητα του NetBeans, οι όποιοι συγκεντρώνουν τις ανταμοιβές τους. Υπάρχουν ποοί τρόποι να βοηθήσετε το έργο ουσιαστικά χωρίς να συνερισφέρετε κώδικα!

Υποστήριξη Κοινότητας, Προώθηση

  • Συμμετοχή σε Λίστες Αλληλογραφίας
    Στο netbeans.org υπάρχουν πολλές λίστες αλληλογραφίας και ομάδες νέων όπου ο καθένας μπορεί να επωφεληθεί από τις γνώσεις κάποιου με την χρήση λογισμικού από at netbeans.org. Για να είστε πραγματικά χρήσιμοι, οι λίστες της κοινότητας χρειάζονται την δική σου γνώμη! Μην διστάσετε να γράψετε - βοηθήστε τους άλλους όταν μπορείτε εσείς!

  • Προωθήστε το NetBeans!
    Άμα είστε χρήστης NetBeans, μιλήστε! Υπάρχει ανταγωνισμός και πολλοί προγραμματιστές δεν έχουν ακούσει για το NetBeans, ή ακόμα χειρότερα, έχουν ακούσει για αυτό αλλά με παρερμηνείες, ή δεν έχουν ακούσει για τις πρόσφατες βελτιώσεις. Βοηθήστε να γίνει γνωστό - μιλήστε για τα ισχυρά σημεία του NetBeans, τις αδυναμίες του, σε blogs, σε επαγγελματικές λίστες αλληλογραφίας, σε περιοδικά και forums. Έχετε βρεί (ή γράψει) κάποιο άρθρο ή αναφορά σχετικά με το NetBeans; Θα θέλαμε πολύ να το διαβάσουμε!

Υποβάλετε Περιεχόμενο, Μεταφράσεις

  • Συνεισφέρετε Περιεχόμενο Ιστότοπου
    Βρήκατε πως να λύσετε ένα use-case; Νέο ή ανανεωμένο περιεχόμενο είναι πάντα ευπρόσδεκτο! Άρθρα τύπου FAQs ή HOWTOs είναι πολύ χρήσιμα, όπως είναι και απεικονίσεις οθόνης (screenshots) και ταινίες Flash. Δείτε τους παρακάτω συνδέσμους για περισσότερες πληροφορίες: Οδηγίες για την συνεισφορά μαθημάτων. Χρησιμοποιείστε αυτό τον χώρο για τις προσφερόμενες επιδείξεις.

  • Μεταφράσεις, Μεταγλωττίσεις
    Ενδιαφέρεστε να μεταφράσετε το περιεχόμενο του ιστότοπου, ή το ίδιο το περιβάλλον εργασίας στην δικιά σας μητρική γλώσσα; Εμείς και οι ομόγλωσσοι σας θα εκτιμήσουμε την βοήθεια! Δείτε τον Γρήγορο Οδηγό Μεταγλώττισης για πληροφορίες. Μερικά μέλη της κοινότητας έχουν δημιουργήσει forums σε διάφορες γλώσσες, και τώρα βοηθούν άλλους να ξεκινήσουν με την Java και το NetBeans.

  • Επεξεργαστείτε το Newsletter, Υποβάλετε Άρθρα για το Edge
    Το εβδομαδιαίο NetBeans Newsletter συντάσσεται από εθελοντές του NetBeans - άμα παρακολουθείτε τα δρώμενα στις λίστες αλληλογραφίας θα μπορούσατε να συνδράμετε στο Newsletter! Το μηνιαίο Edge newsletter αναζητά συνεχώς συντάκτες για να κάνουν μια συνέντευξη, ένα άρθρο... Άμα θέλετε να υποβάλετε ένα κείμενο, παρακαλούμε επικοινωνήστε εδώ. Επίσης άμα έχετε ακούσει για κάποιο γεγονός ή συνέδριο στην περιοχή σας όπου γίνεται χρήση του NetBeans,επικοινωνήστε μαζί μας και θα το δημοσιεύσουμε στο ημερολόγιο γεγονότων.

Υποβολή Αναφορών Σφαλμάτων, Συγγραφή Κώδικα

  • Καταχωρείστε Αναφορά Σφάλματος
    Οι αναφορές Σφαλμάτων είναι πολύ χρήσιμες για τους προγραμματιστές, και ένας ωραίος τρόπος συνεισφοράς στο έργο! Θυμηθείτε να αφιερώνετε αρκετό χρόνο όταν καταχωρείτε μια αναφορά σφάλματος - να είστε σίγουροι ότι μπορείτε να αναπαραγάγετε το πρόβλημα και να είστε πολύ αναλυτικοί στην περιγραφή σας.

  • Καταχωρείστε Αίτημα Βελτίωσης
    Εάν υπάρχει κάτι που θέλετε ή δεν θέλετε να δείτε στο IDE, ή κάτι που νομίζετε ότι χρήζει βελτίωσης ειδοποιήστε μας! Τα αιτήματα βελτιώσεων θα πρέπει να καταχωρούνται στο σύστημα καταγραφής ζητημάτων (issue tracking system), όπως και τα σφάλματα.

  • Συνεισφέροντας Κώδικα και Αρθρωτές Μονάδες (Plugin Modules)
    Όπως προαναφέρθηκε, το NetBeans είναι ένα έργο ανοικτής πηγής - όλος ο πηγαίος κώδικας είναι προσβάσιμος και άν υπάρχει ένα χαρακτηριστικό που θα θέλατε να έχετε ή κάποιο σφάλματα που επιθυμείτε να διορθωθεί, έχετε την ικανότητα να το πραγματοποιήσετε. Μπορείτε να μάθετε περισσότερα για την ανάπτυξη αρθρωτών μονάδων, συνεισφορά σε "μπαλώματα" (patches) και την συνεισφορά αρθρωτών μονάδων. Δημοσιεύστε τα αρθρώματα σας στο Plugin Portal και λάβετε ανάδραση από τα μέλη της κοινότητας!
