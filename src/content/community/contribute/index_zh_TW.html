為 NetBeans.org 作貢獻

您也許知道，NetBeans 是一項開放程式碼計劃。 這意味著公司透過此處提供的資源開發商業產品，而它的發展則主要由社區驅動。 同時，這也意味著，如果您能貢獻知識，協助他人走出困境，您就是對他人有益的人。 如果您提供的是錯誤報告或增強請求，則在應用修復程式或實現增強後，每個人都能得到更好的工具，從中受益。 實際上，對於 NetBeans 而言，您處於主導位置 - 沒有人使用該軟體，一切都無從談起。

您可以透過以下管道為 NetBeans 社區做貢獻，讓整個社區從中受益。 即使並不實際貢獻代碼，您也可以透過多種途徑為該計劃提供幫助！

社區支援與推廣

  • 加入郵件列表
    NetBeans.org 網站上提供了多個郵件列表和新聞群組，讓該軟體的使用者互相交流知識並從中受益。 為了讓這項活動變得真正有益，社區列表需要您的參與！ 請別擔心

  • 推廣 NetBeans！
    如果您是一名 NetBeans 使用者，請大聲說出來！ 外面的競爭激烈，許多開發員未曾聽說過 NetBeans，更糟的是有些人雖然聽說過 NetBeans 但對其有誤解，而有些人則還不知道 NetBeans 的最新改進。 幫忙大力宣傳 - 在部落格、行業郵件列表、雜誌和論壇上談論 NetBeans 的優勢和劣勢。 您發現（或撰寫）過有關 NetBeans 的文章或評論嗎？ 我們很想拜讀一下！

提交內容與譯作

  • 貢獻網站內容
    找到了解決棘手的軟體使用問題的辦法？ 歡迎提供新內容或現有內容的更新！ 諸如「常問問題」和「指南」等文章非常有用，螢幕快照和 Flash 視頻也不錯。 詳細資訊，請訪問以下連結： 關於貢獻教程。 請使用此免費的用於顯示您貢獻的演示文件的 Web 空間。

  • 翻譯、本土化
    您是否有興趣將網站內容或 IDE 本身本土化為您的母語？ 我們和其他開發員都將感激您所提供的協助！ 詳細資訊，請參見本土化快速入門頁面。 一些社區成員已經建立了多種語言版本的論壇，現已開始協助其他人認識 Java 和 NetBeans。

  • 編輯新聞簡訊，向 Edge 提交新聞
    每周一期的NetBeans 新聞簡訊由參與 NetBeans 計劃的志願者們合力編撰 - 如果您正在追蹤郵件列表上的最新事件，則可以編輯一份新聞簡訊！ 每月一期的 Edge 新聞簡訊一直在尋找自願者以撰寫訪談錄、文章、合作夥伴熱點新聞 ... 如果您想提交新聞，請填寫聯絡表。 另外，如果聽到有關所在地區使用 NetBeans 的任何新聞事件或會議，請給我們留言，我們將在事件日曆中公佈出來。

提交錯誤報告、編寫代碼

  • 歸檔錯誤報告
    錯誤報告不僅對於開發員非常有幫助，而且也是為該計劃作貢獻的大好途徑！ 請記住，歸檔錯誤報告時應多花點時間 - 確保可以重現問題並非常明確地予以說明。

  • 歸檔增強請求
    如果想在 IDE 中看到目前所沒有的東西或您覺得能夠帶來改進的東西，請告訴我們！ 與歸檔錯誤的方法相同，應將增強請求歸檔至問題追蹤系統。

  • 貢獻代碼和插件模組
    如前所述，NetBeans 是一項開放程式碼計劃 - 提供所有來源碼，如果想看到某個特性或某個錯誤被修復，您可以馬上將之付諸實現。 您可以瞭解有關開發插件模組、貢獻修補程式和貢獻插件模組的詳細資訊。 請在 Plugin Portal 中發佈您的插件模組，接收社區其他成員的反饋資訊！
