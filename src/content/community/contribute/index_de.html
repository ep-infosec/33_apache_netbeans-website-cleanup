Mitwirken bei netbeans.org

Wie Sie sicherlich wissen ist Netbeans ein Open-Source Projekt. Während viele Firmen kommerzielle Produkte mit Netbeans erstellen, stammt alles was Sie hier finden hauptsächlich von der Community. Wenn Sie etwas von Ihrem Wissen besteuern und damit jemand anderem helfen, profitieren Sie am Ende selbst davon, da auch Ihnen geholfen wird. Wenn Sie Bugs finden oder Verbesserungsvorschläge machen, profitieren alle davon, da wir bessere Software bekommen, wenn die Bugs behoben und die Verbesserungen eingearbeitet werden. Letztendlich bedeutet dass, das Sie bei Netbeans am Steuer sitzen. Ohne die Leute, die die Software benutzen, gäbe es keinen Grund für all das hier.

Hier werden Ihnen einige Wege vorgestellt, wie Sie an der Netbeans Community teilnehmen können. Es gibt viele Möglichkeiten, dem Projekt zu helfen, ohne tatsächlich Code beizusteuern.

Support durch die Community, Promotion

  • Teilnehmen an den Mailinglisten
    Es gibt verschiedene Mailinglisten und Newsgroups auf netbeans.org, wo die Benutzer der Software sich gegenseitig helfen und von den verschiedenen Erfahrungen profitieren können. Um wirklich nützlich zu sein, benötigen diese Listen auch Ihre Beteiligung! Scheuen Sie sich nicht zu Posten, helfen Sie den Anderen wo Sie können!
  • Promoten Sie Netbeans!
    Wenn Sie ein Netbeans-Benutzer sind, erzählen Sie es weiter! Es gibt einen großen Wettbewerb da Draußen und es gibt immer noch einige Entwickler, die noch nie von Netbeans gehört haben. Helfen Sie, Netbeans zu verbreiten - sprechen Sie über die Stärken und Schwächen von Netbeans in Foren, im Blog, auf Mailinglisten oder am Stammtisch.
  • Verlinken Sie Netbeans
    Sie sind Teil der NetBeans-Community! Platzieren Sie ein kleines Netbeans-Icon auf Ihrer Webseite und verlinken Sie es mit netbeans.org.

Verfassen Sie Inhalte oder Übersetzungen

  • Steuern Sie Inhalte für die Webseite bei
    Neue Inhalte oder Updates für bestehende Inhalte unserer Webseite sind immer gerne gesehen. Artikel wie FAQs und HOWTOs sind ebenso wie Screenshots sehr, sehr hilfreich. Folgen Sie den folgenden Links für weitere Informationen: Dokumenten-Wunschliste und Richtlinien zur Teilnahme.
  • Übersetzungen, Lokalisierungen
    Sind Sie daran interessiert, die Webseite oder die IDE selbst in Ihre Muttersprache zu übersetzen? Wir freuen uns sehr über Ihre Hilfe! Lesen Sie Lokalisierung Schnellstart für weitere Informationen.
  • Bearbeiten Sie den Newsletter oder schreiben Sie Nachrichten für The Edge
    Der wöchentliche Netbeans Newsletter wird von verschiedenen an Netbeans beteiligten Freiwilligen zusammengestellt - wenn Sie die aktuellen Ereignisse auf den Mailinglisten verfolgen, könnten auch Sie einen Newsletter schreiben! Der monatliche The Edge Newsletter sucht nach neuen Autoren, die Interviews, Artikel, etc. verfassen. Wenn auch Sie einen Artikel veröffentlichen wollen, nehmen Sie Kontakt auf.

Verfassen Sie Bug-Berichte oder schreiben Sie Code

  • Berichten Sie über einen Bug
    Bug-Berichte sind sehr hilfreich für die Entwickler und ein sehr guter Weg, zum Projekt beizutragen! Bitte denken Sie daran, sich ein wenig Zeit zu nehmen, wenn Sie einen Bug-Bericht ausfüllen. Gehen Sie sicher, dass Sie das Problem reproduzieren können und beschreiben Sie den Fehler sehr ausführlich.
  • Geben Sie einen Verbesserungsvorschlag ab
    Gibt es etwas, dass Sie sich für die IDE wünschen würden, oder etwas, das Sie verbessern könnten? Teilen Sie es uns mit! Verbesserungsvorschläge sollten, ebenso wie Bugs, im Issue-Tracking-System veröffentlicht werden.
  • Bringen Sie Ihren Code ins Projekt
    Wie oben erwähnt, ist Netbeans ein Open-Source-Projekt. Der gesamte Sourcecode ist frei verfügbar, wenn es ein Feature, dass Sie gerne sehen würden, oder einen Bug, den Sie gerne beheben würden gibt, Sie können es möglich machen! Beachten Sie bitte die Informationen zum Beitragen von Patches und Modulen für weitere Informationen.
