程式碼


NetBeans 程式碼是遵循通用開發和發佈授權(CDDL) 要求而發佈。

可以從 Mercurial 儲存庫獲取這些程式碼。NetBeans 的所有開發都利用了本網站上的 Mercurial 伺服器，開發建議和決策均列在公共郵件列表中。

您可以透過多條管道來獲取程式碼。

  • 透過 Mercurial - 詳細資訊，請參見 Mercurial 頁面；
  • 從下載頁面下載已歸檔的快照；
  • 線上瀏覽

如果您有興趣透過程式碼製作軟體的建置，請參見建置/安裝頁面。

此處到底提供哪些內容？


此處提供的內容為 NetBeans IDE 的整個程式碼樹和歷程記錄。通常，有一個「穩定」分支和一個「開發」分支。開發分支是進行持續開發的分支 - 根據開發在發佈周期中所處的具體位置，其穩定性可能或高或低。

有關程式碼在 Mercurial 儲存庫中的物理結構，請參閱 Mercurial 分支和標籤檔案。

此處提供的程式碼完整嗎？還有此處沒有提供的內容？


由於法律方面的原因，某些內容無法在此公佈。一些 NetBeans IDE 部件屬於第三方軟體，或者受授權協定制約，不允許在此公開程式碼。

相關示例如下：

  • Java 編譯器 - NetBeans 中的 Java 剖析器是 Java 語言編譯器 (javac) 的包裝程式。Javac 程式碼遵循 GPL v2 授權證的要求在 OpenJDK 網站上發佈。NB 5.x 及早期版本使用以前版本的 javac 編譯器，遵循更加嚴格的 Sun BCL 授權證。
  • XML 剖析器（Xerces、Crimson、Xalan、DOM、SAX、JAXP...）。- 在 IDE 中用於 XML 的剖析和操控。http://java.sun.com/xml/
  • JavaHelp Runtime library 2.x - IDE 使用 JavaHelp 顯示線上檔案。額外的二進位文件分發可能還包括用於重新建置搜尋資料庫的 JavaHelp 開發庫。http://java.sun.com/products/javahelp/index.html
  • Apache Ant 1.6.x - 基於Java 的建置工具，NetBeans 借助該工具構建系統。http://ant.apache.org/
  • JUnit 3.8.x - 用於在 NetBeans 中執行單元測試的測試框架。http://www.junit.org/index.html
  • Tomcat 5.x.x - 用於 Java Servlet 和 JSP 開發的嵌入式 servlet 容器。http://jakarta.apache.org/tomcat/index.html
  • JSP Standard Tag Library (JSTL) - http://java.sun.com/products/jsp/jstl/

給定 NetBeans 版本包含的未經 CDDL 授權的額外檔案數量不定。請查閱 nbdev 列表進行確認。
