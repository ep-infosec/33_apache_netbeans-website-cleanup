源代码


NetBeans 源代码是遵循 Common Development and Distribution License (CDDL) 的要求而发布。

可以从 Mercurial 存储库获取这些源代码。NetBeans 的所有开发都利用了本站点上的 Mercurial 服务器，开发建议和决策均列在公共邮件列表中。

您可以通过多条渠道获取源代码。

  • 通过 Mercurial - 详细信息，请参见 Mercurial 页面；
  • 从下载页面下载已归档的快照；
  • 在线浏览

如果您有兴趣通过源代码制作软件的内部版本，请参见构建/安装页面。

此处到底提供哪些内容？


此处提供的内容为 NetBeans IDE 的整个源代码树和历史记录。通常，有一个“稳定”分支和一个“开发”分支。开发分支是进行持续开发的分支 — 根据开发在发布周期中所处的具体位置，其稳定性可能或高或低。

有关源代码在 Mercurial 存储库中的物理结构，请参阅 Mercurial 分支和标签文档。

此处提供的源代码完整吗？还有没有此处未提供的内容？


由于法律方面的原因，某些内容无法在此公布。一些 NetBeans IDE 部件属于第三方软件，或者受许可协议制约，不允许在此公开源代码。

相关示例如下：

  • Java 编译器 - NetBeans 中的 Java 分析器是 Java 语言编译器 (javac) 的包装程序。Javac 源代码遵循 GPL v2 许可证的要求在 OpenJDK 站点上发布。NB 5.x 及早期版本使用以前版本的 javac 编译器，遵循更加严格的 Sun BCL 许可证。
  • XML 分析器（Xerces、Crimson、Xalan、DOM、SAX、JAXP...）- 在 IDE 中用于 XML 的分析和操控。http://java.sun.com/xml/
  • JavaHelp 运行时库 2.x - IDE 使用 JavaHelp 显示联机文档。额外的二进制文件分发可能还包括用于重新构建搜索数据库的 JavaHelp 开发库。http://java.sun.com/products/javahelp/index.html
  • Apache Ant 1.6.x - 基于 Java 的构建工具，NetBeans 借助该工具构建系统。http://ant.apache.org/
  • JUnit 3.8.x - 用于在 NetBeans 中执行单元测试的测试框架。http://www.junit.org/index.html
  • Tomcat 5.x.x - 用于 Java Servlet 和 JSP 开发的嵌入式 servlet 容器。http://jakarta.apache.org/tomcat/index.html
  • JSP 标准标记库 (JSTL) - http://java.sun.com/products/jsp/jstl/

给定 NetBeans 版本包含的未经 CDDL 许可的额外文件数量不定。请查阅 nbdev 列表进行确认。
