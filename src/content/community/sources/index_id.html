Sumber


Kode sumber NetBeans tersedia pada Common Development and Distribution License (CDDL).

Kode sumber tersedia dalam repositori Mercurial. Semua pengembangan NetBeans dilakukan menggunakan server Mercurial pada situs ini, dan proposal pengembangan serta keputusannya terjadi pada milis umum.

Anda dapat mengambil sumber tersebut dalam beberapa cara.

  • Melalui Mercurial - lihat halaman Mercurial untuk rinciannya;
  • Download arsip snapshot dari halaman download;
  • Telusuri secara online

Jika Anda tertarik membuat build dari sumbernya, lihat halaman pembangunan/instalasi.

Jadi, tepatnya, apa yang ada di sini?


Apa yang tersedia di sini adalah pohon (tree) sumber dan riwayat IDE NetBeans. Secara umum, ada cabang yang "stabil" dan ada "cabang pengembangan". Cabang pengembangan adalah tempat berlangsungnya pengembangan terus menerus - cabang ini akan menjadi lebih atau kurang stabil tergantung titik siklus rilisnya.

Untuk keterangan mengenai organisasi fisik dari sumber-sumber dalam repositori Mercurial, lihat dokumen Cabang Mercurial dan Labelnya.

Apakah sumber yang ada di sini sudah lengkap? Apakah ada sesuatu yang tidak ada di sini?


Ada beberapa hal yang tidak bisa berada di sini karena alasan hukum. Ada beberapa bagian dari IDE NetBeans yang merupakan perangkat lunak pihak ketiga, atau berada di bawah lisensi yang tidak membolehkan sumbernya dipublikasikan di sini.

Contoh-contoh item dalam pertanyaan ini termasuk:

  • Java Compiler - Pengurai (parser) Java dalam NetBeans adalah pembungkus compiler bahasa Java (javac). Sumber javac tersedia di bawah lisensi GPL v2 dari situs OpenJDK. Versi-versi NB 5.x dan sebelumnya menggunakan versi terdahulu dari compiler javac yang berada di bawah lisensi Sun BCL yang lebih terbatas.
  • XML Parsers (Xerces, Crimson, Xalan, DOM, SAX, JAXP, ...) - digunakan dalam IDE untuk penguraian dan manipulasi XML. http://java.sun.com/xml/
  • JavaHelp Runtime library 2.x - IDE menggunakan JavaHelp untuk menampilkan dokumentasi online. Distribusi binari ekstra dapat juga menyertakan pustaka waktu-pengembangan JavaHelp untuk membangun ulang database pencarian. http://java.sun.com/products/javahelp/index.html
  • Apache Ant 1.6.x - sebuah alat bantu pembangunan berbasis Java yang menjadi sandaran sistem pembangunan (build) NetBeans. http://ant.apache.org/
  • JUnit 3.8.x - kerangka pengujian yang digunakan untuk melakukan tes unit dalam NetBeans. http://www.junit.org/index.html
  • Tomcat 5.x.x - kontainer servlet tertanam untuk Java Servlet dan pengembangan JSP. http://jakarta.apache.org/tomcat/index.html
  • JSP Standard Tag Library (JSTL) - http://java.sun.com/products/jsp/jstl/

Rilis yang diberikan NetBeans dapat memiliki lebih banyak atau lebih sedikit file ekstra yang berlisensi non-CDDL di dalamnya. Tanyakan pada milis nbdev untuk kepastiannya.
