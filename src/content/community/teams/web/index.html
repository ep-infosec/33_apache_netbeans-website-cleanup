The NetBeans.org Web Team

The NetBeans.org web site is an open source community where many people share information about using and developing the NetBeans IDE and NetBeans Platform. The NetBeans.org site offers file version control for our Mercurial source code repository, mailing lists and forums, an issue tracking system for reporting bugs, as well as administrative utilities to manage team member profiles and permissions.

The NetBeans.org site is split into projects: Each project has its own website (at http://module-name.NetBeans.org/), an owner and a team of developers.

Web Content Guidelines and Reports

  • If you are a project owner, or working on a project website, please read our web content guidelines.
  • If you are contributing content to the netbeans.org website itself, please read our www guidelines.
  • If you work on a NetBeans.org project website, logs, error reports & traffic statistics provide insights into the status of your pages.
