List of automated reports

There are several automated procedures that run for netbeans.org websites, and generate reports or logfiles. If you work on a netbeans.org website, the following links may be of interest :

Webalizer Reports
Web Traffic Analysis for each individual netbeans.org module website, as well as some customised reports looking at specific types of traffic like module downloads (nbms). These reports are updated daily.
https://netbeans.org/download/webstats/

404 Broken Link Reports
An analysis of any broken links found, on all netbeans.org module websites.
https://netbeans.org/linkcheck/www-404s.html

testwww -> www Publishing Report
Content on testwww is processed and published on www hourly. Reports of the last run are available here.
https://netbeans.org/download/web-reports/www-publishing.txt
