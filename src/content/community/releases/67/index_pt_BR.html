Informações da versão NetBeans IDE 6.7

O NetBeans IDE é um premiado ambiente de desenvolvimento integrado disponível para Windows, Mac, Linux e Solaris. O projeto NetBeans consiste em um IDE de código aberto e uma plataforma de aplicativos que permite aos desenvolvedores criar rapidamente aplicativos Web, empresariais, móveis e de área de trabalho usando a plataforma Java, bem como JavaFX, PHP, JavaScript e Ajax, Ruby e Ruby on Rails, Groovy e Grails e C/C++.

O projeto NetBeans tem como suporte uma entusiasmada comunidade de desenvolvedorese oferece vastos recursos como documentação e treinamento, bem como uma variada seleção de plug-ins de terceiros.

Visão geral da versão

O NetBeans IDE 6.7 está integrado ao Project Kenai, um ambiente cooperativo no qual os desenvolvedores podem hospedar os projetos de código aberto. Com o Kenai e o NetBeans IDE, uma equipe de desenvolvedores pode criar projetos, verificar, editar, depurar, compilar, analisar e enviar o código, tudo através de uma interface fácil de usar!

Esta versão também tira proveito do sucesso do NetBeans 6.5 com suporte nativo a Maven; integrações ao GlassFish, Hudson e localizador de problemas; e melhorias no Java, PHP, Ruby, Groovy e C/C++. Os destaques da versão 6.7 incluem suporte a JavaScript 1.7, depuração remota de Ruby e integração do Java ME SDK 3.0.

Adquira o NetBeans IDE 6.7.1

A versão para download atualmente disponível é NetBeans IDE 6.7.1, que corresponde a uma atualização do NetBeans IDE 6.7.

O NetBeans IDE 6.7.1 inclui as seguintes alterações:

  • Suporte a JavaFX 1.2
  • Atualização do GlassFish v3 Prelude com correções do pacote #28f
  • Integração de importantes correções baseadas nos comentários dos usuários, especialmente no Editor Java e áreas de análise do projeto

Recursos em destaque

Ferramentas para Connected Developer

  • Crie projetos de código-fonte aberto hospedados no Kenai diretamente no IDE
  • Gerencie seu projeto Kenai e colabore com desenvolvedores da mesma opinião
  • Arquive os relatórios de rastreio de problemas no Jira e Bugzilla

Maven

  • Visualizador gráfico de dependências da biblioteca
  • Auto-completar de código aperfeiçoado para os parâmetros dos plug-ins do Maven
  • Suporte à criação e consumo de serviços Web Java EE
  • Melhorias no navegador e no editor do POM

PHP

  • Suporte a PHPUnit
  • Suporte a Selenium e cobertura de códigos
  • Auto-completar de código no editor PHP

GlassFish

  • Instância do Glassfish v3 Prelude com integração pré-configurada do Derby (Java DB)
  • Suporte a auto-completar de código para o Glassfish v3 Prelude
  • Hierarquia adicionada à visualização em árvore no Services Explorer

Groovy e Grails

  • Suporte personalizado a Grails 1.1
  • Auto-completar de código
  • Desativação da abertura do navegador em execução e configuração de plataformas para uso com projetos Grails
  • Chamada de comandos do Grails ou personalização do script a partir do item do menu de contexto

JavaFX

  • Criação, teste, depuração, criação de perfil e deployment de projetos JavaFX
  • Editor JavaFX com realce de sintaxe, navegação de código-fonte, auto-completar de código, dobramento de código, pop-ups javadoc e detecção de erros.
  • Visualização interativa de elementos visuais
  • Paleta de fragmento de código para transformações, efeitos e animações

Ruby e Rails

  • Depuração remota
  • Melhorias no suporte a Ruby
  • Ações de execução e depuração para casos de teste e conjuntos de aplicativos incluídas no menu de contexto
  • Suporte a testes Shoulda

C/C++

  • Ferramentas para criação de perfil para projetos C++ (com mínima sobrecarga!)
  • Suporte integrado de ferramentas e bibliotecas Qt populares
  • Mais refatorações e geração de código no editor C/C++
  • Visualização de expansão da macro para analisar a saída do pré-processador

Profiler

  • Exportação dos dados de criação de perfil a formatos de arquivo CSV, HTML e XML
  • O HeapWalker oferece suporte a consultas OQL para analisar o conteúdo do despejo de pilha

Depurador

  • Opções de depuração para personalização do processo de depuração
  • Fácil visualização e configuração do código antes do início da sessão de depuração

Java

  • Integração de serviços do Hudson para projetos Ant e Maven
  • As raízes do código gerado são exibidas agora na aba Projetos
  • Biblioteca de Struts migrada para a versão 1.3.8

Java ME / Mobility

  • Distribuído com o Java Platform Micro Edition Software Development Kit 3.0
  • Suporte a Java Card Platform 3.0
  • Suporte total a SVG Rich Components no Visual Mobile Designer
  • Melhorias no SVG Composer

Plataforma NetBeans

  • Melhorias na janela Saída, no catálogo da Central de Atualizações e no desempenho da pesquisa
  • Comportamento melhorado da barra de ferramentas e eliminadores declarativos de tipo MIME
  • Capacidade de alterar a extensão de arquivos
