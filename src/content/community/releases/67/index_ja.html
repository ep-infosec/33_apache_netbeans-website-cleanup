NetBeans IDE 6.7 リリース情報

NetBeans IDE は、受賞歴のある統合開発環境で、Windows、Mac、Linux、および Solaris で使用可能です。NetBeans プロジェクトは、オープンソースの IDE とアプリケーションプラットフォームから構成されます。NetBeans を使用すると、開発者は Java プラットフォームや JavaFX、PHP、JavaScript と Ajax、Ruby と Ruby on Rails、Groovy と Grails、および C/C++ を使用して、Web アプリケーション、エンタープライズアプリケーション、デスクトップアプリケーション、およびモバイルアプリケーションを短時間で作成できます。

NetBeans プロジェクトは活発な開発者コミュニティーによってサポートされ、豊富なドキュメントとトレーニングのリソース、およびさまざまなサードパーティーのプラグインを提供します。

リリース概要

NetBeans IDE 6.7 は、オープンソースプロジェクトをホストする、開発者用の協調環境であるプロジェクト Kenai と統合されています。Kenai と NetBeans IDE を使用すると、開発者チームは、使いやすい単一のインタフェースを介して、プロジェクトを作成し、コードをチェックアウト、編集、デバッグ、構築、検討、およびコミットすることができます。

また、このリリースは、Maven のネイティブサポート、GlassFish、課題追跡と Hudson の統合、および Java、PHP、Ruby、Groovy、C/C++ の機能拡張を備え、実績のある NetBeans 6.5 を基に構築されています。6.7 リリースの注目点には、JavaScript 1.7 のサポート、Ruby のリモートデバッグ、および Java ME SDK 3.0 の統合があります。

NetBeans IDE 6.7.1 の取得

現在入手可能なダウンロードは NetBeans IDE 6.7.1 です。これは、NetBeans IDE 6.7 のアップデートです。

NetBeans IDE 6.7.1 には次の変更点が含まれています。

  • JavaFX 1.2 のサポート
  • バンドル #28f からの修正を含めるための GlassFish v3 Prelude のアップグレード
  • 特に Java エディタとプロジェクトのスキャン領域に関する、ユーザーフィードバックに基づいた重要なバグ修正の統合

主な新機能

Connected Developer ツール

  • IDE 内で Kenai でホストされるオープンソースプロジェクトを直接作成
  • Kenai プロジェクトの管理、および同じ考えを持った開発者との協力
  • Jira と Bugzilla の両方を使用して課題追跡レポートを提出

Maven

  • ライブラリの依存リソースグラフビューア
  • Maven プラグインパラメータのコード補完の改善
  • Java EE、Web サービスの作成および消費のサポート
  • POM エディタおよびナビゲータの拡張

PHP

  • PHPUnit のサポート
  • コードカバレージと Selenium のサポート
  • PHP エディタでの SQL コード補完

GlassFish

  • 事前構成済みの Derby (Java DB) 統合を含む Glassfish v3 Prelude のインスタンス
  • Glassfish v3 Prelude のコード補完のサポート
  • サービスエクスプローラのツリービューへの階層の追加

Groovy および Grails

  • そのまま使用できる Grails 1.1 のサポート
  • コード補完
  • Grails プロジェクトとともに使用するプラットフォームの実行と構成で、ブラウザが開かないようにする
  • コンテキストメニュー項目からの Grails コマンドまたはカスタムスクリプトの呼び出し

JavaFX

  • JavaFX プロジェクトの作成、テスト、デバッグ、プロファイル、配備
  • JavaFX エディタ (構文の強調表示、ソースナビゲーション、コード補完、コードの折り畳み、javadoc のポップアップ、エラー検出)
  • 視覚要素のライブプレビュー
  • 変換、効果、アニメーション用のコードスニペットパレット

Ruby および Rails

  • リモートデバッグ
  • Ruby 定数サポートの改善
  • コンテキストメニューに含まれる、テストケースおよびテストスイートのアクションの実行とデバッグ
  • Shoulda テストのサポート

C/C++

  • C++ プロジェクトの DLight プロファイルツール (最小限のオーバーヘッド)
  • 一般的な Qt ライブラリとツールの統合サポート
  • C/C++ エディタでのリファクタリングおよびコード生成の増加
  • プリプロセッサ出力を解析するためのマクロの展開表示

プロファイラ

  • プロファイルデータを CSV、HTML および XML ファイル形式でエクスポート
  • ヒープダンプの内容を解析するための OQL クエリをヒープウォーカがサポート

デバッガ

  • デバッグプロセスをカスタマイズするデバッガオプション
  • デバッグセッションを開始する前のソースの表示と設定の簡素化

Java

  • Ant および Maven プロジェクト用の Hudson サービス統合
  • 「プロジェクト」タブに表示される生成済みソースのルート
  • version 1.3.8 に移行された Struts ライブラリ

Java ME / Mobility

  • Java Platform Micro Edition Software Development Kit 3.0 に付属
  • Java Card Platform 3.0 のサポート
  • ビジュアルモバイルデザイナの SVG リッチコンポーネントの完全サポート
  • SVG コンポーザの改善

NetBeans プラットフォーム

  • 出力ウィンドウ、アップデートセンターのカタログ、および検索のパフォーマンスの強化
  • ツールバーの動作と宣言による MIME タイプリゾルバの改善
  • ファイル拡張子の変更機能

詳細は、NetBeans IDE 6.7 の新機能および注目機能のページを参照してください。
