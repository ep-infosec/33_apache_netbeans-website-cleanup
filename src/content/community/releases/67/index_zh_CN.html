NetBeans IDE 6.7 发行信息

NetBeans IDE 是一个荣获奖项的集成开发环境，可用于 Windows、Mac、Linux 和 Solaris。NetBeans 项目由开源 IDE 和应用程序平台组成；通过该 IDE 和平台，开发者可以使用 Java 平台以及 JavaFX、PHP、JavaScript 和 Ajax、Ruby 和 Ruby on Rails、Groovy 和 Grails、C/C++ 快速创建 Web、企业、桌面和 Mobile 应用程序。

NetBeans 项目由活跃的开发者社区支持，并且还提供了广泛的文档和培训资源，以及各种供您选用的第三方插件。

发行版概述

NetBeans IDE 6.7 集成了 Project Kenai，Project Kenai 是一个用于开发者托管其开源项目的协作环境。使用 Kenai 和 NetBeans IDE，开发者小组即可通过一个易于使用的界面来创建项目，以及签出、编辑、调试、生成、讨论并提交代码！

该发行版还基于已成功发行的 NetBeans 6.5；它包含对 Maven 的本机支持，集成了 GlassFish、问题跟踪器和 Hudson，并且增强了 Java、PHP、Ruby、Groovy 和 C/C++ 功能。6.7 发行版的重要功能包括支持 JavaScript 1.7 和 Ruby 远程调试，以及集成了 Java ME SDK 3.0。

获取 NetBeans IDE 6.7.1

现在可以下载 NetBeans IDE 6.7.1，它是 NetBeans IDE 6.7 的更新版本。

NetBeans IDE 6.7.1 进行了以下改进：

  • 支持 JavaFX 1.2
  • 升级了 GlassFish v3 Prelude，使其包含修复程序包 #28f 中的修复程序
  • 基于用户反馈（特别是在 Java 编辑器和项目扫描方面的用户反馈），修复了一些重要错误

功能集锦

团队开发工具

  • 直接在 IDE 中创建由 Kenai 托管的开源项目
  • 管理您的 Kenai 项目并与志同道合的开发者进行协作
  • 使用 Jira 和 Bugzilla 对问题跟踪报告进行归档

Maven

  • 库依赖关系图形查看器
  • 改进了 Maven 插件参数的代码完成
  • 支持创建和使用 Java EE 及 Web 服务
  • 增强了 POM 编辑器和导航

PHP

  • PHPUnit 支持
  • 代码覆盖率和 Selenium 支持
  • PHP 编辑器中的 SQL 代码完成

GlassFish

  • 将 GlassFish v3 Prelude 实例与预配置的 Derby (Java DB) 集成在一起
  • 为 GlassFish v3 Prelude 提供了代码完成支持
  • 在服务资源管理器的树视图中添加了分层结构

Groovy 和 Grails

  • 对 Grails 1.1 的快捷支持
  • 代码完成
  • 禁止在运行时打开浏览器，并将平台配置为使用 Grails 项目
  • 从上下文菜单项中调用 Grails 命令或定制脚本

JavaFX

  • 创建、测试、调试、分析和部署 JavaFX 项目
  • JavaFX 编辑器，具有语法突出显示、源导航、代码完成、代码折叠、Javadoc 弹出式窗口和错误检测功能。
  • 实时预览可视元素
  • 用于实现各种变换、效果和动画的代码片段组件面板

Ruby 和 Rails

  • 远程调试
  • 改进了 Ruby 常量支持
  • 上下文菜单中包含了对测试用例和套件的运行和调试操作
  • 支持 Shoulda 测试

C/C++

  • 为 C++ 项目提供了 DLight 性能分析工具（开销降至最低！）
  • 集成了对常用 Qt 库和工具的支持
  • C/C++ 编辑器中具有更多重构和代码生成操作
  • 用于分析预处理程序输出的宏扩展视图

Profiler

  • 将性能分析数据导出为 CSV、HTML 和 XML 文件格式
  • 堆查看器支持 OQL 查询，以分析堆转储的内容

调试器

  • 用于定制调试过程的调试器选项
  • 在启动调试会话前可轻松查看和设置源代码

Java

  • 为 Ant 和 Maven 项目集成了 Hudson 服务
  • 生成的源根目录现在显示在“项目”标签中
  • 已将 Struts 库迁移到版本 1.3.8

Java ME/Mobility

  • 与 Java Platform Micro Edition Software Development Kit 3.0 捆绑在一起
  • 支持 Java Card Platform 3.0
  • 在可视 Mobile 设计器中完全支持 SVG 富组件
  • 改进了 SVG 构成器

NetBeans 平台

  • 增强了“输出”窗口、更新中心目录和搜索性能
  • 改进了工具栏行为和声明性 MIME 类型解析器
  • 能够更改文件扩展名

有关详细信息，请参见 NetBeans IDE 6.7 新增功能和重要功能页。
