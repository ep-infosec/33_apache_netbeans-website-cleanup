Информация к выпуску среды IDE NetBeans 6.9

IDE NetBeans представляет собой отмеченную наградами среду разработки, доступную для ОС Windows, Mac, Linux и Solaris. Проект NetBeans состоит из среды IDE с открытым кодом и платформы приложения, которая позволяет разработчикам ускорить работу над созданием настольных и мобильных приложений, а также веб-приложений и приложений уровня предприятия, используя при этом платформу Java, а также JavaFX, PHP, JavaScript и Ajax, Ruby и Ruby on Rails, Groovy и Grails, а также C/C++.

Проект NetBeans поддерживается активным сообществом разработчиков, которое предлагает обширную документацию и обучающие материалы, а также множество сторонних подключаемых модулей.

Обзор выпуска

Среда IDE NetBeans 6.9 дополнена JavaFX Composer - визуальным инструментальным средством для создания приложений интерфейса JavaFX, аналогичным конструктору GUI Swing для приложений Java SE. Дополнительные функции включают в себя возможность совместного функционирования OSGi для приложений платформы NetBeans, поддержку разработки пакетов OSGi с помощью Maven, поддержку для пакета JavaFX SDK 1.3, платформы Zend PHP и Ruby on Rails 3.0, а также усовершенствования редактора Java, отладчика Java, управления заданиями и другие возможности.

Получение среды IDE NetBeans 6.9

Ключевые возможности

OSGi

  • Разработка пакетов OSGi с помощью Maven
  • Связанный контейнер Felix
  • Использование пакетов OSGi в приложениях RCP NetBeans

Платформа NetBeans

  • Возможность совместного функционирования OSGi (разработка и использование пакетов OSGi в приложениях на основе платформ)
  • Поддержка OSGi Felix 2.0.3, экспериментальная поддержка Equinox
  • Преобразование модулей NetBeans в пакеты OSGi и их запуск в немодифицированном контейнере OSGi
  • Редактор брэндинга для проектов на основе платформ Ant и Maven
  • Приложения на основе платформы NetBeans создают установщики для наиболее распространенных операционных систем
  • Расширенная поддержка для использования веб-служб и подключения к базам данных

 

JavaFX

  • JavaFX Composer для компонентов пользовательского интерфейса в виде форм с состояниями и доступом к различным источникам данных
  • Добавлены и усовершенствованы подсказки редактора и реорганизация
  • Расширенные палитры для фигур, цветов, эффектов и диаграмм JavaFX
  • Усовершенствованное форматирование кода

Java EE

  • Поддержка учета контекстов и зависимостей (CDI) (JSR-299)
  • Поддержка платформы Spring 3.0
  • Связанный сервер GlassFish Server Open Source Edition 3.0.1
  • Легкое восстановление сущностей JPA после изменений базы данных
  • Поддержка веб-служб REST для приложений RCP

Java

  • Группировка точек останова в отладчике Java, отображение истории параметров в отладчике
  • Поддержка процессоров аннотаций в редакторе с возможностью настройки в свойствах проекта
  • Обновленная поддержка Applet и Web Start (JNLP)
  • Улучшенная навигация по анализатору стека операций и URL, действие перехода к переопределенному/реализованному методу

Веб-языки: HTML, CSS, JavaScript

  • Реорганизация и поиск использований для языков, аналогичных CSS и HTML
  • Завершение кода и создание ссылок для атрибутов идентификатора и выбора классов
  • Реорганизация встроенных стилей CSS

 

Java Card

  • Поддержка создания сервлетов Java Card Connected 3, классических и расширенных апплетов
  • Обратная совместимость с Java Card 2.2.x и Java Card Classic
  • Отладка и создание нескольких Java Card Platform/Device
  • Выравнивание с примером реализации Java Card

PHP

  • Поддержка платформы Zend PHP
  • Аннотации "Переопределяет/реализует" и "Переопределен/реализован"
  • Новая программа форматирования со многими правилами форматирования

Ruby

  • Поддержка Ruby on Rails 3.0

C/C++

  • Интеграция модульных тестов в проекты C/C++
  • Расширенная удаленная разработка, включающая в себя удаленную загрузку и просмотр файлов
  • Улучшенная поддержка целевых файлов makefile и Fortran
  • Улучшенный переход по гиперссылкам между переопределенными методами и классами

Общие возможности

  • Значительно увеличено количество подсказок редактора для Java и Ruby
  • Поддержка проверки орфографии в редакторе
  • Пункт меню справки для отчета о неполадках и медленной работе в среде IDE NetBeans
  • Поддержка многочисленных командных серверов на основе Kenai (например, kenai.com и netbeans.org).
