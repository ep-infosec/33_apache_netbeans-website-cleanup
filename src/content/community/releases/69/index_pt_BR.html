Informação da versão do NetBeans IDE 6.9.1

O NetBeans IDE é um premiado ambiente de desenvolvimento integrado disponível para Windows, Mac, Linux e Solaris. O projeto NetBeans consiste em um IDE de código aberto e uma plataforma de aplicativos que permite aos desenvolvedores criar rapidamente aplicativos Web, empresariais, móveis e de área de trabalho utilizando a plataforma Java, bem como JavaFX, PHP, JavaScript e Ajax, Ruby e Ruby on Rails, Groovy e Grails e C/C++.

O projeto NetBeans tem como suporte uma entusiasmada comunidade de desenvolvedorese oferece vastos recursos como documentação e treinamento, bem como uma variada seleção de plug-ins de terceiros.

Visão geral da versão

O NetBeans IDE 6.9.1 apresenta o JavaFX Composer, uma ferramenta de layout visual para desenvolver aplicativos de GUI JavaFX, semelhante ao construtor de GUI Swing para aplicativos Java SE. Destaques adicionais incluem a interoperabilidade para aplicativos da plataforma NetBeans e o suporte para desenvolver pacotes OSGi com o Maven; suporte à estrutura JavaFX SDK 1.3.1, PHP Zend e Ruby on Rails 3.0; assim como aprimoramentos do editor Java, depurador Java, rastreamento de problemas e mais.

Recursos em destaque

OSGi

  • Desenvolver pacotes OSGi com o Maven
  • Container do Felix incorporado
  • Usar pacotes OSGi em um aplicativo NetBeans RCP

Plataforma NetBeans

  • Interoperabilidade OSGi (desenvolver e consumir pacotes OSGi em aplicativos com base na plataforma)
  • Suporte a Felix 2.0.3 OSGi, suporte experimental a Equinox
  • Converter módulos NetBeans em pacotes OSGi e executá-los em um contêiner OSGi não modificado
  • Editor de marcas para projetos com base na plataforma Ant e Maven
  • Os aplicativos com base na plataforma NetBeans geram instaladores para os sistemas operacionais mais comuns
  • Suporte aprimorado para consumir serviços da Web e para se conectar aos bancos de dados

 

JavaFX

  • JavaFX Composer para componentes de forma similar a UI com estados e acessos a diversas fontes de dados
  • Dicas do editor aprimoradas e adicionadas, depurando e refatorando
  • Paletas aprimoradas para Formas, Cores, Efeitos e Tabelas JavaFX
  • Formatação aprimorada de código

Java EE

  • Suporte a Contexts e Dependency Injection (CDI) (JSR-299)
  • Suporte ao Spring Framework 3.0
  • GlassFish Server Open Source Edition 3.0.1 incorporado
  • Fácil regeneração de entidades JPA após a alteração do banco de dados
  • Suporte aos serviços Web REST para aplicativos RCP

Java

  • Agrupamento de interrupção do depurador Java, histórico de parâmetro de anexação de depurador
  • Suporte a processadores de anotações no editor, configurável nas Propriedades do projeto
  • Novo suporte a Applet e Web Start (JNLP)
  • Navegação aprimorada no Stack Trace Analyzer e URLs, ação de Ir para método sobreposto/implementado

Linguagens da Web: HTML, CSS, JavaScript

  • Refatoração e localização de usos para linguagens como o CSS e HTML
  • Conclusão de código e hiperlink para atributos de seletor de ID e classe
  • Estilos CSS de refatoração em-linha

 

Java Card

  • Suporte para criar servlets Java Card Connected 3, applets clássicos e estendidos
  • Compatibilidade retroativa com o Java Card 2.2.x e o Java Card Classic
  • Depuração e criação de múltiplos Java Card Platform/Device
  • Alinhamento com a Implementação de Referência do Java Card

PHP

  • Suporte ao PHP Zend Framework
  • Anotações de "Sobrepõe/Implementa" e "Está sobreposto/Implementado"
  • Novo formatador com muitas regras de formatação

Ruby

  • Suporte ao Ruby on Rails 3.0

C/C++

  • Integração de teste de unidade com projetos C/C++
  • Desenvolvimento remoto aprimorado, incluindo o download e navegação de arquivo remoto
  • Suporte aprimorado para alvos makefile e Fortran
  • Navegação de hiperlink aprimorada entre métodos e classes sobrepostos

Geral

  • Muitas outras dicas do editor para o Java e Ruby
  • Suporte de verificação ortográfica no editor
  • Item do menu Ajuda para reportar erros e lentidão do NetBeans IDE
  • Suporte a vários servidores de equipes com base no Kenai (por exemplo, kenai.com e netbeans.org).
