Instrucións da instalación do IDE NetBeans 6.9.1

Última actualización: 1 de xullo de 2010

Este documento describe como instalar NetBeans IDE 6.9.1 no seu sistema. Vexa as notas da versión do NetBeans IDE 6.9.1 para obter información sobre os sistemas operativos compatíbeis e as configuracións de hardware para o IDE. Para obter máis información das novas características incluídas nesta versión do IDE consulte a páxina de información da versión 6.9.1 do IDE NetBeans.

Software requirido

Requírese o Kit de desenvolvemento de Java SE (JDK) 6.0 Update 13 ou superior para instalar o IDE NetBeans. A versión 6.9.1 do IDE non pode instalarse usando o JDK 5.0. Vostede debe descargar a última actualización do JDK 6.0 desde http://java.sun.com/javase/downloads.

Os paquetes do NetBeans para PHP, Ruby e C/C++ só requiren o Contorno de execución de Java (JRE) 6 para a instalación e execución. Porén, se vai usar as características de Java precisará JDK 6.0.

Opcións de descarga do instalador

No IDE NetBeans páxina de descarga, vostede pode obter un dos varios instaladores, cada un contén o IDE base e ferramentas adicionais.

Existen os seguintes instaladores dispoñíbeis:

  • Java SE. Admite todas as características de desenvolvemento do Java SE estándar así como a plataforma de desenvolvemento de RCP do NetBeans.
  • JavaFX. Inclúe JavaFX SDK 1.3.1 e JavaFX Composer para o desenvolvemento de aplicativos ricos multiplataforma.
  • Java. Fornece ferramentas para o desenvolvemento de aplicativos Java SE e Java ME así como a compatibilidade de plataforma de desenvolvemento RCP de NetBeans. Esta opción de descarga tamén inclúe a edición Open Source do servidor GlassFish 3.0.1, e o software Apache Tomcat 6.0.26.
  • Ruby. Fornece ferramentas para o desenvolvemento con Ruby e compatibilidade con Rails e JRuby. Tamén contén a edición de código aberto do servidor GlassFish 3.0.1.
  • C/C++. Ofrece a compatibilidade de desenvolvemento coas linguaxes C, C++, Qt, Fortran e Ensamblador.
  • PHP. Fornece ferramentas para o desenvolvemento con PHP 5.x, a compatibilidade dos frameworks Zend e Symfony
  • Todo. Esta é unha opción de descarga completa, que contén todos os contornos de execución e tecnoloxías dispoñíbeis para o IDE NetBeans.

Personalizando a túa instalación

Se descarga a opción Java ou "Todo", vostede pode seleccionar exactamente que ferramentas e contornos de execución instalar. Prema en Personalizar na páxina de Benvida do instalador e seleccione as funcionalidades que desexa empregar. Para obter máis información, vexa Instalando o software.

Se quere engadir funcionalidades á súa configuración do IDE de NetBeans, empregue o xestor de complementos de NetBeans. Por exemplo, asuma que descargou o paquete de Ruby. E logo decide probar a funcionalidade de C/C++. Para facer isto, vaia ao Xestor de complementos de NetBeans dende o IDE (seleccione Ferramentas -> Complementos) e engada o paquete C/C++ á súa instalación existente.

Existen algunhas notas importantes sobre a instalación e a configuración de algunhas funcionalidades de NetBeans:

  1. Apache Tomcat 6.0.26 está incluído nas opcións de descarga "Java" e "Todo" pero non se instala de forma predefinida en calquera das dúas opcións. Para instalar o Apache Tomcat desde as descargas Java e "Todo", lance o instalador e seleccione Apache Tomcat 6.0.23 na caixa de diálogo Personalizar.
  2. Para o desenvolvemento con C/C++ (aplicábel para as opcións de descarga de C/C++ e "Todo"), necesita instalar adicionalmente os compiladores e ferramentas. Para máis información vaia a Configurando o IDE NetBeans para C/C++/Fortran
  3. Para Mac OS X, vostede pode personalizar a instalación dende as opcións de descarga Java, Ruby e "Todo" A instalación do IDE dende as opcións de descarga Java SE, PHP e C/C++ non pode personalizarse.
  4. Tamén pode descargar un ficheiro independente da plataforma e executar o ficheiro executábel do IDE NetBeans. Teña en conta que a edición Open Source do servidor GlassFish e Apache Tomcat só están dispoñíbeis cos instaladores dependentes da plataforma.

Comezando a descarga

  1. Vaia a https://netbeans.org/downloads/
  2. Na zona superior dereita da páxina seleccione a linguaxe e plataforma dende a lista despregábel. Vostede pode seleccionar para descargar e empregar un arquivo zip independente da plataforma.
  3. Prema no botón de Descargar dende a opción de descarga que queira instalar.
  4. Garde o ficheiro de instalación ao seu sistema.

Instalando o software

Siga as instrucións nesta sección para instalar o IDE no seu sistema. Estas instrucións de instalación aplícanse a todas as plataformas compatíbeis. Para a lista de plataformas admitidas, vexa as notas de lanzamento.

Microsoft Windows, OS Solaris e Linux

Para instalar o software:

  1. Despois de que a descarga remate execute o instalador.
    • Para Windows, o ficheiro executábel do instalador ten a extensión .exe. Prema dúas veces no ficheiro de instalación para executalo.
    • Para as plataformas Solaris e Linux, o ficheiro do instalador ten a extensión .sh. Para estas plataformas, precisa facer os ficheiros do instalador executábeis empregando a seguinte orde: chmod +x <installer-file-name>
  2. Seleccione as ferramentas e contornos de execución a instalar, leve a cabo os seguintes pasos na páxina de Benvida do asistente de instalación:
    1. Prema en personalizar.
    2. No caixa de diálogo de Personalización da instalación faga as súas seleccións.
    3. Prema en OK.
  3. Na páxina de Benvida do asistente de instalación prema Seguinte.
  4. Na páxina do acordo de licenza, revise o acordo de licenza e prema na caixa de verificación de aceptación, logo prema Seguinte.
  5. Na páxina de instalación do IDE de NetBeans faga o seguinte:
    1. Acepte o cartafol de instalación predefinido para o IDE NetBeans ou especifique outro cartafol.

      Nota: O cartafol de instalación debe estar baleiro e o perfil do usuario que está usando para executar o instalador debe ter permisos para ler/escribir neste cartafol

    2. Acepte a instalación do JDK predefinido para empregar co IDE de NetBeans ou seleccione unha instalación diferente dende a lista despregábel. Se o asistente de instalación non pode atopar unha instalación de JDK compatíbel para empregar co IDE NetBeans, o seu JDK non está instalada na localización predefinida. Neste caso, especifique o camiño ao JDK instalado e prema Seguinte, ou cancele a instalación actual. Logo de instalar a versión do JDK requirido debe reiniciar a instalación.
  6. Se se abre a páxina de instalación da edición Open Source do servidor GlassFish 3.0.1, acepte o cartafol de instalación predefinido ou especifique un cartafol personalizado.
  7. Se está instalando Apache Tomact, na súa páxina de instalación acepte o cartafol de instalación predefinido ou especifique outra localización de instalación.
  8. Na páxina de Resumo, verifique que a lista dos compoñentes a instalar é correcta e que ten o espazo axeitado no seu sistema para realizar a instalación.
  9. Prema en Instalar para comezar a instalación.

Nota: se encontra problemas para instalar o software correctamente, consulte a Resolución de problemas para obter descricións e suxestións para solucionar problemas sen resolver que poden afectar ao proceso de instalación.

Mac OS X

Para instalar o software:

  1. Despois de que a descarga remate, execute o instalador. O ficheiro do instalador ten a extensión .dmg.
  2. Nos paneis que se abren prema na icona do paquete. O paquete ten a extensión .mpkg. O asistente do instalador comezará.
  3. Na páxina de Benvida do asistente de instalación prema en Continuar.
  4. Revise o acordo de licenza e prema en Continuar. Prema en Aceptar na xanela emerxente para aceptar a licenza.
  5. Na páxina de Selección de destino seleccione o dispositivo e prema en Continuar.
  6. Se descargou o paquete "Todo", Java ou Ruby vostede pode personalizar a instalación. No último panel da instalación prema no botón de continuar no botón da parte esquerda do panel. Mostrárase a árbore de produtos.
  7. Seleccione os produtos que quere instalar.
  8. Insira o nome do administrador e o contrasinal do seu sistema e prema en Aceptar para comezar a instalación.

Paquete independente da plataforma

Para executar o software:

  1. Despois de que a descarga do arquivo ZIP independente da plataforma remate, extraíao a calquera cartafol do seu sistema.
  2. Executa o ficheiro executábel localizado no cartafol netbeans/bin.
  3. Acepte o acordo de licenza. O IDE NetBeans iniciarase.

Nota: Se non pode encontrar unha instalación do JDK compatíbel, pode que necesite engadir manualmente o camiño ao cartafol de instalación do JDK realizando o seguinte:

  1. Abra o ficheiro netbeans.conf localizado no cartafol netbeans/etc cun editor de textos.
  2. Introduza a localización da instalación do JDK compatíbel na opción netbeans_jdkhome A localización predefinida en Windows é C:\Program Files\Java\jdk1.6.0.
  3. Garde o ficheiro netbeans.conf e execute o ficheiro executábel do cartafol netbeans/bin.

Nota: A edición de código aberto do servidor GlassFish e Apache Tomcat soamente están dispoñíbeis cos instaladores específicos das plataformas.

Desinstalando o software

Segue as instrucións desta sección para desinstalar o IDE do seu sistema. Estas instrucións de instalación aplícanse a todas as plataformas compatíbeis. Para a lista de plataformas admitidas, vexa as notas de lanzamento.

Microsoft Windows

Se é necesario, vostede pode desinstalar o IDE NetBeans, a edición Open Source do servidor GlassFish 3.0.1 e o Apache Tomcat usando os procedementos seguintes.

Para desinstalar o IDE:

  1. Apague o IDE.
  2. Desde o Panel de control, seleccione Engadir ou eliminar programas, logo seleccione o IDE NetBeans 6.9.1 e prema Eliminar.

Para desinstalar a edición de código aberto do servidor GlassFish 3.0.1:

  1. Apague o IDE.
  2. Desde o panel de control, Seleccione Engadir ou Eliminar programas, logo seleccione a edición de código aberto do servidor GlassFish 3.0.1 e prema Eliminar.

Para desinstalar Apache Tomcat:

  1. Apague o IDE.
  2. Desde o panel de control, Seleccione Engadir ou Eliminar programas, logo seleccione Apache Tomcat 6.0.26 e prema Eliminar.

Solaris OS, Linux

Se é necesario, pode desinstalar o IDE NetBeans, a edición de código aberto do servidor GlassFish 3.0.1, e Apache Tomcat usando os seguintes procedementos.

Para desinstalar o IDE:

  1. Apague o IDE.
  2. No cartafol de instalación do IDE, execute o ficheiro uninstall.sh.
  3. Para desinstalar GlassFish e Tomcat xunto co IDE, na páxina do resumo comprobe que aparecen listadas as versións correctas e marque as casillas de verificación axeitadas.
  4. Na páxina de resumo prema en Desinstalar.
  5. Logo de que a desinstalación remate, prema en Terminar.

Para desinstalar a edición de código aberto do servidor GlassFish 3.0.1:

  1. Apague o IDE.
  2. Localice o cartafol de instalación do servidor de aplicativos.

    Nota: A localización predefinida da instalación é diferente dependendo do tipo de instalación e da plataforma. Na seguinte táboa móstrase o cartafol de instalación predefinido do servidor de aplicativos para varias plataformas.

    Plataforma Instalando como... Instalación de GlassFish 3.0.1
    Windows usuario con permisos de administración %ProgramFiles%\glassfish-3.0.1
    Windows usuario %USERPROFILE%\glassfish-3.0.1
    OS Solaris raíz /opt/glassfish-3.0.1
    OS Solaris, Linux usuario ~/glassfish-3.0.1
    Linux raíz /usr/local/glassfish-3.0.1
  3. No cartafol de instalación do servidor de aplicativos, execute o ficheiro uninstall.sh.
  4. Na páxina de resumo prema en Desinstalar.
  5. Despois de que a desinstalación remate prema en Terminar.

Para desinstalar Apache Tomcat:

  1. Apague o IDE.
  2. Localice o cartafol de instalación do Apache Tomcat e execute o ficheiro uninstall.sh.
  3. Na páxina de resumo prema en Desinstalar.
  4. Despois de que a desinstalación remate prema en Terminar.

Mac OS X

Para desinstalar o IDE e o servidor de aplicativos:

  1. Peche o software.
  2. Arrastre os cartafoles do programa ou os ficheiros ao Lixo. Nota que a localización predefinida para o IDE e o servidor de aplicativos está en /Applications/NetBeans/.

Paquete independente da plataforma

Para instalar os ficheiros do IDE, sigua os pasos descritos na sección Limpando despois dunha instalación incompleta de este documento.

Compatibilidade para a actualización e instalación múltiple

Poden coexistir múltiples instalacións de NetBeans 5.x e 6.x co IDE NetBeans 6.9.1 no mesmo sistema. Non ten que desinstalar as versións anteriores para instalar ou executar o IDE NetBeans 6.9.1.

Se ten unha instalación existente do IDE NetBeans vostede pode seleccionar se desexa importar ou non as opcións de configuración desde o cartafol de usuario existente ao executar IDE NetBeans 6.9.1 por primeira vez.

Nota: Nestes momentos non se poden realizar actualizacións incrementais de versións do IDE. Para actualizar á última versión debe descargar o último instalador.

Resolución de erros

Sobre o cartafol de traballo do instalador

O instalador almacena información sobre as sesións da instalación no cartafol de traballo do instalador. Algúns exemplos de datos son a lista de compoñentes instalados no seu sistema, as mensaxes de rexistro de instalación e os datos de rexistro.

A localización predefinida para o cartafol de traballo do instalador é %USERPROFILE%\.nbi en Windows e ~/.nbi outras plataformas. O subcartafol de rexistro contén os ficheiros de rexistro da instalación.

Limpando as instalacións incompletas

Despois de unha instalación incompleta ou non exitosa, para solucionalo pode limpar o seu sistema das anteriores instalacións.

Para Microsoft Windows, Solaris OS, e Linux:

  1. Elimine o cartafol de traballo do instalador.
  2. Elimine os cartafoles de instalación do IDE e do servidor de aplicativos GlassFish.
  3. Na plataforma Windows, elimine todas as entradas do rexistro nbi* desde as chaves do rexistro seguintes: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Para Mac OS X:

  • Arrastre os cartafoles de programa ou ficheiros ao Lixo.

Desactivando algún software antes da instalación

Mellor desactive o firewall, antivirus e aplicativos de monitorización antes de comezar a instalación. Algún destes aplicativos poden evitar que os compoñentes do IDE sexan configurados correctamente durante o proceso de instalación.

Por exemplo, se ten un aplicativo Monitor de procesos de Logitech {LVPrcSrv.exe} executándose no sistema, deteñao antes de comezar a instalación
