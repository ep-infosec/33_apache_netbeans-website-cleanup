NetBeans 5.5 不仅支持 Java Enterprise Edition 5 平台，而且特别值得一提的是，它还支持 Java 持久性、EJB 3 和 JAX-WS 2.0 规范。

NetBeans IDE 5.5 建立在已成功发行的 NetBeans 5.0 的基础上，同时新增了对 Java EE 5 和 Sun Java System Application Server PE 9 的支持。此外，它还将 Web 2.0的规范应用其中！NetBeans 5.5 可以为 Java SE 6 和 Java EE 5 项目生成 JAX-WS 2.0 工件，同时支持 Java 持久性和 Enterprise Java Beans 3。此外，5.5 更新中心还提供了新的 Subversion 支持模块和更新的 GUI 生成器模块。

功能集锦


NetBeans 5.5 具有多种功能，有助于您在生成功能强大、符合标准的 Java™ 应用程序时最大限度地提高生产率和灵活性：
  • 支持常规的 Java Enterprise Edition™ 5 平台：
    • 部署到 Sun Java System Application Server PE 9 (Glassfish)。
    • 为 Java EE 5 API 和部署描述符提供了代码完成和 Javadoc 功能。
    • 为 Java EE 5 平台更新了 Web 层技术：Servlet 2.5、JavaServer™ Pages 2.1、JavaServer™ Faces 1.2。
    • 提供了 Java EE 5 样例项目，可以帮助您快速入门。

  • 支持 Java 持久性：
    • 通过现有的 DB 结构生成 Java 实体类。
    • 在 IDE 内创建实体类，并允许服务器依赖于 Glassfish 中的 DB-from-Java 功能来生成 DB。
    • 持久性运行时可以用在 Web 应用程序和纯 Java 应用程序中，无需完整的 Java EE 应用服务器。
    • 提供了用于创建完整 Web 应用程序的向导，该应用程序利用 Java 持久性完全支持对数据库执行的创建/读取/更新/删除操作。
    • 对 Java 源代码编辑器增强了以下功能：代码完成、快速错误检查和验证，以及提示功能（尤其是针对 Java 持久性的提示）。
    • 提供了生成常用代码模式的高级用户操作（例如，使用实体管理器）。

  • 支持 Java API for XML Web Services (JAX-WS)（即 Web Service 2.0）：
    • 为 SE 6 和 EE 5 项目生成 JAX-WS 2.0 工件。
    • 通过绑定定制（如异步调用、打包信息）进行配置。
    • 简化并更新了模板向导。
    • 提供了生成常用代码模式的高级用户操作（例如，调用 Web 服务）。

  • 支持 Enterprise Java Beans 3 会话 Bean：
    • 简化了会话 Bean 和消息驱动 Bean 的开发。
    • 对 Java 源代码编辑器增强了以下功能：代码完成、快速错误检查和验证、以及提示功能（尤其是针对 EJB 3 的提示）。

  • 其他改进：
    • 可以轻松、简便地访问 Java EE 5 的 Java BluePrints 解决方案目录。
    • 提供了适用于 JavaServer™ Pages 表达式语言的代码完成功能。
    • 支持 Java EE 应用程序客户端。
    • 支持 Java EE 应用程序中的安全配置。
    • 支持直接通过 IDE 运行 SQL 脚本。

NetBeans 附加软件包

  • NetBeans Enterprise Pack 为创建面向服务的体系结构 (Service Oriented Architecture, SOA) 应用程序提供了支持，其中包括 XML 架构工具、WSDL 编辑器以及用于 Web 服务编制的 BPEL 编辑器。
  • Mobility Pack 可以为启用了 Java™ Micro Edition 平台（Java™ ME 平台）技术的移动设备编写、测试和调试应用程序。
  • NetBeans Profiler 有助您优化应用程序的内存和 CPU 的使用情况。
  • NetBeans C/C++ Development Pack 支持 C/C++ 项目的开发。
  • Visual Web Pack 能够实现以可视方式快速生成基于标准的 Web 应用程序，其中包括对 AJAX 和 JSF 组件的支持。
