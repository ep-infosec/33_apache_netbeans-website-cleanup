5.5.1 功能集锦





NetBeans 5.5.1 的主要功能是支持 Java EE 5 Application Server 9.1。Appserver 9.1 (GlassFish v2) 的 Beta 版本现已发布。

NetBeans 5.5.1 在所支持的平台中还添加了 Windows Vista。

Mobility Pack 包含 J2ME Wireless Toolkit 2.5.1 版。

NetBeans 5.5.1 构建于 NetBeans 5.5 的功能之上；此外，它还修复了一些错误。有关 NetBeans 5.5 功能的完整列表，请参见 5.5 发行版本页面。

NetBeans 附加软件包

    Mobility Pack 可以为启用了 Java™ Micro Edition 平台（Java™ ME 平台）技术的移动设备编写、测试和调试应用程序。


    Visual Web Pack 能够实现以可视方式快速生成基于标准的 Web 应用程序，其中包括对 AJAX 和 JSF 组件的支持。



    NetBeans Enterprise Pack 为创建面向服务的体系结构 (Service Oriented Architecture, SOA) 应用程序提供了支持。
    注意：如果希望使用 Enterprise Pack 中的 SOA 和 BPEL 功能，请使用安装了 NetBeans Enterprise Pack 5.5 的 NetBeans IDE 5.5。我们将不再对 NetBeans Enterprise Pack 5.x 发布任何更新。目前我们进行的所有开发都只针对具有 SOA 功能的 NetBeans 6.0。


    NetBeans Profiler 有助您优化应用程序的内存和 CPU 的使用情况。



    C/C++ Pack 支持 C/C++ 项目的开发。可识别语言的编辑器能够缩排和完成 C/C++ 代码，并可突出显示语法。


其他语言

NetBeans 5.5.1 已本地化为简体中文、日语和巴西葡萄牙语；如果您正在使用其中的一种语言环境，NetBeans 5.5.1 则可以自动为您提供本地化的版本，否则，请选择其他系统和语言以查找相应的语言版本。
