NetBeans Mobility Pack for CLDC 5.5.1 Installation Guide

The NetBeans Mobilty Pack 5.5.1 can be used to rapidly develop applications for Java Micro Edition (Java ME) platform enabled devices like pagers, mobile phones, and set-top boxes. It integrates support for the Mobile Information Device Profile (MIDP) 2.0 and the Connected Limited Device Configuration (CLDC) 1.1.

Prerequisites

Before you can install NetBeans Mobility Pack 5.5.1 you must have the following software packages installed on your computer.

 

Required Software:

  • Java SE Development Kit (JDK) 5.0
  • NetBeans IDE 5.5.1

 

Note: For information about system requirements for the software included in this bundle, see the NetBeans Mobility Pack for CLDC 5.5.1 Release Notes.

Installing the Software

This section contains instructions for installing NetBeans Mobility Pack 5.5.1 on all supported platforms:

Windows

Following are instructions for installing NetBeans Mobility Pack 5.5.1 on the Microsoft Windows platform.

To Install:

  1. When you have downloaded the installer file, netbeans_mobility-5_5_1-win.exe, double-click the installer's icon to launch the installer.
  2. At the Welcome page of the installation wizard, click Next.
  3. Accept the license agreement and click Next.
  4. At the installation directory page of the installation wizard, specify the NetBeans IDE 5.5.1 installation.
  5. The installation wizard searches for compatible JDK installations. If your JDK is not in the default location, you need to specify the location of your Java JDK 1.5.0 or higher. You can also type the path to another installed JDK or use the Browse button to navigate to and select another installation.
  6. When installation is complete, click Finish to exit the wizard.

 

Linux

Following are instructions for installing NetBeans Mobility Pack 5.5.1 on the Linux platform.

To Install:

  1. Navigate to the directory that contains the Mobility Pack installer file you downloaded named netbeans_mobility-5_5_1-linux.bin
  2. If necessary, change the permissions on the installer file to make the binary executable by using the command shown below at a command prompt. $ chmod +x netbeans_mobility-5_5_1-linux.bin
  3. Launch the installer by using the command shown below at a command prompt. $ ./netbeans_mobility-5_5_1-linux.bin
  4. At the Welcome page of the installation wizard, click Next.
  5. Accept the license agreement and click Next.
  6. At the Installation Directory page of the installation wizard, accept the NetBeans IDE directory found by the wizard or select a different IDE installation directory you prefer.
  7. Verify that the installation location is correct and that you have enough free disk space on your system to complete the installation.
  8. Click Install to begin the installation.
  9. When the installation is complete, click Finish to exit the wizard.

Installing Emulator Platforms

Adding a new Emulator Platform

  1. Choose Tools > Java Platform Manager from the main toolbar.
  2. In the Java Platform Manager dialog, click the Add Platform button.
  3. In the Select Platform Type page, choose the Java Micro Edition Platform Emulator. Click Next.
  4. In the Platforms Folders page, the IDE searches for all installed and available Java ME platforms and lists them in the window. Unified Emulator Interface (UEI) compliant platforms are verified and appear with a check in the check box next to the platform name. Platforms that are not UEI-compliant are listed in red. The wizard also identifies which platforms have already been installed.

    5. NOTE: To install platforms that are not UEI-compliant, click the Back button and choose Custom Java Micro Edition Platform Emulator.

  5. In the Platforms page, check the platform emulators you want to install. Click Finish. Then click Close. The selected platform or platforms are installed.

Uninstalling the Software

If you installed the bundled Application Server, the uninstallation removes NetBeans Mobility Pack 5.5.1 from your system.

Windows

To uninstall the Software

  1. Shut down the IDE.
  2. From the Control Panel, start Add or Remove Programs, select NetBeans(TM) Mobility Pack and click Change/Remove.
  3. Follow the instructions in the uninstaller.

Linux

To uninstall the Software

  1. Shut down the IDE.
  2. Navigate to the bin subdirectory of your NetBeans Mobility Pack 5.5.1 installation directory.
  3. Execute the uninstall.sh script.
