NetBeans Mobility Pack for CLDC 5.5.1 安装指南

NetBeans Mobilty Pack 5.5.1 可用于为支持 Java Micro Edition (Java ME) 平台的设备快速开发应用程序，这些设备包括寻呼机、移动电话和机顶盒。该软件既支持移动信息设备配置文件 (Mobile Information Device Profile, MIDP) 2.0，又支持联网的受限设备配置 (Connected Limited Device Configuration, CLDC) 1.1。

先决条件

安装 NetBeans Mobility Pack 5.5.1 之前，您必须首先在计算机上安装以下软件包：

 

所需的软件：

  • Java SE Development Kit (JDK) 5.0
  • NetBeans IDE 5.5.1

 

注意：有关本软件包中所包含软件的系统要求信息，请参见 NetBeans Mobility Pack for CLDC 5.5.1 发行说明。

安装软件

本部分包含有关在所有支持的平台上安装 NetBeans Mobility Pack 5.5.1 的说明。

Windows

以下是在 Microsoft Windows 平台上安装 NetBeans Mobility Pack 5.5.1 的说明。

安装步骤：

  1. 下载安装文件 netbeans_mobility-5_5_1-win.exe 后，双击安装程序图标以启动该安装程序。
  2. 在安装向导的“欢迎”页中，单击“下一步”。
  3. 接受许可证协议，然后单击“下一步”。
  4. 在安装向导的安装目录页中，指定 NetBeans IDE 5.5.1 的安装位置。
  5. 安装向导将搜索兼容的 JDK 安装。如果 JDK 没有位于缺省位置，则需要指定 Java JDK 1.5.0 或更高版本的位置。您也可以键入其他 JDK 的安装路径，或者使用“浏览”按钮导航并选择其他安装。
  6. 安装完成后，单击“完成”退出向导。

 

Linux

以下是在 Linux 平台上安装 NetBeans Mobility Pack 5.5.1 的说明。

安装步骤：

  1. 导航至下载的 Mobility Pack 安装文件 netbeans_mobility-5_5_1-linux.bin 所在的目录。
  2. 如果需要，请在命令提示符下使用以下命令来更改安装文件的权限，以便可以执行该二进制文件： $ chmod +x netbeans_mobility-5_5_1-linux.bin
  3. 在命令提示符下，使用以下命令来启动安装程序： $ ./netbeans_mobility-5_5_1-linux.bin
  4. 在安装向导的“欢迎”页中，单击“下一步”。
  5. 接受许可证协议，然后单击“下一步”。
  6. 在安装向导的安装目录页中，接受该向导找到的 NetBeans IDE 目录，或者选择所需的其他 IDE 安装目录。
  7. 请确保安装位置正确无误，并且系统上有足够的可用硬盘空间用于完成安装。
  8. 单击“安装”开始安装。
  9. 安装完成后，单击“完成”退出向导。

安装仿真器平台

添加新的仿真器平台

  1. 从主工具栏中选择“工具”>“Java 平台管理器”。
  2. 在“Java 平台管理器”对话框中，单击“添加平台”按钮。
  3. 在“选择平台类型”页中，选择“Java Micro Edition 平台仿真器”。单击“下一步”。
  4. 在“平台文件夹”页中，IDE 会搜索所有安装的可用 Java ME 平台并将它们列在窗口中。IDE 将验证与统一仿真器接口 (Unified Emulator Interface, UEI) 兼容的平台，并且在这些平台名称旁边的复选框中会出现复选标记。与 UEI 不兼容的平台将被标记为红色。该向导还可以标识已安装的平台。

    5. 注意：要安装与 UEI 不兼容的平台，请单击“上一步”按钮，然后选择“定制 Java Micro Edition 平台仿真器”。

  5. 在“平台”页中，选中要安装的平台仿真器。单击“完成”。然后，单击“关闭”。系统将安装选定的一个或多个平台。

卸载软件

如果安装了捆绑的应用服务器，则卸载时将从系统中删除 NetBeans Mobility Pack 5.5.1。

Windows

卸载步骤：

  1. 关闭 IDE。
  2. 从“控制面板”中启动“添加或删除程序”，选择 "NetBeans(TM) Mobility Pack"，然后单击“更改/删除”。
  3. 按照卸载程序中的说明执行操作。

Linux

卸载步骤：

  1. 关闭 IDE。
  2. 导航至 NetBeans Mobility Pack 5.5.1 安装目录的 bin 子目录。
  3. 执行 uninstall.sh 脚本。
