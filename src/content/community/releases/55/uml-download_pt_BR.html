NetBeans 5.5, Download do Módulo de modelagem UML e Instruções de instalação

O NetBeans 5.5â„¢, Módulo de modelagem UML fornece os recursos de modelagem UML para o NetBeans IDE. A modelagem UML possibilita que analistas e designers criem aplicativos usando uma linguagem de modelagem padrão. Conseqüentemente, os desenvolvedores podem gerar códigos-fonte a partir do modelo UML e atualizar o modelo a partir das alterações feitas nos seus códigos-fonte.

O módulo de modelagem UML pode ser baixado e instalado através da Central de atualizações do NetBeans. Você pode acessar a Central de atualizações, clicando em Ferramentas > Central de atualizações no menu principal do NetBeans IDE.

Observação para usuários do Mac: O módulo de modelagem UML atualmente não é compatível com a plataforma Macintosh OS X. Consulte as Notas de versão da plataforma Mac OS para obter mais informações sobre os problemas e soluções conhecidos.

Pré-requisito

Esse tópico especifica o pré-requisito que deve ser cumprido antes do início da instalação do NetBeans 5.5, Módulo de modelagem UML.

Software necessário

Antes da instalação do NetBeans 5.5, Módulo de modelagem UML, o software abaixo deve ser instalado:

  • NetBeans IDE 5.5 (daqui em diante denominado IDE)
Observação: Para obter informações sobre os requisitos do sistema do NetBeans 5.5, Módulo de modelagem UML, consulte as Notas de versão do Módulo de modelagem UML.

Fazendo download e instalando o software

Esta seção contém informações para fazer o download e instalar o NetBeans 5.5, Módulo de modelagem UML em todas as plataformas aceitas.

Observação: Se você instalou anteriormente alguma versão beta do Módulo de modelagem UML, não será necessário desinstalá-la. Entretanto, haverá algumas pequenas diferenças na parte posterior do processo de instalação e elas serão indicadas nas etapas abaixo.

  1. Clique em Ferramentas > Central de atualizações no menu principal do IDE.
    O assistente da Central de atualizações se abre.

  2. Na página Selecionar local dos módulos, marque a caixa de seleção Central de atualizações do NetBeans, caso ela ainda não esteja marcada.

  3. Opcionalmente, desmarque todas as outras caixas de seleção para diminuir o número de itens na lista resultante de módulos disponíveis.

  4. Clique em Próximo.
    Uma caixa de diálogo de progresso se abre quando o IDE se conecta à Central de atualizações do NetBeans.

  5. Na página Selecionar módulos a instalar do assistente da Central de atualizações, role para o nó Modelagem UML sob a pasta Recursos.

  6. Selecione o módulo de modelagem UML e clique em Adicionar. 
    Todos os módulos de modelagem UML e seus respectivos módulos de bibliotecas dependentes devem ser mostrados no painel Incluir na instalação, à direta.

  7. Clique em Mais para exibir a página do projeto de modelagem UML.
    Um navegador da Web separado exibe a página do projeto UML.

  8. Retorne ao assistente da Central de atualizações e clique em Próximo.

  9. Leia e aceite as quatro licenças do UML.
    A página Fazer download dos módulos se abre e o progresso é mostrado conforme os módulos são baixados. 

    Observação: Se estiver atualizando a partir de uma versão beta do módulo de modelagem UML, não haverá solicitação para aceitar as quatro licenças do UML.

  10. Depois que os módulos tiverem sido baixados, clique em Próximo.
    A lista de certificados é disponibilizada para a sua visualização.

  11. Clique em Finalizar.
    Se estiver atualizando a partir de uma versão beta do módulo de modelagem UML, a caixa de diálogo Reiniciar o IDE se abrirá. Prossiga para a próxima etapa

    Caso não esteja atualizando a partir de uma versão beta, a instalação do módulo UML continua. Isso leva alguns minutos depois que o assistente da Central de atualizações desaparece. Quando a instalação estiver concluída, você verá a mensagem 'Turning on modules...done' exibida no canto esquerdo inferior da janela do IDE.

  12. Na caixa de diálogo Reiniciar o IDE, escolha Reiniciar o IDE e clique em OK para continuar com a instalação do módulo de modelagem UML.
    Se optar por Instalar depois, o módulo de modelagem UML não será instalado até a próxima vez em que o IDE for reiniciado.

  13. Leia a página Notas de versão para obter informações importantes sobre a versão.

Aumentando as configurações de memória

As configurações de memória padrão do NetBeans devem ser aumentadas quando se estiver trabalhando com projetos UML no IDE. Quando tiver instalado o módulo UML, use as etapas seguintes para garantir o uso de configurações de memória apropriadas no IDE ao trabalhar com projetos UML.

  1. No menu principal do IDE, clique em Ajuda > Sobre e clique na guia Detalhe.

  2. Determine o caminho para o diretório_de_instalação_do_NetBeans55 no qual você instalou o NetBeans 5.5 IDE, localizando o campo Current Dir.
    Por exemplo, o diretório de instalação padrão é C:\Program Files\netbeans-5.5 na plataforma Microsoft Windows.


  3. Altere o diretório para o diretório_de_instalação_do_NetBeans55 que você identificou na etapa anterior.

  4. Altere para o diretório etc/.

  5. Edite o arquivo netbeans.conf.

  6. Na linha netbeans_default_options, altere a opção da linha de comando -J-Xmx128m para -J-Xmx512m.
    Observação: Se depois de completar esta etapa você ainda estiver enfrentando excessos de pilhas ao trabalhar com arquivos grandes, aumente mais esse valor.
