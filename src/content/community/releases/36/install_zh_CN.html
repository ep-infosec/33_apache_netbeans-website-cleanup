NetBeans™ IDE 3.6 安装指南

本部分详细介绍了在以下操作系统上的安装：

  • Windows
  • Solaris™ 操作系统 (Solaris OS) 和 Linux 平台
  • Macintosh OS X
  • 其它操作系统

在 Java™ 2 软件开发工具包 (J2SDK) v. 1.5 Beta 版上安装

有关其它信息（如系统要求以及如何使用不同的启动参数启动 IDE），请参见发行说明。

Windows

在 Microsoft Windows 平台上，您可以选取以下下载文件之一：

  • 自解压安装程序
  • 归档文件发行版本

使用 Windows 安装程序

在下载安装程序文件之后，双击该文件可启动安装向导。通过安装程序，您可以指定：

  • 要在其中运行 IDE 的 Java™ 2 软件开发工具包。
  • NetBeans IDE 的安装目录。
  • 是否将 .java 及 .nbm（NetBeans 模块文件）与 IDE 关联。

要启动 NetBeans IDE，请执行以下操作之一：

  • 双击桌面上的“NetBeans IDE”图标。
  • 从“开始”菜单中选择“NetBeans IDE”|“NetBeans IDE 3.6”|“NetBeans IDE”。

在 Windows 计算机上使用归档文件安装

虽然在 Microsoft Windows 计算机上最好使用上述安装程序进行安装，但您也可以使用 tar.gz 或 zip 文件安装 NetBeans IDE。常见的归档公用程序（如 Winzip）可以处理这两种文件类型。

安装 NetBeans IDE：

  1. 找到已下载的归档文件，然后双击该文件以打开归档文件工具。
  2. 使用归档工具将所有文件提取到一个空目录，如 C:\NetBeans IDE 3.6。

启动 NetBeans IDE：

  1. 导航到 NetBeans IDE 安装目录的 bin 子目录。
  2. 双击 runide.exe 启动 NetBeans IDE。

Solaris™ 操作系统 (Solaris OS) 和 Linux 平台

在 Solaris 和 Linux 平台上，您可以选取以下下载文件之一：

  • 二进制安装程序
  • 归档文件发行版本

使用 Solaris 和 Linux 安装程序

安装 NetBeans IDE：

  1. 从命令提示导航到包含安装程序的目录。
  2. 如有必要，请更改权限，以便能够通过键入以下内容创建二进制可执行文件：
    $ chmod +x NetBeans.bin

    （将 NetBeans.bin 替换为下载的发行版本的实际文件名）。

  3. 通过键入以下内容启动安装程序：
    $ ./NetBeans.bin

    （将 NetBeans.bin 替换为下载的发行版本的实际文件名）。

安装程序将搜索任何已安装的 J2SDK，并提示您应使用哪个 NetBeans IDE。您可以在命令行指定 J2SDK，这可以加快安装进程。例如：

$ ./NetBeans.bin -is:javahome path_to_your_jdk

在 UNIX(R) 平台上使用归档文件安装

要安装 NetBeans IDE，请在您的平台上使用适当的工具将归档文件发行版本解包或解压缩到空目录中。例如，从命令提示键入以下内容：

$ gzip -d NetBeans.tar.gz
$ tar xf NetBeans.tar

（将 NetBeans.tar.gz 和 NetBeans.tar 替换为实际文件名）。

注意：Solaris 用户应该使用 GNU tar 解包 tar.gz 文件，以确保整个归档文件都被解压缩。

要启动 NetBeans IDE，请转到 IDE 安装目录的 bin 子目录，并执行 runide.sh 启动程序脚本。

Macintosh OS X

在 Macintosh OS X 平台上，您可以选取以下下载文件之一：

  • 磁盘映像格式 (.dmg) 文件
  • 归档文件发行版本

注意：Mac OS 用户应该使用 .dmg 文件，该文件包含用于纠正问题 39780 中提及的问题的修补程序。如果您在 Mac OS 上安装了 IDE 的归档文件发行版本，则可能遇到无法正常关闭 IDE 的问题。

使用磁盘映像格式 (.dmg) 文件

在下载映像文件后，双击该文件可将其安装在系统中。

要启动 NetBeans IDE，请运行 NetBeansLauncher 应用程序。

在 Mac OS 平台上使用归档文件安装

安装 NetBeans IDE：

  • 打开 Terminal 应用程序，然后将目录转到 IDE 的安装位置。在命令行上键入以下内容：

    gnutar -xvzf netbeans.tar.gz

    （将 NetBeans.tar.gz 替换为下载的发行版本的实际文件名）。

要启动 NetBeans IDE，请转到 IDE 安装目录的 bin 子目录，并执行 runide.sh 启动程序脚本。

其它操作系统

使用归档文件安装

使用适合于您系统的公用程序解压缩归档文件。

要启动 NetBeans IDE，请导航到 NetBeans IDE 安装目录的 bin 子目录，然后执行适合于您系统的启动程序。以下是可用的启动程序：

runide.exe - Windows
runidew.exe - Windows（无控制台窗口）
runide.sh - UNIX
runideos2.cmd - OS/2
runideopenvms.com - OpenVMS

如果没有适合于特定操作系统的启动程序，则可能需要修改最合适的脚本尝试创建一个启动程序。如果您的计算机支持 JDK 1.4 或更高版本，则应能够运行 IDE。

注意：如果创建启动程序，则可以将其提供给项目。

在 Java™ 2 软件开发工具包 (J2SDK) v. 1.5 Beta 版上安装

您可以在 J2SDK v. 1.5 Beta 版上运行 IDE，但如果它是系统中唯一的 J2SDK，安装程序将无法运行。您还必须安装 J2SDK v. 1.4.1 或兼容版本才能运行安装程序。安装程序运行后，您可以指定 IDE 应在 J2SDK v. 1.5 Beta 版上运行。
