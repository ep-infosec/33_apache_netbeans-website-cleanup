インストールの詳細

  • Windows
  • Solaris[tm] オペレーティングシステム (Solaris OS) および Linux プラットフォーム
  • Macintosh OS X
  • その他のオペレーティングシステム

Java[tm] 2 Software Development Kit (J2SDK), v.1.5 ベータ版を使用する場合のインストール

システム要件や IDE の起動時に使用する各種起動パラメータなどについては、リリースノートを参照してください。

Windows

Microsoft Windows プラットフォームでは、次のいずれかを選択してダウンロードします。

  • 自己解凍型インストーラ
  • アーカイブ版

Windows インストーラによるインストール

ダウンロードしたインストーラファイルをダブルクリックすると、インストールウィザードが起動します。インストーラでは、次の事項を指定することができます。

  • IDE の実行に使用する Java[tm] 2 Software Development Kit
  • NetBeans IDE のインストール先ディレクトリ
  • .java および .nbm (NetBeans モジュールファイル) を IDE に関連付けるかどうか

NetBeans IDE を起動するには、次のいずれかの操作を行います。

  • デスクトップの NetBeans IDE 3.6 アイコンをダブルクリックする。
  • 「スタート」メニューから「NetBeans IDE」>「NetBeans IDE 3.6」>「NetBeans IDE」 を選択する。

Windows マシンでのアーカイブ版によるインストール

アーカイブ版を選択した場合は、tar.gz ファイルまたは zip ファイルを使用し、NetBeans IDE を Microsoft Windows マシンにインストールすることができます。 アーカイブファイルは、Winzip などの一般的なアーカイブユーティリティを使用して解凍することができます。

NetBeans IDE をインストールするには、次の操作を行います。

  1. ダウンロードしたアーカイブファイルの場所で、ファイルをダブルクリックします。アーカイブユーティリティを選択します。
  2. アーカイブユーティリティを使用し、すべてのファイルを空のディレクトリ (例: C:\NetBeans IDE 3.6) に解凍します。

NetBeans IDE を起動する手順は、次のとおりです。

  1. NetBeans IDE のインストールディレクトリの bin サブディレクトリに移動します。
  2. runide.exe をダブルクリックします。NetBeans IDE が起動します。

Solaris[tm] オペレーティングシステム (Solaris OS) および Linux プラットフォーム

Solaris および Linux プラットフォームでは、次のいずれかかを選択してダウンロードします。

  • バイナリインストーラ
  • アーカイブ版

Solaris および Linux インストーラによるインストール

NetBeans IDE をインストールするには、次の操作を行います。

  1. コマンドプロンプトから、インストーラが保存されているディレクトリに移動します。
  2. 必要に応じて、以下のように入力し、バイナリインストーラを実行できるようにアクセス権を変更します。
    $ chmod +x NetBeans.bin

    (NetBeans.bin の部分は、実際にダウンロードしたファイル名に置き換えてください)

  3. 次のように入力してインストーラを起動します。
    $ ./NetBeans.bin

    (NetBeans.bin の部分は、実際にダウンロードしたファイル名に置き換えてください)

インストールされている J2SDK が検索された後、インストーラが起動されます。コマンド行で J2SDK を指定することもできます。これによって、J2SDK の検索時間が短縮され、インストールにかかる時間が短縮される場合があります。以下に例を示します。

$ ./NetBeans.bin -is:javahome path_to_your_jdk

UNIX® プラットフォームでのアーカイブ版によるインストール

NetBeans IDE をインストールするには、次の操作を行います。tar または unzip コマンドを使用し、アーカイブ版を空のディレクトリに展開します。たとえば、コマンドプロンプトから次のように入力します。

$ gzip -d NetBeans.tar.gz
$ tar xf NetBeans.tar

(NetBeans.tar.gz および NetBeans.tar の部分は、実際のファイル名に置き換えてください)

注: Solaris で tar.gz ファイルを使用する場合は、GNU tar を使用すると、アーカイブ全体が確実に展開されます。

NetBeans IDE を起動するには、インストールディレクトリの bin サブディレクトリに移動し、起動スクリプト runide.sh を実行します。

Macintosh OS X

Macintosh OS X プラットフォームでは、次のいずれかを選択してダウンロードします。

  • ディスクイメージファイル (.dmg)
  • アーカイブ版

注: Mac OS では、issue 39780 で説明している問題を修正するためのパッチが含まれている .dmg ファイルを使用してください。また、Mac OS にアーカイブ版の IDE をインストールすると、IDE を正常に終了できないという問題が発生する場合があります。

ディスクイメージファイル (.dmg) によるインストール

ダウンロードしたイメージファイルをダブルクリックし、システムにマウントします。

NetBeansLauncher アプリケーションを実行します。NetBeans IDE が起動されます。

Mac OS プラットフォームでのアーカイブ版によるインストール

NetBeans IDE をインストールするには、次の操作を行います。

  • Terminal アプリケーションを開き、IDE のインストール先となるディレクトリに移動します。コマンド行で次のように入力します。

    gnutar -xvzf netbeans.tar.gz

    (netbeans.tar.gz の部分は、実際にダウンロードしたファイル名に置き換えてください)

NetBeans IDE を起動するには、インストールディレクトリの bin サブディレクトリに移動し、起動スクリプト runide.sh を実行します。

その他のオペレーティングシステム

アーカイブ版によるインストール

システムに応じたユーティリティを使用してアーカイブを展開します。

NetBeans IDE を起動するには、NetBeans IDE のインストールディレクトリの bin サブディレクトリに移動し、システムに応じた起動スクリプトを実行します。使用可能な起動スクリプトは以下のとおりです。

runide.exe - Windows
runidew.exe - Windows (コンソールウィンドウなし)
runide.sh - UNIX
runideos2.cmd - OS/2
runideopenvms.com - OpenVMS

専用の起動スクリプトが存在しないオペレーティングシステムを使用している場合は、既存のスクリプトを修正し、そのオペレーティングシステム用の起動スクリプトを作成する必要があります。JDK 1.4 以上をサポートしているマシンであれば、IDE は実行可能です。

注: 作成した起動スクリプトは、netbeans.org への公開をお願いします。

Java[tm] 2 Software Development Kit (J2SDK), v.1.5 ベータ版を使用する場合のインストール

IDE は、J2SDK v.1.5 を使用して起動できますが、システムにインストールされている J2SDK v.1.5 がベータ版だけである場合は、そのベータ版を使用してインストーラを実行することはできません。インストーラを実行するには、J2SDK v.1.4.1 またはその互換版がインストールされている必要があります。いったんインストーラが実行されれば、IDE の実行に使用する J2SDK として J2SDK v.1.5 ベータ版を指定できるようになります。
