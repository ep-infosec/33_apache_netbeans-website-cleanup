NetBeans IDE 6.5 情報

NetBeans IDE は、受賞歴のある統合開発環境で、Windows、Mac、Linux、および Solaris で使用可能です。NetBeans プロジェクトは、オープンソースの IDE とアプリケーションプラットフォームから構成されます。NetBeans を使用すると、開発者は Java プラットフォームや PHP、JavaScript と Ajax、Ruby と Ruby on Rails、Groovy、および C/C++ を使用して、Web アプリケーション、エンタープライズアプリケーション、デスクトップアプリケーション、およびモバイルアプリケーションを短時間で作成できます。活発な開発者コミュニティーによってサポートされ、さまざまなサードパーティーのプラグインを提供します。ソフトウェア開発者にとって NetBeans IDE はダウンロードが欠かせないツールです。

リリース概要

NetBeans IDE 6.5 は、すべての Java プラットフォーム (Java SE、Java EE、Java ME、JavaFX) が完全にサポートされているだけでなく、PHP、Ajax と JavaScript、Groovy と Grails、Ruby と Ruby on Rails、および C/C++ を使用するソフトウェア開発にも最適なツールです。NetBeans IDE 6.5 release にはいくつかの新機能と拡張機能があります。たとえば IDE 全体を高速検索できる新しいショートカット、さらに使いやすいインタフェース、そして保存時の自動コンパイル機能があります。6.5 release では、Web フレームワーク (Hibernate、Spring、JSF、JPA)、GlassFish アプリケーションサーバー、およびデータベースのサポートも強化されています。

主な新機能

PHP

  • PHP エディタ (コード補完、構文と意味解釈のコードの強調表示)
  • ヒアドキュメント表記と PHTML のサポート
  • Xdebug を使用したデバッグ
  • MySQL データベースのコードスニペットの生成

JavaScript と Ajax

  • FireFox と Internet Explorer での JavaScript のデバッグ
  • HTTP トランザクションの監視
  • JavaScript と CSS エディタの改善
  • JavaScript ライブラリマネージャー (Yahoo UI、Woodstock、jQuery、Dojo、Scriptaculous、Prototype のライブラリを含む)

Java EE と Web の開発

  • Spring、Hibernate、JSF、JSF CRUD ジェネレータ、JPA (Java Persistence API) のサポート強化
  • データベース表または JPA エンティティーからの RESTful Web サービスの作成
  • PHP ファイルへの SaaS サービスのドラッグ＆ドロップ
  • SQL エディタの改善 (コード補完、クエリーの保存と再呼び出しなど)
  • Java Web プロジェクトでの Eclipse プロジェクトのインポートと同期の改善
  • 保存時の自動配備

JavaFX

  • JavaFX プロジェクトの作成、テスト、デバッグ、プロファイル、配備
  • JavaFX エディタ (構文の強調表示、ソースナビゲーション、コード補完、コードの折り畳み、javadoc のポップアップ、エラー検出)
  • 視覚要素のライブプレビュー
  • 変換、効果、アニメーション用のコードスニペットパレット

Groovy および Grails

  • 純粋な Groovy アプリケーションの開発、または Java SE プロジェクトでの Groovy の使用
  • Groovy エディタ (コード補完、強調表示など)
  • Grails Web アプリケーションフレームワーク
  • メタデータを追加しないで既存の Grails アプリケーションを開く

Ruby および Rails

  • テスト結果を表示するための新しいテストランナーインタフェース
  • Rake の改善: Rake ランナーと事前生成した Rake ファイル
  • デバッガ: 条件付きブレークポイントとキャッチポイント
  • Ruby プロジェクトでコマンド行から JVM オプションを指定可能

GlassFish v3 Prelude による Web 開発

  • OSGi に基づいたモジュール式アーキテクチャー
  • 小さなフットプリント、高速の起動と配備
  • JRuby を含むスクリプト作成のサポート

C/C++

  • コード補完、エラーの強調表示、意味解釈の強調表示の改善
  • コールグラフ、メモリーウィンドウ
  • リモート開発
  • アプリケーションのパッケージング (tar ファイル、zip ファイル、または SVR4 パッケージ)

Java ME

  • カスタムコンポーネントをビジュアルデザイナのパレットに追加するための新規プロジェクトウィザード
  • 改善された SVG コンポーザの新しい SVG UI コンポーネントによって優れた UI デザインが可能
  • ビジュアルデザイナのパレットでのデータバインド用の新しいカスタムコンポーネント
  • 難読化ツールを ProGuard 4.2、テストフレームワークを JMUnit 1.1.0 にアップグレード

Java デバッガ

  • UI とワークフローが改善された新しいマルチスレッドのデバッグ
  • 式内のメソッドを選択できるスマートなステップイン機能
  • デッドロックの検出と視覚的な表示

Java SE

  • 保存時の自動コンパイル
  • Eclipse プロジェクトのインポートと同期の改善
  • Java 呼び出し階層
  • Javadoc の解析
  • Swing GUI ビルダーでクラスの簡単な名前または完全修飾名を生成可能

IDE のツールと使いやすさ

  • キャメルケースのコード補完
  • プロジェクトごとの整形設定
  • ファイル、シンボル、型、IDE アクション、オプション、ドキュメントを IDE 全体で高速検索できるショートカット
  • Windows UNC パス (共有ネットワークフォルダ) のサポート
