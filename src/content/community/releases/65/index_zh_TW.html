NetBeans IDE 6.5 發行資訊

翻譯者: David Lin (社群成員)

NetBeans IDE 是一個獲獎的整合式開發環境，適用於 Windows，Mac，Linux 與 Solaris 平台。NetBeans 計畫包含了以下幾個項目 開源的 IDE 以及 應用程式平台 ，讓開發人員能夠快速地使用 Java 建立 網路應用, 企業, 桌面, 以及 行動裝置 等應用程式。 此外，還提供了對 PHP, JavaScript 與 Ajax, Ruby 與 Ruby on Rails, Groovy, 與 C/C++ 等各種技術的支援。整個計畫是由廣大的 開發人員社群 所支援 並且提供了眾多 外掛程式 的選擇。

發行一覽

除了提供對 Java 平台 (Java SE, Java EE, Java ME 與 Java FX) 的完整支援之外 NetBeans IDE 6.5 在開發 PHP, JavaScript 與 Ajax, Ruby 與 Ruby on Rails, Groovy, 與 C/C++ 等各種技術時的理想工具。這次 6.5 版提供了針對網路應用框架的加強支援 (包含了 Hibernate, Spring, JSF, JPA 等)，以及 GlassFish 應用程式伺服器，與資料庫等等。此外還提供了一個在整個 IDE 中都可以使用的快速搜尋功能捷徑，增加了使用者介面的親和力，以及當儲存程式時自動編譯的功能。

功能與特性介紹

PHP

  • PHP 編輯器 (程式碼自動完成, 文法及語義的高亮度支援)
  • 對 heredoc 注釋與 PHTML 的支援
  • 使用 Xdebug 除錯
  • MySQL 資料庫代碼片段自動產生器

JavaScript 與 Ajax

  • FireFox 與 Internet Explorer 上的 JavaScript 除錯
  • 監視 HTTP 交易
  • 改良的 JavaScript 與 CSS 編輯器
  • JavaScript 涵式庫管理器 包含了 Yahoo UI, Woodstock, jQuery, Dojo, Scriptaculous, Prototype 等涵式庫

Java EE 與 網路應用開發

  • 針對 Spring, Hibernate, JSF, JSF CRUD generator, JPA (Java Persistence API) 等技術的加強支援
  • 從資料庫表格或 JPA 實體建立 RESTful 網際網路服務
  • 可直接拖曳 SaaS 服務至 PHP 檔案
  • 改良式的 SQL 編輯器 (程式碼自動完成, 儲存/回呼 查詢指令， 以及更多其他功能)
  • 更好的 Eclipse 專案導入精靈以及 Java 網路專案的同步化功能
  • 儲存時自動部署功能

JavaFX

  • 建立，測試，除錯，優化，以及部署 JavaFX 專案
  • JavaFX 編輯器支援語法高亮度顯示, 程式碼結構導覽器, 程式碼自動完成, 程式碼縮放, 跳出式 javadoc 視窗, 以及錯誤偵測等等。
  • 即時預覽視覺化的元件
  • 程式碼片段調色盤提供了 transformations, effects 與 animations 等實用功能

Groovy 與 Grails

  • 開發純 Groovy 應用程式或在 Java SE 專案中使用 Groovy
  • Groovy 編輯器支援程式碼自動完成, 語法高亮度顯示, 以及更多功能
  • Grails 網路應用程式框架
  • 不用加入額外的中介資料即能開啟現有的 Grails 應用程式

Ruby and Rails

  • 全新的 Test Runner 介面可顯示測試結果
  • 改良過的 Rake 功能：Rake Runner 與一個預先產生的 Rake 檔案
  • 除錯器: 條件化中斷點與 catchpoints
  • Ruby 專案現在能夠直接從命令提示字元接收 JVM 的選項

GlassFish v3 Prelude 支援網路應用開發

  • 模組化， 以 OSGi 為基礎的架構
  • 消耗更少的系統資源，啟動和部署速度更快
  • 支援 scripting，包含了 JRuby

C/C++

  • 改良過的自動程式碼完成, 錯誤自動高亮度顯示, 以及語義高亮度顯示
  • 呼叫圖，記憶體內容視窗
  • 支援遠端開發
  • 提供將應用程式包裝成 tar, zip 檔案或是 SVR4 套件等功能

Java ME

  • 全新的專案精靈支援了加入客製化的新元件到視覺化開發編輯器調色盤的功能
  • 在加強型 SVG Composer 中加入了全新的 SVG UI 元件，讓開發者能夠進行 rich UI 設計
  • 在視覺化開發編輯器中加入了全新的 Data Binding 客製化元件
  • 提供最新的程式碼混淆器 ProGuard 4.2 與測試框架 JMUnit 1.1.0 等版本

Java 除錯器

  • 前新的多執行緒除錯功能並且提供更友善的 UI 與操作流程
  • 在程式碼陳述式中即能使用支援方法選擇的智慧化 Step Into 功能
  • 視覺化的 Deadlock 偵測功能

Java SE

  • 儲存時自動編譯功能
  • 改良式的 Eclipse 專案導入與同步功能
  • Java 呼叫階層
  • 分析 Javadoc 的功能
  • Swing GUI 編輯器現在能夠產生簡化過的或是完整的類別名稱

IDE 工具與可用性

  • CamelCase 程式碼自動完成
  • 針對每個專案都能做客製化的格式設定
  • 全域性的 IDE 快速搜尋捷徑提供了 檔案/符號/型別, IDE 動作, 選項, 與文件等搜尋支援
  • 提供對 Windows UNC 路徑 (共享網路資料夾) 的支援
