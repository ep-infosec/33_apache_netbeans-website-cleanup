Información de Lanzamiento de NetBeans IDE 6.5

NetBeans IDE es un premiado Entorno de Desarrollo Integrado (IDE) disponible para Windows, Mac, Linux, y Solaris. El proyecto NetBeans consiste en un IDE de código abierto y una plataforma de aplicación la cual permite a los desarrolladores crear rápidamente aplicaciones web, empresariales, de escritorio, y móbiles utilizando la plataforma Java, al igual que las plataformas PHP, JavaScript y Ajax, Ruby y Ruby en Rails, Groovy, y C/C++. Está respaldado por una comunidad de desarrollo muy activa y ofrece una diversa selección de extensiones (plugins) de terceros.

Generalidades del Lanzamiento

Además del soporte completo para todas las plataformas Java (Java SE, Java EE, Java ME, y JavaFX), NetBeans IDE 6.5 es la herramienta ideal para el desarrollo de software con PHP, Ajax y JavaScript, Groovy y Grails, Ruby y Ruby en Rails, y C/C++. El lanzamiento 6.5 provee mejoras en el soporte de entornos generales (frameworks) web (Hibernate, Spring, JSF, JPA), el servidor GlassFish, y bases de datos. Adicionalmente incluye un nuevo atajo IDE-wide QuickSearch (Busqueda rápida), un interface de usuario más amigable, y una Compilación automática al Guardar.

Características Destacadas

PHP

  • Editor PHP (Completación de código, resaltado de código sintáctico y semántico)
  • Soporte para notación heredoc y PHTML
  • Depurar usando Xdebug
  • Generadores de trozos de código para bases de datos MySQL

JavaScript y Ajax

  • Depurador JavaScript en FireFox e Internet Explorer
  • Monitor de transacciones HTTP
  • Editores JavaScript y CSS mejorados
  • Administrador de Librerías JavaScript incluyendo librerías Yahoo UI, Woodstock, jQuery, Dojo, Scriptaculous, y Prototype

Java EE y Desarrollo Web

  • Soporte mejorado para Spring, Hibernate, JSF, generador JSF CRUD, JPA (Java Persistence API)
  • Creación de servicios web RESTful a partir de tablas de bases de datos o a partir de entidades JPA
  • Arrastrar y soltar servicios SaaS dentro de archivos PHP
  • Editor SQL mejorado (completación de código, guardar/recordar queries, y más)
  • Importación y sincronización mejoradas de proyectos Eclipse a proyectos Java Web
  • Montaje automático con Guardar (Save)

JavaFX

  • Crear, probar, depurar, perfilar, y montar proyectos JavaFX
  • Editor JavaFX con sintaxis resaltada, navegación en el código fuente, completación de código, pliegue de código, javadoc emergentes, y detección de errores.
  • Previsualización inmediata de elementos visuales
  • Paleta de trozos de código para transformaciones, efectos y animaciones

Groovy y Grails

  • Desarrolle aplicaciones puramente Groovy o use Groovy en sus proyectos Java SE
  • Editor Groovy con completación de código, resaltado, y más
  • Entorno estructurado (framework) de aplicaciones web Grails
  • Abra aplicaciones Grails existentes sin agregar metadata

Ruby y Rails

  • Nuevo interface de Probador de Ejecución (Test Runner) para mostrar los resultados de las pruebas
  • Mejoras Rake: Lanzador Rake (Rake Runner) y archivos Rake pregenerados
  • Depurador: Puntos de ruptura condicionados y puntos de detención
  • Los proyectos Ruby aceptan opciones JVM desde la línea de comandos

Preludio GlassFish v3 Para el Desarrollo Web

  • Modular, arquitectura basada en OSGi
  • Compacto, inicio y montaje rápidos
  • Soporte para escritura, incluyendo JRuby

C/C++

  • Completación de código mejorado, resaltado de errores, y resaltado de semántica
  • Grafo de llamados de funciones (Call Graph), ventana de Memoria
  • Desarrollo remoto
  • Empaquetador de aplicaciones tales como archivos tar, archivos zip, o paquetes SVR4


Java ME

  • Nuevo asistente de proyecto para agregar componentes personalizados a la Paleta Visual de Diseño (Visual Designer Palette)
  • Nuevos componentes SVG UI en el mejorado Estructurador SVG (SVG Composer) permite un muy buen diseño de Interface de Usuario
  • Nuevos Componentes Personalizados de Enlace de Datos (Data Binding Custom Components) en la Paleta Visual de Diseño
  • Herramienta de ofuscación actualizada para ProGuard 4.2 y entorno de desarrollo para JMUnit 1.1.0


Depurador Java

  • Nuevo depurado multi-tarea con Interface de Usuario y fuidez de trabajo mejorados
  • Paso Detallado (Step Into) inteligente con selección de método justo adentro de la expresión
  • Detección de punto muerto con indicador visual

Java SE

  • Compilación automática con Guardar
  • Importación y sincronización mejoradas de proyectos Eclipse
  • Llamados Jerárquicos de Java
  • Analizador Javadoc
  • El constructor Swing GUI puede generar nombres de clases calificados simples o completos


Herramientas y Uso del IDE

  • Completación de código CamelCase
  • Personalizar formatos de configuraciones por cada proyecto
  • Un atajo de Busqueda Rápida (QuickSearch) para archivos/símbolos/tipos, acciones IDE, opciones, y documentos
  • Soporte para Windows UNC paths (carpetas compartidas en red)
