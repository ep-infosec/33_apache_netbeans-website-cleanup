Instrucciones de Instalación de NetBeans IDE 6.5

Última actualización: August 9, 2012

Este documento describe como instalar NetBeans IDE 6.5 en su sistema. Para información acerca de los sistemas operativos soportados y las configuraciones de hardware para el IDE, vea las Notas de Lanzamiento de NetBeans IDE 6.5. Vea Información de NetBeans IDE 6.5 para conocer las nuevas características incluidas en NetBeans IDE 6.5.

Requerimientos de Software

Antes de que instale el IDE, Java SE Development Kit (JDK) 5 Actualización 16 (versión 1.5.0_16) o posterior (incluyendo JDK 6 Actualización 11) debe estar instalado en su sistema. Si usted no tiene una instalación de JDK, no podrá proceder con la instalación.

Para las opciones de descarga PHP, Ruby, y C++, usted puede instalar JRE en lugar de JDK. De todas formas, si planea utilizar alguna característica Java, necesitará JDK.

Nota: El uso de JDK 6 Actualización 12 con NetBeans IDE 6.5 podría causar errores y comportamiento incorrecto del IDE. Por eso, utilice solamente las versiones JDK recomendadas. Para más información, vea las Notas de Lanzamiento de NetBeans IDE 6.5.

Opciones de Descarga del Instalador

En la página de Descarga de NetBeans IDE 6.5, usted puede descargar uno de varios instaladores, cada cual contiene el IDE base y herramientas adicionales.

Los siguientes instaladores están disponibles:

  • Java. Provee herramientas para el desarrollo en aplicaciones Java SE, Java EE y Java ME. Esta opción de descarga también incluye el servidor GlassFish V2 UR2, el servidor GlassFish v3 Prelude, y el software Apache Tomcat.
  • Java SE. Provee todas las características estándares para el desarrollo en Java SE. El soporte para el Desarrollo de Complementos para NetBeans también está incluido.
  • JavaFX. Provee soporte para el desarrollo JavaFX Multi-Platform Rich Application.
  • Ruby. Provee herramientas para el desarrollo Ruby y soporta Rails y JRuby. También contiene el servidor GlassFish v3 Prelude.
  • PHP. Provee herramientas para el desarrollo en PHP. La opción de descarga PHP está disponible en Early Access.
  • C/C++. Soporta el desarrollo en el lenguaje C/C++.
  • All. Esta es una opción de descarga completa de NetBeans IDE 6.5. Contiene todas las aplicaciones y paquetes disponibles para el IDE. Incluye las herramientas para el desarrollo en Servicio de Arquitectura Orientado.

Personalizar su Instalación

Si usted descarga la opciones de descarga Java o "All", usted puede elegir exactamente que herramientas y aplicaciones instalar. Haga click en Customize en la página de bienvenida y seleccione las características que quiera usar. Para más información, vea Instalando el Software.

Si usted quiere agregar funcionalidad a su configuración NetBeans IDE, use el Administrador de Complementos de NetBeans (NetBeans Plugin Manager). Por ejemplo, asuma que usted comienza descargando e instalando el paquete Ruby. Luego usted decide que quiere probar la funcionalidad C/C++. Para hacer esto, vaya al Administrador de Complementos de NetBeans desde el IDE (seleccione Tools > Plugins) y agregue el paquete C/C++ a su instalación existente.

Hay varias notas importantes acerca de la instalación y configuración de algunas características de NetBeans IDE:

  1. Apache Tomcat 6.0.18 está incluido en las opciones de descarga "Java" y "All" pero no está instalado por defecto desde ninguna de estas opciones. Para instalar Apache Tomcat desde la descarga Java o "All", ejecute el instalador y seleccione Apache Tomcat 6.0.18 en el cuadro de diálogo Customize Installation.
  2. Para el desarrollo C/C++ (aplicable para las descargas C/C++ y "All"), usted necesita adicionalmente instalar los compiladores y herramientas. Para más información, refiérase a Instalar y Configurar el Soporte C/C++.
  3. Si usted quiere usar Sun Java System Access Manager, adicionalmente a la opción de descarga "All" para el IDE, usted necesita instalar Java Application Platform SDK Actualización 5 o posterior desde http://java.sun.com/javaee/downloads/index.jsp. Asegúrese de descargar la versión que incluye Access Manager.
  4. Para desarrollar aplicaciones Mobility para aparatos CDC usted necesita descargar Sun Java Toolkit 1.0 para CDC. Después de la instalación usted debe registrar la plataforma CDC en el IDE.
  5. Para Mac OS X, usted puede personalizar la instalación NetBeans IDE desde las opciones de descarga Java, Ruby, y "All". La instalación IDE desde las opciones de descarga Java SE, PHP, y C/C++ no se pueden personalizar.
  6. La instalación de NetBeans IDE para JavaFX está descrita en Instalar NetBeans IDE 6.5 para JavaFX. Para más información acerca de NetBeans IDE para JavaFX, refiérase a Notas de Lanzamiento.

Comenzar la Descarga

  1. Navegue por la página de Descarga de NetBeans IDE 6.5.
  2. En el área superior derecha de la página, seleccione su plataforma desde la lista desplegable.
  3. Haga click en el botón Descarga para las opciones de descarga que quiera instalar.
  4. Guarde el archivo de instalación en su sistema.

Instalar el Software

Siga las instrucciones en esta sección para instalar el IDE en su sistema. Estas instrucciones se aplican a todas las plataformas soportadas. Para una lista de plataformas soportadas, vea las Notas de Lanzamiento de NetBeans IDE 6.5.

Microsoft Windows, Solaris OS, y Linux

Para instalar el software:

  1. Luego que se complete la descarga, ejecute el instalador.
    • Para Windows, el archivo de instalación ejecutable tiene la extensión .exe. Haga doble click en el archivo instalador para ejecutarlo.
    • Para las plataformas Solaris y Linux, el archivo instalador tiene la extensión .sh. Para estas plataformas usted necesita hacer que los archivos de instalación sean ejecutables mediante el uso del siguiente comando:
      chmod +x ./<nombre-de-archivo-instalador>
  2. Para seleccionar las herramientas y aplicaciones a instalar, siga los pasos a continuación en la página de Bienvenida del asistente de instalación:
    1. Haga click en Customize.
    2. En el cuadro de diálogo Customize Installation, haga sus selecciones.
    3. Haga click en OK.
  3. En la página de Bienvenida del asistente de instalación, haga click en Next.
  4. En la página de acuerdo de licencia, revise el acuerdo de licencia, seleccione la casilla de verificación, y haga click en Next.
  5. En la página de instalación de NetBeans IDE, haga lo siguiente:
    1. Acepte el directorio de instalación por defecto para NetBeans IDE o especifique otro directorio.
      Nota: El directorio de instalación debe estar vacío, y el perfil de usuario que usted use para ejecutar el instalador debe tener permisos de lectura y escritura para este directorio.
    2. Acepte la instalación JDK por defecto para usar con NetBeans IDE o seleccione una instalación diferente desde la lista desplegable.
      Si el asistente de instalación no encuentra una instalación JDK compatible para usar con NetBeans IDE, su JDK no está instalado en la dirección por defecto. En este caso, especifique el directorio donde está instalado JDK o cancele la instalación en curso, instale la versión JDK necesaria, y reinicie la instalación.
    3. Haga click en Next.
  6. Si la página del servidor GlassFish V2 UR2 se abre, haga lo siguiente:
    1. Acepte el directorio de instalación por defecto para el servidor o especifique otro directorio de instalación.
      Nota: El directorio de instalación que usted especifique debe estar vacío, y el perfil de usuario que usted está usando para ejecutar la aplicación debe tener permisos de lectura y escritura para este directorio.
    2. Desde la lista de instalaciones JDK compatibles, elija el JDK que quiera que use el servidor o haga click en el botón Browse para especificar otra instalación JDK.
    3. Cambie el nombre de usuario y la clave para el dominio del servidor por defecto o acepte los que están por defecto y haga click en Next.
      El nombre de usuario y la clave por defecto son admin y adminadmin.
    4. Verifique los valores por defecto de los puertos (HTTP, HTTPS, y Admin) para el servidor o modifíquelos si es necesario.
    5. Haga click en Next.
  7. Si la página del servidor GlassFish v3 Prelude se abre, acepte el directorio de instalación por defecto o especifique otra dirección de instalación.
  8. Si la página de instalación de Apache Tomcat se abre, acepte el directorio de instalación por defecto o especifique otra dirección de instalación.
  9. En la página de resumen, verifique que la lista de componentes a ser instalados es correcta y que usted tiene el espacio adecuado en su sistema para la instalación.
  10. Haga click en Install para comenzar la instalación.
  11. Cuando la instalación se complete:
    1. Si el servidor GlassFish ha sido instalado, para registrarlo seleccione la casilla de verificación Register the Sun Java System Application Server and NetBeans IDE después de terminar la instalación.
    2. Si el servidor GlassFish no ha sido instalado, seleccione la casilla de verificación Register the NetBeans IDE después de terminar la instalación para registrar NetBeans IDE.
    3. Haga click en Finish para salir del asistente.
  12. Si usted selecciona la casilla de verificación Register... en el paso anterior, la página Registration se abre en el navegador. Haga click en Register.
  13. Introduzca su cuenta Sun Online o créela en la página www.sun.com, la cual es accesible por la red.

Nota: Si encuentra problemas para completar satisfactoriamente la instalación del software, vea Problemas para descripciones y soluciones sugeridas de problemas no resueltos que podrían afectar el proceso de instalación.

Nota: Si no ha registrado el servidor GlassFish justo después de la instalación, usted puede hacerlo cuando ejecute NetBeans IDE o abra la consola de Administración del servidor GlassFish.

Nota: Si no ha registrado NetBeans IDE 6.5 justo después de la instalación, usted puede hacerlo cuando ejecute NetBeans IDE.

Mac OS X

Para instalar el software:

  1. Después que la descarga se complete, ejecute el instalador. El archivo de instalación tiene la extensión .dmg.
  2. En el panel que se abre haga click en el icono del paquete. El paquete tiene la extensión .mpkg. El asistente de instalación comienza.
  3. En la página de Bienvenida del asistente de instalación, haga click en Continue.
  4. Revise el acuerdo de licencia y haga click en Continue. Haga click en Aceptar en la ventana emergente para aceptar el acuerdo de licencia.
  5. En la página Select a Destination, seleccione el disco y haga click en Continue.
  6. Si descargó las opciones All, Java, o Ruby, usted puede personalizar su instalación. En el último panel del asistente de instalación, presione el botón Customize en la parte inferior izquierda del panel. Se muestra el árbol de productos.
  7. Seleccione los productos que quiera instalar.
  8. Introduzca el nombre y la clave del administrador de su sistema y haga click en OK para comenzar la instalación.

Desinstalar el Software

Microsoft Windows

Si es necesario, usted puede desinstalar NetBeans IDE, el servidor GlassFish V2 UR2, el servidor GlassFish v3 Prelude, y Apache Tomcat usando los siguientes procedimientos.

Para desinstalar el IDE:

  1. Cierre el IDE.
  2. Desde el Panel de Control, seleccione Agregar o Quitar Programas, luego seleccione NetBeans IDE 6.5 y haga click en Quitar.

Para desinstalar el servidor GlassFish V2 UR2:

  1. Cierre el IDE.
  2. Desde el Panel de Control, seleccione Agregar o Quitar Programas, luego seleccione GlassFish V2 UR2 y haga click en Quitar.

Para desinstalar el servidor GlassFish v3 Prelude:

  1. Cierre el IDE.
  2. Desde el Panel de Control, seleccione Agregar o Quitar Programas, luego seleccione GlassFish v3 Prelude y haga click en Quitar.

Para desinstalar Apache Tomcat:

  1. Cierre el IDE.
  2. Desde el Panel de Control, seleccione Agregar o Quitar Programas, luego seleccione Apache Tomcat 6.0.16 y haga click en Quitar.

Solaris OS, Linux

Si es necesario, usted puede desinstalar NetBeans IDE, el servidor GlassFish V2 UR2, el servidor GlassFish v3 Prelude, y Apache Tomcat usando los siguientes procedimientos.

Para desinstalar el IDE:

  1. Cierre el IDE.
  2. En el directorio de instalación del IDE, ejecute el archivo uninstall.sh.
  3. Para desinstalar GlassFish/Tomcat junto con el IDE, en la página Summary asegúrese que las versiones correctas están listadas y chequeadas en las casillas de verificación apropiadas.
  4. En la página Summary, haga click en Uninstal.
  5. Después que se complete la desinstalación, haga click en Finish.

Para desinstalar el servidor GlassFish V2 UR2 o V3 Prelude:

  1. Cierre el IDE.
  2. Encuentre el directorio de instalación del servidor.

    Nota: La dirección de instalación por defecto es diferente dependiendo del tipo de instalación y de la plataforma. La siguiente tabla muestra los directorios de instalación por defecto del servidor.

    Plataforma Instalado Como...

    Instalación del Servidor

    GlassFish V2 UR2

    		 Instalación del Servidor GlassFish v3 Prelude
    Solaris OS 
    		 root /opt/glassfish-v2ur2 /opt/glassfish-v3-prelude
    Solaris OS,
    Linux    		 user ~/glassfish-v2ur2 ~/glassfish-v3-prelude
    Linux root /usr/local/glassfish-v2ur2 /usr/local/glassfish-v3-prelude
  3. En el directorio de instalación del servidor, ejecute el archivo uninstall.sh.
  4. En la página Summary, haga click en Uninstall.
  5. Después que se complete la desinstalación, haga click en Finish.

Nota: El servidor GlassFish es necesario para la funcionalidad SOA. Cuando desinstale el servidor, la funcionalidad SOA es eliminada automáticamente de la instalación de NetBeans IDE.

Para desinstalar Apache Tomcat:

  1. Cierre el IDE.
  2. Encuentre el directorio de instalación por defecto de Apache Tomcat y ejecute el archivo uninstall.sh.
  3. En la página Summary, haga click en Uninstall.
  4. Luego de completarse la desinstalación, haga click en Finish

Mac OS X

Para desinstalar el IDE y el servidor:

  1. Cierre el software.
  2. Arrastre las carpetas o archivos de los programas a la Papelera.
    Note que la dirección por defecto para el IDE y el servidor es /Applications/NetBeans/.

Actualizar y Coexistir con Versiones Anteriores

NetBeans IDE 5.5 o 5.5.1, 6.0, o 6.1 pueden coexistir con NetBeans IDE 6.5 en el mismo sistema. Usted no necesita desinstalar las versiones 5.5 o 5.5.1, 6.0, o 6.1 para poder instalar o ejecutar NetBeans IDE 6.5.

Si usted tiene NetBeans IDE 5.5, 5.5.1, 6.0, o 6.1 instalado usted podrá elegir si o no importar los ajustes de usuario desde los directorios de usuario existentes cuando ejecute NetBeans IDE 6.5 por primera vez.

Problemas

Acerca del Directorio de Trabajo del Instalador

El instalador almacena datos acerca de las sesiones de instalación en el directorio de trabajo del instalador. Ejemplo de estos datos son la lista de componentes instalados en su sistema, lista de mensajes de instalación, y datos de registros.

La dirección por defecto del directorio de trabajo del instalador es %USERPROFILE%\.nbi en Windows y ~/.nbi en otras plataformas. El subdirectorio log contiene los archivos de registro.

Limpieza Después de Instalaciones Incompletas

Después de instalaciones incompletas o no exitosas, usted podría necesitar limpiar su sistema de instalaciones previas.

Para Microsoft Windows, Solaris OS, y Linux:

  1. Borre el directorio de trabajo del instalador.
  2. Borre los directorios de instalación del IDE y del servidor GlassFish.
  3. En la plataforma Windows, borre todas las entradas de registro nbi* desde las siguientes claves de registro:
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall
    HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Para Mac OS X:

  • Arrastre las carpetas de programas a la Papelera.

Deshabilite Algunos Software Antes de Instalar

Es mejor deshabilitar cortafuegos, antivirus, y aplicaciones de monitoreo antes que inicie las instalaciones. Algunas de estas aplicaciones pueden evitar que algunos componentes del IDE sean configurados correctamente durante el proceso de instalación.

Por ejemplo, si usted tiene la aplicación Logitech Process Monitor (LVPrcSrv.exe) ejecutándose en su sistema, apáguela antes de la instalación.
