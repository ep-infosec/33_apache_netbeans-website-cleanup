Informações da versão NetBeans IDE 6.5

O NetBeans IDE é um premiado Ambiente de Desenvolvimento Integrado disponível para Windows, Mac, Linux e Solaris. O projeto NetBeans consiste em um IDE de código aberto e uma plataforma de aplicativos que permite aos desenvolvedores criar rapidamente aplicativos Web, empresariais, móveis e de área de trabalho usando a plataforma Java, bem como PHP, JavaScript e Ajax, Ruby e Ruby on Rails, Groovy e C/C++. Tem como suporte uma entusiasmada comunidade de desenvolvedores e oferece uma variada seleção de plug-ins de terceiros. O NetBeans IDE é um software imprescindível para desenvolvedores.

Visão geral da versão

Além de oferecer total suporte a todas as plataformas Java (Java SE, Java EE, Java ME e JavaFX), o NetBeans IDE 6.5 é a ferramenta ideal para o desenvolvimento de software com PHP, Ajax e JavaScript, Groovy e Grails, Ruby e Ruby on Rails e C/C++. O NetBeans IDE 6.5 oferece vários novos recursos e aprimoramentos, como o atalho QuickSearch geral do IDE, uma interface mais cômoda para o usuário e o recurso automático Compilar ao salvar. A versão 6.5 também inclui suporte aprimorado a frameworks da Web (Hibernate, Spring, JSF, JPA), o servidor de aplicações GlassFish e os bancos de dados.

Recursos em destaque

PHP

  • Editor PHP (auto-completar de código, realce sintático e semântico de código)
  • Suporte à notação heredoc e PHTML
  • Depuração usando o Xdebug
  • Geradores para fragmentos de código do banco de dados MySQL

JavaScript e Ajax

  • Depuração de JavaScript no FireFox & Internet Explorer
  • Monitoração de transação HTTP
  • Editor aprimorado para JavaScript e CSS
  • Gerenciador de bibliotecas JavaScript incluindo bibliotecas Yahoo UI, Woodstock, jQuery, Dojo, Scriptaculous, Prototype

Desenvolvimento Web & Java EE

  • Suporte aprimorado para Spring, Hibernate, JSF, gerador de CRUDs JSF, JPA (Java Persistence API)
  • Criação de serviços Web RESTful a partir de tabelas de banco de dados ou de entidades JPA
  • Recurso de arrastar e soltar serviços SaaS em arquivos PHP
  • Melhorias no Editor SQL (auto-completar de código, salvar/lembrar consultas, e mais)
  • Importação de projeto Eclipse e sincronização para projetos Web Java aprimoradas
  • Recurso automático Implantar ao salvar

JavaFX

  • Criação, teste, depuração, criação de perfil e deployment de projetos JavaFX
  • Editor JavaFX com realce de sintaxe, navegação de código-fonte, auto-completar de código, dobramento de código, pop-ups javadoc e detecção de erros.
  • Visualização interativa de elementos visuais
  • Paleta de fragmento de código para transformações, efeitos e animações

Groovy e Grails

  • Desenvolva aplicativos Groovy puros ou use Groovy em projetos Java SE
  • Editor Groovy com realce e auto-completar de código, e mais
  • Framework de aplicativo Web Grails
  • Abra aplicativos Grails existentes sem adição de metadados

Ruby e Rails

  • Nova interface de execução de teste para exibição dos resultados do teste
  • Melhoria do Rake: Rake Runner e um arquivo pré-gerado do Rake
  • Depurador: pontos de interceptação e pontos de interrupção condicionais
  • Projetos Ruby aceitam opções da JVM da linha de comando

GlassFish v3 Prelude para desenvolvimento Web

  • Arquitetura modular baseada em OSGi
  • Componentes menores, inicialização rápida e deployment
  • Suporte a script, incluindo JRuby

C/C++

  • Auto-completar de código, realce de erro e realce de semântica melhorados
  • Gráfico de chamadas, janela Memória
  • Desenvolvimento remoto
  • Empacotamento de aplicativos como arquivos tar, zip ou pacotes SVR4

Java ME

  • Novos assistentes de projetos para adição de componentes personalizados à paleta do Visual Designer
  • Novos componentes de IU SVG no aprimorado SVG Composer que permitem criação de uma rica IU
  • Novos componentes de vinculação de dados na paleta do Visual Designer
  • Ferramenta de ofuscação atualizada para ProGuard 4.2 e framework de teste para JMUnit 1.1.0

Depurador Java

  • Nova depuração multissegmentada com IU e fluxo de trabalho aprimorados
  • Recurso Passar dentro inteligente com seleção de método dentro da expressão
  • Detecção de deadlocks com indicação visual

Java SE

  • Recurso automático Compilar ao salvar
  • Importação e sincronização aprimoradas de projetos Eclipse
  • Hierarquia de chamada Java
  • Analisar Javadoc
  • O Construtor de GUI Swing pode gerar nomes de classe simples ou totalmente qualificados

Ferramentas e usabilidade do IDE

  • Auto-completar de código com primeira letra em maiúscula (Camel Case)
  • Formatação de configurações por projeto
  • Um atalho QuickSearch geral do IDE para arquivos/símbolos/tipos, ações do IDE, opções e docs
  • Suporte para caminhos UNC do Windows (pastas compartilhadas e de rede)
