NetBeans IDE 6.5 发行信息

NetBeans IDE 是一个荣获奖项的集成开发环境，可用于 Windows、Mac、Linux 和 Solaris。NetBeans 项目由开源 IDE 和应用程序平台组成；通过该 IDE 和平台，开发者可以使用 Java 平台以及 PHP、JavaScript 和 Ajax、Ruby 和 Ruby on Rails、Groovy 和 C/C++ 快速创建 Web、企业、桌面和 Mobile 应用程序。该项目由活跃的开发者社区支持，并且还提供了各种供您选用的第三方插件。

发行版概述

除完全支持所有 Java 平台（Java SE、Java EE、Java ME 和 JavaFX）之外，NetBeans IDE 6.5 还是 PHP、Ajax 和 JavaScript、Groovy 和 Grails、Ruby 和 Ruby on Rails 以及 C/C++ 软件开发的理想工具。6.5 发行版增强了对 Web 框架 (Hibernate, Spring, JSF, JPA)、GlassFish 应用服务器以及数据库的支持。此外，它还包括新增的在 IDE 范围内快速搜索的快捷键、更加友好的用户界面以及自动在保存时编译功能。

功能集锦

PHP

  • PHP 编辑器（代码完成、语法和语义代码突出显示）
  • 支持 Heredoc 表示法和 PHTML
  • 使用 Xdebug 进行调试
  • MySQL 数据库代码片段生成器

JavaScript 和 Ajax

  • 对 FireFox 和 Internet Explorer 进行 JavaScript 调试
  • HTTP 事务监视
  • 改进了 JavaScript 和 CSS 编辑器
  • JavaScript 库管理器，其中包括 Yahoo UI、Woodstock、jQuery、Dojo、Scriptaculous 和 Prototype 库

Java EE 和 Java Web 开发

  • 增强了对 Spring、Hibernate、JSF、JSF CRUD 生成器、JPA（Java Persistence API，Java 持久性 API）的支持
  • 通过数据库表或 JPA 实体创建 REST 风格的 Web 服务
  • 将 SaaS 服务拖放到 PHP 文件中
  • 改进了 SQL 编辑器（代码完成、保存/重新调用查询等功能）
  • 为 Java Web 项目改进了 Eclipse 项目导入和同步
  • 自动在保存时部署

JavaFX

  • 创建、测试、调试、分析和部署 JavaFX 项目
  • JavaFX 编辑器，具有语法突出显示、源导航、代码完成、代码折叠、Javadoc 弹出式窗口和错误检测功能。
  • 实时预览可视元素
  • 用于实现各种变换、效果和动画的代码片段组件面板

Groovy 和 Grails

  • 开发纯 Groovy 应用程序或在 Java SE 项目中使用 Groovy
  • Groovy 编辑器，具有代码完成、突出显示等功能
  • Grails Web 应用程序框架
  • 在不添加 meta 数据的情况下打开现有 Grails 应用程序

Ruby 和 Rails

  • 新增用于显示测试结果的测试运行器界面
  • 改进了 Rake：Rake 运行器和预生成的 Rake 文件
  • 调试器：条件断点和捕捉点
  • Ruby 项目接受命令行 JVM 选项

用于 Web 开发的 GlassFish v3 Prelude

  • 基于 OSGi 的模块化体系结构
  • 占用内存小，启动和部署速度快
  • 支持 JRuby 等脚本

C/C++

  • 改进了代码完成、错误突出显示和语义突出显示功能
  • 调用图形，“内存”窗口
  • 远程开发
  • 将应用程序打包为 tar 文件、zip 文件或 SVR4 包

Java ME

  • 新增用于在可视设计器组件面板中添加定制组件的项目向导
  • 在增强的 SVG 编写器中新增 SVG UI 组件，以便进行富 UI 设计
  • 在可视设计器组件面板中新增数据绑定定制组件
  • 将混淆工具升级到 ProGuard 4.2，将测试框架升级到 JMUnit 1.1.0

Java 调试器

  • 新增具有改进 UI 和工作流的多线程调试功能
  • 通过在表达式中选择适当的方法实现智能步入
  • 可提供可见指示的死锁检测功能

Java SE

  • 自动在保存时编译
  • 改进了 Eclipse 项目导入和同步
  • Java 调用分层结构
  • 分析 Javadoc
  • Swing GUI 生成器可以生成简单或全限定类名

IDE 工具和可用性

  • 驼峰匹配代码完成
  • 按照项目定制格式设置
  • 一个用于在 IDE 范围内快速搜索文件/符号/类型、IDE 操作、选项和文档的快捷键
  • 支持 Windows UNC 路径（共享的网络文件夹）

有关详细信息，请参见 NetBeans IDE 6.5 新增功能和重要功能页。
