Instruções de instalação do NetBeans IDE 6.8

Última atualização: 10 dezembro de 2009

Este documento descreve como instalar o NetBeans IDE 6.8 em seu sistema. Consulte as Notas de Versão do NetBeans IDE 6.8 para obter informações sobre os sistemas operacionais suportados e sobre as configurações de hardware para o IDE. Para saber mais sobre os novos recursos incluídos nesta versão do IDE, consulte a página Informações da versão 6.8 do NetBeans IDE.

Software necessário

Antes de instalar o IDE, você deverá instalar o Java SE Development Kit (JDK) 5.0 Atualização 19 (versão 1.5.0_19) o JDK 6 Atualização 14) em seu sistema. Se não tiver o JDK instalado, não será possível prosseguir com a instalação. É possível fazer o download da versão mais recente do JDK em http://java.sun.com/javase/downloads.

Nas opções de download de PHP, Ruby e C++, você pode instalar o JRE, em vez do JDK. No entanto, se tiver a intenção de usar qualquer recurso Java, você precisará do JDK.

Opções de download do Installer

Na página de download do NetBeans IDE , você pode adquirir um dos diversos instaladores, cada um contendo o IDE básico e as ferramentas adicionais.

Os seguintes instaladores estão disponíveis:

  • Java. Fornece ferramentas para o desenvolvimento de aplicativos Java SE, Java EE e Java ME. Esta opção de download inclui também o GlassFish v3 o software Apache Tomcat 6.0.20.
  • JavaFX. Oferece suporte ao desenvolvimento de avançados aplicativos multiplataformas JavaFX.
  • Java SE. Oferece suporte a todos os recursos de desenvolvimento padrão Java SE, bem como à plataforma de desenvolvimento NetBeans RCP.
  • Ruby. Fornece ferramentas para desenvolvimento Ruby e oferece suporte para Rails e JRuby. Contém também o GlassFish v3.
  • PHP. Oferece ferramentas para desenvolvimento em PHP.
  • C/C++. Oferece suporte para o desenvolvimento nas linguagens C, C++, Fortran e Assembly.
  • Tudo. Trata-se de uma opção de download completo que contém todos tempos de execução e pacotes disponíveis do NetBeans IDE.

Personalizando sua instalação

Se você fez o download da opção Java ou da opção "Tudo", pode escolher exatamente as ferramentas e os tempos de execução para a instalação. Clique em Personalizar na página de boas-vindas do instalador e selecione os recursos que deseja usar. Para obter mais informações, consulte Instalando o software.

Se você deseja adicionar funcionalidade a sua configuração do NetBeans IDE, use o NetBeans Plugin Manager. Por exemplo, suponhamos que você iniciou baixando e instalando o pacote Ruby. Em seguida, você decidiu que deseja experimentar a funcionalidade C/C++. Para fazer isso, acesse o NetBeans Plugin Manager no IDE (selecione Ferramentas > Plug-ins) e adicione o pacote C/C++ à instalação existente.

Há diversas notas importantes sobre a instalação e a configuração de alguns recursos do NetBeans IDE:

  1. O Apache Tomcat 6.0.20 está incluído nas opções de download "Java" e "Tudo", mas não está instalado por padrão nessas opções. Para instalar o Apache Tomcat da opção de download Java ou "Tudo", inicie a instalação e selecione Apache Tomcat 6.0.20 na caixa de diálogo Instalação personalizada.
  2. Para o desenvolvimento de C/C++ (aplicável aos downloads C/C++ e "Tudo"), além disso é necessário instalar os compiladores e as ferramentas. Para obter mais informações, consulte Configurando o NetBeans IDE para C/C++/Fortran.
  3. Para Mac OS X, não é possível personalizar a instalação do NetBeans IDE dos downloads de Java, Ruby e "Tudo". A instalação do IDE dos downloads de JavaSE, PHP e C/C++ não pode ser personalizada.
  4. Também é possível fazer o download de um arquivo zip independente da plataforma e executar o arquivo executável do NetBeans IDE. Observe que os tempos de execução, como os servidores de aplicativos GlassFish e o Apache Tomcat, estão disponíveis somente com instaladores específicos da plataforma.

Iniciando o download

  1. Acesse a página de download do NetBeans IDE.
  2. Na parte superior direita da página, selecione a linguagem e a plataforma na lista suspensa. Você também pode optar por fazer o download e usar somente o arquivo zip independente da plataforma.
  3. Clique no botão Download para selecionar a opção de download que deseja instalar.
  4. Salve o arquivo do instalador no sistema.

Instalando o software

Siga as instruções nesta seção para instalar o IDE no sistema. Estas instruções de instalação aplicam-se a todas as plataformas compatíveis. Para obter uma lista das plataformas compatíveis, consulte as notas de versão.

Microsoft Windows, Solaris OS e Linux

Para instalar o software:

  1. Após concluir o download, execute o instalador.
    • Para Windows, o arquivo executável do instalador tem a extensão .exe. Clique duas vezes no arquivo do instalador para executá-lo.
    • Nas plataformas Solaris e Linux, o arquivo do instalador tem a extensão .sh. Para estas plataformas, você precisa executar os arquivos do instalador com o seguinte comando: chmod +x <installer-file-name>
  2. Para selecionar as ferramentas e os tempos de execução a serem instalados, execute as seguintes etapas na página de boas-vindas do assistente de instalação:
    1. Clique em Personalizar.
    2. Na caixa de diálogo Personalizar a instalação, faça suas seleções:
    3. Clique em OK.
  3. Na página de boas-vindas do assistente de instalação, clique em Avançar.
  4. Na página Contrato de licença, leia o contrato, marque a caixa de seleção para aceitar e clique em Avançar.
  5. Na página de instalação do IDE NetBeans, proceda da seguinte maneira:
    1. Aceite o diretório padrão de instalação do IDE NetBeans ou especifique outro diretório.

      Nota: o diretório de instalação deverá estar vazio e o perfil de usuário que estiver sendo usado para executar o instalador deverá ter permissões de leitura/gravação para esse diretório.

    2. Aceite a instalação padrão do JDK para usar com o NetBeans IDE ou selecione uma instalação diferente na lista suspensa. Se o assistente de instalação não encontrou uma instalação JDK compatível para ser usada com o NetBeans IDE, o JDK não está instalado no local padrão. Nesse caso, especifique o caminho do JDK instalado e clique em Avançar ou cancele a instalação atual. Após a instalação da versão necessária do JDK, você pode reiniciar a instalação.
  6. Se a página de instalação do GlassFish v3 for aberta, aceite o diretório padrão de instalação ou especifique outro local de instalação.
  7. Se estiver instalando o Apache Tomcat, em sua página de instalação, aceite o diretório padrão de instalação ou especifique outro local de instalação.
  8. Na página Resumo, verifique se a lista de componentes a serem instalados está correta e se você tem espaço suficiente no sistema para a instalação.
  9. Clique em Instalar para iniciar a instalação.

Nota: se você encontrar problemas após concluir instalação do software com êxito, consulte Solução de problemas para obter descrições e soluções para as questões não-resolvidas que poderão afetar o processo de instalação.

Mac OS X

Para instalar o software:

  1. Após concluir o download, execute o instalador. O arquivo do instalador tem a extensão .dmg.
  2. Clique no ícone de pacote na janela exibida. A extensão do pacote é .mpkg . O assistente de instalação é iniciado.
  3. Na página de boas-vindas do assistente de instalação, clique em Continuar.
  4. Leia o contrato de licença e clique em Continuar. Clique em Aceitar na janela suspensa para aceitar a licença.
  5. Na página Selecionar um destino. selecione o drive e clique em Continuar.
  6. Se você baixou o pacote Java, Ruby ou Tudo, você pode personalizar a sua instalação. Pressione o botão Personalizar, no lado esquerdo inferior da janela do assistente de instalação, na última janela do assistente de instalação. A árvore de produtos é exibida.
  7. Selecione os produtos que você deseja instalar.
  8. Digite o nome do administrador e a senha no sistema e clique em OK para iniciar a instalação.

Pacote independente de plataforma

Para executar o software:

  1. Após o download do arquivo ZIP independente da plataforma, extraia-o em qualquer pasta do sistema.
  2. Execute o arquivo executável localizado no diretório netbeans/bin.
  3. Aceite o Contrato de Licença. O NetBeans IDE é iniciado.

Nota: observe que os tempos de execução, como o servidor de aplicativos GlassFish e o Apache Tomcat, estão disponíveis somente com instaladores específicos da plataforma.

Nota: se não for possível encontrar a instalação do JDK compatível, talvez seja necessário adicionar manualmente o caminho ao diretório de instalação do JDK:

  1. Abra em um editor de texto o arquivo netbeans.conf localizado em netbeans/etc.
  2. Introduza a localização da instalação do JDK compatível como o valor da opção netbeans_jdkhome.
  3. Salve o arquivo netbeans.conf e execute o arquivo executável em netbeans/bin.

Desinstalando o software

Microsoft Windows

Se necessário, você pode desinstalar o NetBeans IDE, o servidor de aplicativos GlassFish v3 e o Apache Tomcat usando os seguintes procedimentos.

Para desinstalar o IDE:

  1. Feche o IDE.
  2. No Painel de controle, selecione Adicionar ou remover programas e então selecione NetBeans IDE 6.8 e clique em Remover.

Para desinstalar o GlassFish v3:

  1. Feche o IDE.
  2. No Painel de controle, selecione Adicionar ou remover programas e, então, selecione GlassFish v3 e clique em Remover.

Para desinstalar o Apache Tomcat:

  1. Feche o IDE.
  2. No Painel de controle, selecione Adicionar ou remover programas e então selecione Apache Tomcat 6.0.20 e clique em Remover.

Solaris OS, Linux

Se necessário, você pode desinstalar o NetBeans IDE, o GlassFish v3 e o Apache Tomcat usando os seguintes procedimentos.

Para desinstalar o IDE:

  1. Feche o IDE.
  2. No diretório de instalação do IDE, execute o arquivo uninstall.sh.
  3. Para desinstalar o GlassFish/Tomcat e o IDE ao mesmo tempo, na página Resumo assegure-se de que as versões listadas são as corretas e as caixas apropriadas estão marcadas.
  4. Na página Resumo, clique em Desinstalar.
  5. Quando a desinstalação for concluída, clique em Concluir.

Para desinstalar o GlassFish v3 :

  1. Feche o IDE.
  2. Localize o diretório de instalação do servidor de aplicações.

    Nota: o local padrão de instalação será diferente, dependendo do tipo de instalação e da plataforma. A tabela a seguir exibe o diretório padrão de instalação do servidor de aplicativos para várias plataformas.

    Plataforma Instalando como... Instalação do GlassFish v3
    Windows usuário com direitos de administrador %ProgramFiles%\glassfish-v3
    Windows usuário %USERPROFILE%\glassfish-v3
    Solaris OS  raiz /opt/glassfish-v3
    SO Solaris, Linux usuário ~/glassfish-v3
    Linux raiz /usr/local/glassfish-v3
  3. No diretório de instalação do servidor de aplicações, execute o arquivo uninstall.sh.
  4. Na página Resumo, clique em Desinstalar.
  5. Quando a desinstalação for concluída, clique em Concluir.

Para desinstalar o Apache Tomcat:

  1. Feche o IDE.
  2. Localize o diretório de instalação do Apache Tomcat e execute o arquivo uninstall.sh.
  3. Na página Resumo, clique em Desinstalar.
  4. Quando a desinstalação for concluída, clique em Concluir

Mac OS X

Para desinstalar o IDE e o servidor de aplicações:

  1. Feche o software.
  2. Arraste as pastas ou arquivos de programas para a Lixeira. Observe que o local padrão do IDE e do servidor de aplicações é /Applications/NetBeans/.

Pacote independente de plataforma

Para desinstalar os arquivos do IDE, siga as etapas descritas na seção Limpeza após instalações incompletas deste documento.

Atualizando e coexistindo com versões anteriores

O NetBeans IDE 5.x, 6.0, 6.1, 6.5 e 6.7 podem coexistir com o NetBeans IDE 6.8 no mesmo sistema. Você não precisa desinstalar as versões anteriores para instalar ou executar o NetBeans IDE 6.8.

Se você possui o NetBeans IDE 5.x, 6.0, 6.1, 6.5 ou 6.7 instalado, terá a opção de importar ou não as configurações de usuário do diretório existente ao executar o NetBeans 6.8 da primeira vez.

Solução de problemas

Sobre o diretório de trabalho do instalador

O instalador armazena dados sobre as sessões de instalação no diretório de trabalho do instalador. Exemplos desses dados são a lista de componentes instalados no sistema, as mensagens do registro de instalação e os dados de registro.

O local padrão do diretório de trabalho do instalador é %USERPROFILE%\.nbi no Windows e ~/.nbi em outras plataformas. O subdiretório do registro contém os arquivos de registro de instalação.

Limpeza após instalações incompletas

Após instalações incompletas ou malsucedidas, talvez seja necessário limpar o sistema das instalações anteriores.

Para Microsoft Windows, Solaris OS e Linux:

  1. Exclua o diretório de trabalho do instalador.
  2. Exclua os diretórios de instalação do IDE e do servidor de aplicações GlassFish.
  3. Na plataforma Windows, exclua todas as entradas do registro menos a nbi* de todas as chaves de registro: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Para Mac OS X:

  • Arraste as pastas ou arquivos de programas para a Lixeira.

Desabilitando alguns produtos de software antes da instalação

É melhor desabilitar o firewall, o antivírus e os aplicativos de monitoramento antes de iniciar a instalação. Alguns desses aplicativos podem evitar que os componentes do IDE sejam configurados corretamente durante o processo de instalação.

Por exemplo, se você possuir o aplicativo Logitech Process Monitor (LVPrcSrv.exe) no sistema, desative-o antes da instalação.
