NetBeans IDE 6.8 リリース情報

NetBeans IDE は、受賞歴のある統合開発環境で、Windows、Mac、Linux、および Solaris で使用可能です。NetBeansは、オープンソースのIDEとアプリケーションプラットフォームから構成されます。NetBeansを使用すると、開発者はJavaプラットフォームやJavaFX、PHP、JavaScriptとAjax、RubyとRubyonRails、GroovyとGrails、およびC/C++をて、Webアプリケーション、エンタープライズアプリケーション、デスクトップアプリケーション、およびモバイルアプリケーションを短時間で作成できます。

NetBeansプロジェクトは活発な開発者コミュニティーによってサポートされ、豊富なドキュメントとトレーニングリソース、およびさまざまなサードパーティーのプラグイン提供します。

リリース概要

NetBeans IDE 6.8 は、Java EE 6 の仕様を完全にサポートした最初の IDE です。また、JSF 2.0/Facelets、Java Persistence 2.0、Web アプリケーションでの EJB の使用を含む EJB 3.1、RESTful Web サービス、および GlassFish v3 のサポート機能も改善されています。最新の JavaFX SDK 1.2.1 を使用した開発や、新しい PHP 5.3 release または Symfony フレームワークを使用した PHP Web アプリケーションの作成に使用することもお勧めします。

オープンソースプロジェクトをホストするための共同作業環境であるプロジェクトKenaiの独自のに、新しくJIRAの完全なサポートが追加され、インスタントメッセンジャーおよび課題追跡ツールの統合も改善されました。IDE の Maven およびデータベースの統合にも機能が追加されました。エディタ、および Ruby、Groovy、C/C++ などのプロジェクトのツール統合も改善されました。

主な新機能

JavaEnterpriseEdition6

  • Java EE 6 および Java EE 6 Web プロファイルを使用した Web プロジェクト、Web アプリケーションでの EJB
  • EJB 3.1 のサポート、EJB プロジェクトファイルウィザードでのシングルトンセッションタイプのサポート
  • RESTful Web サービス (JAX-RS 1.1)、GlassFish Metro 2.0 Web サービス (JAX-WS 2.2)、JAXB 2.2
  • Java Persistence JPA 2.0、GlassFish v3 アプリケーションサーバーを使用した配備、デバッグ、およびプロファイル

JavaServerFaces2.0(Facelets)を使用したWebプロジェクト

  • Facelets 向けのコード補完、エラーヒント、名前空間補完、ドキュメントポップアップ、およびタグの自動インポート
  • JSF フォームと HTML フォームのジェネレータを含む、Facelets ライブラリ、複合コンポーネント、式言語のエディタサポート
  • エンティティーから JSF フォームおよび JSF データ表を生成する、カスタマイズ可能な JSF コンポーネントパレット
  • エンティティーからカスタマイズ可能な CRUD (作成/読み取り/更新/削除) JSF ページを生成する、新規ファイルウィザード
  • 配備記述子の代わりに注釈を幅広く使用

JavaFX

  • 最新の JavaFX SDK 1.2.1 のサポートを追加
  • コード補完の改善
  • エディタのヒント: インポートの修正、囲む、抽象メソッドの実装など
  • ナビゲーションの改善: ハイパーリンク、型に移動、使用状況を検索

Kenai.com:ConnectedDeveloper

  • JIRA の完全なサポート (アップデートセンターからのプラグイン)
  • メンバーやプロジェクトの情報をより詳細に表示し、検索機能とナビゲーションが改善され、プロジェクト共有がより簡単になったプロジェクトダッシュボード
  • インスタントメッセンジャー統合の改善: オンライン参加、Kenai メンバー間でのプライベートチャットおよびグループチャット、コード/ファイル/課題/スタックトレースへのリンクをメッセージに簡単に追加
  • 課題追跡統合の改善

PHP

  • PHP 5.3 の完全なサポート: 名前空間、ラムダ関数、クロージャー、構文の追加: NEWDOC、三項演算子、ジャンプラベル、__callStatic()
  • Symfony フレームワークのサポート: Symfony プロジェクト、Symfony コマンド、ショートカット、YAML ファイル内での PHP 構文の色分け
  • リモート PHP アプリケーションからの PHP プロジェクトの作成
  • PHPUnit、コードカバレージ、FTP/SFTP 統合の改善、スキャンおよび索引付けからの PHP プロジェクトフォルダの除外

Maven

  • Maven 原型カタログからの新規プロジェクトと Java EE 6、Groovy、および Scala プロジェクトのサポートの改善
  • 依存リソースグラフでのカスタマイズ可能な依存リソースの除外
  • Maven CheckStyle プラグイン
  • Kenai.com にホストされた Maven プロジェクトに対する「Kenai から更新」操作

Ruby

  • ディスパッチャーを使用した Rails 2.3.4 アプリケーション、JRuby 1.4、Ruby 1.9 デバッグ、および RSpec 1.2.7 の作成のサポート
  • 名前変更リファクタリング、型推測、ナビゲーションの改善
  • Rails サーバーの引数の指定
  • 引数を使用したファイルの実行とデバッグ、およびプロジェクトに含まれないファイルの実行とデバッグ

C/C++

  • プロファイル: 新しいマイクロステートアカウンティングインジケータ、スレッドマップビュー、ホットスポットビュー、メモリーリークビュー、同期の問題ビュー
  • リモート開発時のより速い同期
  • gdbserver への接続、および実行中のプロセスへのより簡単な接続のサポート

その他の改善

  • Java デバッガ: 名前で変数ツリー内のオブジェクトをマーク付けして、式で参照
  • データベース統合: SQL エディタでの、DELETE、DROP、UPDATE 文、および予約キーワードのコード補完
  • Groovy 1.6.4 と Grails: AST 変換によるメソッドを含む、コード補完の改善

最新の開発ビルドの機能の詳細については、NetBeansIDEのNewandNoteworthyページ参照してください。
