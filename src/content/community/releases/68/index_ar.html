معلومات إصدار NetBeans IDE 6.8

يعد NetBeans IDE بيئة تطوير متكاملة حائزة على جوائز، وهو متوفر لأنظمة التشغيل Windows وMac وLinux وSolaris. يتألف مشروع NetBeans من بيئة IDE مفتوحة المصدر ونظام أساسي للتطبيقات يتيح للمطورين إمكانية الإنشاء السريع لتطبيقات الويب والمؤسسات وأجهزة سطح المكتب وكذلك تطبيقات الأجهزة المحمولة باستخدام نظام Java الأساسي وكذلك أنظمة JavaFX وPHP وJavaScript وAjax وRuby وRuby on Rails وGroovy وGrails وC/C++.

ويتولى دعم مشروع NetBeans جماعة مطورين بالغة النشاط تقدم وثائق وموارد تدريب شاملة، علاوةً على خيارات متنوعة من المكونات الإضافية من جهات خارجية.

نظرة عامة على الإصدار

يعد NetBeans IDE 6.8 بيئة IDE الأولى التي تقدم دعمًا لمواصفات Java EE 6 بأكملها مع دعم محسن لكل من JSF 2.0/Facelet وJava Persistence 2.0 وEJB 3.1، شاملاً استخدام مكونات EJB في تطبيقات الويب وخدمات الويب RESTful وGlassFish v3. كما نوصي باستخدامه في أعمال التطوير باستخدام أحدث برامج JavaFX SDK 1.2.1، ولإنشاء تطبيقات الويب PHP بالإصدار الجديد PHP 5.3 أو Symfony Framework.

إن ما تمكنا من تحقيقه من تكامل فريد مع موقع Project Kenai، والذي يمثل بيئة تعاونية لاستضافة المشروعات ذات المصدر المفتوح، يعززه الآن دعم كامل لـ JIRA، إضافة إلى التكامل المحسن مع برامج الرسائل الفورية وتعقب المشكلات. كما قمنا أيضًا بإضافة ميزات إلى تكامل Maven وقاعدة البيانات ببيئة IDE، إلى جانب تحسين تكامل المحرر والأدوات الخاصة بمشروعات Ruby وGroovy وC/C++.

الوثائق

استخدم المستندات التالية لبدء العمل باستخدام NetBeans IDE 6.8.

  • ملاحظات حول الإصدار
  • إرشادات التثبيت
  • البرامج التعليمية الحديثة

أبرز السمات

Java Enterprise Edition 6

  • مشروعات الويب باستخدام Java EE 6 وملفات التعريف Java EE 6 Web ومكونات EJB في تطبيقات الويب.
  • دعم EJB 3.1، كما يدعم معالج ملف مشروع EJB نوع جلسة العمل Singleton
  • خدمات الويب (RESTful (JAX-RS 1.1 وخدمات الويب GlassFish Metro 2.0 (JAX-WS 2.2) وJAXB 2.2
  • Java Persistence JPA 2.0، مع النشر وتصحيح الأخطاء وجمع المعلومات باستخدام خادم التطبيقات GlassFish v3

مشروعات الويب مع JavaServer Faces 2.0 (Facelets)

  • استكمال التعليمات البرمجية وتلميحات الأخطاء واستكمال مساحة الاسم والقوائم المنبثقة للوثائق والاستيراد التلقائي للعلامات الخاصة بـ Facelets
  • دعم محرر مكتبات Facelets والمكونات المركبة ولغة التعبيرات، بما في ذلك وحدات إنشاء نماذج JSF وHTML
  • تقوم لوحة ألوان مكونات JSF القابلة للتخصيص بإنشاء نماذج JSF وجداول بيانات JSF من الوحدات
  • يقوم معالج New File (ملف جديد) بإنشاء صفحات JSF قابلة للتخصيص (إنشاء/قراءة/تحديث/حذف) CRUD من الوحدات
  • استخدام التعليقات التوضيحية على نطاق أوسع بدلاً من أدوات وصف النشر

JavaFX

  • دعم إضافي لأحدث إصدار JavaFX SDK 1.2.1
  • استكمال محسن للتعليمات البرمجية
  • تلميحات المحرر: تتضمن إصلاح عمليات الاستيراد والإحاطة وتنفيذ الأساليب المجردة والمزيد غير ذلك
  • تنقل محسن: ارتباطات تشعبية وانتقال إلى النوع وبحث عن استخدامات

Kenai.com: قناة اتصال للمطور

  • دعم كامل لأداة JIRA (مكون إضافي من مركز التحديثات)
  • لوحة معلومات للمشروعات تعرض مزيدًا من التفاصيل حول الأعضاء والمشروعات مع إمكانيات محسنة للتنقل والبحث ومشاركة أسهل للمشروعات
  • تكامل محسن لبرامج الرسائل الفورية: تواجد على الإنترنت ومحادثات خاصة وجماعية مع أعضاء Kenai وسهولة إضافة ارتباطات إلى التعليمات البرمجية / الملفات / المشكلات / عمليات التتبع المكدسة للوصول إلى الرسائل
  • تكامل محسّن لبرنامج تعقب المشكلات

PHP

  • دعم Full PHP 5.3: مساحات الاسم ودوال lambda وإغلاقها وإضافات إلى بناء الجملة: NOWDOC والشروط الثلاثية وتسميات الانتقال السريع و__callStatic()
  • دعم Symfony Framework: مشروعات Symfony وأوامر Symfony والاختصارات وألوان بناء جملة PHP في ملفات YAML
  • إنشاء مشروع PHP من تطبيق PHP بعيد
  • PHPUnit وتغطية التعليمات البرمجية وتحسينات تكامل FTP/SFTP، واستثناء مجلدات مشروع PHP من المسح/الفهرسة

Maven

  • مشروع جديد من كتالوج النماذج الأولية لـ Maven ودعم محسن لمشروعات Java EE 6 وGroovy وScala
  • استثناء تبعية قابل للتخصيص في الرسوم البيانية للتبعية
  • المكون الإضافي Maven CheckStyle
  • إجراء "التحديث من Kenai" لمشروعات Maven التي تستضيفها Kenai.com

Ruby

  • تقديم الدعم لإنشاء تطبيقات Rails 2.3.4 باستخدام وحدات الإرسال وJRuby 1.4 وتصحيح أخطاء Ruby 1.9 وRSpec 1.2.7
  • إعادة بناء تعليمات برمجية محسّنة لإعادة التسمية واستنتاج النوع والتنقل
  • تحديد الوسيطات لخوادم Rails
  • تشغيل/تصحيح أخطاء الملف ذي الوسيطات، وأيضًا بالنسبة للملفات التي لا تعد جزءًا من مشروع

C وC++

  • جمع المعلومات: مؤشر حساب وقت الحالات متناهية الصغر (Microstate Accounting) وعرض خريطة مؤشرات الترابط وعرض النقاط الفعّالة وعرض حالات تسرب الذاكرة وعرض مشكلات المزامنة
  • مزامنة أسرع أثناء التطوير عن بُعد
  • دعم إرفاق gdbserver وإمكانية إرفاق أسهل بالعمليات قيد التنفيذ بالفعل

تحسينات متنوعة

  • مصحح أخطاء Java: يميز كائنًا في شجرة المتغيرات باسم للإشارة إليه في التعبيرات
  • تكامل قاعدة البيانات: استكمال التعليمات البرمجية في محرر SQL الآن أيضًا لعبارات DELETE وDROP وUPDATE وكذلك الكلمات الأساسية المحجوزة.
  • Groovy 1.6.4 وGrails: استكمال محسن للتعليمات البرمجية، يشمل الأساليب المقدمة بواسطة عمليات التحويل AST Transformations.

تعرّف على المزيد حول ميزات أحدث إصدار للتطوير في صفحة الجديد والجدير بالملاحظة حول NetBeans IDE!
