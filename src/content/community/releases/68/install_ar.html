إرشادات تثبيت NetBeans IDE 6.8

تاريخ آخر تحديث: 7 ديسمبر 2009

يوضح هذا المستند كيفية تثبيت NetBeans IDE 6.8 على النظام. يُرجى مراجعة ملاحظات الإصدار NetBeans IDE 6.8 للحصول على معلومات حول التكوينات الافتراضية لأنظمة التشغيل والأجهزة بالنسبة لـ IDE. لمعرفة المزيد حول الميزات الجديدة المضمنة في هذا الإصدار من IDE، انظر صفحة معلومات إصدار NetBeans IDE 6.8 .

البرامج المطلوبة

قبل تثبيت IDE، يلزم تثبيت Java SE Development Kit (JDK) 5.0 Update 19 (الإصدار 1.5.0_19) أو JDK 6 Update 14 أو إصدار أحدث على النظام. إذا لم يكن JDK مثبتًا، فستتعذر متابعة عملية التثبيت. يمكن تنزيل أحدث إصدارات JDK من الموقع http://java.sun.com/javase/downloads.

بالنسبة لخيارات تنزيل PHP وRuby وC/C++K، يمكنك تثبيت JRE بدلاً من JDK. على أنه إذا كنت تخطط لاستخدام أي من سمات Java، فستحتاج إلى JDK.

البداية

خيارات تنزيل المثبت

في صفحة تنزيل NetBeans IDE، يمكن الحصول على أحد المثبتات المتعددة، والتي يشتمل كل منها على بيئة IDE أساسية وأدوات إضافية.

تتوفر المثبتات التالية:

  • Java. يوفر أدوات لتطوير تطبيقات Java SE وJava EE وJava ME. يشتمل خيار التنزيل هذا على GlassFish v3 وبرنامج Apache Tomcat 6.0.20.
  • JavaFX. يوفر دعمًا لتطوير التطبيقات الغنية بالميزات متعددة الأنظمة الأساسية JavaFX.
  • Java SE. يوفر جميع الميزات القياسية لتطوير Java SE. ويشمل ذلك دعمًا لتطوير NetBeans Platform RCP.
  • Ruby. يوفر أدوات لتطوير Ruby ودعم Rails وJRuby. كما يشتمل أيضًا على برنامج GlassFish v3.
  • PHP. يوفر أدوات لتطوير PHP.
  • C/C++. يدعم التطوير بلغات C وC++ وQt وFortran واللغات التجميعية.
  • All (الكل). يعد ذلك خيار تنزيل كاملاً، يشتمل على جميع أوقات التشغيل والحزم المتوفرة لبيئة NetBeans IDE.

البداية

تخصيص عملية التثبيت

إذا قمت بتنزيل خيار التنزيل Java أو "All" (الكل)، فسيمكنك اختيار الأدوات وأوقات التشغيل التي تود تثبيتها بشكل محدد. انقر فوق Customize (تخصيص) في صفحة الترحيب بالمثبت وحدد السمات التي تريد استخدامها. لمزيد من المعلومات، انظر تثبيت البرنامج.

إذا أردت إضافة وظيفة إلى تكوين NetBeans IDE، فاستخدم NetBeans Plugin Manager (إدارة المكون الإضافي NetBeans). وعلى سبيل المثال، لنفترض أنك تريد البدء في تنزيل حزمة Ruby وتثبيتها. ثم أردت عندئذ أن تجرب استخدام وظيفة C/C++. وللقيام بذلك، انتقل إلى NetBeans Plugin Manager (إدارة المكون الإضافي NetBeans) من IDE (اختر Tools (أدوات) > Plugins (مكونات إضافية)) وقم بإضافة حزمة C/C++ إلى التثبيت الموجود لديك.

هناك العديد من الملاحظات الهامة حول تثبيت بعض ميزات NetBeans IDE وتكوينها:

  1. يكون Apache Tomcat 6.0.20 مضمنًا في خياري تثبيت "Java" و"All" إلا أنه لا يتم تثبيته افتراضيًا من أي منهما. لتثبيت Apache Tomcat من تنزيل Java أو "All"، قم بتشغيل المثبت وتحديد Apache Tomcat 6.0.20 في مربع الحوار Customize Installation (تخصيص التثبيت).
  2. بالنسبة لتطوير C/C++ (يمكن تطبيقه على تنزيلات C/C++ و"All")، ستحتاج كذلك لتثبيت برامج التحويل البرمجي والأدوات. لمزيد من المعلومات، راجع تكوين NetBeans IDE لـ C/C++/Fortran.
  3. بالنسبة لنظام التشغيل Mac OS X، يمكن تخصيص تثبيت NetBeans IDE من تنزيلات Java وRuby و"All". لا يمكن تخصيص تثبيت IDE من تنزيلات JavaSE وPHP وC/C++.
  4. يمكنك أيضًا تنزيل ملف مضغوط غير مرتبط بالنظام الأساسي وتشغيل الملف التنفيذي NetBeans IDE. لاحظ أن أوقات التشغيل، مثل خوادم تطبيق GlassFish وApache Tomcat تتوفر فقط مع المثبتات الخاصة بالنظام الأساسي.

البداية

بدء التنزيل

  1. انتقل إلى صفحة تنزيل NetBeans IDE.
  2. في الناحية اليسرى العلوية من الصفحة، حدد اللغة والنظام الأساسي من القائمة المنسدلة. كما يمكن اختيار تنزيل الملف المضغوط غير المرتبط بالنظام الأساسي واستخدامه.
  3. انقر فوق الزر Download (تنزيل) الخاص بخيار التنزيل الذي ترغب في تثبيته.
  4. احفظ ملف المثبت على النظام لديك.

البداية

تثبيت البرنامج

اتبع الإرشادات الواردة في هذا القسم لتثبيت بيئة IDE على النظام. تنطبق إرشادات التثبيت هذه على جميع الأنظمة الأساسية المدعومة. للحصول على قائمة بالأنظمة الأساسية المدعومة، انظر ملاحظات حول الإصدار.

Microsoft Windows وSolaris OS وLinux

لتثبيت البرنامج:

  1. بعد استكمال التنزيل، قم بتشغيل المثبت.
    • بالنسبة لنظام Windows، يتميز الملف التنفيذي للمثبت بالامتداد .exe. انقر نقرًا مزدوجًا فوق ملف المثبت لتشغيله.
    • بالنسبة للنظامين الأساسيين Solaris وLinux، يتميز ملف المثبت بالامتداد .sh . وبالنسبة لهذين النظامين الأساسيين، يجب أن تجعل ملفات المثبت ملفات تنفيذية عن طريق استخدام الأمر التالي: chmod +x <installer-file-name>
  2. لاختيار أدوات وأوقات التشغيل المطلوب تثبيتها، قم بإجراء الخطوات التالية في صفحة الترحيب بمعالج التثبيت:
    1. انقر فوق Customize (تخصيص).
    2. في مربع حوار Customize Installation (تخصيص التثبيت)، حدد ما تريد من خيارات.
    3. انقر فوق OK (موافق).
  3. في صفحة الترحيب بمعالج التثبيت، انقر فوق Next (التالي).
  4. في صفحة اتفاقية الترخيص، راجع اتفاقية الترخيص وحدد خانة اختيار القبول ثم انقر فوق Next (التالي).
  5. في صفحة تثبيت NetBeans IDE، قم بإجراء ما يلي:
    1. اقبل دليل التثبيت الافتراضي الخاص بـ NetBeans IDE أو حدد دليلاً آخر. ملاحظة: يجب أن يكون دليل التثبيت فارغًا، كما يجب أن يشتمل ملف تعريف المستخدم المستعمل في تشغيل المثبت على أذونات قراءة/كتابة لذلك الدليل.
    2. اقبل تثبيت JDK الافتراضي للاستخدام مع NetBeans IDE أو حدد تثبيتًا مختلفًا من القائمة المنسدلة. إذا لم يعثر معالج التثبيت على تثبيت لـ JDK متوافق للاستخدام مع NetBeans IDE، فلن يتم تثبيت JDK في الموقع الافتراضي. وفي هذه الحالة، حدد مسار أداة JDK المثبتة أو ألغ التثبيت الحالي ثم قم بتثبيت إصدار JDK المطلوب وأعد تشغيله.
    3. انقر فوق Next (التالي).
  6. في حالة فتح صفحة تثبيت GlassFish v3، اقبل دليل التثبيت الافتراضي أو حدد موقع تثبيت آخر.
  7. في حالة تثبيت Apache Tomcat، اقبل دليل التثبيت الافتراضي في صفحة التثبيت الخاصة به، حدد موقع تثبيت آخر.
  8. في الصفحة ملخص، تحقق من صحة قائمة المكونات التي سيتم تثبيتها ومن تخصيص مساحة كافية على النظام للتثبيت.
  9. انقر فوق Install (تثبيت) لبدء التثبيت.

    10. ملاحظة: إذا صادفتك مشكلات تعوق استكمال تثبيت البرنامج، فراجع استكشاف الأخطاء وإصلاحها للحصول على أوصاف وحلول بديلة مقترحة لمشكلات لم يتم حلها قد تؤثر على عملية التثبيت.

    Mac OS X

    لتثبيت البرنامج:

    1. بعد استكمال التنزيل، قم بتشغيل المثبت. يتميز ملف المثبت بالامتداد .dmg .
    2. في اللوحات التي ستفتح انقر فوق رمز الحزمة. تتميز الحزمة بالامتداد .mpkg . يبدأ معالج التثبيت.
    3. في صفحة الترحيب بمعالج التثبيت، انقر فوق Continue (متابعة).
    4. مراجعة اتفاقية الترخيص وانقر فوق Continue (متابعة). ثم انقر فوق Accept (قبول) في الإطار المنبثق لقبول الترخيص.
    5. في صفحة Select a Destination (تحديد وجهة)، حدد محرك الأقراص وانقر فوق Continue (متابعة).
    6. إذا قمت بتنزيل All (الكل) أو Java أو مجموعة Ruby، فيمكنك تخصيص التثبيت. في لوحة معالج التثبيت الأخيرة، اضغط على الزر Customize (تخصيص) في الجانب السفلي الأيمن من اللوحة. ويتم عرض شجرة المنتجات.
    7. حدد المنتجات التي تريد تثبيتها.
    8. أدخل اسم المسؤول وكلمة مرور النظام وانقر فوق OK (موافق) لبدء التثبيت.

    حزمة غير مرتبطة بالنظام الأساسي

    لتشغيل البرنامج:

    1. بعد استكمال تنزيل ملف ZIP غير المرتبط بالنظام الأساسي، قم باستخراجه إلى أي مجلد على النظام.
    2. قم بتشغيل الملف التنفيذي الموجود في الدليل netbeans/bin.
    3. اقبل اتفاقية الترخيص. يبدأ NetBeans IDE.

    ملاحظة: لاحظ أن أوقات التشغيل، مثل خادم تطبيقات GlassFish وApache Tomcat، تتوفر فقط مع المثبتات الخاصة بالنظام الأساسي.

    ملاحظة: إذا تعذر العثور على تثبيت JDK متوافق، فقد تحتاج لإضافة المسار يدويًا إلى دليل تثبيت JDK:

    1. افتح ملف netbeans.conf الموجود في netbeans/etc في محرر نص.
    2. أدخل موقع تثبيت JDK المتوافق كقيمة الخيار netbeans_jdkhome .
    3. احفظ ملف netbeans.conf وقم بتشغيل الملف التنفيذي في netbeans/bin.

    البداية

    إزالة تثبيت البرنامج

    Microsoft Windows

    عند الضرورة، يمكن إزالة تثبيت NetBeans IDE وGlassFish v3 وApache Tomcat عبر اتباع الخطوات التالية.

    لإزالة تثبيت IDE:

    1. أوقف تشغيل IDE.
    2. من لوحة التحكم، حدد "إضافة أو إزالة البرامج" ثم حدد NetBeans IDE 6.8 وانقر فوق "إزالة".

    لإزالة تثبيت GlassFish v3:

    1. أوقف تشغيل IDE.
    2. من لوحة التحكم، حدد "إضافة أو إزالة البرامج" ثم حدد GlassFish v3 وانقر فوق "إزالة".

    لإزالة تثبيت Apache Tomcat:

    1. أوقف تشغيل IDE.
    2. من لوحة التحكم، حدد "إضافة أو إزالة البرامج" ثم حدد Apache Tomcat 6.0.20 وانقر فوق "إزالة".

    Solaris OS وLinux

    عند الضرورة، يمكن إزالة تثبيت NetBeans IDE وGlassFish v3 وApache Tomcat بالاستعانة بالخطوات التالية.

    لإزالة تثبيت IDE:

    1. أوقف تشغيل IDE.
    2. في دليل التثبيت IDE، قم بتشغيل ملف uninstall.sh .
    3. لإزالة تثبيت GlassFish/Tomcat مع IDE، تأكد في صفحة Summary "الملخص" من عرض الإصدارات الصحيحة ثم حدد خانات الاختيار الملائمة.
    4. في الصفحة Summary، انقر فوق Uninstall (إزالة التثبيت).
    5. عند استكمال التثبيت، انقر فوق Finish (إنهاء).

    لإزالة تثبيت GlassFish v3 :

    1. أوقف تشغيل IDE.
    2. حدد موقع دليل تثبيت خادم التطبيقات.

      ملاحظة: يختلف موقع التثبيت الافتراضي حسب نوع التثبيت والنظام الأساسي. ويوضح الجدول التالي دليل تثبيت خادم التطبيقات الافتراضي للعديد من الأنظمة الأساسية.

      النظام الأساسي التثبيت بصفة... تثبيت GlassFish v3
      Windows مستخدم ذو حقوق مسؤول %ProgramFiles%\glassfish-v3
      Windows مستخدم %USERPROFILE%\glassfish-v3
      Solaris OS  الجذر /opt/glassfish-v3
      Solaris OS، Linux مستخدم ~/glassfish-v3
      Linux جذر /usr/local/glassfish-v3
    3. في دليل تثبيت خادم التطبيقات، قم بتشغيل ملف uninstall.sh .
    4. في الصفحة Summary، انقر فوق Uninstall (إزالة التثبيت).
    5. بعد استكمال عملية إزالة التثبيت، انقر فوق Finish (إنهاء).

    لإزالة تثبيت Apache Tomcat:

    1. أوقف تشغيل IDE.
    2. قم بتحديد موقع دليل تثبيت Apache Tomcat وتشغيل الملف uninstall.sh .
    3. في صفحة Summary، انقر فوق Uninstall (إزالة التثبيت).
    4. بعد استكمال عملية إزالة التثبيت، انقر فوق Finish (إنهاء)

    Mac OS X

    لإزالة تثبيت IDE وخادم التطبيقات:

    1. أوقف تشغيل البرنامج.
    2. اسحب مجلدات أو ملفات البرنامج إلى Trash (سلة المهملات). لاحظ أن الموقع الافتراضي لبيئة IDE وخادم التطبيقات هو /Applications/NetBeans/.

    حزمة غير مرتبطة بالنظام الأساسي

    لإزالة تثبيت ملفات IDE، اتبع الخطوات الموضحة في قسم إزالة المخلفات بعد عمليات التثبيت غير المكتملة من هذا المستند.

    البداية

    الترقية والتواجد مع الإصدارات السابقة

    يمكن أن تتواجد الإصدارات 5.x و6.0 و6.1 و6.5 و6.7 من NetBeans IDE مع NetBeans IDE 6.8 على نفس النظام. ومن ثم لا تلزم إزالة تثبيت الإصدارات السابقة أو تشغيل NetBeans IDE 6.8.

    إذا كان الإصدار 5.x أو 6.0 أو 6.1 أو 6.5 أو 6.7 من NetBeans IDE هو المثبت، فسيكون بوسعك اختيار استيراد إعدادات المستخدم أم لا من دليل المستخدم الموجود عند تشغيل NetBeans IDE 6.8 للمرة الأولى.

    البداية

    استكشاف الأخطاء وإصلاحها

    نبذة حول دليل عمل المثبت

    يقوم المثبت بتخزين بيانات حول جلسات عمل التثبيت في دليل عمل المثبت. ومن بين الأمثلة على هذه البيانات قائمة المكونات المثبتة على النظام ورسائل سجل التثبيت وبيانات التسجيل.

    الموقع الافتراضي لدليل عمل المثبت هو %USERPROFILE%\.nbi على نظام Windows و ~/.nbi على الأنظمة الأساسية الأخرى. يشتمل الدليل الفرعي للسجل على ملفات سجل التثبيت.

    إزالة المخلفات بعد عمليات التثبيت غير المكتملة

    بعد عمليات التثبيت غير الكاملة أو غير الناجحة، قد يلزم تنظيف النظام من مخلفات عمليات التثبيت السابقة.

    بالنسبة لأنظمة Microsoft Windows وSolaris OS وLinux:

    1. احذف دليل عمل المثبت.
    2. احذف دلائل التثبيت الخاصة بـ IDE وخادم التطبيقات GlassFish.
    3. في النظام الأساسي لـ Windows، احذف جميع إدخالات التسجيل nbi* من مفاتيح التسجيل التالية: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

    بالنسبة لنظام Mac OS X:

    • اسحب مجلدات أو ملفات البرنامج إلى Trash (سلة المهملات).

    تعطيل بعض البرامج قبل التثبيت

    يُفضل تعطيل تطبيقات جدار الحماية ومكافحة الفيروسات والمراقبة قبل بدء التثبيت. حيث قد تحول بعض هذه البرامج دون تكوين مكونات IDE بصورة صحيحة أثناء عملية التثبيت.

    على سبيل المثال، إذا كان تطبيق Logitech Process Monitor (LVPrcSrv.exe) قيد التشغيل على النظام، فقم بإيقافه قبل التثبيت.
