Informations de version sur NetBeans IDE 6.8

NetBeans IDE est un environnement de développement intégré (EDI) de premier ordre pour Windows, Mac, Linux et Solaris. Le projet NetBeans consiste en un EDI Open Source et en une plate-forme d'application permettant aux développeurs de créer rapidement des applications Web, d'entreprise, de bureau et mobiles à l'aide de la plate-forme Java, ainsi que de JavaFX, PHP, JavaScript et Ajax, Ruby et Ruby on Rails, Groovy et Grails et C/C++.

Le projet NetBeans est soutenu par une communauté de développeurs dynamique et propose différentes ressources de documentation et de formation, ainsi qu'un choix varié de plug-ins tiers.

Présentation de la version

NetBeans IDE 6.8 est le premier EDI à prendre en charge l'intégralité de la spécification Java EE 6, avec prise en charge améliorée de JSF 2.0/Facelets, Java Persistence 2.0, EJB 3.1 et notamment l'utilisation des EJB dans des applications Web, les services Web RESTful et GlassFish v3. Nous recommandons également cette version pour développer avec la dernière version de JavaFX SDK 1.2.1 et pour créer des applications Web PHP avec la nouvelle version de PHP 5.3 ou avec le framework Symfony.

Notre intégration unique de Project Kenai, un environnement collaboratif permettant d'héberger des projets Open Source, offre désormais la prise en charge complète de JIRA, ainsi que l'intégration d'un programme de messagerie instantané amélioré et d'un outil de suivi des problèmes. Nous avons également ajouté des fonctionnalités à l'intégration des bases de données et Maven de l'EDI et avons amélioré l'intégration de l'éditeur et des outils des projets Ruby, Groovy et C/C++.

Principales fonctionnalités

Java Enterprise Edition 6

  • Projets Web avec Java EE 6 et les profils Web Java EE 6, EJB dans les applications Web
  • Prise en charge d'EJB 3.1, l'assistant des fichiers de projet EJB prend également en charge le type de session Singleton
  • Services Web RESTful (JAX-RS 1.1), services Web GlassFish Metro 2.0 (JAX-WS 2.2), JAXB 2.2
  • Java Persistence JPA 2.0, déploiement, débogage et profilage avec le serveur d'application GlassFish v3

Projets Web avec JavaServer Faces 2.0 (Facelets)

  • Complétion de code, indications d'erreurs, achèvement d'espaces de noms, fenêtres contextuelles de documentation et importation automatique de balises pour Facelets
  • Prise en charge de l'éditeur des bibliothèques Facelets, des composants composites, des langages d'expression, notamment des générateurs de formulaires JSF et HTML
  • La palette de composants JSF personnalisable génère des formulaires JSF et des tables de données JSF à partir d'entités
  • L'assistant de création de fichiers génère des pages JSF CRUD (Create/Read/Update/Delete) personnalisables à partir d'entités
  • Utilisation élargie des annotations au lieu des descripteurs de déploiement

JavaFX

  • Ajout de la prise en charge de la dernière version de JavaFX SDK 1.2.1
  • Complétion de code améliorée
  • Indications de l'éditeur : Réparer les éléments importés, Entourer avec, Implémenter des méthodes abstraites, etc
  • Navigation améliorée : Hyperliens, Aller au type, Rechercher les utilisations

Kenai.com : développeur connecté

  • Prise en charge complète de JIRA (plug-in du centre de mise à jour)
  • Tableau de bord des projets avec des informations plus détaillées sur les membres et les projets, une recherche et une navigation améliorées et un partage des projets plus simple
  • Intégration améliorée de la messagerie instantanée : présence en ligne, discussion privée et en groupe avec les membres du projet Kenai, liens faciles à ajouter vers du code/des fichiers/des problèmes/des suivis de piles vers les messages
  • Intégration améliorée de l'outil de suivi des problèmes

PHP

  • Prise en charge complète de PHP 5.3 : espaces de noms, fonctions et fermetures lambda, ajouts de syntaxes : NOWDOC, conditions ternaires, étiquettes de renvoi, __callStatic()
  • Prise en charge du framework Symfony : projets Symfony, commandes Symfony, raccourcis, coloriage syntaxique PHP dans les fichiers YAML
  • Création d'un projet PHP à partir d'une application PHP distante
  • PHPUnit, couverture du code, amélioration de l'intégration FTP/SFTP, exclusion des dossiers de projet PHP du balayage et de l'indexation

Maven

  • Nouveau projet du catalogue d'archétypes Maven et prise en charge optimisée des projets Java EE 6, Groovy et Scala
  • Exclusion personnalisable des dépendances dans le graphe des dépendances
  • Plug-in CheckStyle de Maven
  • Action de "mise à jour de Kenai" pour les projets Maven hébergés par Kenai.com

Ruby

  • Prise en charge de la création d'applications Rails 2.3.4 avec dispatcheurs, débogage JRuby 1.4, Ruby 1.9 et RSpec 1.2.7
  • Refactorisation de renommage, inférence de type et navigation améliorées
  • Spécification des arguments pour les serveurs Rails
  • Exécution/Débogage de fichiers avec arguments et de fichiers ne faisant pas partie d'un projet

C et C++

  • Profilage : nouvel indicateur de comptabilité (Microstate Accounting), vue de liste de threads, vue des points actif, vue des pertes de mémoire, vue de synchronisation des problèmes
  • Synchronisation accélérée au cours du développement à distance
  • Prise en charge du rattachement de gdbserver et rattachement simplifié aux processus déjà en cours d'exécution

Améliorations diverses

  • Débogueur Java : attribuer un nom à un objet dans l'arborescence des variables pour y faire référence dans les expressions
  • Intégration des bases de données : la complétion de code dans l'éditeur SQL prend maintenant également en charge les instructions DELETE, DROP et UPDATE, ainsi que les mots-clés réservés
  • Groovy 1.6.4 et Grails : complétion de code améliorée, notamment les méthodes introduites par l'intermédiaire des transformations AST
