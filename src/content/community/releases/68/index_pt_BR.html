Informações da versão 6.8 do NetBeans IDE

O NetBeans IDE é um premiado ambiente de desenvolvimento integrado disponível para Windows, Mac, Linux e Solaris. O projeto NetBeans consiste em um IDE de código aberto e uma plataforma de aplicativos que permite aos desenvolvedores criar rapidamente aplicativos Web, empresariais, móveis e de área de trabalho usando a plataforma Java, bem como JavaFX, PHP, JavaScript e Ajax, Ruby e Ruby on Rails, Groovy e Grails e C/C++.

O projeto NetBeans tem como suporte uma entusiasmada comunidade de desenvolvedorese oferece vastos recursos como documentação e treinamento, bem como uma variada seleção de plug-ins de terceiros.

Visão geral da versão

O NetBeans 6.8 é o primeiro IDE a oferecer suporte completo para toda a especificação do Java EE 6 com suporte aprimorado para o JSF 2.0/Facelets, Java Persistence 2.0, EJB 3.1, incluindo o uso de EJBs em aplicações Web, serviços Web RESTful e o GlassFish v3. Nós o recomendamos também para o desenvolvimento com o JavaFX SDK 1.2.1 mais recente, e para criar aplicativos Web PHP com a nova versão 5.3 do PHP ou com o Symfony Framework.

Nossa integração único do Projeto Kenai, um ambiente colaborativo para hospedar projetos de código-fonte aberto, agora vem acompanhado com o suporte JIRA completo, e um sistema aprimorado de mensagens instantâneas e com a integração de rastreamento de questões. Nós também adicionamos recursos ao IDE do Maven e a integração com banco de dados, e a integração aprimorada do editor e ferramentas para projetos Ruby, Groovy e C/C++.

Recursos em destaque

Java Enterprise Edition 6

  • Projetos Web com perfis Java EE 6 e Java EE 6 Web, com aplicativos Web no EJBs.
  • Suporte ao EJB 3.1, o assistente de arquivo de projeto EJB também suporta o tipo de sessão Singleton
  • Serviços Web RESTful (JAX-RS 1.1), serviços Web GlassFish Metro 2.0 (JAX-WS 2.2), JAXB 2.2
  • Java Persistence JPA 2.0, implementação, depuração e perfilamento com o servidor de aplicativos GlassFish v3

Projetos Web com JavaServer Faces 2.0 (Facelets)

  • Conclusão de código, dicas de erros, conclusão do namespace, caixa suspensa de documentação, e importação automática de identificadores para Facelets
  • Suporte de editor para bibliotecas Facelets, componentes compostos, linguagem de expressão, incluindo geradores para formulários JSF e HTML
  • Paletas personalizadas de componentes JSF geram formulários JSF e tabelas de banco de dados JSF de entidades
  • Novo assistente de Arquivo gera página JSF de CRUD personalizável (criar/ler/atualizar/excluir) de entidades
  • Uso mais amplo de anotações ao invés de descritores de implementação

JavaFX

  • Adicionado o suporte para o JavaFX SDK 1.2.1 mais recente
  • Conclusão aprimorada de código
  • Dicas do editor: Corrigir importações, Circundar como, Métodos de implementações abstratas, e mais
  • Navegação aprimorada: Hiperlinks, Ir para tipo, Encontrar utilizações

Kenai.com: Desenvolvedor conectado

  • Suporte completo ao JIRA (plug-in da central de atualizações)
  • Painel do projeto com mais membros e detalhes do projeto, pesquisa e navegação aprimorada, compartilhamento mais fácil de projeto
  • Integração aprimorada de mensagens instantâneas: presença on-line, bate-papo privado e em grupo com membros do Kenai, fácil adição de links ao código / arquivo / questões/ rastreamento em pilha para mensagens
  • Integração aprimorada do rastreador de questões

PHP

  • Suporte completo ao PHP 5.3; namespaces, funções e fechamentos lambda, adições de sintaxe: NOWDOC, condições ternárias, rótulos de pulo, __callStatic()
  • Suporte ao Symfony Framework: projetos Symfony, comandos Symfony, atalhos, colorir a sintaxe PHP em arquivos YAML
  • Criar um projeto PHP a partir de um aplicativo PHP remoto
  • PHPUnit, Cobertura de Código, aprimoramentos de integração FTP/SFTP, excluir pastas de projetos PHP da varredura/indexação

Maven

  • Novo projeto a partir de um catálogo de arquetipo Maven e suporte aprimorado para projetos Java EE 6, Groovy e Scala
  • Exclusão personalizável de dependência em um gráfico de dependência
  • Plug-in Maven CheckStyle
  • Ação de "Atualizar a partir do Kenai" para projetos Maven hospedados no Kenai.com

Ruby

  • Suporte para criar aplicativos Rails 2.3.4 com dispatchers, JRuby 1.4, depuração Ruby 1.9 e RSpec 1.2.7
  • Refatoração de renomeação, inferência de tipo e navegação aprimorados
  • Especificação de argumentos para servidores Rails
  • Executar/Depurar arquivo com argumentos, também para arquivos que não fazem parte de um projeto

C e C++

  • Perfilamento: Novo indicador Microstate Accounting, vista de Mapa de Thread, vista de Pontos-chave, vista de Vazamento de Memória, vista de Problemas de Sincronização
  • Sincronização mais rápida durante o desenvolvimento remoto
  • Suporte para anexar gdbserver e para a anexação mais fácil aos processos já em execução

Aprimoramentos diversos

  • Depurador Java: marcar um objeto na árvore de variáveis com um nome a ser referido nas expressões
  • Integração com o banco de dados: a conclusão de código no editor SQL agora também para as declarações DELETE, DROP, UPDATE e para palavras-chave reservadas
  • Groovy 1.6.4 e Grails: conclusão aprimorada de código, incluindo métodos introduzidos através de Transformações AST
