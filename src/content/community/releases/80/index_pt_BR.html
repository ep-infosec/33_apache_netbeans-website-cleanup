Informações sobre o NetBeans IDE 8.0.1

O NetBeans IDE 8.0 fornece analisadores de código e editores prontos para uso para trabalhar com as tecnologias Java 8 mais recentes -- Java SE 8, Java SE Embedded 8 e Java ME Embedded 8. O IDE também possui uma faixa de novos aprimoramentos que melhoram ainda mais seu suporte para Maven e Java EE com o PrimeFaces; novas ferramentas para HTML5, em particular para AngularJS; e aprimoramentos para o suporte a PHP e C / C++.

O NetBeans IDE 8.0 está disponível em inglês, português do Brasil, japonês, russo e chinês simplificado.

O download mais recente disponível é o NetBeans IDE 8.0.1, que é uma atualização do NetBeans IDE 8.0. Ele oferece o seguinte:

  • Suporte ao Servidor WildFly e ao GlassFish 4.1
  • Melhorias para refatoração de JavaScript
  • Melhorias para projetos em HTML5 (Grunt, Karma, Bower)
  • Integração de patches recentes e Mais

Novas Funcionalidades em Destaque

Java

  • Suporte a JDK 8: ferramentas e aprimoramentos do editor para o trabalho com Perfis, Lambdas e Streams.
  • Suporte ao Java SE Embedded para implantar, executar, depurar ou criar perfis de aplicações Java SE em um dispositivo incorporado, como Raspberry PI, diretamente do NetBeans IDE.
  • Suporte ao Java ME Embedded 8.
  • Diversos aprimoramentos do Java Editor, como muitas novas dicas sobre Java, Javadoc mostrado como dica de ferramenta, aperfeiçoamentos na renomeação instantânea e exclusões de autocompletar código.
  • Integração aprimorada com JavaFX Scene Builder.

Java EE

  • Novo geradores de código PrimeFaces, para gerar aplicações base CRUD PrimeFaces completas com conexão para bancos de dados.
  • Suporte ao Servidor WildFly e ao WebLogic 12.1.3
  • Bundle do GlassFish atualizado para 4.1
  • Suporte a Tomcat 8.0.9 e TomEE, incluindo configuração e origens de dados JPA, JSF e JARXS prontas para uso.
  • O assistente de Cliente de Modelo de Facelets pergunta quais seções devem ser geradas.
  • Autocompletar código para seções de modelo, e dentro de componentes compostos JSF.
  • Integração de CDI aprimorada no arquivo beans.xml, com autocompletar código para classes e estereótipos alternativos.
  • Mais flexibilidade para dicas de JPA, que podem ser desativadas, ativadas e alteradas quanto ao nível de advertência.

Maven

  • Novo alternador de layout do gráfico Maven.
  • O Navegador mostra metas de POM.
  • Recursos de Web adicionais mostrados na janela Projetos.
  • Desempenho melhorado de Maven ao re-executar aplicações Web.
  • Desempenho melhorado do Maven ao "Aplicar Alterações de Código" durante a depuração.

JavaScript

  • Suporte a RequireJS
  • Navegação do AngularJS via hiperlinks da view para os controladores.
  • Muitos aprimoramentos para AngularJS na funcionalidade autocompletar código entre artefatos, tais como autocompletar código na view para as propriedades definidas no controladores.
  • Suporte para autocompletar código em modelos do Knockout.
  • Novo suporte ao editor para criar widgets e plug-ins do JQuery.
  • Há suporte para a depuração do código JavaScript executado no mecanismo de script Nashorn (JDK 8+).

HTML5

  • Suporte para depuração do Android 4.4 Webkit para Cordova 3.3+.
  • Novo suporte para o executor de teste do Karma.
  • Novo suporte para o build do Grunt.
  • O Avatar.js está disponível no Gerenciador de Plug-ins, via Ferramentas | Plug-ins. Após a instalação, você estará apto a criar projetos Avatar.js.

PHP

  • Novo suporte para o testador do Nette.
  • Novo suporte para o Corretor do CS PHP.

  • Suporte do Novo Assistente de Projeto para o Composer.

  • Nova Console do GDB para ativar o uso do modo GDB da linha de comandos.

  • Novo design de interface de usuário para gerenciar Grupos de Projeto.
