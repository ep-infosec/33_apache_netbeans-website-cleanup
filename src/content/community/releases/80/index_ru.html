Сведения о NetBeans IDE 8.0.1

NetBeans IDE 8.0 предлагает готовые решения по анализу и редактированию кода, предназначенные для работы с новейшими технологиями Java 8 — Java SE 8, Java SE Embedded 8 и Java ME Embedded 8. Кроме того, IDE содержит ряд улучшений, расширяющих поддержку Maven и Java EE с PrimeFaces, новые инструменты для HTML5, в частности для AngularJS, и обеспечивает усовершенствованную поддержку PHP и C/C++.

IDE NetBeans 8.0 доступна на английском, бразильском португальском, японском, русском и упрощенном китайском языках.

Новейшая версия, доступная для загрузки: NetBeans IDE 8.0.1. Это обновление NetBeans IDE 8.0, которое содержит:

  • Поддержка WildFly Server и GlassFish 4.1
  • Улучшенный рефакторинг JavaScript
  • Улучшения проектов HTML5 (Grunt, Karma, Bower)
  • Интеграция новейших исправлений и другое

Обзор новых функциональных возможностей

Java

  • Поддержка JDK 8: улучшенные инструменты и средства редактирования для работы с Profiles, Lambdas и Streams.
  • Поддержка Java SE Embedded для развертывания, запуска, отладки и профилирования приложений Java SE, установленных на встроенных устройствах, например Raspberry PI, непосредственно с NetBeans IDE.
  • Поддержка Java ME Embedded 8.
  • Улучшенная работа Java Editor: множество новых подсказок Java, отображение Javadoc в виде всплывающих подсказок, улучшенная функция быстрого переименования файлов и исключения автозаполнения кода.
  • Улучшенная интеграция с JavaFX Scene Builder.

Java EE

  • Новые генераторы кода PrimeFaces, предназначенные для создания приложений CRUD на основе схемы PrimeFaces с подключением к базе данных.
  • Поддержка WildFly Server и WebLogic 12.1.3.
  • Пакет GlassFish обновлен до версии 4.1.
  • Поддержка Tomcat 8.0.9 и TomEE, включая готовые конфигурации и источники данных JPA, JSF и JAXRS.
  • Запрос мастера клиента шаблонов Facelets на создание разделов.
  • Автозаполнение кода для разделов шаблонов и в составных компонентах JSF.
  • Расширенная интеграция CDI в файле beans.xml с автозаполнением кода для альтернативных классов и стереотипов.
  • Больше возможностей работы с подсказками JPA: отключение, включение и изменение уровня оповещения.

Maven

  • Новый переключатель графического макета Maven.
  • Навигатор отображает цели POM.
  • В окне "Проекты" отображаются дополнительные веб-ресурсы.
  • Улучшенная производительность Maven при повторном запуске веб-приложений.
  • Улучшенная производительность функции Maven "Применить изменения кода" при отладке.

JavaScript

  • Поддержка RequireJS.
  • Навигация AngularJS через гиперссылки от представления к контроллерам.
  • Оптимизация работы AngularJS при автозаполнении кода между артефактами, например автозаполнение кода в представлении для свойств, определенных в контроллерах.
  • Поддержка автозаполнения кода в шаблонах Knockout.
  • Новая функция редактора для создания виджетов и плагинов JQuery.
  • Поддержка откладки кода JavaScript, исполняемого в интерпретаторе Nashorn (JDK 8+).

HTML5

  • Поддержка отладки Android 4.4 WebKit для Cordova 3.3+.
  • Новая поддержка исполнителя тестов Karma.
  • Новая поддержка сборки Grunt.
  • Avatar.js доступен из диспетчера подключаемых модулей (Сервис | Подключаемые модули). После установки вы можете создавать проекты Avatar.js.

  • Новая поддержка тестировщика Nette.

  • Поддержка компоновщика в мастере новых проектов.
