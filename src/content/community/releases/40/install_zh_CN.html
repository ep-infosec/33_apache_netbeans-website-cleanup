NetBeans™ IDE 4.0 安装说明

本部分详细介绍了 IDE 在以下操作系统上的安装：

  • Windows
  • Solaris™ 操作系统 (Solaris OS)
  • Linux
  • Macintosh OS X
  • 其他操作系统
  • 归档文件发行版本

有关系统要求和其他信息（如使用不同的启动参数启动 IDE），请参见发行说明。

Windows

在 Microsoft Windows 计算机上，最好使用自解压的安装程序来安装 NetBeans IDE。

使用 Windows 安装程序

要安装 NetBeans IDE，请执行以下步骤：

  1. 在下载安装程序文件后，双击安装程序的图标以启动安装向导。
  2. 指定 NetBeans IDE 的安装目录。
  3. 指定将运行 IDE 的 Java™ 2 软件开发工具包。

启动 NetBeans IDE

要启动 IDE，请执行以下操作之一：

  • 双击桌面上的 "NetBeans IDE" 图标。
  • 从“开始”菜单中，选择 "NetBeans IDE 4.0" > "NetBeans IDE"。

Solaris™ 操作系统 (Solaris OS)

在运行 Solaris™ 操作系统的计算机上，最好使用二进制安装程序来安装 NetBeans IDE。

使用 Solaris 安装程序

要安装 NetBeans IDE，请执行以下步骤：

  1. 浏览到包含安装程序的目录。
  2. 如有必要，请在命令提示符下键入以下内容，以便更改安装程序文件的权限，使二进制文件成为可执行文件：
    $ chmod +x your_binary_executable

    注意：确保将 your_binary_executable 替换为下载的二进制文件的实际文件名。

  3. 在命令提示符下键入以下内容以启动安装程序：
    $ ./your_binary_executable

    注意：仍要确保将 your_binary_executable 替换为下载的二进制文件的实际文件名。

安装程序搜索已安装的 JDK，并提示您指定安装程序应使用的 JDK。您也可以在命令行中指定安装程序应使用的 JDK。例如：
$ ./your_binary_executable -is:javahome path_to_your_jdk

启动 NetBeans IDE

要启动 IDE，请执行以下步骤：

  1. 浏览到安装目录的 bin 子目录。
  2. 键入 ./netbeans 以执行启动程序脚本。

Macintosh OS X

在 Macintosh OS X 上，NetBeans 是以本机应用程序包的形式提供的。最好使用压缩包 (.tar.gz) 文件来安装 NetBeans IDE。

使用压缩包 (.tar.gz) 文件

要安装 NetBeans IDE，请执行以下步骤：

  1. 将压缩包下载到系统上的所需位置。
  2. 双击该文件，将内容解压缩到系统上。

启动 NetBeans IDE

要启动 IDE，请执行以下步骤：

  • 双击 NetBeans 应用程序图标。

其他操作系统

如果使用的操作系统没有安装程序，您仍然可以使用 NetBeans IDE，方法是：下载并安装归档文件发行版本。有关详细信息，请参见下面的归档文件发行版本一节。


归档文件发行版本

如果要测试内部开发版本，或者由于使用的操作系统没有安装程序，您还可以安装 IDE 的归档文件发行版本。

使用归档文件发行版本

要安装 NetBeans IDE，请执行以下步骤：

  1. 浏览到包含归档文件的目录。
  2. 使用适合系统的实用程序解压缩归档文件。

启动 NetBeans IDE

要启动 NetBeans IDE，请执行以下步骤：

  1. 浏览到 NetBeans IDE 安装目录的 bin 子目录。
  2. 执行适合系统的启动程序。

归档文件发行版本中包含以下启动程序：

netbeans.exe - Windows
netbeansw.exe - Windows（无控制台窗口）
netbeans - UNIX、Linux
macosx_launcher.dmg - Mac Os
netbeans.cmd - OS/2

如果没有适合操作系统的特定启动程序，则可能需要修改相应脚本以创建一个启动程序。不过，如果您的计算机支持 JDK 1.4.2 或更高版本，则应该能够运行 IDE。

注意：如果创建了启动程序，您可以将其提供给 NetBeans 项目。有关详细信息，请参见 Contributing to netbeans.org。
