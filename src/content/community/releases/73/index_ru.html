Сведения о версии IDE NetBeans 7.3

NetBeans IDE 7.3 позволяет разработчикам создавать и отлаживать многофункциональные веб-приложения и мобильные приложения, используя последние версии стандартов HTML5, JavaScript и CSS. Разработка самых современных многофункциональных веб-приложений становится возможной благодаря таким инструментам, как инспектор страниц, редактор CSS-стилей, полностью обновленный редактора JavaScript, новый отладчик JavaScript и др. Среди дополнительных возможностей версии 7.3 можно отметить усовершенствованную поддержку языков Groovy, PHP, JavaFX и C/C++ в среде IDE.

IDE NetBeans 7.3 доступна на английском, бразильском португальском, японском, русском и упрощенном китайском языках.

Обзор новых функциональных возможностей

Проект Easel: разработка приложений на HTML5

  • Проект приложения на HTML5 с поддержкой тестирования JavaScript
  • Существенные улучшения редактора JavaScript
  • Инспектор страниц и редактор визуальных стилей CSS
  • Отладчик JavaScript
  • Встроенный браузер на базе WebKit; полная интеграция с браузером Chrome
Полный снимок экрана

Java

  • Отладочные операторы: Новая панель навигации показывает текущее размещение в редакторе
  • Обновленный вид элементов и иерархии
  • Новые подсказки и способы реорганизации кода
  • Фильтры для результатов поиска вхождений
  • Рабочая вкладка редактора POM в редакторе pom.xml

Java EE

  • Инструмент тестирования Java Persistence JPQL
  • Усовершенствованная разработка сервиса REST

JavaFX

  • Поддержка FXML/Scene Builder
  • Поддержка JDK 7u6 и более поздних версий для Windows, Mac и Linux
  • Значительное улучшение редактирования FXML с более точным завершением кода, созданием ошибок и т.д.
  • Добавлено действие создания контроллера FXML при работе с файлами FXML

PHP

  • Парсеры для аннотаций пространства имен (Symfony 2, Doctrine 2 и др.)
  • Базовая интеграция с инструментом компоновки (диспетчер зависимостей для PHP)
  • Автозавершение кода в Twig (с документацией)
  • Согласование фигурных скобок Smarty для связанных тегов
  • Ошибки парсера Smarty для несогласованных тегов

Groovy

  • Интеграция Groovy 2.0
  • Поддержка в проектах Java EE
  • Поддержка тестирования JUnit в Groovy
  • Реализация базового набора методов реорганизации кода (поиск вхождений, переименование для типа класса)

Инструмент профилирования

  • Выборочное профилирование памяти
  • Сравнение снимков ЦП
  • Поддержка виртуальной машины Linux ARM

C/C++

  • Снижение использования ресурсов памяти наполовину для больших проектов
  • Увеличение скорости парсера
  • Сохранение и перемещение индекса парсера вместе с проектом

Платформа NetBeans

  • Фильтрация содержимого диалогового окна "Параметры"
  • Меню "Создать файл" и "Открыть последний файл", новое сочетание клавиш быстрого доступа CTRL+ALT+T
  • Главная панель инструментов отображает новый раскрывающийся список для случаев переполнения данных
