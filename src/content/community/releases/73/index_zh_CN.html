NetBeans IDE 7.3 发行信息

借助 NetBeans IDE 7.3，开发者可以使用最新的 HTML5、JavaScript 和 CSS3 标准来创建和调试功能丰富的 Web 应用程序和 Mobile 应用程序。使用页检查器和 CSS 样式编辑器、经过彻底修订的 JavaScript 编辑器以及新型 JavaScript 调试器等，开发者将获得一流的丰富 Web 开发体验。7.3 具有的其他突出优点包括不断改进的 IDE 对 Groovy、PHP、JavaFX 和 C/C++ 的支持。

NetBeans IDE 7.3 提供了英语、巴西葡萄牙语、日语、俄语和简体中文版本。

新增功能集锦

项目框架：HTML5 应用程序开发

  • HTML5 应用程序项目，包含 JavaScript 测试支持
  • 得到大幅改进的 JavaScript 编辑器
  • 页检查器和可视 CSS 样式编辑器
  • JavaScript 调试器
  • 嵌入式 WebKit 浏览器；与 Chrome 深入集成
完整屏幕快照

Java

  • 面包屑：新的导航栏显示编辑器中当前嵌入的内容
  • 更新了“显示成员”视图和“显示分层结构”视图
  • 一组新的提示和重构
  • 用于查找用例结果的过滤器
  • pom.xml 编辑器中的“有效 POM”编辑器选项卡

Java EE

  • Java 持久性 JPQL 测试工具
  • REST 服务开发增强功能

JavaFX

  • FXML/场景构建器支持
  • Windows、Mac 和 Linux 上的 JDK 7u6+ 支持
  • FXML 编辑通过提供准确的代码完成、错误标记等众多功能得到了大幅改进
  • FXML 文件上添加了 FXML 生成控制器操作

PHP

  • 针对有命名空间标注的解析器（Symfony 2、Doctrine 2 等）
  • 基本编写器集成（PHP 的依赖关系管理器）
  • Twig 代码完成（含文档）
  • 匹配相关标记的 Smarty 花括号
  • 不匹配标记的 Smarty 解析器错误

Groovy

  • Groovy 2.0 集成
  • Java EE 项目支持
  • Groovy JUnit 测试支持
  • 实施了基本重构（查找用例，类类型的重命名）

分析器

  • 采样内存性能分析
  • CPU 快照比较
  • 支持 Linux ARM 计算机

C/C++

  • 大型项目的内存使用量至少降低一半
  • 解析器速度提升
  • 解析器的索引可以随项目保留并重新定位

NetBeans 平台

  • “选项”对话框内容过滤
  • “新建文件/打开近期的文件”菜单，以及新的快捷键 CTRL+ALT+T
  • 主工具栏显示针对溢出情况的新下拉列表

有关此发行版本功能的详细信息，请参见 NetBeans IDE 7.3 新增功能和重要功能页。
