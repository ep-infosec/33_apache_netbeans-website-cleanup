NetBeans IDE 7.3リリース情報

NetBeans IDE 7.3は、開発者が最新のHTML5、JavaScriptおよびCSS3標準を使用して、リッチなWebアプリケーションやモバイル・アプリケーションの作成とデバッグを行えるよう支援します。開発者は、ページ・インスペクタとCSSスタイル・エディタ、全面的に刷新されたJavaScriptエディタ、新しいJavaScriptデバッガなどによる最先端のリッチなWeb開発体験を期待できます。7.3ではまた、IDEによるGroovy、PHP、JavaFXおよびC/C++のサポートが引き続き強化されていることも注目に値します。

NetBeans IDE 7.3は、英語、ポルトガル語(ブラジル)、日本語、ロシア語および簡体字中国語で使用できます。

新機能の概要

Project Easel: HTML5アプリケーション開発

  • HTML5アプリケーション・プロジェクトとJavaScriptテストのサポート
  • JavaScriptエディタの大幅な改善
  • ページ・インスペクタとビジュアルCSSスタイル・エディタ
  • JavaScriptデバッガ
  • 組込みのWebKitブラウザ(Chromeと緊密に統合)
フル・スクリーンショット

Java

  • ブレッドクラム: 新しいナビゲーション・バーにエディタでの現在の埋込みを表示
  • メンバー表示ビューと階層表示ビューの更新
  • 一連の新しいヒントとリファクタリング
  • 使用状況の検索結果のフィルタ
  • pom.xmlエディタの「有効なPOM」エディタ・タブ

Java EE

  • Java Persistence JPQLテスト・ツール
  • RESTサービス開発の強化

JavaFX

  • FXML/Scene Builderのサポート
  • Windows、MacおよびLinuxでのJDK 7u6+のサポート
  • 正確なコード補完、エラーのマーク、その他の機能により、FXML編集を大幅に改善
  • FXMLコントローラ生成アクションをFXMLファイルに追加

PHP

  • 名前空間アノテーション用のパーサー(Symfony 2、Doctrine 2など)
  • 基本的なコンポーザの統合(PHP用の依存性マネージャ)
  • Twigコード補完(ドキュメントあり)
  • Smartyで関連タグの照合をサポート
  • 不一致のタグのSmartyパーサー・エラー

Groovy

  • Groovy 2.0の統合
  • Java EEプロジェクトでのサポート
  • Groovy JUnitテストのサポート
  • 基本的なリファクタリングを実装(クラスの型の使用状況を検索、名前を変更)

プロファイラ

  • サンプル・メモリー・プロファイリング
  • CPUスナップショット比較
  • Linux ARMマシンのサポート

C/C++

  • 大規模なプロジェクトのメモリー使用量を半分以下に削減
  • パーサーの高速化
  • パーサーの索引を保持してプロジェクトに再配置することが可能

NetBeansプラットフォーム

  • オプション・ダイアログの内容のフィルタリング
  • 「新規ファイル」/「最近のファイルを開く」メニューと新しいショートカット[Ctrl]+[Alt]+[T]
  • オーバーフローの場合に新しいドロップダウン・リストをメイン・ツールバーに表示

このリリースの機能の詳細は、NetBeans IDE 7.3の新機能と注目すべき機能のページを参照してください。
