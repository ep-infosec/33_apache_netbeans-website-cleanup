Информация по среде IDE NetBeans 6.1

Среда IDE NetBeans представляет собой стандартную модульную интегрированную среду разработки (Integrated Development Environment; IDE), написанную на языке программирования Java. Проект NetBeans состоит из интегрированной среды разработки с отрытым исходным кодом и платформы приложения, которая может использоваться в качестве общей платформы для создания приложений любого типа.

Обзор выпуска

В состав выпуска среды IDE NetBeans 6.1 входит несколько новых компонентов и расширений, например, дополнительные функции правки JavaScript, поддержка использования веб-платформы Spring, более тесная интеграция с MySQL и оптимизация совместного использования библиотек для связанных проектов. Заявленная поддержка Ruby/JRuby была расширена и теперь включает новый редактор с функциями быстрого ввода, диспетчер платформ Ruby, поддержку быстрой отладки JRuby и многие другие новые возможности и усовершенствования.

По запросам пользователей возвращены функции создания шаблонов компонентов и JSF CRUD, отсутствовавшие в выпуске 6.0. Кроме того, в качестве подключаемых модулей доступны более ранние версии новых модулей, например, поддержка ClearCase.

Особенностью данного выпуска также является оптимизированная производительность, в частности, ускоренный запуск (до 40%), более низкое потребление памяти и сокращенное время реакции программы при работе с крупными проектами. Ниже приводится список возможностей, предоставляемых в этом выпуске.

Получение среды IDE NetBeans 6.1

Документация

В качестве руководства по началу работы со средой IDE NetBeans 6.1 используйте следующие документы.

  • Заметки к выпуску
  • Инструкции по установке

Ключевые возможности

  • Поддержка JavaScript
    • Семантическое выделение
    • Дополнение кода и анализ типов
    • Быстрый ввод и семантические проверки
    • Реорганизация

  • Увеличение производительности
    • Ускоренный запуск (на 40%)
    • Расширенные функции интеллектуального анализа для ускорения дополнения кода
    • Более низкое потребление памяти

  • Поддержка платформы Spring
    • Связанная библиотека Spring Framework 2.5
    • Мастера для файлов настройки XML и веб-контроллеров MVC для Spring
    • Дополнение кода для имен компонентов, классов и свойств Java
    • Поддержка веб-платформы MVC для Spring в веб-проекте
    • Дополнительная информация...

  • Поддержка нового MySQL в Database Explorer
    • Регистрация серверов MySQL
    • Просмотр, создание и удаление баз данных
    • Простой запуск из средства администрирования для MySQL

  • Поддержка компонентов Java
    • Шаблоны компонентов в навигаторе
    • Генерация свойств компонента
    • Редактор информации о компоненте

  • Генератор JSF CRUD
    • Генерация приложения JavaServer Faces CRUD из классов объектов
    • Поддержка всех типов отношений между объектами ("один к одному", "один ко многим", "многие к одному" и "многие ко многим")
    • Поддержка всех типов первичных ключей (по одному столбцу, составной и генерируемый)

  • Поддержка Ruby/JRuby
    • Расширения редактора, включая новые подсказки и варианты быстрого ввода
    • Поддержка JRuby, быстрая отладка
    • Диспетчер платформ
    • Оптимизация интеграции сервера и баз данных в проектах Rails

  • Дополнение кода: документация Javadoc
    • Поддержка стандартных тегов: @param и т.д.
    • Дополнение кода для параметров, исключений и т.д.

  • Поддержка распространенных веб-интерфейсов API
    • Простое создание приложений mashup
    • Функции перетаскивания в веб-службах POJO, Servlet, JSP и RESTful а также генерация средой IDE NetBeans полного кода для доступа к службам
    • Поддержка веб-интерфейсов API, например, Google, Facebook, Yahoo и YouTube

  • Поддержка веб-служб RESTful
    • Компоновка веб-служб RESTful, совместимых с JSR311, с помощью Jersey
    • Тестирование приложений с помощью нового клиента REST
    • Использование мастеров для создания сервисов REST из классов объектов JPA и генерации клиентских заглушек JavaScript из WADL

  • Совместное использование проектов (совместно используемые библиотеки)
    • Определение зависимостей для библиотек с использованием относительных путей к библиотеке (для типов проектов Java и Web по умолчанию, а также для всех типов проектов Java EE)
    • Упрощенная процедура совместного использования проектов с другими членами группы, например, посредством систем управления версиями
    • Активация автоматической компоновки проектов

  • Новые подключаемые модули
    • Управление версиями ClearCase
    • Поддержка AXIS
    • Поддержка пользовательского интерфейса SOAP
    • Поддержка платформ Hibernate (Дополнительная информация...)

  • Java Mobility
    • Эмулятор Mpowerplayer MIDP для приложений MIDP в ОС Mac X (доступен в центре подключаемых модулей)
    • SVG Composer для пользовательских компонентов SVG
    • Оптимизированная стабильность и дополненная документация

  • Предварительный просмотр функций в выпусках выше 6.1
    • Поддержка редактора для PHP: дополнение кода, синтаксическое выделение, навигация, отладка и многое другое
    • Отладчик JavaScript
