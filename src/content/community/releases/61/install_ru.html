Указания по установке среды IDE NetBeans 6.1

Дата последнего обновления: August 9, 2012

В этом документе описывается процедура установки среды IDE NetBeans 6.1. Для получения информации о поддерживаемых операционных системах и комплектации оборудования для среды IDE см. Заметки к выпуску среды IDE NetBeans 6.1. Для получения информации о новых возможностях среды IDE NetBeans 6.1 см. Информация по среде IDE NetBeans 6.1.

Требуемое программное обеспечение

Перед установкой IDE необходимо установить Java SE Development Kit (JDK) 5, обновление 14 (версия 1.5.0_14) или выше (включая JDK 6, обновление 4). В противном случае установить IDE будет невозможно. Для получения дополнительной информации о требуемой версии JDK см. "Требуемое программное обеспечение" в документе Заметки к выпуску среды IDE NetBeans 6.1.

Варианты загрузки программы установки

На странице загрузки среды IDE NetBeans 6.1 можно загрузить одну из нескольких программ установки, каждая из которых содержит основные и дополнительные средства IDE.

Доступны следующие программы установки:

  • Web & Java EE. Предоставляет средства для разработки веб-приложений и приложений Java SE и Java EE. Этот вариант загрузки также включает сервер приложений GlassFish V2 UR2 и программное обеспечение Apache Tomcat.
  • Mobility. Поддерживает разработку приложений Java ME и включает пакет Mobility Pack для создания приложений MIDP для мобильных устройств. Учтите, что для работы некоторых компонентов Mobility, например, мастера создания беспроводных соединений, требуются функции из пакета Web & Java EE. Если планируется использовать эти компоненты, выберите вариант загрузки "All".
  • Java SE. Предоставляет все стандартные компоненты для разработки Java SE. Также включена поддержка разработки подключаемых модулей NetBeans.
  • Ruby. Предоставляет средства для разработки приложений Ruby и включает поддержку Rails и JRuby.
  • PHP. Предоставляет средства для разработки PHP. Вариант загрузки "PHP" доступен в режиме "Early Access".
  • C/C++. Поддерживает разработку приложений на языке C/C++.
  • All. Это наиболее полный вариант загрузки для среды IDE NetBeans 6.1. Он содержит все среды выполнения и пакеты, доступные для IDE. Кроме того, он включает средства для разработки сервисно-ориентированной архитектуры и построения диаграмм на унифицированном языке моделирования.

Настройка параметров установки

При выборе вариантов загрузки "Web & Java EE" или "All" можно точно указать средства и среды выполнения, которые требуется установить.

Для варианта загрузки "Web & Java EE" на странице "Welcome" выберите требуемые серверы.

Для варианта загрузки "All" нажмите "Customize" на странице "Welcome" и выберите требуемые компоненты. Для получения дополнительной информации см. раздел Установка программного обеспечения.

Для добавления функциональных возможностей к среде IDE NetBeans используется диспетчер подключаемых модулей NetBeans. Например, предположим, что сначала был загружен и установлен пакет "Ruby". После этого было принято решение попробовать функциональность C/C++. Для этого из IDE перейдите к диспетчеру подключаемых модулей NetBeans (выберите "Tools > Plugins") и добавьте к существующей установке пакет "C/C++".

Ниже приводится ряд важных примечаний относительно установки и настройки некоторых компонентов среды IDE NetBeans:

  1. Apache Tomcat 6.0.16 включен в варианты загрузки "Web & Java EE" и "All", но он устанавливается по умолчанию только из варианта загрузки "Java EE". Для установки Apache Tomcat из пакета "All" запустите программу установки и выберите "Apache Tomcat 6.0.16" в диалоговом окне "Customize Installation".
  2. Для разработки на C/C++ (применимо для вариантов загрузки "C/C++" и "All") необходимо установить компиляторы и средства. Для получения дополнительной информации см. раздел Установка и настройка поддержки C/C++.
  3. Если требуется использовать Sun Java System Access Manager, то помимо варианта загрузки "All" для IDE необходимо установить Java Application Platform SDK, обновление 4 или выше по адресу http://java.sun.com/javaee/downloads/index.jsp.
  4. Для разработки приложений Mobility для устройств CDC необходимо загрузить и установить Sun Java Toolkit 1.0 для CDC. После установки необходимо зарегистрировать платформу CDC в IDE.
  5. Для ОС Mac X настройка параметров установки среды IDE NetBeans не поддерживается.

Начало загрузки

  1. Перейдите к странице загрузки среды IDE NetBeans 6.1.
  2. Выберите соответствующую платформу из выпадающего списка в верхней правой части страницы.
  3. Нажмите кнопку "Download", соответствующую выбранному варианту загрузки для установки.
  4. Сохраните файл программы установки.

Установка программного обеспечения

Для установки IDE следуйте указаниям, приведенным в этом разделе. Данные указания по установке применимы ко всем поддерживаемым платформам. Список поддерживаемых платформ приводится в разделе Заметки к выпуску среды IDE NetBeans 6.1.

ОС Microsoft Windows, Solaris и Linux

Для установки программного обеспечения:

  1. После завершения загрузки запустите программу установки.
    • Для Windows исполняемый файл программы установки имеет расширение .exe. Дважды щелкните по файлу программы установки для запуска.
    • Для платформ Solaris и Linux файл программы установки имеет расширение .sh. Для этих платформ необходимо выполнить следующую команду для исполняемых файлов программы установки:
      chmod +x . /<installer-file-name>
  2. Для выбора средств и среды исполнения выполните следующие действия на странице "Welcome" мастера установки:
    1. Нажмите "Customize".
    2. В диалоговом окне "Customize Installation" выберите требуемые варианты.
    3. Нажмите "OK".
  3. На странице "Welcome" мастера установки нажмите "Next".
  4. На странице "License Agreement" прочтите лицензионное соглашение, установите флажок для принятия его условий и нажмите "Next".
  5. На странице установки среды IDE NetBeans выполните следующие действия:
    1. Выберите каталог для установки среды IDE NetBeans по умолчанию или укажите другой каталог.
      Примечание: Каталог установки должен быть пустым, а профиль пользователя, в котором выполняется программа установки, должен предусматривать полномочия на чтение и запись для этого каталога.
    2. Выберите установку JDK для использования со средой IDE NetBeans по умолчанию или укажите другую установку из выпадающего списка.
      Если мастер установки не нашел совместимую установку JDK для использования со средой IDE NetBeans, JDK не будет установлен в местоположение по умолчанию. В этом случае укажите путь к установленному JDK или отмените текущую установку, установите требуемую версию JDK и перезапустите эту установку.
    3. Нажмите "Next".
  6. При появлении страницы установки сервера приложений GlassFish выполните следующие действия:
    1. Выберите каталог для установки сервера по умолчанию или укажите другой каталог.
      Примечание: Выбранный каталог установки должен быть пустым, а профиль пользователя, в котором выполняется программа установки, должен предусматривать полномочия на чтение и запись для этого каталога.
    2. Из списка совместимых установок JDK выберите JDK для использования сервером приложений, либо нажмите кнопку "Browse" для указания другой установки JDK.
    3. Измените имя пользователя и пароль для домена сервера по умолчанию или выберите значения по умолчанию, а затем нажмите "Next".
      По умолчанию используется имя пользователя admin и пароль adminadmin.
    4. Проверьте значения портов по умолчанию (HTTP, HTTPS и Admin) для сервера и при необходимости измените их.
    5. Нажмите "Next".
  7. При появлении страницы установки Apache Tomcat выберите каталог установки по умолчанию или укажите другое местоположение для установки.
  8. На странице "Summary" проверьте правильность списка компонентов, которые требуется установить, а также наличие достаточного пространства в системе для установки.
  9. Для запуска процесса установки нажмите "Install".
  10. После завершения установки:
    1. Если был установлен сервер приложений GlassFish, для его регистрации установите флажок "Register the Sun Java System Application Server and NetBeans IDE after finishing the installer".
    2. Если сервер приложений GlassFish не был установлен, установите флажок "Register the NetBeans IDE after finishing the installer" для регистрации среды IDE NetBeans.
    3. Для выхода из мастера нажмите "Finish".
  11. Если в предыдущем шаге был установлен флажок "Register…", в обозревателе откроется страница "Registration". Нажмите "Register".
  12. Введите учетную запись Sun Online или создайте ее на странице www.sun.com, на которую можно перейти по этой ссылке.

Примечание: При возникновении проблем во время установки программного обеспечения см. раздел Устранение неполадок, в котором содержится описание соответствующих процедур и возможных решений проблем, влияющих на процесс установки.

Примечание: Если сервер приложений GlassFish был зарегистрирован после установки некорректно, это можно исправить при запуске среды IDE NetBeans или посредством консоли администратора сервера приложений GlassFish.

Примечание: Если среда IDE NetBeans 6.1 была зарегистрирована после установки некорректно, это можно исправить при запуске среды IDE NetBeans.

ОС Mac X

Для установки программного обеспечения:

  1. После завершения загрузки запустите программу установки. Файл программы установки имеет расширение .dmg.
  2. На открывшейся панели щелкните значок пакета. Пакет имеет расширение .mpkg. Запускается мастер установки.
  3. На странице "Welcome" мастера установки нажмите "Continue".
  4. Прочтите лицензионное соглашение и нажмите "Continue". Для принятия условий лицензионного соглашения нажмите "Accept" во всплывающем окне.
  5. На странице "Select a Destination" выберите диск и нажмите "Continue".
  6. Если были загружены пакеты "All" или "Java EE", можно настроить параметры установки. В нижней левой части последней панели мастера установки нажмите кнопку "Customize". Появится дерево программ.
  7. Выберите программы, которые требуется установить.
  8. Введите соответствующее имя администратора и пароль и нажмите "OK" для запуска установки.

Удаление программного обеспечения

Microsoft Windows

При необходимости путем выполнения следующей процедуры можно удалить среду IDE NetBeans, сервер приложений GlassFish и Apache Tomcat.

Для удаления IDE:

  1. Завершите работу IDE.
  2. На панели управления выберите "Add or Remove Programs", затем выберите среду IDE NetBeans 6.1 и нажмите "Remove".

Для удаления сервера приложений GlassFish:

  1. Завершите работу IDE.
  2. На панели управления выберите "Add or Remove Programs", затем выберите GlassFish V2 UR2 и нажмите "Remove".

Для удаления Apache Tomcat:

  1. Завершите работу IDE.
  2. На панели управления выберите "Add or Remove Programs", затем выберите Apache Tomcat 6.0.16 и нажмите "Remove".

ОС Solaris и Linux

При необходимости путем выполнения следующей процедуры можно удалить среду IDE NetBeans, сервер приложений GlassFish и Apache Tomcat.

Для удаления IDE:

  1. Завершите работу IDE.
  2. В каталоге установки IDE запустите файл uninstall.sh.
  3. Для удаления GlassFish/Tomcat вместе с IDE на странице "Summary" проверьте правильность версий и соответствующих флажков.
  4. Нажмите "Uninstall" на странице "Summary".
  5. После завершения удаления нажмите "Finish".

Для удаления сервера приложений GlassFish:

  1. Завершите работу IDE.
  2. Определите каталог установки сервера приложений.

    Примечание: Местоположение установки по умолчанию зависит от типа установки и платформы. В следующей таблице приводятся каталоги установки сервера приложений по умолчанию.

    Платформа Установка как... Установка SOA
    ОС Solaris
    		 root /opt/glassfish-v2ur2
    ОС Solaris,
    Linux    		 user ~/glassfish-v2ur2
    Linux root /usr/local/glassfish-v2ur2
  3. В каталоге установки сервера приложений запустите файл uninstall.sh.
  4. Нажмите "Uninstall" на странице "Summary".
  5. После завершения удаления нажмите "Finish".

Примечание: Сервер приложений GlassFish необходим при использовании функциональности SOA. После удаления сервера приложений функциональность SOA автоматически удаляется из установленной среды IDE NetBeans.

Для удаления Apache Tomcat:

  1. Завершите работу IDE.
  2. Определите каталог установки Apache Tomcat и запустите файл uninstall.sh.
  3. Нажмите "Uninstall" на странице "Summary".
  4. После завершения удаления нажмите "Finish".

ОС Mac X

Для удаления IDE и сервера приложений:

  1. Завершите работу программного обеспечения.
  2. Перетащите папки или файлы программы в корзину.
    Учтите, что местоположением для IDE и сервера приложений по умолчанию является /Applications/NetBeans/.

Обновление и сосуществование предыдущих версий

Одновременно со средой IDE NetBeans 6.1 в одной системе могут существовать выпуски среды IDE NetBeans 5.5, 5.5.1 или 6.0. Для установки или запуска среды IDE NetBeans 6.1 удаление версий 5.5, 5.5.1 или 6.0 не требуется.

Если в системе также установлены версии среды IDE NetBeans 5.5, 5.5.1 или 6.0, при первом выполнении среды IDE NetBeans 6.1 можно импортировать пользовательские параметры настройки из предыдущего каталога пользователя.

Устранение неполадок

О рабочем каталоге программы установки

Программа установки сохраняет данные о сеансах установки в рабочем каталоге. Эти данные включают список установленных в системе компонентов, сообщения файла регистрации установки и информацию из системного реестра.

Местоположением рабочего каталога программы установки по умолчанию является %USERPROFILE%\.nbi для ОС Windows и ~/.nbi для других платформ. Файлы регистрации установки содержатся в подкаталоге файла регистрации.

Очистка после незавершенных установок

После незавершенных или неуспешных установок может потребоваться очистка системы.

Для ОС Microsoft Windows, Solaris и Linux:

  1. Удалите рабочий каталог программы установки.
  2. Удалите каталоги установки IDE и сервера приложений GlassFish.
  3. На платформе Windows удалите все записи nbi* из следующих разделов системного реестра:
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall
    HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Для ОС Mac X:

  • Перетащите папки или файлы программы в корзину.

Отключение программного обеспечения перед установкой

Перед началом установки рекомендуется отключить брандмауэр, антивирусное программное обеспечение и приложения мониторинга. Некоторые из этих приложений могут стать причиной некорректной настройки компонентов среды IDE в течение процесса установки.

Например, если в системе выполняется приложение монитора процессов Logitech (LVPrcSrv.exe), перед установкой его рекомендуется отключить.
