NetBeans IDE 6.1 情報

NetBeans IDE は、Java プログラミング言語で作成された、規格に準拠したモジュール方式統合開発環境です。NetBeans プロジェクトは、オープンソースの IDE と、あらゆる種類のアプリケーションを構築する汎用フレームワークとして使用できるアプリケーションプラットフォームから構成されます。

リリース概要

NetBeans IDE 6.1 release では、豊富な JavaScript 編集機能、Spring Web Framework の使用のサポート、MySQL との統合の強化、および依存プロジェクト間でのライブラリの共有方法の改善など、さまざまな新機能および拡張機能が提供されています。高く評価されている Ruby/JRuby のサポートは、新しいエディタ簡易修正、Ruby プラットフォームマネージャー、JRuby 高速デバッグのサポート、およびその他の多くの新機能と修正によって拡張されました。

多くの要望により、6.0 release で削除された Bean パターンと JSF CRUD 生成機能が復活しました。また、ClearCase サポートのような新しいモジュールの先行バージョンが、プラグインとして利用可能です。

このリリースでは、パフォーマンスの改善、特に起動時間の短縮 (最大 40%)、メモリ消費量の削減、および大きなプロジェクトを操作するときの応答速度の改善なども提供されています。このリリースの機能の一覧については、次を参照してください。

主な新機能

  • JavaScript のサポート
    • 意味解釈の強調表示
    • コード補完と型分析
    • 簡易修正と意味解釈のチェック
    • リファクタリング

  • パフォーマンスの強化
    • 最大で 40% 速い起動時間
    • 高性能の構文解析によるコード補完の高速化
    • メモリ消費量の削減

  • Spring Framework のサポート
    • 付属の Spring Framework 2.5 ライブラリ
    • XML 構成ファイルおよび Spring Web MVC コントローラーのウィザード
    • Bean 名、Java クラス、およびプロパティーのコード補完
    • Web プロジェクトでの Spring Web MVC Framework のサポート
    • さらに詳しく...

  • データベースエクスプローラーでの新規 MySQL サポート
    • MySQL サーバーの登録
    • データベースの表示、作成、および削除
    • MySQL 管理ツールの簡易起動

  • Java Beans のサポート
    • ナビゲータでの Beans パターン
    • Bean プロパティーの生成
    • BeanInfo エディタ

  • JSF CRUD ジェネレータ
    • エンティティークラスからの JavaServer Faces CRUD アプリケーションの生成
    • すべての種類のエンティティー関係をサポート (一対一、一対多、多対多)
    • すべての種類の主キーをサポート (単一列、複合、および生成済み)

  • Ruby/JRuby のサポート
    • エディタの拡張 (新しいヒントと修正を含む)
    • JRuby 高速デバッグのサポート
    • プラットフォームマネージャー
    • Rails プロジェクトでのサーバーとデータベース統合の改善

  • Javadoc のコード補完
    • 標準タグのサポート (@param など)
    • パラメータや例外などのコード補完

  • 一般的な Web API のサポート
    • マッシュアップアプリケーションの容易な作成
    • POJO、サーブレット、JSP、および RESTful Web サービスへのドラッグアンドドロップ操作。NetBeans IDE はサービスにアクセスするためのコードをすべて生成します
    • Google、Facebook、Yahoo、YouTube などの Web API のサポート

  • RESTful Web サービスのサポート
    • Jersey を使用した、JSR311 準拠の RESTful Web サービスの構築
    • 新しい REST テストクライアントを使用した、アプリケーションのテスト
    • ウィザードを使用して JPA エンティティークラスから REST サービスを作成し、WADL から JavaScript クライアントスタブを生成

  • 共有プロジェクト (共有可能ライブラリ)
    • 相対ライブラリパスを使用してライブラリの依存関係を指定 (デフォルトの Java プロジェクト、Web プロジェクト、およびすべて種類の Java EE プロジェクト)
    • バージョン管理システムを使用する場合など、より簡単なチームのメンバー間でのプロジェクトの共有
    • プロジェクトの自動構築の有効化

  • 新しいプラグイン
    • ClearCase バージョン管理
    • AXIS のサポート
    • SOAP UI のサポート
    • Hibernate Framework のサポート (さらに詳しく...)

  • Java Mobility
    • MacOS X 上での MIDP アプリケーション用 Mpowerplayer MIDP エミュレーター (プラグインセンターから入手可能)
    • SVC カスタムコンポーネント用の SVG コンポーザー
    • ドキュメントと持続性の改善

  • 6.1 以降の機能のプレビュー版
    • PHP のエディタサポート: コード補完、構文の強調表示、ナビゲーション、デバッグ、およびその他
    • JavaScript デバッガ



詳細は、NetBeans IDE 6.1 New & Noteworthy ページを参照してください。
