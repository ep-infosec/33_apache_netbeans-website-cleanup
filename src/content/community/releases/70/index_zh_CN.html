NetBeans IDE 7.0 发行信息

NetBeans IDE 是一个荣获奖项的集成开发环境，可用于 Windows、Mac、Linux 和 Solaris。NetBeans 项目由开源 IDE 和应用程序平台组成；通过该 IDE 和平台，开发者可以使用 Java 平台、PHP、JavaScript 和 Ajax、Groovy 和 Grails 以及 C/C++ 快速创建 Web、企业、桌面和 Mobile 应用程序。

NetBeans 项目由活跃的开发者社区支持，并且还提供了广泛的文档和培训资源，以及各种供您选用的第三方插件。

发行版概述

NetBeans IDE 7.0 为使用 JDK 7 开发者预览按照建议的 Java SE 7 规范进行开发提供了语言支持。该发行版还提供了与 Oracle WebLogic Server 的增强集成以及对 Oracle Database 和 GlassFish 3.1 的支持。其他重要功能包括 Maven 3 和 HTML5 编辑支持；用于改进 Swing GUI 开发的新 GridBagLayout 设计器以及 Java 编辑器增强功能等。

NetBeans IDE 7.0 提供了英语、巴西葡萄牙语、日语、俄语和简体中文版本。

功能集锦

JDK 7

  • Project Coin 支持
  • 编辑器增强功能：代码完成、提示

WebLogic Server

  • 更轻松快捷地部署到 WebLogic
  • 显示部署的应用程序和资源的新服务器运行时节点
  • JSF 与服务器库集成在一起

Oracle Database

  • 简化的连接向导
  • JDBC 驱动程序的引导安装
  • 编辑和部署存储过程




GlassFish

  • GlassFish 3.1 支持
  • 用于远程 GlassFish 的域重新启动和日志查看器
  • 启用和禁用部署的应用程序

Java

  • Maven 3 支持
  • JUnit 4.8.2 集成和各种 JUnit 增强功能
  • 库和 Java 平台中的 Javadoc 的远程 HTTP URL 支持
  • 改进的新 GridBagLayout 可视定制器

Java EE

  • 改进的 CDI、REST 服务和 Java 持久性支持
  • 新的 Bean 验证支持
  • JSF 组件库支持，包括捆绑的 PrimeFaces 库
  • JSF 中的改进表达式语言编辑功能，包括代码完成、重构和提示





Web 语言

  • HTML5 编辑支持
  • JSON 格式化程序

PHP

  • 生成 PhpDoc
  • 重命名重构、安全删除重构
  • PHP 5.3 - 别名支持

C/C++

  • 从用户的现有二进制文件中轻松导入项目
  • 新的项目类型，其中，用户的源文件位于远程系统上

NetBeans 平台

  • 用于在层中生成操作注册的标注
  • 性能增强以及与 Profiler 紧密集成在一起
  • 其他 NetBeans API 更改

常规

  • 编辑器中的文字换行
  • 增强的 Profiler 集成
  • 在 IDE 和其他程序之间切换时，降低了外部更改检查的影响。
