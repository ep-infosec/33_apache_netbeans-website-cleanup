Informações sobre o NetBeans IDE Versão 7.1

O NetBeans IDE é um premiado ambiente de desenvolvimento integrado disponível para Windows, Mac, Linux e Solaris. O projeto NetBeans consiste em um IDE de código aberto e uma plataforma de aplicativos que permite aos desenvolvedores criar rapidamente aplicativos web, empresariais, móveis e de área de trabalho utilizando a plataforma Java, bem como PHP, JavaScript e Ajax, Groovy e Grails, e C/C++.

O projeto NetBeans tem como suporte uma entusiasmada comunidade de desenvolvedorese oferece vastos recursos como documentação e treinamento, bem como uma variada seleção de plug-ins de terceiros.

Visão geral da versão

O NetBeans IDE 7.1 introduz o suporte a JavaFX 2.0 ativando todo o ciclo de desenvolvimento de compilação/depuração/perfil para aplicativos JavaFX 2.0. Esta versão também oferece aprimoramentos significativos ao GUI Builder, suporte a CSS3 e ferramentas para a depuração visual das interfaces de usuário Swing e JavaFX. Os destaques adicionais incluem o suporte integrado do Git com o IDE, novos recursos de depuração PHP, diversos aprimoramentos JavaEE e Maven e mais.

O NetBeans IDE 7.1 está disponível em inglês, português do Brasil, japonês, russo e chinês simplificado.

Novos recursos em destaque

JavaFX

  • Suporte a JavaFX 2.0
  • Suporte a todos os três modelos de implementação: Área de trabalho, Applet, JNLP
  • Pré-carregadores para melhorar o carregamento do aplicativo
  • Controles de IU personalizados usando CSS3

Java

  • Depurador Visual novo
  • Suporte a Gap no personalizador de GridBagLayout
  • Inspeção e refatoração
  • Destaque e formatação de Javadoc aprimorados

Linguagens da web

  • Suporte para CSS3
  • Preenchimento automático de código e documentação para novos elementos do CSS3
  • Propriedades específicas do navegador

Java EE

  • Suporte a implementação em cluster e de instância para o GlassFish
  • Desenvolvendo aplicativos Java EE 6 para implementação no servidor WebLogic
  • Mais de 50 melhorias de CDI no editor Java
  • Suporte ao conjunto de componentes do JSF
  • Melhoria em persistência de Java, serviços web, EJB, WebLogic e mais

PHP

  • Depuração de PHP aprimorado
  • Recursos de grupos de teste do PHPUnit
  • Suporte para modelos Smarty
  • Uploads mais rápidos com o recurso de manter ativo para cliente (S)FTP

Plataforma NetBeans

  • Novas APIs do sistema de janelas
  • Comportamento aprimorado do TopComponentGroup
  • Designer de layout visual de janelas
  • Componentes de multivisualização conectáveis

Controle de versão

  • Suporte integrado ao Git
  • Guia Histórico na janela do editor de arquivos
  • Bloqueio/desbloqueio de arquivos no diretório de trabalho do Subversion
  • Suporte para ramificações e marcações do Mercurial
