Release 6.5

Released on November 20th, 2008

The 6.5 release provides enhanced support for web frameworks (Hibernate, Spring, JSF, JPA), the GlassFish application server, and databases. Additionally it includes a new IDE-wide QuickSearch shortcut, a more user-friendly interface, and automatic Compile on Save.

  • General Info about NetBeans IDE 6.5
  • Download NetBeans IDE 6.5

Release 6.1

Released on April 24th, 2008

The 6.1 release provides several new features and enhancements, such as rich JavaScript editing features, support for using the Spring web framework, tighter MySQL integration, Support for Ruby/JRuby was enhanced with new editor quick fixes, a Ruby platform manager, fast debug support for JRuby and others...

  • General Info about NetBeans IDE 6.1
  • Download NetBeans IDE 6.1

Release 6.0

Released on December 3rd, 2007

The focus of NetBeans IDE 6.0 is improved developer productivity through a smarter, faster editor, and the integration of all NetBeans products into one IDE.

Release 5.5.1

Released on April 25th, 2007

The 5.5.1 update is a bug fix release for the core. It includes a couple of new features and enhancements for the C/C++ pack and the Enterprise pack.

  • General Info about NetBeans IDE 5.5.1

Release 5.5

Released on October 30th, 2006

NetBeans 5.5 supports the Java Enterprise Edition 5 platform, most notably Java Persistence, the Sun Java System Application Server PE 9, EJB 3 and JAX-WS 2.0 specifications. A new Subversion module and a GUI builder update are both available on the 5.5 Update Center.

  • General Info about NetBeans IDE 5.5
  • Download NetBeans IDE 5.5

Release 5.0

Released on January 31st, 2006
NetBeans 5.0 introduced plug-in developer support, the intuitive GUI builder Matisse, redesigned CVS support, Sun Application Server 8.2, Weblogic9 and JBoss 4 support, and many editor enhancements.
