NetBeans IDE 7.2リリース情報

NetBeans IDEは、受賞歴のある統合開発環境で、Windows、Mac、LinuxおよびSolarisで使用可能です。NetBeansプロジェクトは、オープン・ソースのIDEとアプリケーション・プラットフォームから構成され、これを使用すると、開発者はJavaプラットフォームの他、PHP、JavaScript、Ajax、GroovyとGrails、およびC/C++を使用して、Webアプリケーション、エンタープライズ・アプリケーション、デスクトップ・アプリケーションおよびモバイル・アプリケーションを短時間で作成できます。

NetBeansプロジェクトは活発な開発者コミュニティによってサポートされており、豊富なドキュメントとトレーニング・リソース、および選択可能な様々なサードパーティのプラグインが提供されています。

リリース概要

NetBeans IDE 7.2では、Javaエディタの新しい静的コード分析機能とさらに進化したプロジェクト・スキャンにより、パフォーマンスとコーディング機能が大きく向上しました。このリリースには、JavaFXフォームを視覚的に作成するためのScene Builderとの統合、複数のPHPフレームワークのサポート、Groovyサポートの更新、そしてJava EE、Maven、C/C++およびNetBeansプラットフォームにおけるその他数多くの機能拡張などの特徴も含まれています。

NetBeans IDE 7.2は、英語、ポルトガル語(ブラジル)、日本語、ロシア語および簡体字中国語で使用できます。

新機能の概要

パフォーマンス

  • プロジェクト・スキャンがバックグラウンドで行われるようになったため、エディタの機能やナビゲーションがブロックされない
  • プロジェクト・スキャン速度の向上
  • リモート・ファイルシステムでのパフォーマンスの向上

Java

  • FindBugsのサポート
  • TestNGテスト・フレームワークのサポート
  • 新しいリファクタリングとヒント
  • 様々なMavenサポート拡張機能
  • 様々なデバッガ拡張機能

Java EE

  • Amazon Elastic Beanstalkのサポート
  • Java Persistence名前付き問合せとJPQL文のコード補完
  • EJBおよびCDIでのエディタ・ヒントの充実
  • JSFコンポーネント・スイートのサポートの向上および更新

JavaFX

  • Scene Builderの相互運用性
  • FXMLおよびFX CSSサポートの向上
  • FX-in-Swingプロジェクト
  • 即時の「WebStartとして実行」および「ブラウザで実行」のためのデフォルト構成
  • プロジェクト自動更新と、向上したFX SDK/RT自動検出

PHP

  • PHP 5.4のサポート
  • PHP編集: 「使用の修正」アクション、注釈サポート、NeonおよびApache構成ファイルの編集、その他
  • Symfony2、Doctrine2およびApiGenフレームワークのサポート
  • FTPリモート同期
  • HudsonでのPHPプロジェクトの実行のサポート

GroovyとGrails

  • スキャン、コード補完および「型に移動」アクションの速度の大幅な改善
  • Groovyバージョン1.8.6への更新
  • コード補完およびエディタ・ヒントの機能向上

C/C++

  • 新しいC++11標準のサポート
  • gdbのプリティ・プリンティング、ウォッチポイントおよびキャッチポイントのサポート
  • 部分的および逐次の使用状況検索
  • 解析パフォーマンスを向上し、メモリー消費を削減

NetBeansプラットフォーム

  • ウィンドウ・システムでのタブの配置を構成可能
  • タブ制御のカスタム実装のサポート
  • ナビゲータ・パネルを宣言登録するための新しい@NavigatorPanel.Registration
  • コマンドライン・オプションを宣言登録するための新しい@Arg注釈
  • 新しい@NbBundle.Messagesをフィールドで利用可能

その他

  • Subversion 1.7組込みサポート
  • 「プロジェクト内を検索」機能の向上
  • 新しくなったユーザー・ディレクトリとキャッシュ・ディレクトリのデフォルトの場所
  • Windows用の64ビット版起動ツール

このリリースの機能の詳細は、NetBeans IDE 7.2の新機能と注目すべき機能のページを参照してください。
