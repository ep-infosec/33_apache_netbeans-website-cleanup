NetBeans IDE 7.2 发行信息

NetBeans IDE 是一个荣获奖项的集成开发环境，可用于 Windows、Mac、Linux 和 Solaris。NetBeans 项目由开源 IDE 和应用程序平台组成；通过该 IDE 和平台，开发者可以使用 Java 平台以及 PHP、JavaScript、Ajax、Groovy 和 Grails、C/C++ 快速创建 Web、企业、桌面和移动应用程序。

NetBeans 项目由活跃的开发者社区支持，并且还提供了广泛的文档和培训资源，以及各种供您选用的第三方插件。

发行版概述

NetBeans IDE 7.2 提供了显著改进的性能和编码体验，其中包含 Java 编辑器中的新增静态代码分析功能以及更加智能的项目扫描。该发行版还包含一些重要功能，例如与场景构建器集成以便以可视方式创建 JavaFX 窗体；支持多个 PHP 框架；更新的 Groovy 支持以及 Java EE、Maven、C/C++ 和 NetBeans 平台中的许多其他增强功能。

NetBeans IDE 7.2 提供了英语、巴西葡萄牙语、日语、俄语和简体中文版本。

新增功能集锦

性能

  • 现在项目扫描在后台执行，不会妨碍编辑器功能和导航
  • 提高了项目扫描速度
  • 改进了远程文件系统的性能

Java

  • FindBugs 支持
  • TestNG 测试框架支持
  • 新增了重构和提示
  • 各种 Maven 支持增强功能
  • 各种调试器增强功能

Java EE

  • Amazon Elastic Beanstalk 支持
  • 用于 Java 持久性命名查询和 JPQL 语句的代码完成功能
  • EJB 和 CDI 的更多编辑器提示
  • 针对 JSF 组件套件的改进和更新支持

JavaFX

  • 场景构建器互操作性
  • 改进了 FXML 和 FX CSS 支持
  • FX-in-Swing 项目
  • 即时 "Run as WebStart"（以 WebStart 方式运行）和 "Run in Browser"（在浏览器中运行）的默认配置
  • 项目自动更新，改进了 FX SDK/RT 自动检测功能

PHP

  • 支持 PHP 5.4
  • PHP 编辑："Fix Uses"（修复用法）操作、标注支持、Neon 和 Apache 配置文件编辑等等
  • 支持 Symfony2、Doctrine2 和 ApiGen 框架
  • FTP 远程同步
  • 支持在 Hudson 上运行 PHP 项目

Groovy 和 Grails

  • 极大地提高了扫描、代码完成和 "Go to Type"（转至类型）操作的速度
  • 已更新到 Groovy 版本 1.8.6
  • 代码完成和编辑器提示的增强功能

C/C++

  • 支持新的 C++11 标准
  • 对 gdb 的美化、监测点和捕获点支持
  • 部分和增量查找用例
  • 提高了解析性能并减少了内存消耗

NetBeans 平台

  • 可配置窗口系统中的标签定位
  • 支持标签控件的定制实现
  • 新 @NavigatorPanel.Registration，可用于以声明方式注册导航器面板
  • 新 @Arg 标注，可用于以声明方式注册命令行选项
  • 可用于字段的新 @NbBundle.Messages

其他

  • Subversion 1.7 内置支持
  • 改进了 "Find in Projects"（在项目中查找）功能
  • userdir 和 cachedir 的新默认位置
  • 适用于 Windows 的 64 位启动程序

有关此发行版本功能的详细信息，请参见 NetBeans IDE 7.2 新增功能和重要功能页。
