Informações da Release do NetBeans IDE 7.2

O NetBeans IDE é um premiado ambiente de desenvolvimento integrado disponível para Windows, Mac, Linux e Solaris. O projeto NetBeans consiste em um IDE de código aberto e em uma plataforma de aplicações que permite aos desenvolvedores criar rapidamente aplicações web, aplicações corporativas, aplicações móveis e aplicações de área de trabalho utilizando a plataforma Java, bem como PHP, JavaScript, Ajax, Groovy e Grails e C/C++.

O projeto NetBeans tem como suporte uma entusiasmada comunidade de desenvolvedorese oferece vastos recursos como documentação e treinamento, bem como uma variada seleção de plug-ins de terceiros.

Visão Geral da Release

O Netbeans IDE 7.2 oferece um melhor desempenho e experiência de codificação, com novos recursos de análise de código estático no Editor de Java e varredura do projeto mais inteligente. A release também inclui recursos notáveis como integração com Scene Builder para criar forms do JavaFX visualmente; suporte para vários frameworks PHP; atualizado suporte Groovy; e muitas outras melhorias em Java EE, Maven, C/C ++ e Plataforma NetBeans.

O NetBeans IDE 7.2 está disponível em inglês, português do Brasil, japonês, russo e chinês simplificado.

Novas Funcionalidades em Destaque

Desempenho

  • Varredura do projeto agora em segundo plano, não bloqueando recursos do editor e navegação
  • Velocidade de varredura do projeto melhorada
  • Desempenho melhorado em sistemas de arquivos remotos

Java

  • Suporte na Localização de Bugs
  • TestNG - suporte de framework de teste
  • Novas refatorações e dicas
  • Várias melhorias de suporte de Maven
  • Várias melhorias do Depurador

Java EE

  • Suporte do Amazon Elastic Beanstalk
  • Autocompletar código para consultas com nome de Java Persistence e instruções JPQL
  • Mais dicas do editor para EJB e CDI
  • Suporte melhorado e atualizado para suites do componente JSF

JavaFX

  • Interoperabilidade do Scene Builder
  • Suporte CSS de FXML e FX melhorado
  • Projetos FX no Swing
  • Configurações default para Executar como WebStart e Executar no Browser instantâneo
  • Atualização automática do projeto e detecção automática de SDK/RT de FX melhorado

PHP

  • Suporte para PHP 5.4
  • Edição de PHP: corrige ação de Usos, suporte a anotações, edição de arquivos de Configuração Neon e Apache e muito mais
  • Suporte para frameworks Symfony2, Doctrine2 e ApiGen
  • Sincronização remota para FTP
  • Suporte para projetos PHP em execução no Hudson

Groovy e Grails

  • Melhoria significante de varredura, autocompletar código e velocidade de ação Ir para Tipo
  • Atualizado para Groovy versão 1.8.6
  • Melhorias de dicas do editor e de autocompletar código

C/C++

  • Novo suporte de C++11 padrão
  • Suporte para pretty printing, watchpoints e catchpoints para gdb
  • Localização de Usos Parcial e Incremental
  • Desempenho de parsing melhorado e consumo de memória reduzido

Plataforma NetBeans

  • O posicionamento de guias no sistema de janelas pode ser configurado
  • Suporte para implementações personalizadas do controle de guias
  • Novo @NavigatorPanel.Registration para registrar painéis do navegador declarativamente
  • Nova anotação @Arg declarativamente para registrar as opções da linha de comandos
  • Novo @nbbundle.messages nos campos disponíveis

Diversos

  • Suporte do Subversion 1.7 incorporado
  • Funcionalidade Localizar nos Projetos melhorada
  • Novas localizações default para userdir e cachedir
  • Acionador de 64-bits para Windows
