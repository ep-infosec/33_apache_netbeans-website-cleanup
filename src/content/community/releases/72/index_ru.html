Сведения о версии IDE NetBeans 7.2

IDE NetBeans представляет собой завоевавшую награды интегрированную среду разработки. Доступны версии среды для Windows, Mac, Linux и Solaris. Проект NetBeans состоит из интегрированной среды разработки с открытым кодом и платформы приложений, которые позволяют разработчикам быстро создавать корпоративные, настольные, мобильные и веб-приложения с использованием платформы Java на языках PHP, JavaScript, Ajax, Groovy, Grails и C/C++.

Проект NetBeans поддерживается активным сообществом разработчиков, которое предоставляет подробную документацию и обучающие материалы, а также большой выбор разнообразных сторонних подключаемых модулей.

Обзор выпуска

IDE NetBeans 7.2 обеспечивает значительное улучшение производительности и процесса создания кода благодаря новым возможностям статического анализа кода в редакторе Java и более интеллектуальному сканированию проектов. Выпуск также включает в себя достойные внимания функциональные возможности, такие как интеграция с Scene Builder для визуального создания форм JavaFX, поддержка нескольких платформ PHP, обновленная поддержка Groovy и множество других усовершенствований Java EE, Maven, C/C++ и платформы NetBeans.

IDE NetBeans 7.2 доступна на английском, бразильском португальском, японском, русском и упрощенном китайском языках.

Обзор новых функциональных возможностей

Производительность

  • Фоновое сканирование проектов без блокирования функций редактора и навигации
  • Ускоренное сканирование проектов
  • Улучшенная производительность в удаленных файловых системах

Java

  • Поддержка FindBugs
  • Поддержка среды тестирования TestNG
  • Новые способы реорганизации кода и подсказки
  • Различные усовершенствования поддержки Maven
  • Различные усовершенствования отладчика

Java EE

  • поддержка Amazon Elastic Beanstalk
  • Дополнение кода для именованных запросов Java Persistence и операторов JPQL
  • Дополнительные подсказки в редакторе для EJB и CDI
  • Улучшенная и обновленная поддержка для комплектов компонентов JSF

JavaFX

  • Возможность взаимодействия с Scene Builder
  • Улучшенная поддержка FXML и FX CSS
  • Проекты с поддержкой FX-in-Swing
  • Конфигурации по умолчанию для немедленного выполнения команд Запуск от имени WebStart и Запуск в браузере
  • Автоматическое обновление проектов и улучшенное автоматическое обнаружение FX SDK/RT

PHP

  • Поддержка PHP 5.4
  • Редактирование PHP: команда исправления использований, поддержка аннотаций, редактирование файлов конфигураций Neon и Apache и многое другое
  • Поддержка сред Symfony2, Doctrine2 и ApiGen
  • Удаленная синхронизация с FTP
  • Поддержка реализации проектов PHP в среде Hudson

Groovy и Grails

  • Значительное ускорение сканирования, завершения кода и выполнения команды Переход к типу
  • Обновление до версии Groovy 1.8.6
  • Усовершенствования завершения кода и подсказок редактора

C/C++

  • Поддержка нового стандарта C++11
  • Поддержка структурной распечатки, точек отслеживания и точек захвата для GDB
  • Частичный и последовательный поиск случаев использования
  • Повышение производительности анализа и уменьшение объема потребляемой памяти

Платформа NetBeans

  • Возможность настройки расположения вкладок в системе окна
  • Поддержка нестандартных реализаций элемента управления вкладками
  • @NavigatorPanel.Registration - новая возможность для декларативной регистрации панелей навигатора
  • @Arg - новая аннотация для декларативной регистрации параметров командной строки
  • @NbBundle.Messages - новая возможность, доступная для полей

Прочее

  • Встроенная поддержка Subversion 1.7
  • Улучшенная функциональность поиска в проектах
  • Новые местоположения по умолчанию для userdir и cachedir
  • Средство запуска 64-разрядной версии для Windows
