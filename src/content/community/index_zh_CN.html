什么是 NetBeans？

NetBeans 是一个开源项目，拥有广泛的用户基础和成长中的社区，以及全球100多个（仍不断增加）合作伙伴。Sun Microsystems 于2000年6月创建了 NetBeans 开源项目并且一直以来充当该项目的主要赞助商。

目前有两款产品：NetBeans IDE 和 NetBeans Platform。

NetBeans IDE 是一个开发环境 - 是程序员撰写、编译、测试、调试、性能分析和部署程序的工具。这个 IDE 是用 Java 编写的，但却可以支持任何编程语言。另外还有许多扩展 NetBeans IDE 功能的模块。NetBeans IDE 一个免费的产品，而且没有任何使用限制。

NetBeans Platform 是一个模块化的且可扩展的基础，用作创建大型桌面应用程序的软件基石。ISV 合作伙伴提供增值的插件，它们很容易集成到 IDE 中，并且也可用以开发其本身的工具和解决方案。

两款产品都是开源的，可免费用于商用和非商用目的。源代码在遵循 Common Development and Distribution License (CDDL) 的前提下可以重新再利用。

什么是 netbeans.org？

netbeans.org 是 Netbeans 的官方网站，NetBeans 开源社区的家 - 在这里的人们都致力于建立一个世界级的 IDE。netbeans.org 允许来自全球 160 多个国家的用户连接到 NetBeans 周围的资源和人们。在这里，您可以下载 NetBeans 的最新版本，浏览在线支持文档，强化您的 Java 知识，跟踪最新报道，加入邮件列表，贡献代码，了解涉及同一个项目的人们，建立联系，等等。

NetBeans 中文社区

为了更方便中文用户直接获取 NetBeans 的相关信息，支持文档，最新下载等，为了更多的中文用户能够加入到 NetBeans 这个大家庭里面并且做出自己的贡献，从而扩大 NetBeans 中文社区的规模，我们特别建起这个中文的门户网站 zh-cn.netbeans.org，欢迎大家的访问并期待更多的支持！

加入 NetBeans 社区

成为 NetBeans 团队的一部分，使 NetBeans 的明天更美好。通过注册，您将直接为 NetBeans 的成功做出了贡献，我们也会及时通知您关于 NetBeans 的最新的消息。请在 NetBeans 的官方网站免费注册。

为 NetBeans 做贡献

NetBeans 是一个开源项目，随时欢迎您的参与和支持！有许多为其做贡献的方式：在邮件列表里回答用户的问题，本地化 IDE 或翻译网站内容，帮助测试，贡献代码使 NetBeans IDE 变的更强！详情请见如何为 NetBeans 做贡献。
