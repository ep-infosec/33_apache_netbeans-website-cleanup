NetBeans User Groups -- Near You

You would like to see more NetBeans-related talks and have a chat with other Developers who speak your language? Start or join a local JUG! Check the current list on the NetBeans User Group wiki.

You know of a Java or NetBeans User group that is not listed here? Add it to the wiki or send us more details and we will list them here, so fellow community members can join!

If there is no NetBeans User group for your area, you can join a Java User Group, and find people there with whom you can found a NetBeans User Group.
