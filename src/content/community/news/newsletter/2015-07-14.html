Tutorials

Getting Started with HTML5 Applications

This tutorial teaches you how to create JavaScript/HTML5 projects in NetBeans IDE and some of the features in the IDE that support the use of JavaScript and CSS in your project. In this tutorial you will create a simple HTML5 application that uses the jQuery JavaScript library to modify a list in a web page.

Working with CSS Style Sheets in an HTML5 Application

HTML5 applications typically combine HTML, CSS and JavaScript to create applications that are run in a browser and that are displayed on a variety of devices, including smartphones, tablets and laptops. This tutorial teaches you how NetBeans IDE provides tools that can help you work with CSS rules to modify the layout of an application. The IDE also provides support for creating and using Sass and LESS CSS preprocessors in your application.

Debugging and Testing JavaScript in an HTML5 Application

In this tutorial, you learn how the IDE provides tools that can help you debug and test JavaScript files in the IDE, using the free NetBeans Chrome Connector plugin, as well as the Jasmine testing framework and the Karma or the JS Test Driver test runners.
