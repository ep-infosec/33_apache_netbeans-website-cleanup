Patch2 para o NetBeans IDE 6.9 já está disponível

A equipe do NetBeans disponibilizou um novo patch, que é uma atualização para o NetBeans IDE 6.9.1. O pacote inclui correções de bugs nos módulos para C/C++, Plataforma do IDE, Java, Depurador Java, Aplicações Web Java, Maven, Mobilidade, OSGI, PHP e Plataforma RCP.

Para obter as correções, o NetBeans IDE deve estar instalado e funcionando. Você também pode baixar as correções através do Gerenciador de plugin do IDE.
