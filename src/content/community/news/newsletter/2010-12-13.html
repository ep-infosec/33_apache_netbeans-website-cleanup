Issue # 466 - Dec 13, 2010

Project News

Cobertura Support for Maven Projects in NetBeans IDE

NetBeans IDE now supports code coverage for Maven projects through the Cobertura Maven plugin. Learn how to display test coverage in the IDE, and for NetBeans RCP applications on Maven.

Improvements to NetBeans 7.0 PHP Editor Performance

The NetBeans PHP team has been implementing fixes to performance issues in the PHP editor for NetBeans 7.0. (Download the nightly builds to get the changes.) To continue to improve the editor, the team wants to receive more performance use cases. Read the team blog for details.

Plugin: Gmail Notify

This NetBeans plugin enables notification in the IDE when new Gmail messages become available.

Articles

Interview: Caoyuan Deng on the NetBeans Scala Plugin

Developer Caoyuan Deng speaks about the origin of the Scala plugin for NetBeans, the current status and future of the project.

Swiss Air Traffic Control Takes Off on the NetBeans Platform

Last year, SkyGuide, a Swiss aviation software company went through the NetBeans Platform training course and ported several of their applications to the NetBeans Platform. In this follow-up article, software engineer Julien Piaser introduces Crystal, a NetBeans Platform application that manages air traffic information.

Community

The Spam Stops Here! Interview with Holger Stenzhorn

Holger Stenzhorn, a NetBeans community moderator, gives an account of his spam-fighting days on the NetBeans mailing lists.

New Version of Kojo (Scala Learning Environment) Available

Created in Scala on the NetBeans Platform, Kojo is a learning environment that teaches children the basics of programming and helps them to explore concepts in Math, Sciences and the Arts. Learn more about new features in the latest release of this impressive open-source project.

Tutorials

JDK 7 Support in NetBeans IDE 7.0 Tutorial

A short guide to using JDK 7 features in NetBeans IDE 7.0. Also includes information on setting up the JDK.

Devoxx Video: Creating Lightweight Apps with Vanilla Java EE 6

Revisit JavaEE expert Adam Bien's Devoxx talk, now in in video, and a related blog entry.

Using the Embedded EJB Container to Test Enterprise Applications

A tutorial on how using EJB 3.0 technology in the NetBeans IDE to develop an enterprise application.
