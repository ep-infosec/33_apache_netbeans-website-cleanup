项目新闻

DTrace GUI 1.1 现已可以获得

DTrace GUI 插件的新版本 (1.1版) 可以在 NetBeans 6.8 升级中心获得.当前版本的 DTrace GUI 显示了七个 Chime ，它们可以被 PHP 开发者和用户用来调试和调节 PHP 程序. 你可以通过 工具 > 插件 > 可用插件 > DTrace 来安装该插件.

新的 Java Card 插件 Beta 测试!

Java Card 是用来开发在 Smart Cards 上面运行 Java 程序的一个平台. NetBeans 中新的 Java Card 插件可以在 开发版本中获得, 我们希望可以得到来自真正用户的反馈 ! 请帮助我们来测试这个 beta 版本的插件并告诉我们你的想法.
