Project News

NetBeans IDE 7.0 Testers Wanted - Register for NetCAT 7.0!

Do you enjoy playing with software, hunting for defects? If you are an experienced NetBeans IDE user and you have a few hours per week from November through February, you can join other community volunteers in the NetCAT 7.0 program. Get your NetBeans bug(s) fixed and make friends along the way. Register for NetCAT 7.0!


Plugin: PHP Nette Framework

Ondrej Brejla's PHP Nette Framework plugin provides support for the Nette Framework, a powerful PHP framework for rapid and easy creation of high quality and innovative web applications. The plugin provides built-in support for the "Latte" template language with syntax highlighting and code completion.

Training

NetBeans Platform at Oracle Technology Network Developer Day Vienna, Nov. 11

Get an introduction to the NetBeans Platform at Oracle's Developer Day in Vienna, Austria on November 11. There will be a session to explore concepts behind the NetBeans Platform and a hands-on lab to get experience creating modular Swing applications on the NetBeans Platform.

NetBeans Platform Lab at Devoxx 2010, Nov. 15

Join Geertjan Wielenga and Anton Epple for a NetBeans Platform hands-on lab at Devoxx 2010. The lab will provide a comprehensive overview of the NetBeans Platform, how it works, and how to get started using it.
