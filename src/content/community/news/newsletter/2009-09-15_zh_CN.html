和 Danny Coward 一起看 JDK 7 的新功能

在这个 Java 技术视频中，Danny Coward，Sun Microsystems 的客户端软件主架构师，重点介绍了 JDK 7 中的一些新功能。你将看到演示和样例代码——他还会讲到，作为一个本地交响乐团的协奏小提琴家，为什么是一种极好的逃避。

Java EE 6 和 NetBeans IDE 6.8

Geertjan 的博客分享了关于在即将到来的 NetBeans IDE 6.8 中的 Java EE 6 的最新新闻。主要的重点在于“Web 应用程序”项目类型上，因为 EJB 现在也可以直接在 Web 应用程序中创建了。如果你选择了 JSF 框架，你可以可以选择 Facelets（缺省）或 JSP 作为页面语言。请到 Geertjan 的博客去了解更多。

直接在 IDE 中获得新的样例项目

你知道可以从“新建项目”菜单打开样例项目——但你知道实际上还可以将更多样例直接载入到 IDE 吗？NetBeans 文档组已经完全更新了在线样例类别：所有的样例现在都由 kenai.com 代管。在线浏览列表，下载 zip 压缩的项目——或者使用 NetBeans Kenai 集成直接在 IDE 中打开它们！

技术提示：定制代码折叠
