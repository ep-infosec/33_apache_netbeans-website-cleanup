Java SE 8：在 HTTP 链接检查器中使用线程

本教程向你展示如何使用 Java 平台标准版 8（Java SE 8）和 NetBeans 8 以 Thread 类创建一个链接检查器。对于小应用程序，你可以根据 Thread 对象的定义，用其定义的 Runnable 接口或其本身添加一个线程。
