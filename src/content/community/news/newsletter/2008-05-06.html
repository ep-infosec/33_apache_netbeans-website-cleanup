Issue # 340 - May 06, 2008

Project News

MySQL-GlassFish Bundle Now Available for NetBeans 6.1 IDE!

The NetBeans IDE 6.1 with MySQL-GlassFish bundle is available for download. The bundle consists of GlassFish V2 Update 2 and the MySQL Community Server 5.0, and is supported on Solaris, Linux, Windows and Mac OS X.


Sun & NetBeans Announce Launch of NetBeans IDE Early Access for PHP

NetBeans IDE Early Access for PHP includes intelligent editing features such as prioritized code completion, instant rename, mark occurrences, dynamic code templates and easy navigation. Get an easy start to PHP development with features such as debugging support, deployment to the local server, dynamic help and sample projects. Learn more about the Early Access PHP plugin.

Check Out the Java SE 6 Update 10 Beta

Java SE 6 Update 10 Beta is an update release that introduces new features and enhancements aimed at improving the end-user experience. The release has significant performance and footprint improvements including the new JavaKernel. The Java SE team wants to make NetBeans IDE users aware of these Java improvements and is interested in feedback. Learn more about Java SE 6 update 10 Beta.
