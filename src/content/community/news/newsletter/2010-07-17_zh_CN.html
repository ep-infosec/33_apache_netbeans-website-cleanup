文章

NetBeans 平台上的鞋设计计费

在这次采访中，Dave Irving 讲了 iCadSuite：该程序从鞋设计系统中获取平整化的 2D 几何图形，再算出要生产一双这样的鞋所需的费用！改程序使用 NetBeans 平台作为框架。

UberConf：OSGi、Java EE 和 Clojure

Arun Gupta 在 UberConf 上主持了一个会议，是关于将 GlassFish 作为用于开发 OSGi、Java EE 和混合应用程序的简易执行环境（看看他的幻灯片）。他还分享了从使用 NetBeans IDE 的 Clojure 插件中所学到的东西。

博客

NetBeans 6.9 性能分析器：增强的堆遍历器

试试在 NetBeans 6.9 性能分析器中增强的堆遍历器：它让用户能更容易地将 Java 帧垃圾回收根节点匹配到相应的本地变量。

使用 NetBeans、MySQL 和 YUI 图标跟踪你的跑步里程

Arun Gupta 正在做一个程序以跟踪他的跑步里程。想法是使用不同的 Web 框架构建相同的应用程序，将它们部署到 GlassFish，再进行比较！这次他使用 Apache Wicket、GlassFish、NetBeans、MySQL 和 YUI Charts 来监控进度。

用于 PHP 项目的“立即分析性能！”

你知道在 NetBeans 6.9 中，“立即分析性能！”操作也能用于 PHP 项目了吗？该工具栏按钮让你生成一份 CPU 性能快照以让你明白在你认为 IDE 慢的时候，发生了什么事。它将帮助你分析性能瓶颈，在你发送错误报告时，请附上该快照。

对于想实现一个 NetBeans 插件以在 NetBeans IDE 中为他们最喜爱的 PHP 框架添加支持的开发人员来说，该页面是一个起点。（需要 Java 知识。）

Metro 2.0.1 在此

新发布的 Metro 2.0.1 含有 Jax-WS (RI) 2.2.1。它已被集成到 GlassFish 3.0.1，但也能和诸如 Tomcat 的其他容器一起工作。来自水族馆博客的 Alexis Moussine-Pouchkine 说：“当前最佳的开发人员体验极可能是使用 NetBeans 6.9。”

NetBeans 电子商务教程

在这个多部分的教程中，你讲学习如何创建简单而强大的电子商务应用程序，它展示了 Java Web 和 EE 开发的各种重要功能。这样下来，你将习惯于 NetBeans IDE，并能够将它应用到自己的开发目的中。

NetBeans 平台上的瓦加杜古

Geertjan Wielenga 举行了又一场 NetBeans 平台培训，这次是在布基纳法索的瓦加杜古——通过 Skype 和 WebEx！所以无论你在哪儿，只要想在 2、3 或 5 天内上手并在 NetBeans 平台上跑起来，请立即发邮件给 users at edu dot netbeans dot org 以了解详情。对教育机构是免费的！

Dobbs 博士：JavaFX JDBC 数据源

本文通过一系列数据源组件，向你介绍 JavaFX 对 JDBC 和各种数据格式的内建支持。多亏有了 NetBeans JavaFX 创作器，你不必编写大量的数据处理代码，就能做成很多事。
