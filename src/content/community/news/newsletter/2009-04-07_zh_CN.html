NetBeans 向导是如何工作的？

一致的对话框布局是 NetBeans 平台的承诺。一种方式是通过向导 API（Wizard API），它使得你的应用有一个标准的外观。这份教程描述了如何使用此 API。

在线讨论会：MySQL, NetBeans 以及 Java RESTful Web 服务

本在线讨论会展示了如何使用最近版本的 NetBeans IDE 进行 RESTful Web 服务的开发，其客户端部署于 GlassFish 与 MySQL 上。讨论如何使用 JAX-RS 开发了一个简单 RESTful Pet Catalog 服务的过程，以及如何构建两个此服务的示例客户端，一个使用 Dojo AJAX 框架，另一个使用 JavaFX。
