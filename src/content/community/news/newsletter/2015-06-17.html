Plugin

HTML5 Fun Pack

Did you know that HTML5 now includes attributes for validation and error handling? Did you know you can build code completion features into your website via HTML elements? HTML is richer than you might think, thanks to HTML5. Use this new plugin to populate your HTML Component Palette in the IDE with new elements and attributes and get started learning about the enhancements in HTML via NetBeans.

New Location of the NetBeans Module Maven Plugin

Users of the NetBeans Maven plugin should take note of its new location, https://github.com/mojohaus/nbm-maven-plugin, from where also the related documentation will be found.

MVC with NetBeans Platform and JavaFX

In this article by Rob Terpilowski, in Seattle, you will learn about the creation of a small application based on the NetBeans Platform and JavaFx, which will monitor the prices of stocks and update the UI in real time as prices are updated.

Extending the FindBugs Architecture

FindBugs is a key code quality tool for Java based projects. Gualtiero Testa discusses various FindBugs plugins and how you can set them up in NetBeans IDE.
