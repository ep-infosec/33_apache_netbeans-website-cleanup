Nouvelles de projet

NetBeans 6.5 est maintenant disponible en téléchargement!

Netbeans.org est fier d'annoncer la disponibilité de l'EDI 6.5 !
NetBeans IDE 6.5 permet le développement simplifié et rapide d'applications web, entreprise, bureau et mobile avec PHP, JavaScript, Java, C/C++, Ruby et Groovy. La version est actuellement disponible en anglais, japonais, chinois simplifié et portugais brésilien. Supporté par une vibrante communauté de développeurs et offrant une grande sélection de plug-ins tiers, l'EDI Netbeans est un must pour tout développeur !

  • Télécharger NetBeans 6.5
  • Fonctionnalités de Netbeans 6.5
  • Tour guidé sur Netbeans 6.5 en vidéo
Comme toujours, nous saluons et encourageons les retours d'expérience dans l'utilisation de l'EDI Netbeans. Partager vos enseignements sur nos listes de diffusion et nos forums; Si vous avez un blog concernant NetBeans, ajoutez-le sur Planet NetBeans.

Sun Tech Days en Chine - Guangzhou (19 novembre ), Beijing (21-22 novembre)

Les Sun Tech Days continuent de tenir en avant-garde la formation des développeurs. Si vous développez en Java, éditez des scripts, travaillez avec MySQL et sur OpenSolaris avec NetBeans ou bien si vous passez votre temps à développer des RIAs, cette conférence est la vôtre. Formation

Séquence vidéo : Introduction à l'EDI NetBeans version 6.5

Découvrez les fonctionnalités de l'EDI Netbeans 6.5 et regardez une brève introduction sur l'édition, la compilation et l'exécution des programmes Java. Découvrez aussi comment utiliser la productivité des fonctionnalités de l'EDI.


Mobile Distillery Celsius v5, Alembic v1 Tour - Londres, GB (27 novembre)

Vous cherchez des solutions pour développer facilement en une fois et porter automatiquement vos contenus et services sur la plus large gamme de périphériques possible ? Rejoignez le Mobile Distillery Tour en GB pour apprendre à concevoir et porter automatiquement les applications mobiles innovantes sur plus de 1000 périphériques à travers le monde, en quelques clics directement dans l'EDI Netbeans. L'inscription est gratuite !

Séquence vidéo : Construisez un diaporama Flickr avec PHP

Cette démo montre les nouvelles fonctionnalités PHP de Netbeans 6.5 à travers la création d'une simple application pour accéder à Flickr. Vous verrez donc comment transformer les images Flickr en un simple diaporama en utilisant JavaScript et CSS.


Développement itératif rapide d'applications web Java avec GlassFish V3 et NetBeans

Vince Kraemer montre comment les développeurs d'applications web utilisant Java peuvent exploiter certaines des nouvelles fonctionnalités de Netbeans 6.5 et GlassFish V3 pour accroître leur productivité.

Guide utilisateur de JavaScript pour NetBeans 6.5

Le déboggeur JavaScript de Netbeans fournit un ensemble d'outils qui aident à vérifier, gérer et éditer un code JavaScript. Il offre une expérience commune d'utilisateur aussi bien pour Firefox qu'Internet Explorer. Ce tutoriel montre comment déboguer une fonctionnalité JavaScript dans des applications web en utilisant le déboggeur de l'EDI Netbeans 6.5.

Chart FX 7 pour Java Desktop Beta

Chart FX 7 pour Java Desktop donne maintenant toute la puissance de Chart FX aux développeurs d'application Swing dans Netbeans 6.x en intégrant le générateur d'interface graphique Swing. Visitez le site web de the Chart FX pour télécharger des informations et une liste complète des fonctionnalités.
