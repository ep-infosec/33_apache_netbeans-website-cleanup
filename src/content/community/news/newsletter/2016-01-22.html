Project News

Plans for NetBeans IDE 8.2

ECMAScript 6 support, NodeJS enhancements, Oracle JET features, PHP7 support, Docker, and editor multicarets... are some of the many features being planned for the next release of NetBeans IDE. Check out the schedule and keep up to date with another exciting NetBeans release!

Partners

Everlaw: Angular 2, TypeScript, and NetBeans IDE

The architecture of Angular 2 is completely different to previous releases - it is component based and there are no controllers or scopes. The app code is loaded with a SystemJS loader. A component is a class annotated with @Component, which includes the HTML fragment (template). Find out how NetBeans IDE supports this combination with thanks to its partner Everlaw.

IDR Solutions: From Mercurial to Git with NetBeans IDE

NetBeans partner IDR Solutions explains the details of how they switched versioning and build systems, in clearly explained and illustrated steps.

Plugin

Plugin: Darcula LAF for NetBeans

The excellent new plugin that integrates Darcula into NetBeans IDE has been further updated and is now available in the NetBeans IDE 8.1 Plugin Manager, under the Tools menu.

Books

Murach's Beginning Java with NetBeans

Murach's "Beginning Java with NetBeans" is specifically for beginning programmers and programmers with limited experience who want to learn Java at a professional level. By the time you finish this book, you’ll have all the core Java skills that you need and you will have learned to use NetBeans too!
