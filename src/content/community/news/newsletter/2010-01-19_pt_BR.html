Novidades do projeto

Beta-teste do novo plugin Java Card!

Java Card é uma plataforma para o desenvolvimento de programas Java executados em cartões inteligentes. Os novos plugins para o NetBeans Java Card já estão disponíveis nos pacotes de desenvolvimento, e nós gostaríamos de obter feedback de usuários reais! Ajude a testar a versão beta deste novo plugin e diga o que achou.

DTrace GUI 1.1 disponível para download

Uma nova versão (1.1) do plugin GUI DTrace já está disponível a partir da Central de Atualizações do NetBeans 6.8. A versão atual do DTrace GUI tem sete novas exibições Chime que podem ser utilizados por programadores PHP e usuários para depurar e melhorar aplicações escritas em PHP. Vá em Ferramentas > Plugins> Plugins Disponíveis > DTrace e instale o plugin.

Treinamento para fãs da Plataforma NetBeans!
