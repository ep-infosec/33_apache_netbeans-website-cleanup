Trabalhando com uma fonte de dados RESTful WS no JavaFX Composer

Um aplicativo JavaFX pode obter e exibir dados de um Web Service RESTful ou outras fontes de dados remoto. Para simplificar o acesso a dados provenientes de várias fontes em diversos formatos, JavaFX Composer unifica o formato dos dados no lado do cliente para que seja mais fácil para você usar uma fonte de dados remota.

Trabalhando com injeção e Qualificação no CDI

O NetBeans IDE 6.9 fornece suporte embutido para Contextos e Injeção de dependência. CDI, especificado pela JSR-299, é parte integrante do Java EE 6. Este tutorial demonstra como você pode usar injeção de CDI para injetar classes ou interfaces em outras classes. Ele também mostra como aplicar qualificadores CDI no seu código, assim você pode especificar que tipo de classe deve ser injetado em um determinado ponto de injeção.
