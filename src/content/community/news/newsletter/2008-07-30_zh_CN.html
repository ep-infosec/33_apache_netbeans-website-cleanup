项目新闻

现已提供用于 NetBeans IDE 的 OpenLaszlo 插件

OpenLaszlo 是一个基于 Flash 的富 Internet 应用程序 Web 框架，它允许开发者使用 XML 和 JavaScript 生成 RIA 应用程序而无需懂得 Flash。用于 NetBeans IDE 的 OpenLaszlo 插件和 OpenLaszlo 4.1.1 库捆绑在一起，并包括样例应用程序、模板、.lzx 文件支持，以及更多。该插件能和 NetBeans IDE 5.5.1、6.0 和 6.1 一起工作。

用 NetBeans IDE 创建一个移动应用去参加 Sprint 的 Instinct 竞赛

Sprint 最近宣布了 Instinct Java 开发者竞赛，这是它的首次开发者赛事，并仅仅着眼于 Samsung Instinct 触摸屏电话。快速入门指南已经提供，其中包括一份是讲用 Sprint 的 WTK 和 NetBeans IDE 进行开发！阅读教程，然后开始争取赢得 25,000 美元的机会！这次竞赛的截止日期是 8 月 29 日。祝你好运！

下载用于 NetBeans IDE 6.1 Java 编辑器的 DZone 参考卡片

这款最新的 DZone RefCardz（一张能让你在 NetBeans IDE Java 编辑器中无所不能的作弊单）列出了全部的键盘快捷键，并提供了一份在编辑器中最佳处理 Java 代码的全面阐述，涵盖查看、导航、源处理以及重构。今天就去获取你的 NetBeans IDE 6.1 Java 编辑器 RefCardz 吧！
