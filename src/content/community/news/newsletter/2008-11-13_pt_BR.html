Treinamento

Demonstração Hibernate para NetBeans 6.1 e 6.5

Confira o suporte ao Hibernate do NetBeans neste vídeo. Nós vamos passar pela criação de POJOs, mapeamento de POJOs para banco de dados utilizando arquivos de mapeamento do Hibernate, e então mostrar os dados em uma aplicação web JSF criada com o Visual Web Designer.

Olá Mundo JavaServer Faces com NetBeans e GlassFish

Arun Gupta mostra em seu blog como criar uma aplicação simples de Java Server Faces utilizando o NetBeans 6.1. Ele cobre a criação de uma aplicação JSF, populando um widget JSF com um Managed Bean, utilizando uma Unidade de Persistência com widgets JSF, regra de navegação entre múltiplas páginas, mensagens de validação de erros simples, e injeção de um bean dentro de outra classe.

Dica Técnica do NetBeans 6.5: Compile ao salvar

Este vídeo demonstra a característica *Compile ao salvar* que foi introduzida no NetBeans 6.5. Poupe tempo fazendo mudanças incrementais para sua aplicação e teste-executando estas alterações sem reconstruir e reimplantar toda a aplicação.
