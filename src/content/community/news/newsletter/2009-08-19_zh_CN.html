NetBeans 中 Java 商店支持 Griffon

Griffon 是一个用于创建 Java Swing 应用，Web 启动应用和 Applets 的框架，它使用了Groovy 语言。针对 NetBeans IDE 6.7.1 的 Griffon 插件现在已经准备就绪。该插件允许你在 NetBeans IDE 中创建 Griffon 应用，从 IDE 中运行其他使用 Griffon 的 Groovy 脚本并将应用打包成一个 JAR 文件，使应用通过 Java 商店进行分发。
