Project News

Download NetBeans 6.9 Release Candidate 1!

The NetBeans Team is pleased to announce the first release candidate build of NetBeans 6.9. Download the build and provide feedback by taking the NetCAT survey or joining discussions on the NetBeans mailing lists and forums. The final release of NetBeans 6.9 is planned for June.


Survey: Is NetBeans 6.9 Ready for FCS?

Is NetBeans 6.9 ready for a stablized release? Download the latest Release Candidate build, take the NetBeans Community Acceptance Survey and tell us about your experience! The survey will be opened until June 8th.


Community

NetBeans Community Poll Launches!

NetBeans.org launches a new feature this week--NetBeans Community Polls! The polls are quick glimpses at what NetBeans users and developers think about current technology trends, including topics within the NetBeans community. Visit NetBeans.org to cast your vote for this month's question.

We also welcome suggestions for future polls. Serious or light-hearted, topical or quirky. Send your poll ideas.

Training

Creating RESTful Service Clients in NetBeans Modules

Starting in NetBeans IDE 6.9, native REST support is available in NetBeans Module projects. You can now directly generate RESTful client code in a NetBeans module, and also insert Jersey RESTful client code in a Java application. In this tutorial, you create a NetBeans Platform application that consumes the Twitter "What Are You Doing" service, displaying a list of your Twitter friends' status messages.

Application States In JavaFX Composer

This tutorial shows tools and a possible work-flow for working with states in JavaFX. A state represents a set of property values that apply at one point of time. The application switches to other states (with other property values) every time the user clicks a button, chooses a menu item, or similar.

Using the Annotation Processors Support in the NetBeans IDE

This tutorial demonstrates how to attach annotation processors to your project and use them while working on your code in the NetBeans IDE.

Articles

3D Game Development on the NetBeans Platform

The first alpha of the upcoming jMonkeyEngine 3 has been released. This popular 3D game engine now comes with a user-friendly game development tool, the jMonkeyPlatform, which is built on the NetBeans Platform.

Cycling Power Analyzer on the NetBeans Platform

Dean Yeazel is the senior software engineer for Saris Cycling Group and the lead for PowerAgent, a tool for cyclists who use bike computers. With PowerAgent, cyclists can analyze and get a summary of their metrics: average power, average speed, etc.
