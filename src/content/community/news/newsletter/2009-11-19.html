JProfiler 6.0 Released, Supports NetBeans 6.8

NetBeans Strategic Partner ej-technologies has announced the release of JProfiler 6.0. New features in JProfiler 6.0 include reworked thread profiling, a method statistics view, exceptional method run analysis, tree maps, auto-tuning as well as support for FreeBSD x86 and Linux PPC. JProfiler 6.0 supports NetBeans 6.8 including Glassfish v3 and Tomcat servers. A 10-day trial version is available.
