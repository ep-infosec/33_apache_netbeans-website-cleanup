NetBeans Plugin: LDAP Explorer

A NetBeans module for exploring LDAP services from within the IDE. The module provides a simple and quick interface for basic interaction with LDAP services.

Tutorials

Static Code Analysis in the NetBeans IDE Java Editor

The NetBeans IDE 7.2 Java Editor introduces static code analysis feature for finding potential problems and detecting inconsistencies in your source code. In this tutorial, you will learn how to perform static analysis of your Java code using FindBugs and NetBeans Java Hints without actually running your applications.

Java SE 7: Reviewing JVM Performance Command Line Tools

Java SE 7 includes a number of tools you can use to monitor your Java Virtual Machine (JVM). In this tutorial, learn to use the jps command and the new jcmd command to get information about currently running JVMs on a system. Then, use the process information with jstat to monitor JVM garbage collection activity locally and remotely.


Video: Editing PHP 5.4 with NetBeans IDE

This screencast shows NetBeans IDE code completion for PHP 5.4 innovations such as short array syntax and traits; it also shows how to create a PHP project that uses the PHP 5.4 built-in server.

Gap Editing Support in the NetBeans GUI Builder

This tutorial demonstrates the use of gap editing directly in the NetBeans IDE's GUI Builder.
