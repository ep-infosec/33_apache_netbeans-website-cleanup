Project News

New Feature: Advanced JavaScript Troubleshooting Tools

Working with JavaScript? A new feature being worked on for the next release of NetBeans is an analysis tool for Knockout.js applications. Get a quick overview of it here and learn about one of the great new features planned for NetBeans IDE 8.1.

Plugin

New: HTML5 Fun Pack

Did you know there's a "video" tag and an "audio" tag in HTML, since HTML5 was released? And did you know there's now support for calendars, code completion, and drag & drop -- natively in HTML? Install the HTML5 Fun Pack into NetBeans IDE and start experimenting with HTML as an application framework, today!
