Issue # 589 - May 27, 2013

Project News

Register for Free Webinar on Java EE 7

The "Introducing Java EE 7" webinar is a free online event where you can connect with Java users from all over the world as you learn about the power and capabilities of Java EE 7. It will be broadcast on Wednesday, June 12, 2013 and Thursday, June 13, 2013.

Join NetCAT for NetBeans IDE 7.4

For the upcoming NetBeans IDE 7.4, the start of the NetBeans Community Acceptance Testing program (NetCAT) has been announced. Get involved with the program, develop on the bleeding edge, and report on your experiences directly to the NetBeans team.

Articles

PCWorld.com: Office Suite on the NetBeans Platform

Anthony Goubard, a freelance Java developer, took 30 days to build and launch a basic open source office suite on the NetBeans Platform. Many YouTube movies, screencasts, and information about this project is available. Join in today!

Jelastic.com: Integration with NetBeans IDE

The Jelastic team reports on why they created a Jelastic plugin for NetBeans IDE. With the new Jelastic plugin, you can work with your development, test and production deployments without leaving your favorite IDE.

JAXenter.com: NetBeans IDE 7.3 and HTML5 support

More and more enterprise applications are moving to the HTML5 platform and NetBeans IDE wants to help ease that transition. Read a recently published article on JAXenter.com about the new HTML5 features in NetBeans IDE.

Calendar

30 May 2013: Joeffice Launch Party in Amsterdam

There has been quite a lot of press coverage on the launch of Joeffice, the NetBeans Platform based open source office suite, including in Slashdot, PC World, webwereld.nl, IT wire, SF gate, and Le monde informatique. Come to the launch party and find out what all the buzz is about.

3 June - 5 June 2013: Java Croatia Conference

The HrOUG (Croatian Oracle User Group) and HUJAK (Croatian Java User Association) continue the tradition of organizing the International Java Conference in Croatia. Great speakers and content. If you're in Croatia, be sure to join in!

1 July 2013: JayDay Munich

Adam Bien, Gerrit Grunwald, Simon Ritter, and many more rock star speakers will be in Munich, Germany, on 1 July 2013 at JayDay! Register today!
