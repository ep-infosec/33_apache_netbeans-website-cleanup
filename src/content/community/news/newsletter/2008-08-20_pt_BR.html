Novidades do projeto

Ajude a testar o projeto Eclipse Project Importer para o NetBeans 6.5 Milestone 1

Atendendo a pedidos, o Eclipse Project Importer foi melhorado e agora faz parte da distribuição padrão no NetBeans IDE 6.5 Milestone 1. Se você trabalha em uma equipe mista de IDE e tem colegas utilizando o Eclipse, nós o encorajamos a testar a importação de projetos Eclipse e fornecer seu feedback sobre sua experiência da seguinte maneira:

Preenchendo um "issue"; selecione o componente 'ide' e o subcomponente 'eclipse project import'. Ou
Escrevendo para nbusers com o assunto "Eclipse Project Importer".

Ibon Urrutia - Construindo aplicativos móveis com TagsMe, não requer Java

TagsME é uma plataforma de desenvolvimento de aplicativos para dispositivos móveis. O editor GUI do TagsMe, construido sobre o NetBeans Platform pelo desenvolvedor Java e membro do Dream Team do NetBeans Ibon Urrutia, não requer conhecimentos de Java para iniciar a criação de aplicativos móveis.

Criando e consumindo componentes de Serviços Web RESTful usando jMaki

Você pode reutilizar clientes stubs de RESTful em múltiplas aplicações web envolvendo o stub em componentes jMaki, também chamado de widget jMaki. Este tutorial mostra a você como criar e consumir componentes jMaki construindo projetos de serviços web RESTful no NetBeans.

Qual é seu signo? - Introdução ao Framework JavaServer Faces

Este tutorial, o primeiro de uma série, demonstra os fundamentos do framework JavaServer Faces (JSF) para codificar aplicações web no NetBeans. Você vai criar uma aplicação web chamada jAstrologer que solicita ao usuário seu nome e data de aniversário e retornará informações como o signo zodíaco, pedra da sorte e horóscopo.

Shapes, binding e animação no JavaFX e NetBeans

JavaFX funciona com gráficos e animações de forma super simples. E graças ao novo plugin do NetBeans você pode construir gráficos ricos interativamente. Neste rápido demo por Josh Marinacci, aprenda o básico da criação de shapes, binding a variáveis e simples animações.
