刊号 # 242 - May 30, 2013

项目新闻

参与针对 NetBeans IDE 7.4 的 NetCAT

对即将到来的 NetBeans IDE 7.4，NetBeans 社区验收测试计划（NetCAT）的启动已经公布。参与这项计划，进行尖端开发，并将你的体验直接汇报给 NetBeans 团队。

注册 Java EE 7 的免费网上研讨

“正在到来的 Java EE 7”网上研讨是一次免费的在线事件，你可以和来自世界各地的 Java 用户联线，并了解 Java EE 7 的强大与能力。它将于 2013 年 6 月 12 日星期三以及 13 日星期四进行广播。

文章

JAXenter.com：NetBeans IDE 7.3 和 HTML5 支持

越来越多的企业应用程序正迁移到 HTML5 平台，NetBeans IDE 希望提供帮助以简化那种转变。请阅读 JAXenter.com 上新近发表的有关 NetBeans IDE 中的 HTML5 新功能的文章。

Jelastic.com：与 NetBeans 集成

Jelastic 团队报道了他们为什么要给 NetBeans IDE 创建一个 Jelastic 插件。用这个新的 Jelastic 插件，你可以不必离开最喜欢的 IDE 来进行开发、测试和产品部署。

PCWorld.com：NetBeans 平台上的办公套件

Anthony Goubard，一个自由职业的 Java 开发人员，用 30 天的时间在 NetBeans 平台上构建并启动了一个基本的开源办公套件。可获取关于该项目的很多 YouTube 电影、屏播和信息。今天就加入吧！

日程表

2013 年 7 月 1 日：慕尼黑 Java 日

Adam Bien、Gerrit Grunwald、Simon Ritter 已经很多巨星演讲人将与会德国慕尼黑 2013 年 7 月 1 日的 Java 日！今天就注册吧！

2013 年 6 月 3 日 - 6 月 5 日：Java 克罗地亚大会

HrOUG（克罗地亚 Oracle 用户组）和 HUJAK（克罗地亚 Java 联合会）延续了在克罗地亚组织国际 Java 大会的传统。演讲人和内容都非常好。如果你在克罗地亚，请一定参加！

2013 年 5 月 30 日：阿姆斯特丹的 Joeffice 启动聚会

已经有相当多关于 Joeffice（基于 NetBeans 平台的办公套件）启动的新闻报道见诸包括 Slashdot、PC World、webwereld.nl、IT wire、SF gate 和 Le monde informatique 等媒体。敬请光临启动聚会，来看看都有些什么。
