Novidades do projeto

Patch 1 para o NetBeans IDE 6.8 está disponível

A equipe do NetBeans disponibilizou um novo patch, que é uma atualização para o NetBeans IDE 6.8. O patch inclui algumas correções de bugs. Para obter as correções, o NetBeans IDE 6.8 precisa estar instalado e rodando. Obtenha o patch usando o gerenciador de Plugins do IDE.

Dê seu feedback sobre a nova funcionalidade do PHP: símbolo de erro

A equipe do NetBeans PHP quer seu feedback sobre um recurso experimental que pode fazer parte do NetBeans 6.9: símbolo de erro. Se você tem um arquivo PHP com qualquer erro, o arquivo pode ser marcado diretamente na janela Projetos. Este recurso é útil? Diga à equipe.

Treinamento

Swing e Twitter com Java e NetBeans

Neste tutorial Alfonso Romero descreve como construir uma aplicação personalizada para o Twitter usando Java e NetBeans, ao invés de usar a interface padrão da web.

Suporte RTSP em aplicativos JavaFX

O Suporte Real-Time Streaming Protocol (RTSP) está disponível no SDK do JavaFX desde a versão 1.2. Este documento esclarece quais plataformas suportam esta funcionalidade, e fornece recomendações básicas para possibilitar o streaming de mídia em seu aplicativo.

Adicionando um seletor de arquivos para um aplicativo Java

Aprenda a adicionar um seletor de arquivos para um aplicativo Java, utilizando o componente javax.swing.JFileChooser e o construtor de interfaces gráficas do NetBeans.

Artigos

Vídeo: JavaFX & NetBeans IDE na Jfokus 2010

Desenvolvedores do Epsilon, um empresa sueca de consultoria em tecnologia e sistemas, são fãs do JavaFX e do NetBeans IDE. Na apresentação do Jfokus 2010, eles demonstrarão um projeto e irão discutir o uso das duas tecnologias.

JavaFX Composer - Um grande passo para o JavaFX?

O JavaFX Composer é um novo plugin visual do editor NetBeans 6.8 (similar ao do Projeto Matisse para Swing). O DZone entrevistou membros da equipe de desenvolvimento do JavaFX para ficar sabendo dos comentários dos usuários da ferramenta. Mais ferramentas como o JavaFX Composer podem levar a um aumento de aplicativos baseados em JavaFX, tais como o aplicativo de visualização de medalhas do site da Olimpíada de Inverno de 2010.
