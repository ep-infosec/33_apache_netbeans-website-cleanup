New Spring Boot Integration for NetBeans IDE

Aggelos Karalias, one of the attendees at the recent NetBeans Day Greece, has made a brilliant plugin for NetBeans IDE that provides code completion for Spring Boot configuration properties. No more typos, if you're using Spring Boot with NetBeans IDE!

NetBeans UML Plugin and Open Source Lessons Learned

The Open Source Software Development Center at the University of Belgrade has released a new version of their UML plugin for NetBeans, called "easyUML". Try it out, while reading about what the team learned while working on this plugin, together with tips and tricks for collaborating between open source projects and universities around the world.
