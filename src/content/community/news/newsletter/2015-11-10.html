Project News

Released: NetBeans IDE 8.1

The NetBeans team is happy and proud to announce the release of NetBeans IDE 8.1. As always, it is free and open source. For the first time, Node.js tools, including a debugger, have been integrated.

Calling All NetBeans Plugin Owners!

Are you promoting your technology via NetBeans IDE by means of a plugin? That's great and we thank you for your contribution to the NetBeans community. Read this article to find out how to promote your technology via NetBeans IDE 8.1!

NetBeans 8.1 mit überarbeitetem Java-Profiler

Die quelloffene Java-IDE NetBeans ist in Version 8.1 erschienen. Das Release bringt u. a. neue Tools für HTML5, JavaScript, Node.js, KnockoutJS und AngularJS, sowie einen überarbeiteten Java-Profiler mit sich.

Oracle JET: Overview of Data Visualization Components

As part of the initial public release of Oracle JET, the brand new JavaScript toolkit by Oracle, we're very excited to introduce the first set of data visualizations targeted for the JavaScript developer, supported by the NetBeans plugin for JET.
