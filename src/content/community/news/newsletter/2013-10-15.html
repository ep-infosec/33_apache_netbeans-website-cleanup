Issue # 609 - Oct 15, 2013

Project News

NetBeans Day 2013

Maybe you missed NetBeans Day this year. In the first part of a new series, find out what James Gosling and others discussed during NetBeans Day in San Francisco!

Plugin: CoffeeScript

Install this plugin and you will have advanced syntax highlighting, error checking, and much more for CoffeeScript, a language that compiles into JavaScript.

Community

Poll: Your IDE for PrimeFaces Applications

Do you develop PrimeFaces applications? Take the PrimeFaces poll and let their team know which IDE you use for daily development.

Reflections on JavaOne 2013 by the NetBeans Community (Part 2)

The NetBeans community turned out in full strength at NetBeans Day and JavaOne 2013. In this article, Shai Almog, Thomas Kruse, and Zoran Sevarac reflect on their experiences during the conference.

Video: JavaFX with CoC, DI, and SceneBuilder

Watch NetBeans Dream Team member Adam Bien as he shows how to build enterprise JavaFX applications with Convention over Configuration, Dependency Injection, Maven, NetBeans and Scene Builder.

Articles

Don't Reinvent the Wheel, Use the NetBeans Platform

After trying using NetBeans Platform, Djamel Torche reports that he is impressed by what it provides in features, component/code re-usability and best practices. Read the results of his research here.

New NetBeans Application: CaseLnk

CaseLnk Case Management System, by CaseForge Technology, is a new NetBeans application for improving office productivity, user collaboration, and customer satisfaction. It supports cases, documents, tasks, events, processes, and contacts.

New NetBeans Application: Validation Manager

Javier Ortiz reports on Validation Manager, which is a tool created on the NetBeans Platform to handle all the cumbersome paperwork of regulated environment validations, including validation plans, protocols, executions, and exceptions, while keeping everything in one place and, best of all, paperless.

Tutorials

Java SE 8: Lambda Quick Start

This free webcast (Date: Wednesday, October 16th, 2013 Time: 8:00 am US/Pacific time) provides an introduction to the forthcoming lambda expressions syntax added to Java 8. Learn how lambda expressions replace anonymous inner classes.

Java EE 7: JMS 2.0 Producer & Receiver

This tutorial covers Java Messaging Service 2.0 (JMS 2.0), a new API for sending and receiving messages in Java Platform, Enterprise Edition 7 (Java EE 7) web applications by using GlassFish and NetBeans.
