项目新闻

GlassFish ESB v2.1 已发布——包括 NetBeans 6.5

开源项目 OpenESB 已经发布了最新版本：GlassFish ESB v2.1。此次发布包括了作为设计工具的 NetBeans IDE 6.5.1 和作为运行时的 GlassFish。下载包含用于 NetBeans 6.5.1 的最新 SOA 工具，比如 BPEL 编辑器、WSDL 编辑器，以及一些向导，它们还不能从 NetBeans 下载站点得到。立即下载吧！

社区同意 NetBeans 6.7 为黄金时间做好准备

我们在此高兴地宣布 NetBeans IDE 6.7 社区验收调查的结果：144 名应答者中的 79% 同意 NetBeans 6.7 发布候选版已经足够稳定，可以推出！NetBeans 工程师修复了 3 个最严重的报出的问题，我们计划在不久后通过更新中心推送其他修复。我们正在和最关键的应答者联系，以确认所有的重要问题都被覆盖。总之，这是来自社区的对 NetBeans 6.7 最好的新闻。感谢你们的专业反馈！


文章

Kenai.com：电影版

你准备启动一个开源项目，但从来都抽不出时间设置问题追踪器和版本控制系统吗？借助 NetBeans 和 Kenai，你能为你的团队获得免费的项目代管，以及一个完全集成的开发工具。在这个简短的屏幕快照系列中，Ken Ganfield 演示如何创建和分享一个新项目，如何打开并加入一个现有项目，以及如何使用问题追踪器集成。

NetBeans 平台如何帮助半导体的验证

Geertjan Wielenga 和 Chad Dorkin 谈论了一个 NetBeans 平台的真实使用案例：一个用于设计和验证半导体芯片的工具！Chad 选择在 NetBeans 平台上构建该验证工具，因为他需要应用程序可伸缩且模块化。他还很好地使用了 NetBeans 窗口系统，就像你能在屏幕快照中看到的那样——快去看看！

博客

云计算和 NetBeans RCP 造就美军防御项目

梦之队成员 Fabrizio Giudici 在参加用于一级方程式赛车的实时遥测服务（RTTS）时首次遇见 Dennis Reedy——Dennis 现正从事对评估美军汽车生存力进行仿真的工作！其核心服务在云中代管，客户端构建在 NetBeans 平台上。在博客中，Fabrizio 和我们分享了他和 Dennis 关于该既酷又真实的项目的对话。

Netbeans、JavaScript、Ant、Hudson 和 JSLint：我的天！

在博客中，Ari Shamash 描述了他如何将 JavaScript 代码质量工具 JSLint 集成到他的持续构建/集成环境。他解释了如何修改 NetBeans 项目的属性和 build.xml 文件，并添加一个 Ant 目标使 JSLint 任务成为常规构建的一部分。

社区

用于 NetBeans 6.7 的 Erlang 和 Scala 插件已发布

邓草原宣布用于 NetBeans IDE 6.7（RC3 或更好）的 Erlang 插件（版本 1）发布。它现在用 Scala 而非 Java 编写，并支持关键字、函数和变量的代码完成。也请看看他在本月早些时候发布的 Scala 插件！

Java EE 模式和最佳实践——示例资源库已启动

正如先前所宣布的，Kenai.com 项目 Java EE 模式和最佳实践现已启动。下载用于 Java EE 5 和 6 平台的 NetBeans 即用示例和模板。所有项目都在 Glassfish v2/v3、部分在 openEJB 和 JBoss 5 上测试过。使用 NetBeans IDE 6.7 聊天和 Kenai 论坛讨论 Java EE 的最佳实践，今天就加入吧！

通缉学生项目！

Peter、Philip 和 Kate 的故事，他们启动了另一个学生项目。但在这一次，一切都不同！他们不必建立另一个错误追踪器、维克、版本资源库，或邮件列表。相反，他们在 NetBeans IDE 中创建了一个项目，然后把它分享到 Kenai.com。在 Petr Dvorak 的博客中阅读他们的故事！
