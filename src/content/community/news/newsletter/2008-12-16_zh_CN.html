项目新闻

NetBeans IDE 6.5的Patch1现在可用了

NetBeans.org发布了一个新的补丁，这是一个NetBeans IDE 6.5 的更新。该修补程序包括如下模块的错误修复：自动更新服务，Ant， BlueJ项目支持, C / C + + ，数据库，图形用户界面生成器，GlashFish v3 Prelude，GlashFish v3 Prelude -JRuby， IDE平台，Java， JavaE E， W eb应用， Java Profiler， NetBeans 6.5 PHP平台, RESTful Web Services， Ruby and Rails， SaaS的服务支持和UML 。这个补丁适用于英文，日文，简体中国和巴西葡萄牙语语言。 如果要取得修正，必须安装和运行NetBeans IDE。您可以通过IDE的插件管理器下载修补程序。

NetBeans 6.5 IDE的繁体中文版现在可用了！

“NetBeans翻译项目”很自豪地宣布NetBeans IDE6.5的繁体中文版现在可用了！该发行和日语，简体中国，巴西葡萄牙语和其他社区语言一起加入了多国语言版NetBeans 6.5的清单; 它包括了NetBeans平台的Java ， Web和JavaEE用户界面模块的本地化。 此本地化是台湾太阳培训中心，太阳全球化团队以及NetBeans台湾社区的成员的共同努力。了解更多关于此发行。


新版本的NetBeans DTrace 图形用户界面插件可用

新 版本（ 0.5 ）的DTrace的图形用户界面现在可以通过NetBeans 6.5更新中心下载了。新的DTrace的图形用户界面有30多个新的编钟显示器，可用于可视化的输出DTrace。您可以使用新的编钟显示调整和调试在OpenSolaris用PHP，Ruby，Python， Javascripts ，Java， C和C + +语言写的应用程序。本质上说， DTrace的图形用户界面可用于优化OpenSolaris上的整个软件/网络堆栈。
