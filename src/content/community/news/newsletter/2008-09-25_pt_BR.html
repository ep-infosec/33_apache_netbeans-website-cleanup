Agenda

NetBeans World Tour pingando em São Paulo! (1o. de outubro)

Uma nova temporada do NetBeans World Tour está há menos de 1 semana para iniciar! Em 1o. de outubro, a jornada inicia-se na vibrante São Paulo, e segue por outros 11 destinos. Sua cidade está na lista? Visite a página do World Tour paraconhecer os próximos locais e datas, informações sobre o registro, e uma espiada na agenda deste ano. O registro para o NetBeans Day é gratuito!


Novidades do projeto

Baixe o plugin NetBeans Dynamic Tracing GUI (DTrace)

Uma nova versão do plugin NetBeans DTrace GUI está disponível para download através da Central de Atualizações do NetBeans. Algumas novas funcionalidades e destaques nesta versão incluem:

  • Chime (a visualização de saída do DTrace) embarcada no plugin NetBeans DTrace GUI
  • Uma nova e melhorada interface visual
  • Funciona com o NetBeans 6.0, 6.1, e a futura versão 6.5

Aprenda mais na página do plugin NetBeans DTrace GUI.

Webinar: Maximizando a colaboração com o NetBeans e o Codebeamer

Junte o software Intland, o Departamento de Agricultura dos Estados Unidos (USDA) e os especialistas em tecnologia do NetBeans para um Webinar "on-demand" sobre como o USDA usou o gerenciamento do ciclo de vida de aplicações (ALM - Application Live cycle Management) da Intland, Codebeamer, e a plataforma cliente rica do NetBeans (RCP) para criar um ambiente colaborativo para pesquisas científicas.

Ajude a traduzir o próximo NetBeans 6.5!

A base para traduzir a próxima versão do NetBeans 6.5 iniciou dia 11 de Agosto, a Comunidade de localização e tradução do NetBeans iniciou a tradução focada na versão 6.5. Para juntar-se a este esforço, assine a lista de discussão, apresente-se, e encontre outros interessados em traduzir o NetBeans. Seu trabalho irá ajudar no esforço para fazer o projeto NetBeans acessível aos desenvolvedores por todo o mundo!

Pergunte aos especialistas: JavaFX Preview (de 18 a 22 de agosto)

A recente liberação do JavaFX Preview foi projetada para ajudar os iniciantes a se familiarizarem com o JavaFX. Tem uma pergunta sobre o JavaFX Preview? Poste a partir de 18 de agosto, domingo, e receba respostas de três membros chave da equipe de engenheiros JavaFX da Sun: Joshua Marinacci, Martin Brehovsky e Lawrence McDonough.

Comunidade

Podcast do NetBeans - Episódio 45

Neste episódio do podcast, confira a entrevista completa de Lloyd Dunn com Ted Leung do Python, gravada depois da aparição do Leung na OSCON. Outros itens incluem uma curta conversa com o pesquisador de usuários do NetBeans Jakub Franc, uma nova dica de API Design com Jaroslav Tulach, um projeto e plugin do NetBeans do mês, e como sempre: o quebra-cabeça NetBeans.

Envie e teste sua aplicação móvel com 10 horas de experimento gratuito do DeviceAnywhere!

A integração do DeviceAnywhere com o NetBeans permite aos desenvolvedores do Java móvel acessar soluções excelentes para desenvolvimento e testes de aplicações móveis baseadas em Java. Para celebrar sua parceria com o NetBeans, DeviceAnywhere esta dobrando seu experimento gratuito - de 5 para 10 horas! Isto é uma oferta exclusiva para os usuários do NetBeans. Através deste teste gratuito estendido, os desenvolvedores podem executar remotamente todos os testes em dispositivos reais - a partir do conforto de seus desktops.

Treinamento

Nandini Ramani introduz o JavaFX Preview SDK

A Diretora de Engenharia Nandini Ramani introduz o JavaFX Preview SDK e mostra como usar o NetBeans 6.1 com o JavaFX para executar exemplos e criar uma aplicação.


Dica tecnológica: Opções de linha de comando do instalador do NetBeans

Uma referência útil da equipe do Instalador do NetBeans - uma tabela de opções de linha de comando do instalador do NetBeans, com exemplos e descrições para as opções listadas.

Tutorial Web Service End-to-End: Aplicação Móvel do Dilbert

Utilize o NetBeans para criar uma aplicação cliente que consome um web service. A aplicação captura diariamente as tiras cômicas do Dilbert via um web-service, converte os gráficos para um formato space-saving, e mostra a tira em um dispositivo móvel.

Criando uma aplicação desktop Java com banco de dados

Este tutorial o guiará na criação de uma aplicação Java desktop com banco de dados completa, que permite aos usuários navegar e editar registros de clientes e histórico de compras. O tutorial mostra como personalizar o código gerado pelo IDE para interagir com múltiplas tabelas relacionadas. A aplicação final inclui um visualizador principal para navegação por registros de clientes e históricos, uma função de busca, caixas de diálogos separadas para adição e edição de registros.

Utilizando Spring 2 e Facelets juntos com o NetBeans 6.1

Uma contribuição de Siegfried Bolz à comunidade - Aprenda a utilizar o Spring Web Framework com o Plugin Facelets no NetBeans 6.1.
