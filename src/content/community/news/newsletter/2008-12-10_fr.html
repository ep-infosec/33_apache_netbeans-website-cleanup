Articles

Importer les projets Eclipse dans l'EDI Netbeans

L'EDI NetBeans 6.5 présente un assistant amélioré d'import de projet Eclipse. Cet assistant vous permet d'utiliser Netbeans pour travailler sur des projets d'application web et Java bureau qui sont créés dans Eclipse ou MyEclipse. Une fois le projet importé, vous pouvez utiliser soit Netbeans ou Eclipse pour modifier le projet ou les deux!
