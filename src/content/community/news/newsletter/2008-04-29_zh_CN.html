NetBeans IDE 6.1 现在可以下载了！

NetBeans.org 骄傲的宣布 NetBeans IDE 6.1 现在可以下载了！NetBeans 6.1 目前已经支持更加广泛的开源脚本技术，提供了更好的性能。这一发行版除 Java 技术外支持更多的扩展语言，提供了对于C/C++、JavaScript和Ruby（包括 Ruby On Rails）丰富特性的支持发行版细节概览：

  • 支持更高级的 JavaScript 编辑
  • 更好的性能（提高40%的启动速度）
  • 内建对 Spring Framework 的支持
  • 集成了对 RESTful Web Service 的支持
  • 数据库浏览器支持新版本的 MySQL 数据库
  • 增强了对 Ruby/JRuby 的支持
了解更多关于 NetBeans IDE 6.1 发行版的信息： NetBeans IDE 6.1 发布页面 NetBeans IDE 6.1 教程和文档 NetBeans IDE 6.1 特性列表 一如既往，我们欢迎并期待着您通过我们的邮件列表和您的Blog带给我们您的反馈信息。要发现为什么 NetBeans 6.1 是您唯一需要的 IDE：现在就下载它吧！
