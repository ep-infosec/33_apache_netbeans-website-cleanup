Novidades do projeto

Veja já hoje o amanhã com NetBeans IDE 6.8

O NetBeans IDE 6.8™ oferece o melhor suporte para todo a especificação Java™ Platform Enterprise Edition 6 e à plataforma Sun GlassFish Enterprise Server™ v3. Simplifique o desenvolvimento de aplicações Java com recursos da linguagem Java EE 6: menos configuraões XML e mais desenvolvimento POJO; implantação fácil ao GlassFish V3. O NetBeans IDE continua a ser a ferramenta escolhida para as linguagens de desenvolvimento mais "tops" de linha, e esta versão inclui novos recursos e melhor suporte para PHP 5.3 e ao framework Symfony, as últimas novidades do JavaFX™ SDK 1.2.1, C/C++, Ruby, Maven e muito mais. Integração com o Project Kenai, um ambiente colaborativo para hospedagem de projetos open-source, agora com suporte completo ao JIRA, bem como integração melhorada ao registro de bugs e requisitos. Como sempre, a Plataforma NetBeans fornece uma estrutura de aplicação solidificada que pode poupar anos de desenvolvimento. Suportando as primeiras melhorias do Java EE 6, o NetBeans IDE 6.8 aumenta a produtividade do desenvolvedor e permite aos desenvolvedores tirar vantagem das mais recentes funcionalidades de linguagens e plataforma. NetBeans IDE 6.8 está disponível em Inglês, Português Brasileiro, Japonês e Chinês Simplificado.

Destaques da Versão

Java Enterprise Edition 6

  • JavaServer Faces™ 2.0 para interfaces web e a capacidade de usar EJB™ em aplicações web
  • Java Persistence JPA 2.0 e suporte a web services RESTful
  • Implantação, depuração e profiling com GlassFish v3

    JavaServer Faces 2.0 (Facelets)
  • Auto-completar de código, dicas sobre erros, auto-completar de namespace, documentação em janelas popup, e tag de importação automática para Facelets
  • Editor com suporte para as bibliotecas Facelets, componentes compostos, linguagem de expressão

    JavaFX
  • Suporte para o JavaFX SDK 1.2.1
  • Navegação melhorada, auto-completar de código e dicas do editor

    PHP
  • Suporte ao PHP 5.3
  • Suporte ao framework Symfony

    Kenai.com: Desenvolvedor conectado
  • Suporte completo ao JIRA
  • Integração melhorada aos registros de bugs e requisitos

    Maven
  • Suporte melhorados para projetos Java EE 6, Groovy, Scala
  • Exclusão de dependência personalizável no gráfico de dependências

    Ruby
  • Suporte para Rails 2.3.4, JRuby 1.4, depuração Ruby 1.9, RSpec 1.2.7
  • Melhoria na refatoração de renomear, inferência de tipos e navegação

    C/C++
  • Profiling: novo indicador de micro estados e visualização detalhada de Threads
  • Sincronização acelerada durante o desenvolvimento remoto
