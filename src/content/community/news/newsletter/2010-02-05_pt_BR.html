Introdução ao desenvolvimento Java Card

Java Card é uma plataforma executada em smart cards e dispositivos pequenos que cabem na palma da sua mão. Uma característica interessante do novo Java Card 3 é o suporte nativo à Servlets. Você pode escrever uma aplicação web usando APIs comuns que funcionam em um cartão inteligente!
