Correções de bugs para o NetBeans 6.0 e 6.0.1 disponíveis

O NetBeans disponibilizou um novo pacote que é uma pequena atualização ao NetBeans 6.0 e 6.0.1. O pacote inclui muitas correções de bugs nos módulos para C/C++, Aplicação Composite, na platforma do IDE, Java Profiler, Java, NetBeans 6.0, Plataforma, Subversion, UML, Visual JSF, aplicações Web e Serviços Web. Artigos

SIP e Java no NetBeans 6.1

A tecnologia SIP (Session Initiation Protocol) está por trás de muitos serviços populares de hoje em dia, tais como VoIP, mensagens instantâneas e conferências via web. A base de sustentação do SIP é o SIP Servlet, definido inicialmente na JSR-116 e desde então atualizada na JSR-289. Geertjan Wielenga entrevista dois desenvolvedores que criaram o suporte ao SIP no NetBeans 6.1.

Encontro do NetBeans, Oracle & JetBrains na batalha de IDE 2008
