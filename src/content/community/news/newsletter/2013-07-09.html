Issue # 595 - Jul 09, 2013

Project News

Jelastic and JRebel in NetBeans Podcast

The latest NetBeans Podcast features Anton Arhipov (JRebel) and Dmitry Sotnikov (Jelastic), as well as Jan Pirek from the NetBeans Team.

Plugin: NetBeans Cheat Sheets

Two new windows are provided by this plugin, as cheat sheets for viewing code templates and key bindings while coding in the NetBeans editor.

Articles

NetBeans 7 + JBoss 7 = Productive Combo

NetBeans 7.3.1 comes with JBoss 7 integration, similar to GlassFish v3/v4. Adam Bien shows how to get started!

JFXtras RadialMenu for NetBeans Platform

In part 1 and part 2 of an ongoing series about JavaFX and the NetBeans Platform, NASA engineer Sean Phillips walks through adapting an open source JavaFX RadialMenu to replace the NetBeans Platform system menu.

Data analysis on the NetBeans Platform

"All the infrastructure that makes NetBeans a great Java development IDE is available as a foundation for any type of desktop application," reports the Sharp Statistics blog. Read all about it, with screenshots included.

Blogs

Configuring GlassFish To Serve Offline HTML 5 Applications

HTML 5 offline applications have to specify a cache.manifest file, containing a list of resources to be cached in the browser. Adam Bien shows how to workaround a GlassFish limitation for these kinds of applications.

How to Add an Action to Package Nodes

In the dev@platform.netbeans.org mailing list recently there was a question: “How can I to add actions to the Java package node?” So I gave it a try and the solution is straight forward.

Tutorials

Java Platform, Enterprise Edition 7: Using the File Upload Component in JSF 2.2

This tutorial covers how to use the file upload component introduced in JavaServer Faces 2.2 (JSF 2.2). It also demonstrates validation with the file upload component.


Video: How to Build a Minecraft Mod with NetBeans

Arun Gupta's son explains how to build a minecraft mod during a Lightning Talk at the Silicon Valley JUG.
