调试并测试 HTML5 应用中的 JavaScript

本教程介绍了如何使用 NetBeans IDE 来进行 JavaScript 调试与测试，并可以通过 NetBeans Chrome Connector 插件、Jasmine 测试框架、Karma 或 JS Test Driver 进行自动化测试。

HTML5 应用中使用 CSS

HTML5 应用是以 HTML、CSS 与 JavaScript 组合创建并跑在各种设备的浏览器中的应用，设备包括了智能手机、平板以及笔记本。本教程介绍了如何使用 NetBeans IDE 提供的工具来进行 CSS 布局。NetBeans IDE 也提供了使用 Sass 以及 LESS CSS 预处理的功能。
