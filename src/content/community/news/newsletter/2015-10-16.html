Project News

Available: Release Candidate 2 of NetBeans IDE 8.1

The second release candidate of NetBeans IDE 8.1 is available: NetBeans IDE 8.1 RC 2. Notable enhancements include the bundled GlassFish Server Open Source Edition, which has been updated to the latest version 4.1.1, as well as the bundled Apache Tomcat server, which has been updated to the version 8.0.27, as well as 25 stopper fixes. Try it out and give feedback today!
