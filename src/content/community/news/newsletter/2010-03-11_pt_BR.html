Introdução à Plataforma NetBeans utilizando Maven

Este documento é uma rápida demonstração de como criar uma aplicação simples da Plataforma NetBeans. Você vai usar o Maven para criar o aplicativo da Plataforma NetBeans e o módulo. Você usará o kit de ferramentas UI Swing e o construtor GUI "Matisse" para criar um componente da janela.

Assistência de código em C/C++

Assistência de código é um conjunto de recursos do IDE que ajuda a navegar e editar o código fonte: Auto-completar de código, busca de utilizações, hiperlinks a declarações, visualizar a estrutura atual do arquivo na janela do Navegador, investigar uma hierarquia de herança de classe, visualizar hierarquia de inclusão de cabeçalho, exibir uma hierarquia de tipos, e assim por diante. Este documento mostra como configurar a assistência de código ao criar um projeto C/C++ de código existente.
