Novidades do projeto

Faça download do NetBeans IDE 6.1 em Japonês, Chinês simplificado & Português do Brasil!

O NetBeans IDE 6.1 está agora disponível em Japonês, Chinês simplificado, e Português do Brasil! Toda a gama de Packs do NetBeans e Pacotes podem ser baixados nestes idiomas, assim como em inglês. TagsMe GUI: O novo aplicativo NetBeans Platform

O editor TagsMe GUI é um aplicativo NetBeans Platform que Ibon Urrutia, membro do NetBeans Dream Team, e outros desenvolveram para fornecer um editor gráfico para seu framework JavaME, o TagsMe. Veja em ação no NBTV. Geertjan Wielenga deu uma breve olhada por cima do aplicativo.

Treinamento

Criando portlets para sites web com o NetBeans

Crie portlets e forneça conteúdo dinâmico através de widgets arrastar-e-soltar no NetBeans. O portlet de exemplo neste tutorial utiliza o widget jMaki Tabbed View, recupera RSS feeds, e utiliza links estáticos da Central de Programação Java da Sun.

Introdução ao framework Spring

O framework Spring é um aplicativo de código aberto muito popular que pode tornar o desenvolvimento Java mais fácil, e este tutorial fornece os passos para construir um aplicativo MVC web simples utilizando o framework.

Utilizando o Hibernate em um aplicativo Visual Web JSF

Aprenda como adicionar o suporte ao Hibernate ao NetBeans e utilizá-lo com componentes JSF em um aplicativo Visual Web.

Construindo relacionamento de recursos do Rails à moda RESTful

O framework Ruby on Rails tem um poderoso subframework de modelo conhecido como ActiveRecord. O framework ActiveRecord analisa o esquema do banco de dados e fornece automaticamente a maioria dos métodos necessários para trabalhar com os dados. Neste tutorial aprenda a utilizar os métodos do ActiveRecord para especificar relacionamentos um-para-muitos entre duas classes de modelo e utilizar o suporte ao REST (Representational State Transfer) para recursos aninhados.

Blogs

Blog do Maven por Milos Kleint

Milos Kleint, o engenheiro do NetBeans que trabalha no suporte ao Maven para o IDE, fez um blog que é uma boa opção para os interessados em informações sobre o suporte ao Maven pelo NetBeans, como em seu mais recente artigo Maven POM inheritance view.

Suíte do módulo de desenvolvimento de aplicativo SIP está disponível

Os módulos de desenvolvimento de aplicativo SIP estão agora disponíveis na central de atualização Beta do NetBeans com o NetBeans 6.1. Leia as instruções de instalação, incluindo telas, no blog de Vince Kraemer.

Aplicativos de banco de dados do mundo real em Java

Patrick Keegan iniciou uma série de artigos em seu blog mostrando como criar aplicativos desktop de banco de dados com características padrão mas complexas. Nas cinco partes que ele forneceu até agora, ele cobriu os relacionamentos um-para-muitos e muitos-para-um, utilizando caixas de diálogos separadas para entrada de dados, adicionando ações, criando campos de pesquisa, e muito mais. As séries começam com Input on a New Desktop Java Database Tutorial. Boa leitura!
