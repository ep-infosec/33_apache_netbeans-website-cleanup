Issue # 671 - Jan 13, 2015

Project News

NetBeans Top 5 Highlights of 2014!

What were you key insights and learnings with NetBeans this year? In and around the NetBeans community, here are the key events of 2014!

Plugin

Debug Plot for NetBeans IDE

Watch and listen as William Shackleford introduces you to a very useful and free plugin he has created to visualize variables in the NetBeans debugger.

Articles

InfoWorld: "Review: The big 4 Java IDEs compared"

"With a clean design and good support for both SE and enterprise applications, NetBeans is probably the best choice for all-around Java development." Read InfoWorld's thorough comparision of NetBeans and its competitors!

Three Interesting Recipes from "NetBeans IDE 8 Cookbook"

Highlights of the newly released "NetBeans IDE 8 Cookbook", highly recommended for anyone working with NetBeans IDE, published over the last few months, full of new info, well explained!

Training

Course: NetBeans with Java

Free! An excellent course for anyone working with NetBeans IDE, covers all the basics, as well as many advanced features. It is unbelievable that a course this good is available for free, make use of the opportunity today.

Course: Learn Java NetBeans on Android

Free! Start learning Java and NetBeans on your Android phone today. A step by step guide is available that can be installed on Android, with many instructions and a thorough guide to Java with many illustrations and insights into NetBeans IDE.

Calendar

NetBeans Day Netherlands: 12 February 2015

Are you in the Netherlands? Come to a free event next month organized by the NetBeans community, featuring Adam Bien, Timon Veenstra, Geertjan Wielenga, and others! Learn about NetBeans (sessions on Java, Maven, Java EE, JavaScript, and new Python features), AgroSense, PiDome, and more!
