博客

在 NetBeans IDE 中开始 Prado 编程

PRADO (PHP 面向对象快速程序开发) 是用来开发 PHP 5 Web 程序的一个基于组件和事件驱动的编程框架. 阅读更多关于 Petr Pisl 怎样在 NetBeans IDE 中创建一个基于 Prado 框架的 Hello World 程序.

NetBeans 6.5 平台的10大新功能

Geertjan Wielanga 为你概述 NetBeans 6.5 平台的10大新功能. 该平台是支撑 NetBeans IDE 和其它相似的大模块化的 Swing 程序的基础结构, 因为本博客不仅仅只是适合于 NetBeans IDE 的.

练习

NetBeans IDE 6.1 和 6.5 的 Hibernate 示例

在本截屏教程中发现 NetBeans 对 Hibernate 支持. 我们通过创建 POJOs, 使用Hibernate mapping 文件将 POJOs 映射到数据库 , 然后在一个使用 Visual Web 设计器创建的 JSF web 程序中显示数据.

NetBeans 和 GlassFish 的Hello JavaServer Faces World 程序

Arun Gupta 在他的博客中展示了如何使用 NetBeans 6.1 创建一个简单的 Java Server Faces 程序. 它提及了如何创建 JSF 应用程序,使用 Managed Bean 构建 JSF widgets,在多页间导航的规则,简单错误验证消息,往另一个类中注入一个 bean.

NetBeans IDE 6.5 技术提示: 保存即编译

本截屏教程展示了 NetBeans IDE 6.5 中已经介绍了的保存即编译功能. 你对你的程序进行多次更改并且无需重新编译和发布整个程序就可以测试运行更改后的程序,这是非常节省时间的.
