项目新闻

Java SE 6 Update 10 Beta 发布

Java SE 6 Update 10 Beta 是为了提高用户使用体验而加入新的增强功能的升级版本。该版本提供了包含 JavaKernel 在内的重要性能提升。Java SE 团队希望 NetBeans IDE 用户了解这些改进并得到反馈意见。

Sun & NetBeans 宣布发布支持 PHP 的NetBeans IDE 预览版

支持 PHP 的NetBeans IDE 预览版包括智能编辑特性，例如优先代码完成，快速重命名，标记事件，动态代码模板以及方便的导航。

NetBeans 6.1 IDE 捆绑 MySQL-GlassFish 发布！

NetBeans 6.1 IDE 捆绑 MySQL-GlassFish 安装包可以下载了。此版本捆绑了 GlassFish V2 Update 2 和 the MySQL Community Server 5.0，支持 Solaris，Linux，Windows 和 Mac OS X。
