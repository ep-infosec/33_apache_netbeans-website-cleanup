Project News

Are you getting the best coding experience when using NetBeans IDE 7.1 or NetBeans IDE 7.1.1? Are there features or enhancements the NetBeans team should consider in future releases? Tell us in this short survey. It should only take about 3-5 minutes to complete.

Plugin: JTable Print Preview

Try out a recently updated plugin providing a print preview for JTables in NetBeans IDE.

Articles

Discussion: Why is Java EE 6 better than Spring?

Arun Gupta summarizes the advantages of Java EE 6 over Spring and unleashes an ongoing discussion! If you're wondering which of the two frameworks to use, this article, together with the many comments that follow, is a good place to start.

NetBeans PHP Team: New JavaScript Editor

Petr Pisl, a lead engineer in the NetBeans PHP Team, describes the work being done in creating a brand new JavaScript Editor for NetBeans IDE. He shows several screenshots of new features and asks for your help and feedback.

API Design Tip: Invisible Abstract Methods

Having an abstract class with a package private invisible abstract method effectively prevents anyone from subclassing it. Read on for a discussion on this topic by NetBeans API architect Jaroslav Tulach.

Tutorials
