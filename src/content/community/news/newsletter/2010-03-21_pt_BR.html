Dica técnica: declaração goto em arquivos CSS

O próximo NetBeans IDE 6.9 irá permitir que você navegue por uma classe de CSS ou declaração de seleção de id de qualquer classe ou atributo ID de uma tag HTML. Passe o mouse sobre uma classe ou uma seleção ID e clique com o Ctrl pressionado para abrir o arquivo CSS apropriado que contém a declaração.

Expondo um banco de dados MySQL com RESTful Web Services

Este tutorial mostra como usar o NetBeans IDE para expor um banco de dados MySQL como uma ponta da extremidade de um serviço web RESTful. Você vai implantar o "sakila", banco de dados exemplo no GlassFish usando o plugin RESTful Web Services. Neste tutorial, você aprenderá a criar uma conexão de banco de dados MySQL, uma nova fonte de dados usando essa conexão, classes de entidade do banco de dados MySQL, e serviços web RESTful a partir dessas classes de entidade.
