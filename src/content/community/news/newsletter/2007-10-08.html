Project News

Download Portal Pack 2.0 Beta

Portal Pack 2.0 Beta is now available. New features in this release include:

* Support for both Portlet 1.0 (JSR 168) and 2.0 (JSR 286 Public Draft 1)
* Option for adding a new Public Render Parameter through UI
* A new wizard to create portlet filters, and more.


NetBeans OpenLaszlo Support Plugin

The NetBeans OpenLaszlo Support plugin adds support for OpenLaszlo, a Flash-based Rich Internet Application web framework that allows you to develop RIA applications using XML and Javascript on the UI and Java on the server, without any knowledge of Flash itself. Current features include:

* Bundling the OpenLaszlo 4.0.2 library into NetBeans's libraries list
* Recognizing .lzx files as XML and opening them in the XML editor with full Navigator support
* Providing XML code completion, and more.

NetBeans at SubConf 2007, Munich, Germany

Join NetBeans Developers Arseniy Kuznetsov, Tomas Stupka, and Maros Sandor at the Subversion Conference 2007, October 16, in Munich, Germany.

NetBeans IDE 6.0 comes with built-in support for Subversion. Their presentation: Subversion and NetBeans: Integration Matters, will focus on the advantages of tight integration of Subversion tooling into common IDE workflow. Learn how to work effectively on versioned projects including initial setup, an optimized 'Edit, Diff, Commit' cycle, solving common source code archeology cases such as searching for commits and making clean rollbacks or backports--and more.
