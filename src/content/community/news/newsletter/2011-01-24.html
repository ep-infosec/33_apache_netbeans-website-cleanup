Project News

NetBeans Plugin: SCSS ("Sassy CSS") Support

SCSS is one of two syntaxes available for Sass, which is an extension of CSS that adds power and elegance to the basic language. Learn more about Sass here. This NetBeans community-contributed plugin adds support for editing SCSS files.

APIGen Plugin: PHP Project API Generator

The APIGen NetBeans plugin integrates API documentation generator APIGen into the NetBeans IDE and allows you to generate API documentation for PHP projects.

Romanian localization for the NetBeans Platform is back on track with new translator Stan Ioan Eugen. With 93% localization completed on the NetBeans Platform 7.0, join Eugen to get the project to 100%!
