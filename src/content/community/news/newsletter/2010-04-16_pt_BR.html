Um estudante certificado da Plataforma NetBeans criou um novo plugin que ajuda a reorganizar as declarações de importação. Ele funciona no NetBeans IDE 6.7.1 ou 6.8.

Programação de banco de dados no JavaFX com Java DB

Quem disse que programar JDBC com Java era difícil? Com JavaFX você pode construir aplicações ricas de mídia com muito menos código do que você faria com Swing, mas você ainda pode usar sua base de código Java já existente. Isso inclui bancos de dados via JDBC, mas existem alguns truques que você precisa saber e que vamos explorar aqui.

DZone: Novas ferramentas para desenvolvedores OSGi

A próxima versão do NetBeans IDE 6.9 traz novas ferramentas para desenvolvedores OSGi, especificamente para aqueles que querem usar o Swing como seu toolkit UI. A Plataforma NetBeans 6.9 entende pacotes OSGi e você será capaz de importar pacotes OSGi em sua plataforma de aplicações NetBeans.

Lista completa de macros para o editor Java do NetBeans

No editor Java do NetBeans, você pode criar macros clicando no botão "Iniciar gravação de macro". Também é possível editar macros na janela Opções, na aba Editar > Macros. Uma sintaxe especial de macro é usada para definir essas macros - aqui você encontra a lista completa.
