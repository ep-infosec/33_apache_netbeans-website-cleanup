Calendar

NetBeans Day Paris - March 19, 2007

Brush up on your French for the next stop on the NetBeans worldTour: Paris! Mingle with NetBeans experts and enthusiasts. Watch cool demos of available and upcoming features. Pick up a few IDE tips & tricks. Get the latest on NetBeans, and more. Reserve your spot today AND show up early!

NetBeans Day San Francisco - Register Now!

Registration is open for NetBeans Software Day 2007, which will take place on Monday, May 7, at the Moscone Center in San Francisco. Discover the latest news about the NetBeans IDE and platform; meet James Gosling, Jonathan Schwartz, Rich Green and other Java luminaries; watch demos of the next generation of NetBeans software, and more. NetBeans Day is free, so register now and save the date!

Project News

Advanced Refactoring Talk at JavaOne

Attending JavaOne and interested in refactoring? Make a note to drop in on a session with Tom Ball. His JavaOne proposal: "TS-9861: Advanced Java Programming Language Refactoring: Pushing the Envelope" has just been accepted. Do you have a JavaOne talk that's received the green light? Community

NetBeans Governance Board Elections - Voting Begins

The nominations are in for the NetBeans Governance Board; now it's time to cast your votes for the community members you wish to elect to the board. Visit the Board Elections page to learn more about the candidates and voting instructions. The voting period runs through March 21, 2007, midnight in the last time zone.

Contribute to NetBeans with the NetBeans Community Docs Program

Sign up to contribute to NetBeans! The NetBeans Community Docs program lets you write about NetBeans in any form you want: tutorials, white papers, FAQs, tips and tricks, blogs or Flash demos. Visit the NetBeans Community Docs Wiki to learn how to contribute content and even request contributions.

Articles

Translating NetBeans 5.5 for Taiwanese Users

First came Brazilian Portuguese, now the Translation Project has helped deliver the NetBeans 5.5 IDE in Traditional Chinese. Why is this latest release unique? The translation was a solo effort by James Yu, a PhD candidate in molecular and medical pharmacology.

Training

Free Access to Select Online Training with Sun Learning Services

Sun Learning Services is offering a selection of FREE web-based courses as a "benefit of membership" associated with the Sun Developer Network. Take advantage of Sun's expertise with free access to recently released online training courses such as: Introduction to Developing Rich Client Applications, Overview of Java Application Security, Overview of XML, Web Services Enabling Technologies, and more. New courses are added each month, so check back frequently.

Community Docs Contribution: Component Creation at Runtime Tutorial

Sapankumar Parikh has contributed a tutorial to the Community Docs Wiki: The Component Creation at Runtime Tutorial. Understand the process of creating components and their action events at run time.

Visual Web Pack: Developing a Media Management Application

Learn how to develop a media management application using the Visual Web Pack. This article demonstrates how to design the pages and write the code for an application that stores and manipulates media files. The article comes with a working application that you can download and use.

Blogs

Building Ruby Apps with Derby and NetBeans

Brian Leonard's tutorial shows how to rapidly produce a web application with NetBeans jRuby and Derby. Does it look familiar? It's a blow-by-blow of Roumen Strobl's Ruby on Rails Flash demo, but with a twist--Leonard uses the Derby database that comes with JDK 6 in place of MySQL. (Thanks to Charles Ditzel for the tip.)

Swing Application Framework & Swing Databinding Demo

Curious about how NetBeans 6 will support JSR-295 and JSR-296? Watch Roumen's latest (and funny) Flash demo about the Matisse GUI Builder.
