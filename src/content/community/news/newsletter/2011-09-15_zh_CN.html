项目新闻

NetBeans @ JavaOne 2011

在 JavaOne 2011 上，NetBeans 将倍受关注。这里列举了一些 NetBeans 在今年圣弗朗西斯科举行的 JavaOne 2011 大会相关主题。

NetCat 项目

Dhawal Saiya 对最近开始的 NetBeans IDE 7.1 NetCAT 项目做了简单报告。NetCAT 项目是一个 Beta 测试项目，看看发生了什么吧。

插件：Antlr Editor Support

Nikolay Krasilnkov 介绍了他编写的一个 NetBeans 插件，用于编辑 Antlr 文件。该插件现已支持 NetBeans IDE 7.0 与 7.0.1。支持语法着色以及一些很酷的特性来帮助你更好地在 NetBeans IDE 里编辑 Antrl 文件。

社区

NetBeans IDE 7.0.1 中的内存使用量

NetBeans 用户 Mark Barnes 写到：“自从安装了 NB 7.0.1 (from 6.9)，我发现内存使用情况比以前好多了。”

小贴士：NetBeans Java 代码自动完成

在 NetBeans Java 编辑器中，代码完成功能有着比你想象中多的功能。阅读本文，了解很多非常酷的代码片段其实是可以通过代码自动完成框来生成的。

Twitter：NetBeans PHP Rocks

@msafi: “如果我在开发一个 PHP IDE，但偶然之间我使用了 NetBeans。我将停止我的 IDE 开发，因为相比之下，NetBeans PHP 支持简直是太帅了！”
