练习

使用 Schliemann 里支持未知语言

泛型语言框架（Generic Languages Framework，代号 Schliemann）提供了一种简单的方法来在 NetBeans IDE 内支持您的语言。这份教程展示了如何支持普通的 STEP Physical File (ISO 10303 Part21) 格式，这种格式是用于 CAD 与 Product Structure Exchange 交换信息的。

添加 Web 框架支持

NetBeans，Ruby 与 AppleScript

"Mac Developer Tips" 博主 John Muchow 描述了如何在 Mac 上使用 NetBeans 与 rb-appscript（一种桥接 Ruby 与 Apple Event Manager 的脚步语言）控制 scriptable 应用程序。这份介绍展示了如何在 Mac OS X 上使用 Ruby 编写 AppleScript 的脚步应用。
