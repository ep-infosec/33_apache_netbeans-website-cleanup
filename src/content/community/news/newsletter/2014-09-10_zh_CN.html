练习

Java SE 8：在接口中实现默认方法

Java SE 8 之前，Java 中的接口只能包含方法声明，而不能有实现，并且任何实现接口的非抽象类必须提供实现。这条限制使得扩展现有接口和 API 几乎不可能。为了克服该限制，一个新概念，叫做默认方法，在 Java SE 8 中引入了。默认方法是在接口中完全实现的方法，它们用关键字 default 声明。因为默认方法有一些默认的实现，它们可以帮助扩展接口而不破坏现有代码。
