文章

和 NetBeans 平台与时俱进

当系统和软件架构师 Hermann 找不到一个与时俱进的应用程序以适合于在他的项目中常常牵涉到的所有角色的要求时，他用 NetBeans 平台写了一个自己的：“针对功能，我使用了大量的 NetBeans 平台模块，否则就要自己完成。而且我还不确定能否实现出和从 NetBeans 平台中所得到的（而且免费！）一样的品质等级。”

和 JavaFX 开发人员 Fabrizio Giudici 一起观鸟

Reviews Interactive 最近采访了 JavaFX 开发人员 Fabrizio Giudici：一位在 JavaFX 社区中杰出且受到高度尊敬的博客，他经常为 Java.net 和 DZone 写文章。Fabrizio 说他对在移动设备上使用 JavaFX 特别感兴趣，并且由于 JavaFX，仅在一周内，他就为他最新的用于野鸟观察家的应用程序创造了可工作的原型。

eWeek 点评：NetBeans IDE 6.7 提供了与 Kenai 项目的高效集成

“最新版的 NetBeans IDE 以 x.7 发布，但它包括了这么多的新功能，有理由得到完全的版本提升。可能最大的改进之处就是完全集成了 Kenai 项目：Sun 为开源写作建造的站点，而这正是本篇点评将要关注的。”

社区

让你的名字列在 NetBeans 荣誉中！

你可能听说过 NetCAT：NetBeans 社区 Beta 测试计划。在像 NetBeans 这种开源项目中，毫不奇怪可通过很多其他方式做贡献：你可以像 NetFIX 队员一样帮助我们修复错误，或者甚至在新的 NetDEV 计划下，为将来的 NetBeans IDE 6.8 合作开发新功能。想让你的手沾满源代码并实现你最喜欢的增强吗？今天就加入 NetDEV 吧！


带 JavaFX 的 NetBeans IDE 6.7.1 现已提供！

NetBeans 团队高兴地宣告 NetBeans IDE 6.7.1 的发布，它带有 JavaFX 1.2 支持和对 GlassFish v3 集成的升级。它还包括对 Java 编辑器和社区成员所要求的项目扫描方面的重要改进。了解此新发布的更多信息，并在今天就下载 NetBeans IDE 6.7.1！


从 Kenai 获取 NetBeans-即用的 Java EE 示例

Java EE 专家和梦之队成员 Adam Bien 开始在 Kenai 上分享他的 Java EE 示例项目集——而且他还在添加新的！我们不知道有多少人用过 NetBeans Kenai 集成来签出并运行 Adam 的 NetBeans-即用的示例项目。但我们肯定知道在超过 100 名成员推动下，这是最流行和发展最快的 Kenai 项目之一。今天就加入，并跟着专家学习吧！

如果你不能参加今年的 JavaOne 大会去了解 NetBeans 和所有其他优秀的 Sun 技术，还有一个机会去跟上一些你错过的重要会议和内容表：我们将告诉你到哪儿去找来自今年大会中广受欢迎的下载、动手实验室、视频和播客！

免费 JavaFX 编程（激情无限！）课程

“免费 15 周 JavaFX 编程（激情无限！）”在线课程的第二期将在 2009 年 8 月 25 日开始。整个课程免费，并且任何注册到邮件别名的人都可以在线听课。参与者将学习演示材料，完成已提供的动手实验室，并做每周的家庭作业。你可随意使用课堂邮件别名来问（和答）JavaFX 问题！

本地化这个：NetBeans 平台应用程序

再也没有借口了！本地化构建与 NetBeans 平台上的应用程序实际上很简单，因为你还是可以用 NetBeans 平台应用程序的 GUI 表单编辑器。在这篇通俗易懂的教程中，用 NetBeans IDE 6.1、6.5 和 6.7 学习国际化的全部知识。
