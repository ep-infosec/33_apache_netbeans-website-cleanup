Issue # 438 - May 11, 2010

Project News

Updated NetBeans Samples Catalog Now Available!

The NetBeans Samples Catalog has been updated. Visit the catalog's main page on NetBeans.org to find sample applications packaged as NetBeans IDE projects that are used in NetBeans tutorials. You can also contribute your own sample apps.

Training

Getting Started with JavaFX Composer

JavaFX Composer is a visual layout tool that lets you quickly build JavaFX GUI applications. This tutorial provides a quick introduction to building JavaFX applications using the Composer's visual editor.


Using JavaFX UI Controls

Learn how to easily create user interfaces with the new JavaFX UI controls and how to make these interfaces cool and fancy by applying CSS.

NetBeans Platform Quick Start Using OSGi

This document is a quick demonstration of how to integrate OSGi bundles in a NetBeans Platform 6.9 application. You import the OSGi bundles constituting a Sudoku Game, together with the related required Equinox bundles.

Working with Events in CDI

NetBeans IDE 6.9 provides built-in support for Contexts and Dependency Injection (CDI). This tutorial demonstrates how to integrate Java EE events into an application with CDI using the IDE.

Blogs

Feature Preview: Values of PHP Constants

In NetBeans IDE 6.9, values of PHP constants will be displayed in the Navigator window, and in the documentation and Code Completion popups. This new feature is available for testing in the daily development builds for NetBeans 6.9 for PHP.

"OSGi Support in NetBeans Getting Pretty Sweet"

James Gosling, famed Java creator notes improvements to NetBeans IDE in the 6.9 Beta release, as well as to JavaFX 1.3.

Articles

Yahoo Query Language Console in NetBeans IDE

Developer Wagner Santos' YQL Console for NetBeans plugin helps developers test queries and functionalities using Yahoo APIs.

Seismic Data Visualizer on the NetBeans Platform

An interview with Andrey Ilinykh, software architect for INTViewer, a NetBeans Platform based platform for viewing, analyzing, and quality checking seismic data.
