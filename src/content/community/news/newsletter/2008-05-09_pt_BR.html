Novidades do projeto

Pacote MySQL-GlassFish agora disponível para NetBeans 6.1!

O NetBeans 6.1 com o pacote MySQL-GlassFish está disponível para download. O pacote contém o GlassFish V2 Update 2 e o MySQL Community Server 5.0, e tem suporte no Solaris, Linux, Windows e Mac OS X.


A Sun e o NetBeans anunciam o lançamento do NetBeans IDE Early Access para PHP

O NetBeans IDE Early Access para PHP inclui características inteligentes de edição como auto-completar de código priorizado, renomear instantâneo, ocorrências de marcas, modelos de código dinâmico e fácil navegabilidade. Inicie facilmente o desenvolvimento PHP com características como suporte à depuração, implantação ao servidor local, sistema de ajuda dinâmica e projetos de exemplo. Aprenda mais sobre o plugin Early Access PHP.

Confira o Java SE 6 Update 10 Beta

Java SE 6 Update 10 Beta é uma atualização que introduz novas características e melhoramentos com o intuito de aprimorar a experiência do usuário final. Esta versão tem significantes melhorias de performance e em sua base incluindo o novo JavaKernel. A equipe do Java SE gostaria de deixar os usuários do NetBeans cientes destas melhorias e estão interessados no feedback. Leia mais sobre o Java SE 6 update 10 Beta.
