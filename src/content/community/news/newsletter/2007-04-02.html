GWT Support Plugin Graduates from Java.net Incubator

The GWT4NB project aims to enable developers to take advantage of both the superior support for creating Web Applications built into the NetBeans IDE and the power of GWT (Google Web Toolkit). GWT is an open source Java software development framework that makes writing AJAX applications easy. Write your front end in the Java programming language, and the GWT compiler converts your Java classes to browser-compliant JavaScript and HTML.

Using IRB in NetBeans

The NetBeans 6.0 Ruby module includes the IRB (Interactive Ruby Shell) Console. Generally, IRB is started from the command prompt in the directory containing your scripts. However, in NetBeans, it's launched from the NetBeans installation directory. Brian Leonard has some tips on using IRB in NetBeans.
