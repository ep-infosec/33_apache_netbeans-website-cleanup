Télécharger la SDK de JavaFX Preview

JavaFX Preview SDK est une version avant-première de la plateforme JavaFX. Destiné aux scripteurs web et aux nouveaux adeptes, il donne une prévisualisation de l'exécution et des outils dont on a besoin pour créer d'extraordinaires Applications Riches Internet (RIA) sur la plateforme Java(TM). La version standard comprend Preview SDK, l'EDI NetBeans 6.1 avec JavaFX, le Projet Nile et le Java Runtime Environment 6 Update 10 Beta.

Commencer avec le contrôle JMX dans l'EDI NetBeans

Ayez une petite description de l'utilisation de la technologie JMX dans l'EDI NetBeans. Le module JMX de NetBeans intègre la technologie JMX dans un flux de travaux dans l'EDI et vous permet de dévélopper les applications de management, d'ajouter le management aux applications existantes, de développer des applications manager et de contrôler l'état de la machine virtuelle.
