社区

本周语言社区的关注点：中文

NetBeans 的开发者们是一群不同的个体，他们使用着各种各样的编程语言甚至连母语都大不相同。来自全世界的用户是怎样交换着他们的编程思想和提供相互间的支持呢？ 在接下来的几周中，我们将向您介绍NetBeans社区多种语言的的门户网站，讨论组，博客和邮件列表或着是与其有关的NetBeans IDE 和 NetBeans 平台的使用。这周，我们有幸通过中文社区开发者收集了一系列有用的站点。

Ted Neward ，Tim Boudreau 和 Geertjan Wielenga谈论NetBeans平台

Ted Neward 就有关 NetBeans 平台方面采访了 Tim Boudreau 和 Geertjan Wielenga。在第一部分中，他们向我们介绍了插件的开发，事实上这是非常的简单。点击第二个预览图标你将了解到更多有关在NetBeans平台上构建 Swing 应用程序的信息，除此之外你还可以获知他们和 Jaroslav Tulach 合著的《 Rich Client Programming: Plugging into the NetBeans Platform 》。

博客

Java EE 5, JBPM, JBoss Seam, Netbeans 和 Glassfish V2 的整合

JBPM 是 JBoss的一个工作流引擎，它允许你在控制处理时对事件触发和任务执行作出反应。这篇博客描述了如何导入 JBoss “todo” 示例程序做为 NetBeans 项目以及如何在 Glassfish v2 web 服务器上运行它。

NetBeans IDE 上的 PHP： 处理类属性变量（Class Property Variables）

在这段视频中，Petr Pisl 将向你们展示一个简单的应用程序实例--在 NetBeans IDE 6.5的PHP是如何处理类属性变量的，除此之外你还将看到其它的编辑特性。切换到全屏模式可看到更多的细节。

练习

NetBeans IDE 6.5 中Web 应用程序的安全性

这个文档将通过一些基本原理教你如何加强一个部署到 Tomcat 或 the Sun Java System Application Server上的 web 应用程序的安全性。你将了解到如何使用 window 登录配置安全认证和在 web 服务器上建立一个具有安全身份的用户。

教学环节：代码模板的备份

NetBeans 可以让你根据个人爱好自定义插入代码块的模板。你还可以输入适当的缩写和某个自定义的快捷键，此后它将为你自动扩展为完整的代码片段。在这个教学环节中，Kristian Rink 将向你演示如何在不同的 IDE 安装程序保留你个人的自定义模板。

NetBeans IDE 6.5 上 Ajax 的特性介绍

这个介绍是有关在NetBeans IDE 上 Ajax 展现的一些特性--当你在使用 Ajax 相关技术进行工作时它允许你的程序运行得更快更有效。当获知一些有关 Ajax 的初级功能后，你构建一个简单的应用程序时它将从事于文本域的自动完成。

用NetBeans IDE 创建一个简单的 JavsFx 应用程序

这个教程将带你快速浏览如何使用 NetBeans IDE 6.1 开发 JavaFX 应用程序。当你完成这个教程中的程序后，你将知道如何使用NetBeans IDE 创建，演练，运行和部署 JavaFx 应用程序。
