Comunidade

Metro 2.0.1 está aqui

O recém-lançado Metro 2.0.1 inclui JAX-WS (RI) 2.2.1. Está integrado ao GlassFish 3.0.1, mas também trabalha com outros conteiners, tais como Tomcat. Alexis Moussine-Pouchkine do blog Aquarium diz: "A melhor experiência para o desenvolvedor neste momento é provavelmente utilizando o NetBeans 6.9".

Criando suporte à frameworks PHP no NetBeans

Esta página é um ponto de partida para os desenvolvedores que gostariam de implementar um plugin que adiciona suporte ao NetBeans para PHP o seu framework favorito no IDE NetBeans. (É necessário conhecimento em Java.)

Treinamento

O tutorial de e-commerce do NetBeans

Nesta parte do tutorial, você aprenderá a criar uma aplicação simples, mas eficaz para o comércio electrônico. O tutorial demonstra várias características importantes do Java EE e desenvolvimento web. Ao fazê-lo, você vai se familiarizar com o IDE NetBeans e tornar-se capaz de aplicá-la a seus propósitos de desenvolvimento.

Ouagadougou na Plataforma NetBeans

Geertjan Wielenga entregua outra formação na Plataforma NetBeans, desta vez em Ouagadougou, Burkina Faso - via Skype e WebEx! Assim, onde quer que esteja, se você quiser começar na Plataforma NetBeans em 2, 3 ou 5 dias escreva para users@edu.netbeans.org para obter detalhes sobre isto. Grátis para as instituições de ensino!

Dr. Dobbs: A fonte de dados JDBC JavaFX

Este artigo apresenta o suporte interno do JavaFX ao JDBC e vários formatos de dados através de um conjunto de componentes de fonte de dados. Graças ao JavaFX Composer do NetBeans, você pode obter muita coisa sem ter que escrever um monte de código para manipulação de dados.

Artigos

Custo de projeto de sapatos na Plataforma NetBeans

Nesta entrevista, Dave Irving fala sobre iCadSuite: Esse aplicativo tem uma geometria plana 2D a partir de um sistema de design de calçados, e trabalha sobre os custos para produzir um par destes sapatos! O aplicativo usa a Plataforma NetBeans como framework.

UberConf: Java EE, OSGi e Clojure

Arun Gupta apresentou uma sessão no UberConf sobre o GlassFish como um ambiente de fácil execução para a implantação do OSGi, Java EE e aplicações híbridas (veja os slides). Ele também fala sobre o que aprendeu lá e sobre como usar o plugin para o NetBeans IDE Clojure.

Blogs

Analisador de perfil do NetBeans 6.9: HeapWalker melhorado

Confira o HeapWalker melhorado no Analisador de Perfil do NetBeans 6.9: agora é fácil para os usuários encontrarem as raízes de quadros de Garbage Collections para uma variável local correspondente.

Acompanhe as suas milhas em execução usando o NetBeans, MySQL, e YUI Gráficos

Arun Gupta está trabalhando em um aplicativo para controlar suas milhas em execução. A idéia é construir a mesma aplicação Web utilizando frameworks, implantá-los no GlassFish, e compará-los! Desta vez ele está utilizando o Apache Wicket, GlassFish, NetBeans, MySQL, e Gráficos YUI para monitorar o progresso.

Análise de perfil agora! Para projetos PHP

Você sabia que a ação "Análise de perfil agora!" também está disponível para projetos PHP no NetBeans 6.9? Esta barra de ferramentas permite que você gere um perfil da CPU instantânea para entender o que está acontecendo se você achar que o IDE está lento. Ela irá ajudá-lo a analisar gargalos de desempenho, e quando você enviar um relatório de bug, anexar a foto também.
