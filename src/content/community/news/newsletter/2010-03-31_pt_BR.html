Novidades do projeto

Desenvolvimento na plataforma NetBeans JavaFX

Uma rápida olhada em como a equipe do NetBeans está trabalhando para integrar componentes JavaFX para a plataforma NetBeans. Envie o seu feedback e casos de uso.

Versão Sappy adiciona o suporte de acessibilidade para o NetBeans

Desenvolvido por uma equipe da Southern Illinois University, Sappy no NetBeans é um conjunto de módulos que permite aos programadores com deficiência visual a utilizar o NetBeans.

Treinamento

Dica de tecnologia: poupar espaço nas janelas do NetBeans

Você usa a barra de ferramentas, mas odeia desperdiçar o espaço vertical necessário para mostrar uma barra de ferramentas normal? Jesse Glick mostra como inserir os botões na barra de menu.

Dica de tecnologia: WSDL personalizado no Java EE 6

Jeff Rubinoff compartilha uma solução alternativa para os usuários que se deparam com problemas quando utilizam o NetBeans para gerar um arquivo WSDL para um serviço Web Java EE 6 e, em seguida, personaliza o arquivo WSDL.

Adicionando pesquisa de ajuda on-line para aplicações NetBeans Java SE

PDFHelp é uma alternativa livre para PDF em Java Help online. Este tutorial mostra como usá-lo para adicionar ajuda com suporte à pesquisa à suas aplicações NetBeans.

Criar um relógio digital no JavaFX

Aprenda a programar um relógio estilo LED para uso em suas aplicações JavaFX. Este tutorial abrange uma série de tópicos importantes, incluindo animação, ligação de dados e interação com a linguagem de programação Java.


Artigos

Integração do OpenOffice para Aplicações da Plataforma NetBeans

Você está desenvolvendo uma aplicação desktop e quer tirar proveito de recursos do OpenOffice? Geertjan Wielenga escreve sobre como integrar as duas ferramentas.

O Sistema de construção da Plataforma NetBeans

Nesta terceira parte do artigo, Hermien Pellissier discute a estrutura do sistema de construção da plataforma NetBeans. Ela oferece dicas para personalizar o comportamento da compilação, introduz a estrutura de projetos da Plataforma NetBeans, e muito mais.

Oracle TechCast: JavaFX - Iniciando

O Oracle TechCasts está vivo, com especialistas falando sobre as novas ferramentas, tecnologias e tendências no desenvolvimento de aplicações. Os ouvintes podem fazer perguntas via Twitter, Facebook ou o media player durante estes eventos ao vivo. Neste TechCast, Nandini Ramani, Diretor de Desenvolvimento Java da Oracle, fala sobre inovação, evangelismo, e a aprovação na área de JavaFX.
