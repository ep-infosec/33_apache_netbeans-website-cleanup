Project News

New Features in NetBeans PHP Debugger

A new set of NetBeans PHP features now makes debugging PHP projects a more comfortable experience. Get details from the NetBeans PHP team blog.

NetBeans Plugin: Switch File Read/Write Access

This featured NetBeans module allows users to change between the read-only and write mode of a file currently opened in the editor. The latest update works with NetBeans IDE 7.

Java 7 Questions & Answers

Henrik Stahl, Senior Director of Oracle's Java Platform Group, is responding to user questions about Java 7. Read the latest threads, and submit your questions.

Griffon Plugin for NetBeans IDE 7.0.1

There's an update to the NetBeans Griffon Plugin for NetBeans 7.0.1. The latest change lets users create Matisse GUI forms in Griffon applications.

NetBeans Git Plugin Update: Small Changes, Big Impact

The recent NetBeans Git plugin update includes a number of enhancements that makes a big impact. Check out Adam Bien's entry for details.

Getting Started with Xdebug and NetBeans IDE 7

This extensive post takes a chapter-by-chapter look at David Heffelfinger's Java EE 6 Development with NetBeans 7 book. The reviewer's conclusion? "...particularly beneficial for developers with little or no Java EE experience who want to use NetBeans 7 to generate Java EE 6 applications."
