Introdução ao contextos e injeção de dependência e JSF 2.0

Contextos e injeção de dependência (JSR-299) fornecem uma arquitetura que permite aos componentes Java EE, como servlets, beans corporativos e JavaBeans existirem dentro do ciclo de vida de uma aplicação com escopo bem definido. Esta é uma demonstração de como criar um projeto com JSF 2.0 com suporte ao CDI no NetBeans IDE 6.9.

Como usar States em um aplicativo JavaFX

Em JavaFX, um "state" representa um conjunto de valores de propriedade em seu aplicativo. Se você alternar para outro "state", os novos valores são atribuídos a todos os componentes. Usando "states", é fácil criar múltiplas telas, efeitos de zoom, barras laterais, etc. Este blog também contém algumas das melhores práticas de como trabalhar com o "master state".
