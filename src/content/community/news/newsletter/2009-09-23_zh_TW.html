社群

第十七屆NetBeans董事會成員-開放提名!

到了選擇新的第十七屆NetBeans董事會成員的時候了! 藉由參與整個提名過程, 來選擇您認為適合擔任董事會成員的NetBeans社群會員吧. 您甚至可以提名自己! 學習更多相關的資訊有關於提名的流程. 整個提名的截止日期為9/30禮拜三, 於最後一個時區的午夜。

適用於NetBeans IDE 6.7.1的Java EE 5工具套件更新第七版釋出

Java EE 5 工具套件已經更新成適合NetBeans IDE 6.7.1與NetBeans Portal Pack 3.0.2 來使用了. 這次的套件已經開放可以從Java EE下載網頁來下載.

適用於NetBeans IDE的JOnAS伺服器外掛釋出

JonbAS是一個伺服器外掛, 允許將JOnAS應用伺服器緊密的整合到NetBeans IDE中. 它讓使用者能夠避免掉進行繁複且耗時的手動與重複性任務, 像是部署或者反部署一個應用程式在每個階段中. 今天就下載它, 使用它, 並且讓該外掛的作者Baptiste Mesta得知您的想法吧.

NetBeans平台現在被翻譯成羅馬尼亞文了!

NetBeans平台開發人員與夢幻團隊成員Emilian Bold最近釋出了一個羅馬尼亞文版本的NetBeans平台! 這次的釋出讓建立以羅馬尼亞文為基礎的NetBeans平台應用程式變的更加容易, 而且也為NetBeans IDE翻譯成羅馬尼亞文的計畫墊下了一個很好的基礎! 您可以到這邊下載目前的版本.

文章

讓NetBeans IDE 6.8從資料庫中產生JSF 2.0網頁

讓我們繼續Java EE文章系列, Java EE團隊架構師以及首席工程師David Konecny介紹了接下來將成為NetBeans IDE 6.8的一個特性, 全新重寫的JSF網業產生精靈. 當您從一個資料庫中選擇了表格, 精靈將會自動產生持續性entity類別, 階段性企業bean, Java Server Faces網頁, 以及需要的Java Server Faces管理beans來作為開發視覺化層以及資料層的基礎邏輯.
