Patch3 para o NetBeans IDE 6.9.1 já está disponível

A equipe do NetBeans disponibilizou um novo patch, que é uma atualização para o NetBeans IDE 6.9.1. O pacote inclui correções de bugs nos módulos para C/C++, Platforma IDE, Java e PHP. Obtenha mais informações sobre os patches do NetBeans IDE 6.9. Para obter as correções, o NetBeans IDE deve estar instalado e funcionando. Você pode baixar as correções através do Gerenciador de Plugin do IDE.
