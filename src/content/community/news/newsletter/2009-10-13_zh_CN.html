NetBeans 6.8 M2 中的 JSF 2.0 的支持

NetBeans 6.8 M2 中的 JSF 2.0 的支持

NetBeans 6.8 Milestone 2 对于 JavaServer Faces 2.0 的支持体现在用来编辑 Facelet 页面的功能强大的编辑器, 它包含了对 JSF 标签的语法高亮和错误检查, 文档支持, 和对 EL 表达式的代码完成功能, 核心 Facelet 库和命名空间.

开始学习 Java EE 6 应用程序

让 NetBeans IDE 令你开始学习 Java EE 6 规范的新功能: 在本教程中你将创建一个简单的包含一个 EJB 3.1 无状态 session bean 的 Java EE 6 web 程序. 生成的 session bean 使用了 JPA 2.0 中的 Criteria API 部分的 queries. 然后再创建一个命名的 managed bean 用来获得 session facade 和一个使用了JSF 2.0 指定的 Facelets View 框架的表现层.

在 NetBeans IDE 编写 JUnit 测试

测试一个程序是开发周期中的一个基本部分, 并且编写和维护单元测试可以保证你源代码中每个单独方法都可以正确运行. NetBeans IDE 对 JUnit 单元测试框架的整合支持使你可以快速容易地创建 JUnit 测试和测试用例.
