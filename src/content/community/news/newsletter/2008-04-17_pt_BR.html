Novidades do projeto

Faça download do NetBeans 6.1 Release Candidate!

O NetBeans.org orgulha-se de anunciar a disponibilidade do NetBeans 6.1 Release Candidate! Este lançamento inclui novas características e melhorias para a edição de JavaScript, utilizando o framework web Spring, com uma integração mais próxima ao MySQL, compartilhamento de bibliotecas entre projetos dependentes, e muitos outros destaques e correções. Faça o download do Release Candidate hoje e nos diga o que você achou em nossas listas de discussão. Lembre-se de escrever no seu blog sobre o IDE e poderá ganhar $500 dólares! O lançamento da versão final do NetBeans 6.1 está agendada para o final de abril.

O NetBeans 6.1 Release Candidate está aí e nós gostaríamos de receber seu feedback. Faça download e utilize este lançamento e depois responda à pesquisa de nossa Comunidade de Aceitação. O prazo final para a pesquisa foi na quarta-feira, 16 de abril à meia noite do último fuso horário.

Projeto de tradução do NetBeans apresentado no SouJava

Wagner Santos fez uma palestra sobre o projeto de tradução do NetBeans para português na última reunião do SouJava na Globalcode - São Paulo. Wagner mostrou as vantagens de se utilizar o IDE em português e também explicou que com o IDE traduzido a adoção ao Java como ferramenta de desenvolvimento será maior por órgãos governamentais, e principalmente, por estudantes universitários e secundaristas. Conheça e junte-se ao projeto de tradução do NetBeans.


Treinamento

Construindo uma sala de bate papo com o componente Ajax Transaction Dynamic Faces

Este tutorial fornece instruções para construir uma sala de bate papo feita com Ajax utilizando a biblioteca dos componentes Dynamic Faces.

Criando uma aplicação web Ruby com o NetBeans 6.1

Aprendar como utilizar o suporte ao Ruby no NetBeans 6.1 para criar e executar uma simples aplicação web com banco de dados. Completando os passos deste tutorial, você aprenderá a fazer as seguintes tarefas: - Utilizar tarefas Rake e migração de arquivos para criar e atualizar tabelas de banco de dados - Utilizar o gerador de andaime para gerar uma aplicação web básica de banco de dados para criar, ler, atualizar, apagar (CRUD) - Editar visualizações para sintonizar melhor as páginas web

Guia de início rápido para o desenvolvimento de CLDC/MIDP no NetBeans

Aprenda o básico da utilização do NetBeans para criar uma aplicação Java™ Platform, Micro Edition (Java™ ME platform), Mobile Information Device Profile (MIDP). Este tutorial também oferece uma visualização rápida nas diversas características de mobilidade para o desenvolvimento CLDC/MIDP.
