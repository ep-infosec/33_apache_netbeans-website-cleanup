项目新闻

NetBeans IDE 6.8 Beta 已可下载！

NetBeans 团队高兴地宣布 NetBeans IDE 6.8 Beta 已经推出，这是第一个对整个 Java EE 6 规范提供支持的 IDE！Java EE 6 的亮点包括对 JSF 2.0/Facelets、Java 持久化 2.0、可在 Web 应用程序中使用 EJB 的 EJB 3.1、RESTful Web 服务，以及 GlassFish v3 的支持。此次发布还包括对 PHP、JavaFX、Kenai 项目、Maven、Ruby 和 C/C++ 的改进支持。

Kenai.com 上的 Java EE 和 MonkeyBars！

开发人员可以在 Kenai.com 上找到很多炫酷的项目，包括 Adam Bien 的 Java EE 模式和最佳实践，它是这个服务上成长最快的 Java EE 项目之一。如果你到了 MonkeyBars 里，请加入 Monkeybar 项目以了解如何用该库编写使用 JRuby 和 Swing 的 GUI 应用的更多信息。另外，如果想从 Kenai.com 发掘更多，请看看利用 Kenai Web API 的各种方式。

VisualVM 1.2 已可下载

VisualVM 1.2 包括许多新功能，包括：

  • 抽样的 CPU 和内存分析器
  • 重新设计的带动态工具提示的图表
  • 对 jstatd 连接的增强支持
  • 可定制的代理设置
  • 使用最新的 NetBeans 平台和 NetBeans Profiler 6.7.1
此次发布还扩展了插件的 API，修复了若干错误，并提供了日文和简体中文版。
