Edição # 1 - Nov 20, 2010

Novidades do projeto

A equipe do NetBeans tem o prazer de anunciar o NetBeans IDE 7.0 Beta

  • Download do NetBeans IDE7.0 Beta
  • Navegue pelos vídeos e tutoriais do NetBeans

O NetBeans IDE 7.0 Beta apresenta suporte ao desenvolvimento Java SE 7 com a plataforma JDK 7. O lançamento também oferece uma maior integração com o Oracle WebLogic Server, bem como suporte para o Oracle Database e GlassFish 3.1.

Outros destaques incluem Maven 3 e supote ao HTML5; um novo designer para o GridBagLayout para melhorar o desenvolvimento de telas Swing; suporte melhorado para a execução e depuração de aplicações Java EE autônomas; e melhorias no edtor Java. Destaques do lançamento:

  • JDK 7
    • Suporte ao Projeto Coin
    • Melhorias no editor: conclusão de código, dicas
  • WebLogic Server
    • Implementação simplificada e rápida para WebLogic
    • Novo nós de servidor em execução para exibir aplicações e recursos implantados
    • Integração JSF com bibliotecas do servidor
  • Banco de dados Oracle
    • Assistente de conexão simplificado
    • Instalação assistida para o driver JDBC
    • Edição e implantação de Stored Procedures
  • GlassFish
    • Suporte ao GlassFish 3.1
    • Reinício de domínio e visualização de log para GlassFish remoto
    • Ativar e desativar aplicativos implantados
  • Java
    • Suporte ao Maven 3
    • Integração JUnit 4.8.2 e várias melhorias do JUnit
    • Suporte à URLs HTTP remotas para Javadoc em bibliotecas e plataformas Java
    • Novo visual melhorado para o GridBagLayout
  • Java EE
    • Suporte melhorado para CDI, serviços REST e persistência Java
    • Novo suporte para validação de beans
    • Suporte para as bibliotecas de componentes JSF, incluindo bibliotecas PrimeFaces embutidas
    • Edição de Expression Language melhorada no JSF, incluindo a conclusão de código, refatoração e dicas
  • Linguagens Web
    • Suporte ao HTML5
    • Formatador JSON
  • PHP
    • Gerar phpDoc
    • Refatoração Renomear e Exclusão Segura
    • PHP 5.3 - Suporte para aliases
  • C/C++
    • Importação fácil de projeto já existente do usuário
    • Novo tipo de projeto quando a fonte de arquivos do usuário estão localizados remotamente
  • Plataforma NetBeans
    • Anotações para a geração de registros de Ação na camada
    • Melhorias de desempenho e integração com o Profiler
    • Mudanças adicionais na API do NetBeans


    • NetBeans IDE 7.0 Beta está disponível em Inglês, Português, Japonês e Chinês Simplificado, bem como em vários outros idiomas traduzidos pela comunidade. Visite a página NetBeans IDE 7.0 - Novidades e Destaques para obter mais informações e atualizações do desenvolvimento.


    A versão final do NetBeans IDE 7.0 está previsto para março de 2011. Entretanto, gostariamos que você fizesse download do Beta e nos dê seu feedback sobre o que achou. Você também pode participar de discussões sobre o NetBeans IDE em nossas listas de discussão e fóruns, e também siga o NetBeans no Twitter para atualizações da equipe.
