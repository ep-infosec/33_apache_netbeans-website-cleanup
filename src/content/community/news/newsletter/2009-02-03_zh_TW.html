期號 # 6 - Feb 03, 2009

專案新聞

NetBeans IDE 6.5 的 Patch 2 正式推出！

NetBeans.org 已釋出 NetBeans IDE 6.5 最新的修正檔。
此次修正了以下模組的錯誤：C/C++、資料庫、檔案編輯、GlassFish v3 Prelude、Groovy 與 Grails、Hibernate 支援功能、IDE 平台、Java、JavaEE、Java 調整器、JBoss 程式伺服器、Mercurial、原生式存取( Native Access )、NetBeans 6.5 PHP 平台、RESTful 網路服務、Ruby 與 Rails、Subversion、UML、視覺行動設計器( Visual Mobile Designer ) 及網路應用程式。此修正檔已有英文、日文、簡體中文、巴西葡語四種語言版本。

社群

凍蒜！ NetBeans 管理委員會提名開始。

又到了選舉 NetBeans 管理委員會的時候了。這個三人組成的委員會( 2 位社群成員與 1 位昇陽指定) 是為了確保 NetBeans 專案可以在一個公平、公開的制度下運作。下個半年任期的提名期為：1/28 至 2/11。按此連結得知如何提名。

訓練

Screencast：Wicket 虛擬導引

聽過 Wicket，卻不知道怎麼用它嗎？這次為您介紹的這個網路開發架構是不需要任何基礎的。在影片的最後，您會發現何以 Wicket 可與其他的架構相媲美；同時，您也會知道如何在 NetBeans 中使用之。


用 MMAPI混合 MIDI、音調與採樣音頻

這個教學是由 Vikram Gopal 來說明如何使用行動媒介API ( MMAPI )與 NetBeans 來同時混合、播放多媒體。


將 YUI ( Yahoo User Interface) 可排序資料表單加入 NetBeans 網路專案

感謝Troy Giunipero 在此示範如何把 Yahoo 資料表單整合進 PHP 與 Java 程式之中。而此方便性正歸功於 NetBeans 。

在 NetBeans 編寫 JUnit 測試

此教學介紹了一些在NetBeans 寫作與執行 JUnit 測試的基礎。使用者建立了 JUnit 3 和 JUnit 4 的元件測試，並測試了一個Java 類別函式庫專案。

小提示：一”鍵”之差

CVS ( 版本控制系統 )裡，變更版、最新版傻傻分不清楚？最後發現只是一個”空白”的差異。感謝 Hubert Ikkink 展示了如何開啟 NetBeans 中的”空白”辨識功能來分辨其中的差異。
