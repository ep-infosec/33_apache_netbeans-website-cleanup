Novidades do projeto

Junte-se ao NetCAT e ajude a melhorar o NetBeans 6.5!

Você gosta da qualidade do NetBeans e quer manter isto para os futuros usuários? Ou você acha que o IDE deveria resolver alguns problemas? Seja lá qual for a sua atitude você pode devolver algo para o seu IDE favorito ajudando a testar a versão 6.5 do NetBeans. Se você tiver ao menos 2 horas por semana para dispensar em Julho e Agosto, registre-se no NetCAT 6.5!

Aprenda a implantar, executar e testar um aplicativo composto com este tutorial que fornece uma visão geral de um simples aplicativo composto de processamento de empréstimos.

Características de edição de JavaScript no NetBeans

O NetBeans 6.1 oferece melhorias na edição de JavaScript modelada no suporte já existente para Java, Ruby e outras linguagens. Este guia fornece uma visão geral destas características como a coloração de sintaxe, marcação de ocorrências, renomeação instantânea, refatoração e muito mais!

Construindo relatórios com o iReport da JasperSoft

Giulio Toffoli, arquiteto do iReport e fundador do projeto, guia-o através de uma demonstração de como criar relatórios utilizando o NetBeans e o plugin do iReport. Guilio também decidiu recentemente mover o projeto iReport para a plataforma NetBeans. Ouça o que ele tem a dizer sobre porque o NetBeans foi a melhor escolha.

Guia de configuração da Plataforma de Emulação CDC do NetBeans

O NetBeans para dispositivos móveis suporta o desenvolvimento JavaME CDC, mas carece de uma plataforma CDC embutida. Confira os passos para baixar, instalar, e registrar o Toolkit Sun Java para CDC e outros Kits de desenvolvimento de Software (SDK) populares para o desenvolvimento CDC.
