Agenda

Veja o NetBeans em ação na turnê "Java na estrada"!

Java na estrada: Códigos do litoral é uma turnê que cruzará 20 cidades nos Estados Unidos mostrando o compromisso da Oracle para tudo o que for relacionado ao Java. Venha assistir os técnicos Java demonstrarem o mais recente software Java, como o IDE NetBeans! A primeira parada do passeio é Nova York, em 14 de junho. Visite a página da turnê para conferir os destinos e datas perto de você.

Novidades do projeto

As 5 mais da plataforma NetBeans 6.9

Os desenvolvedores de aplicações desktop Java Swing podem conferir as últimas melhorias na Plataforma NetBeans 6.9, tais como melhorias na API, mais exemplos de projetos, e um novo Guia do Desenvolvedor da Plataforma NetBeans. Veja os detalhes no blog do Geertjan.

Suporte ao OAuth no NetBeans IDE 6.9

OAuth é um protocolo de segurança que permite aos usuários autorizar pedidos de cliente para acesso à seus recursos web. O engenheiro do NetBeans Milan Kuchtiak demonstra o suporte ao OAuth no NetBeans usando um aplicativo cliente do Twitter.

Blogs

Introdução criando editores de fluxo de trabalho

Este tutorial mostra como iniciar um aplicativo editor de fluxo de trabalho com a Plataforma NetBeans. O aplicativo irá mostrar uma cena onde você pode criar novos nós e arrastar os nós para conectá-los.

JavaFX Composer com suporte à Grid Container

O JavaFX 1.3 traz o novo contêiner de grade que facilita a organização dos controles em formulários. Este blog apresenta uma visão geral de como tirar o máximo desta nova funcionalidade do JavaFX Composer no NetBeans.

Treinamento

Aplicando anotações @Alternative

Este tutorial demonstra como utilizar a anotação @Alternative para configurar aplicações para diferentes implantações, e também mostra como usar as anotações do ciclo de vida gerenciado de um bean.

Tutorial sobre o Plugin Code Coverage

O plugin do NetBeans Code Coverage fornece uma maneira interativa para ver os resultados compreendidos nos testes dentro do NetBeans. Isso permite que aos desenvolvedores identificar rapidamente as partes de código Java que não são abrangidos por seus testes unitários.

Dicas para desenvolver aplicações JavaFX TV

A natureza da plataforma de televisão afeta a maneira como você cria interfaces eficazes para funcionar nela. Este artigo descreve algumas das maneiras em que a plataforma de televisão difere da plataforma desktop e oferece dicas e orientações para a criação de aplicações eficazes para TV.
