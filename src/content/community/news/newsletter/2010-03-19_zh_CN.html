用 WinSCP 进行远程 C/C++ 开发

如果你没有在 Windows 上安装任何 C 编译器，只需设置 NetBeans 去连接 Linux 机就可进行远程 C/C++ 开发！用这种方式，你仍可以从诸如语法高亮和调用图导航的可视化 IDE 功能中得到好处。从这篇博客中了解更多。

技术提示：用于 CSS 文件的转到声明

即将到来的 NetBeans IDE 6.9 将允许你从 HTML 标记的任何 class 或 id 属性导航到 CSS 类或 id 选择器声明。将鼠标指针移到 class 或 id 选择器上方，然后 Ctrl-单击就可以打开包含了该声明的 CSS 文件。
