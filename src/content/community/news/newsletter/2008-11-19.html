Project News

NetBeans IDE 6.5 Now Available for Download!

NetBeans.org is proud to announce the availability of NetBeans IDE 6.5!

NetBeans IDE 6.5 offers simplified and rapid development of web, enterprise, desktop, and mobile applications with PHP, JavaScript, Java, C/C++, Ruby, and Groovy. The release is currently available in English, Japanese, Simplified Chinese and Brazilian Portuguese. Supported by a vibrant developer community and offering a diverse selection of third-party plugins, the NetBeans IDE is a must-download for developers!

Training

Screencast: Introduction to NetBeans IDE 6.5

Discover the NetBeans IDE 6.5 editor features and get a short introduction on how to edit, compile and run Java programs, and how to use the IDE's productivity features.


Screencast: Building a Flickr Slideshow with PHP

This demo shows off the new PHP features of NetBeans 6.5 by walking through building a simple application to access Flickr. You will then see how to turn your Flickr images into a simple slideshow using JavaScript and CSS.


JavaScript Debugger User's Guide for NetBeans IDE 6.5

The NetBeans JavaScript debugger provides a set of tools that help to inspect, monitor and edit JavaScript code. It also offers a common user experience for both Firefox and Internet Explorer. This tutorial demonstrates how to debug JavaScript functionality in web applications using the NetBeans IDE 6.5 debugger.

Faster Iterative Java Web App Development with GlassFish V3 and NetBeans 6.5

Vince Kraemer demonstrates how web app developers using Java can leverage some of the new features of NetBeans 6.5 and GlassFish V3 to increase their productivity.
