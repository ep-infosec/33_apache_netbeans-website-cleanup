OpenLaszlo un plugin pour l'EDI NetBeans disponible

OpenLaszlo est un framework web d'application Riche Internet basé sur Flash qui permet aux dévéloppeurs de construire des applications RIA en utilisant XML et JavaScript sans aucune connaissance de Flash. Le plugin OpenLaszlo pour l'EDI NetBeans est fournit avec les librairies OpenLaszlo 4.1.1 intégrées et inclus des applications démos, des modèles, le support du fichier .lzx, et plus. Le plugin fonctionne avec les EDI NetBeans 5.5.1, 6.0 and 6.1.
