網路研討會： 使用 NetBeans， JRuby-on-Rails 與 GlassFish 來開發以 MySQL 作為資料庫的 Java 應用程式

一起來加入 Arun Gupta， 昇陽公認的"GlassFish 玩家"， 在這個網路研討會中， 他將會帶領我們使用最新版本的 NetBeans IDE，加上 JRuby-on-Rails 框架，來快速並簡單地開發和部署以 MySQL 及 GlassFish 為基礎的應用程式。 何時開始： 2009 年 3 月 26 日， 星期四， 早上 10 點(太平洋夏令時間)， 下午 1 點(東部夏令時間), 或是 17:00 (格林威治標準時間)。 (此次的研討會大約有 45 分鐘左右，並緊接著 Q&A。)

訓練

建立一個 RESTful 的網路服務以及 JavaFX 客戶端程式

學習如何使用 NetBeans IDE 與 GlassFish 和 MySQL 來建立一個 RESTful 網路服務。 您也將會學到如何使用 NetBeans IDE 的 JavaFX 支援來執行該網路服務的 JavaFX 客戶端程式。

學習如何部署 PHP 到 OpenSolaris 伺服器端的網路堆疊

這個入門教學展示了如何以另一種方式在您的電腦上設定 PHP，MySQL，Apache 以及 XDebug。 並且教您如何部署 PHP 應用程式到在 Virtual Box 中， 安裝在 OpenSolaris 上預先設定好的伺服器端 PHP 網路堆疊。

NetBeans 平台 CRUD 應用程式開發入門
