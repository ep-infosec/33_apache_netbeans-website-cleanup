Issue # 700 - Aug 11, 2015

Project News

Released: NetBeans IDE 8.1 Beta

With new Node.js functionality as its key feature, NetBeans IDE 8.1 Beta has been released. Look through the release notes and try out (free) NetBeans IDE 8.1 Beta today!

Calendar

August: NetBeans in Aachen

EuregJUG Maas-Rhine, the cross-border, English-language Java User Group announces: "We are happy to announce our second talk: 'Free Open Source Tools for Maven, HTML5, IoT, and Java EE' by Geertjan Wielenga." Come to this event on 25 August and find out what's new in NetBeans IDE.

September: NetBeans in Oslo

Attend this session at JavaZone in Oslo during 9 and 10 September if you want to quickly kickstart your understanding of the entire JavaScript ecosystem in the space of an hour, with demos done in NetBeans IDE.

October: NetBeans in London

How can you make sure that your next Web application supports not only the desktop browser but also all the other devices and screen sizes? And that it can efficiently talk to the back-end services in a bidirectional way? Find out all the answers at JAX London on 14 October.

Plugin

Markdown Support

Learn about a NetBeans IDE plugin that adds Markdown support. The plugin provides basic syntax highlighting, an HTML preview, and HTML export when you are editing Markdown documents.

Blogs

Getting Started with Java EE

In a few simple steps, Michael Muller shows you how to get started creating a Java EE web application in NetBeans IDE.

Maven and Java EE

David Delabassee reports that "it's probably fair to say that Maven has been, at least in the Java landscape, one of the main actors" in the build system ecosystem. He then refers to an article showing how Maven and Java EE are deeply integrated in NetBeans. Read it to find out why NetBeans is the best of breed choice for Maven developers.

Node.js and JavaScript

Join us, Node.js developers everywhere, NetBeans is a lightweight and free tool with a special new focus on Node.js application development! Node.js is the key feature of NetBeans IDE 8.1, which has been released in its Beta release. Find out the basic features of how Node.js works in NetBeans IDE 8.1.
