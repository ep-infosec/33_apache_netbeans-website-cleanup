项目新闻

Portal Pack 3.0 Beta现在发布了

Portal Pack 3.0 Beta二进制文件现在可以下载了。这些版本已经解决大的bug，增强了对Liferay Portal Server在tomcat中运行的目录部署和调试支持。并且，下面这些服务器现在也被支持： • Project WebSynergy • Liferay Portal Server 5.1.x and later • Sun Java System Portal Server 7.1/7.2 • OpenPortal Portlet Container

NetBeans赢得Developer.com的2009产品奖

NetBeans以在Developer.com2009年度奖产品中赢得五个奖项开始了2009年！NetBeans 在这些类别中获得了最高的投票：Java Tool, Open Source Tool, Development Platform, Development Utilities, and Wireless/Mobile Tool. 一个大的“Thank you”献给NetBeans社区和支持者以感谢帮助赢得这些奖项！
