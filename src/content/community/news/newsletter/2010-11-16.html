Patch2 for NetBeans IDE 6.9 Now Available

The NetBeans team has released a new patch, which is an update to NetBeans IDE 6.9.1. The patch includes bug fixes in modules for C/C++, IDE Platform, Java, Java Debugger, Java Web Applications, Maven, Mobility, OSGI, PHP, and RCP Platform.

To obtain the fixes, the NetBeans IDE must be installed and running. You can also download the fixes through the IDE's Plugin Manager.

Oracle Technology Network (OTN) Developer Day is a free developer event specifically designed for the enterprise Java developers, to allow all attendees to learn how to enrich their development experience with the latest developments in the Java technology.
