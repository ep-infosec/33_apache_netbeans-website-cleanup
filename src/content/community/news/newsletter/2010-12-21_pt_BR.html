Novidades do projeto

Melhorias de desempenho no editor PHP do NetBeans 7.0

A equipe NetBeans PHP implementou correções para problemas de desempenho no editor PHP do NetBeans 7.0. (Baixe o nightly builds para obter as mudanças.) Para continuar a melhorar o editor, a equipe quer receber mais casos de uso sobre o desempenho. Leia o blog da equipe para saber mais.

Suporte ao Cobertura para projetos Maven no NetBeans IDE

Agora o NetBeans suporta a cobertura de código para projetos Maven através do plugin Cobertura Maven. Aprenda a exibir testes cobertos no IDE, e em aplicações RCP NetBeans no Maven.

Plugin: Notificador Gmail

Nova versão do Kojo disponível (Ambiente de aprendizagem em Scala)
