Issue # 538 - May 21, 2012

Project News

jQuery Support in NetBeans IDE

jQuery support is coming to the NetBeans IDE. Get a preview of the new feature and find out how to try it in the IDE.

JavaFX, SceneBuilder and NetBeans IDE

What else is coming up in NetBeans IDE 7.2? Scene Builder integration in JavaFX support. Toni Epple gives a short overview of what this new feature offers.

Training

Free Workshop: Modern Java Development (May 30, California)

Stop by Oracle HQ in Redwood Shores, California, for a FREE day of sessions and hands-on labs that expose you to the world of Java. Discover the productivity that Java EE 6 provides; get a sneak peek into Java EE 7 and how it enables the cloud; and learn about Java 7, 8 and JavaFX 2.0 and more.

Webinar - Embedding MySQL within MySQL: Connector/J Tips

In this on-demand webinar, Java and MySQL expert Mark Matthew shares tips on how to make your Java application's MySQL database high performing, zero-admin, and fault tolerant.

Tutorials

JavaFX 2.0 Game Tutorial Series

This multi-part tutorial series takes users through the process of developing a game using JavaFX 2.0.

Vert.x Project with Maven and NetBeans 7.1

Learn how to set up a Vert.x (Mod) project to work with Maven and NetBeans IDE 7.1 on Mac OS X and Java 7.

Articles

Getting Started With RESTEasy and NetBeans IDE 7.1

RESTEasy allows users to easily and quickly create REST based services with Java. Developer and blogger Chad Lung shows how to get a minimal REST service running using NetBeans IDE 7.1 and RESTEasy.

NetBeans Usability Tips

What are your favorite NetBeans tips? Developer and blogger Dustin Marx shares a few tips and tricks that he finds useful, as well as others compiled by NetBeans users and developers around the blogosphere.
