Introdução à Plataforma NetBeans usando Maven

Uma demonstração rápida de como criar um aplicativo simples da Plataforma NetBeans e um módulo usando Maven para construção. Você vai usar os arquétipos do Maven, o conjunto de ferramentas UI Swing e "Matisse" GUI Builder. O aspecto modular da Plataforma NetBeans permite desenvolver e expandir a funcionalidade do seu aplicativo simplesmente adicionando novos módulos.

Componentes Composite - Java EE 6 leve

Ligação de dados com JavaFX Composer

Este tutorial mostra como definir ligações JavaFX usando o JavaFX Composer no NetBeans IDE 6.9. Você vai aprender como criar um aplicativo que permite aos usuários inserir um texto que contém itens de lista separados por vírgulas. A entrada do usuário será automaticamente analisada e carregada à uma exibição de lista utilizando uma ligação JavaFX.

Dica técnica: Melhorias no suporte ao framework PHP

Na próxima versão do NetBeans IDE 6.9, o auto-completar de código em exibição de arquivos (ou modelos) agora também oferece variáveis do controlador (ou ações). Este novo recurso funciona tanto para Symfony quanto para Zend Frameworks. Pessoas que tenham usado uma solução semelhante vão querer dar uma olhada nesta nova funcionalidade.
