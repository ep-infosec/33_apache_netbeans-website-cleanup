Novidades do projeto

NetBeans 6.1 disponível!

NetBeans.org se orgulha em anunciar a disponibilidade do NetBeans IDE 6.1! O NetBeans 6.1 suporta uma extensa faixa de tecnologias de scripting de código aberto e oferece melhorada performance. Esta entrega estende o suporte a linguagem além da tecnologia Java, fornecendo um conjunto rico de funcionalidades para C/C++, JavaScript e a linguagem Ruby, incluindo Ruby on Rails. Os destaques desta versão incluem: * Suporte de edição JavaScript avançada * Melhoria de performance (até 40% mais rápido na inicialização) * Suporte embutido ao Framework Spring * Suporte integrado para serviço web RESTful * Novo suporte ao MySQL no explorador de banco de dados * Melhorias significantes do suporte ao Ruby/JRuby Aprenda mais sobre a versão 6.1 do NetBeans: NetBeans IDE 6.1 Página da versão NetBeans IDE 6.1 Tutoriais e documentação NetBeans IDE 6.1 Lista de características Como sempre, nós gostariamos de receber seu feedback em nossas listas e em seus blogs. Descubra porque o NetBeans 6.1 é o único IDE que você precisa: Faça o download hoje mesmo!

O projeto Greenfire de Adam Bien ganhou o prêmio de inovação JAX

O especialista em Java EE e membro do Time dos Sonhos do NetBeans, Adam Bien, é um dos vencedores do prêmio JAX! O projeto GreenFire, sua aplicação para controle eficiente de energia dos sistemas de aquecimento, ganhou o quarto prêmio e o prêmio de audiência. Parabéns Adam! Vai participar do JavaOne e quer saber mais sobre este projeto? Então verifique a sessão de Curiosos do Assunto (BOF) do Adam no qual ele fala sobre o GreenFire e a extensibilidade do JavaEE.
