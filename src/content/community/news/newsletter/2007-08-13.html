Issue # 304 - Aug 13, 2007

Calendar

Register for NetBeans Day Boston - September 11

The first stop on the NetBeans worldTour 2007-2008 is Boston! Attend NetBeans Day Boston and discover the latest in developer tools, and how NetBeans can boost your productivity regardless of what type of Java technology development you do or what language you work in. Meet other NetBeans users and experts, and see demos of the next generation of NetBeans software. Learn more about what to expect on NetBeans Day, and remember: Registration is free!


NetBeans at Javagruppen (Denmark) - August 20 & 23

Join NetBeans evangelist Roman Strobl in Aarhus (August 20) and Copenhagen (August 23), Denmark, for an introduction to the NetBeans IDE. Discover the newest features within the NetBeans IDE for desktop, web and mobile development, and watch demos that highlight these features in action.

Project News

Developer Edition of NetBeans OpenGL-Pack Released

A developer release of the OpenGL-Pack for the Netbeans 6.0 IDE is now available for download. Features include: OpenGL Shading Language Compiler/Linker integration and syntax highlighting, Compile and Link arbitrary shaders, and more. Check out Adam Bien's blog entry for additional information about the release.

Community

NetBeans Platform: Extending JFugue Music NotePad

When Michal Fapso, a student at the Faculty of Information Technology in the Czech Republic, needed a project to work on for his term paper on Graphical User Interfaces in Java, he found the open source JFugue Music NotePad and extended it with very powerful editing features. He received full marks, and contributed his code back to the project. And in the process, he learned about the NetBeans Platform. Read all about it in Geertjan's blog.

NetBeans Podcast Episode #33

Tune in to hear Roman and Gregg talk about everything from the status of NetBeans 6.0 to bugs in the JDK. There is also discussion of Ruby, freeform projects, project Jarvis, and more!

Training

Using Java Persistence API Within a Visual Web Application

Learn how to combine a visual web application in the NetBeans 6.0 IDE with the Java Persistence API. This article shows you how to simultaneously use the Java Persistence API and visual web components to retrieve database records and display and modify the data.


Integrating & Using SQE Plug-in for NetBeans IDE 6.0

A tutorial contribution to Project Grizzly from Antek Baran: Learn how to setup and use NetBeans IDE 6.0 with the SQE plug-in. The SQE plug-in contains the popular static analysis tools: FindBugs, PMD and CheckStyle along with Dependency Finder.

The Divas Write About jMaki 0.9.7

Release Candidate 2 (0.9.7) of the jMaki framework was released on August 10. The Tutorial Divas have posted two blog entries that show how to use the new data model in RC2 to dynamically load data from a database, use the JSON API to format it into the data model format and load it into a jMaki widget that uses that particular data model. In their first blog entry, they use the Spry Accordion widget in a Visual Web application. In their second entry, they show how simple it is to swap in the Yahoo Tabbed View widget.

Blogs

Creating "Pretty" URLs with NetBeans and Visual Web Pack

Ever noticed those ugly URLs that JavaServer Faces produces when you navigate around a JavaServer Faces web application? Check out David Botterill's blog entry in which he writes about turning those ugly URLs into pretty ones.

Double Feature: Two Screencasts from Arun Gupta

Screencast #Web5: Creating Mashups with jMaki - Mashing up the Maps shows how jMaki allows you to embed and interact with map widgets in your application using NetBeans IDE. In Screencast #Web6: First JRuby app in GlassFish, see how a JRuby application can be developed using NetBeans IDE and deployed on GlassFish.

Perspectives in the NetBeans IDE

Shakthi Anuradha has created a plugin module for NetBeans IDE 6.0 that provides a perspectives feature. It allows you to easily save and restore window positions in the IDE. His blog entry includes a link to a Flash demo that shows it in action.
