Creating web service clients of various types is a breeze in NetBeans. Winston Prakash shows how easy it is to create a client for a location finder service and Geertjan Wielenga shows how to integrate the client into a NetBeans module. Read more ...

Tutorial: Performing Inserts, Updates & Deletes

This tutorial shows you how to use NetBeans Visual Web Pack 5.5 to build a web application that can create, retrieve, update, and delete database rows. The application provides a drop-down list of master data along with a synchronized detail table. Users of the application can add to, update, and delete the records in the detail table and from its associated database. Read more ...
