Java SE 7：回顾 JVM 性能命令行工具

Java SE 7 包含的一些工具可以让您来监视 Java 虚拟机（JVM）。在本教程中，学习使用 jps 命令和新 jcmd 命令来获取有关系统当前正在运行的 JVM 信息。然后，使用这些信息与 jstat 可监视本地和远程的 JVM 垃圾回收状态。
