Issue # 500 - Aug 16, 2011

Project News

Still Time to Do the NetBeans 7 Satisfaction Survey!

Thank you to all those who have completed the NetBeans 7 Satisfaction Survey! If you haven't done so yet -- you are very strongly encouraged to complete it too, only about 5 minutes should be needed, as this survey gives you a real opportunity to give direction to the NetBeans project.

Community

Dream Team: "First Patch" Program

Tom Wheeler, from the NetBeans Dream Team, which is a group of external NetBeans enthusiasts, has now officially announced the first patch program, aiming to help get you started contributing patches to NetBeans IDE.

Articles

AgroSense Farm Manager 1.0.9

A new release of AgroSense Farm Manager, the Maven based NetBeans Platform project. This time, a Flamingo-based toolbar is included. Do you want to contribute to the project too? It is open sourced and you're welcome to participate.

NetBeans API: org.netbeans.editor.SideBarFactory

In a series introducing lesser known NetBeans APIs, the SideBarFactory is shown in action, based on code from Andrzej Olszak, a NetBeans Platform developer from Denmark.

Blogs

PHP Blog: Keep-Alive for Internal S/FTP Client

The NetBeans PHP team have yet another announcement in their very active blog on new PHP features. The upload-on-save functionality is now much more faster and more friendly for your remote servers!

JUnit in NetBeans IDE 7.0

What's new with JUnit support in NetBeans IDE 7.0? An Italian blogger tells you all about it.

Jaroslav Tulach: The Story of NetBeans IDE 7.0.1

NetBeans API architect discusses some interesting problems that arose around the NetBeans IDE 7.0.1 bugfix release.

NbAndroid

Tonny Kohar from Indonesia gets you up and running with Android development in NetBeans IDE!
