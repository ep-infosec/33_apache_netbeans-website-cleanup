Issue # 325 - Jan 22, 2008

Calendar

iReport Webinar: Building Powerful Reports in Minutes (Jan. 23)

Recently, JasperSoft announced the integration of iReport and the NetBeans 6.0 IDE. Now, in a one-hour webinar with iReport creator Giulio Toffoli, get an overview of the capabilities of iReport, recent enhancements and future plans, and a demo of the forthcoming iReport for NetBeans. Register today!

Free Java EE 5 Architectures & Design Talk in Munich (Jan. 24)

Join Adam Bien and Munich's Java User Group and NetBeans User Group for a discussion and demo of patterns, best practices and approaches for developing maintainable Java EE 5/SE applications using NetBeans 6.0.

Project News

Ideas for Dapper's NetBeans Plugin Competition

By now you know of Dapper's NetBeans Plugin competition. But are you still looking for inspiration? Geertjan Wielenga has three scenarios that you can implement for a chance to win the competition's cool prizes. Check it out, and please thank Geertjan when you win.

Call Graph for C/C++ Code Now Visible in NetBeans

An experimental version of the C/C++ Call Graph module is now available for download from the NetBeans Plugin Portal. The team responsible for the module also wants your feedback. Send comments to Alexander.Simon@sun.com or users@cnd.netbeans.org.

Community

Three Wins for NetBeans in Developer.com Product Awards!

The NetBeans IDE won the Development Tool and Java Tool categories in the Developer.com Product of the Year 2008 Awards! In both categories, NetBeans' out-of-the-box experience stood out for nominators, as did improvements to version 6.0. The NetBeans Mobility Pack also nabbed a win as the top Wireless/Mobile Development Tool. Congratulations to NetBeans!

NetBeans Podcast Episode #39

Tune in to listen as Roman and Gregg discuss the latest, including: cool new plugins for NetBeans 6.0, some of the features in NetBeans 6.1, upcoming Webinars, their favorite databases, and much more.

Training

Developing Secure Applications With Sun Java System Access Manager & NetBeans IDE

This tutorial describes how to extend the authorization of Sun Java System Access Manager with a hierarchy-based resource model and interoperable Web services for .NET applications. The NetBeans IDE is used for development, deployment, and testing of the service endpoint.

Updated: Create a Rails 2.0 Ruby Weblog in 10 Minutes

The tutorial divas have posted an update to the popular Ruby Weblog in 10 Minutes tutorial; it shows how to use the new Rails 2.0 framework from the NetBeans IDE.

Using Client-side Rendering Features in Selector Components

In this tutorial, use the NetBeans Visual Web JSF framework to create an application that uses a multi-select listbox. When you choose one or more items from the listbox, the application shows the selected items in a text area using Ajax functionality.

Blogs

Cobertura and Hudson with NetBeans RCP Projects

Platform developer Fabrizio Giudici blogs about how to integrate NetBeans RCP projects with Cobertura and Hudson.

New: NetBeans Support Blog

The NetBeans Support Team has started a blog that many NetBeans users can turn to for useful tips & tricks, issue workarounds and product updates. Recent entries include a compilation of links for Visual Web JSF Development and a quick guide for tracking NetBeans 6.0 plugin owners. Make sure to check it out.
