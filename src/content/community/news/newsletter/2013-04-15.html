Project News

SASS and LESS Support in NetBeans Editor

Current development builds of NetBeans IDE now support two CSS frameworks in the Editor: SASS and LESS.

Newly Released: Ad-hoc Project Plugin

This plugin lets you treat any folder as a project; it provides a Favorites node that collects frequently edited files and a way to search files in sub-folders by type. No set-up or configuration is needed.

Handling Forms in Java EE 6 by Using Servlets and CDI

This tutorial shows the basics of Contexts and Dependency Injection (CDI) for Java EE 6 by developing a sign-up form that uses servlets to process its data and inject Java beans.
