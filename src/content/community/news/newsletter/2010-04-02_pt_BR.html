Disponível o plugin ICEfaces para NetBeans IDE 6.8

O ICEfaces lançou recentemente um plugin de integração ICEfaces para o NetBeans IDE 6.8. O versão inclui:

  • Suporte para ICEfaces 1.8.2a no Java EE 5 ou projetos J2EE 1.4 (JSF JSF 1.1 ou 1.2)
  • Suporte para ICEfaces 2.0 Alpha 2 em projetos JavaEE 6 (JSF 2.0)

    Saiba mais e obtenha tutoriais no site do ICEfaces.

    • Plugin de navegação LDAP para o NetBeans

    Uma dica do blogueiro Mathieu: o desenvolvedor Allan Lykke Christensen lançou um plugin para o NetBeans de um navegador LDAP. O módulo fornece uma interface simples e rápida para a interacção com os serviços de base LDAP. Saiba mais sobre o projeto aqui .
