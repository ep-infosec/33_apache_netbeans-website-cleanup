技术贴士: 组织自动导入声明的插件

NetBeans 平台认证培训的一个学生创建了一个新的插件, 它可以帮助你识别导入(import)声明. NetBeans IDE 6.7.1 或者 6.8 中都可以使用该插件.

使用 Java DB 进行 JavaFX 的数据库编程

谁说 Java 的 JDBC 编程是困难的? 使用 JavaFX 创建多媒体程序时比使用 Swing 时的代码要少得多, 而且你还可以调用已经存在的 Java 代码库. 这也包括了使用 JDBC 来连接数据库, 但是有些技巧你还是需要注意的, 我们会向你讲述.

DZone: OSGi 开发者的新的酷的工具

NetBeans IDE 跟着下来的 6.9 版本为 OSGI 开发者带来了新的工具, 特别是那些使用 Swing 作为界面的开发者. NetBeans Platform 6.9 明白 OSGi bundle 是什么, 而你可以将 OSGI bundle 导入到你的 NetBeans 平台程序中.

技术贴士: 打包的乐趣

在使用 NetBeans 来创建一个 JavaFX 程序时, 它创建了一个 JNLP 文件, 一个 JavaFX JAR 文件, 和一个在浏览器中运行的默认的 HTML 文件. 这对于以单独方式或者在浏览器中运行是不错的, 但你想将它打包成一个程序并部署到像 Glassfish 那样的服务器时, 你应该怎样做呢? Matt Warman 分享了他的一些经验.
