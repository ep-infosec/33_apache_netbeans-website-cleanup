Terminal para o Yahoo Query Language no NetBeans IDE

Wagner Santos, desenvolvedor do plugin YQL Console para o NetBeans ajuda os desenvolvedores a testar queries e funcionalidades utilizando a API do Yahoo.

Visualizador de dados sísmicos na Plataforma NetBeans

Uma entrevista com Andrey Ilinykh, arquiteto de software para INTViewer, uma plataforma baseada no NetBeans para visualização, análise e verificação da qualidade dos dados sísmicos.

Treinamento

Trabalhando com eventos no CDI

O NetBeans IDE 6.9 fornece suporte embutido para Contextos e Injeção de Dependência (CDI). Este tutorial demonstra como integrar eventos Java EE em um aplicativo com o CDI usando o IDE.

Introdução à Plataforma NetBeans usando OSGi

Este documento é uma demonstração rápida de como integrar pacotes OSGi em uma aplicação da plataforma NetBeans 6.9. Você vai importar os pacotes OSGi constituindo um jogo de Sudoku, juntamente com os respectivos pacotes Equinox necessários.

Usando controles UI do JavaFX

Aprenda a criar facilmente interfaces de usuário com os novos controles UI do JavaFX e como melhorar essas interfaces aplicando CSS.

Introdução ao JavaFX Composer

JavaFX Composer é uma ferramenta gráfica de manipulação de layout que lhe permite criar rapidamente aplicações GUI JavaFX. Este tutorial oferece uma introdução rápida à construção de aplicações JavaFX utilizando o editor visual Composer.

Funcionalidade: Valores de constantes PHP

No NetBeans IDE 6.9, os valores de constantes do PHP será exibido na janela Navegador, na documentação e janelas de auto-completar de código. Esta nova funcionalidade está disponível para testes nas compilações diárias de desenvolvimento para NetBeans 6.9 para PHP.
