Baixe o JavaFX Preview SDK

JavaFX Preview SDK é uma pré-estréia da plataforma JavaFX. Destinado a criadores de scripts para web e iniciantes, ele oferece uma prévia do runtime e as ferramentas necessárias para criar incríveis Aplicações Ricas de Internet (RIA - Rich Internet Applications) na plataforma Java(TM). O pacote liberado inclui o Preview SDK, NetBeans IDE 6.1 com JavaFX, o Projeto Nile, e o Java Runtime Envirionment 6 Update 10 Beta.

Iniciando com o monitoramento JMX no NetBeans

Veja uma pequena introdução de utilização da tecnologia JMX no NetBeans.
