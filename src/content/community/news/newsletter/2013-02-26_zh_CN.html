NetBeans IDE 7.3 发布

NetBeans IDE 7.3 诞生了！7.3的发布使开发人员能够创建和调试基于最新的 HTML5，JavaScript 和CSS3 标准的 Web 和移动应用。开发人员可以使用及时页面预览器查看丰富的 Web 和移动应用开发页面的预期效果，Chrome 浏览器集成，CSS3 样式辑器，基于最新的 Nashorn JavaScript 引擎重写的 JavaScript 编辑器，一个新的 JavaScript 调试器等等。
