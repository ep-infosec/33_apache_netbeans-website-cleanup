Training

Top 10 Interesting NetBeans IDE Java Shortcuts

Here's a grab bag of things that Geertjan Wielenga just has added to the shortcut card (some new, some old that hadn't been included before) that you might find interesting too.

I/O Profiling for C/C++ Projects on Solaris

The I/O Usage tool in the NetBeans IDE shows a graphical view of the reads and writes of your C/C++ project. You can run the IDE locally on your PC or Mac and set up your project to use a Solaris system as the remote development host to benefit from this DTrace feature.

Creating a Touch Enabled SVG UI for Java ME Devices

This tutorial shows you how to develop rich user interfaces for Java ME based touch screen devices. The Rich SVG UI widgets described here are supported on devices with or without touch screens.

Community

Java Mobility on Mac, Solaris, and Linux

Karol Harezlak's Microemulator enables basic support for Java ME CLDC development in NetBeans on Solaris, Mac OS and Linux. The project has found a new home on dev.java.net -- here you find download links as well as installation instructions.

Next Phase for NetBeans PoWeR

NetBeans PoWeR (Platform Wapper Repository) is a semi-experimental initiative for providing pre-cooked NBM artefacts for the NetBeans Platform, and wrapping some popular third-party libraries, similar to Eclipse Orbit. Have a look and tell community member Fabrizio Giudici what you think!

New Cool Flashy NetBeans Platform Video!

Gephi is a visualization and exploration platform for all kinds of networks and complex systems, dynamic and hierarchical graphs. Swing and the NetBeans Platform in action, pretty cool!

Blogs

JavaFX Composer Blog

Follow the NetBeans JavaFX Composer blog and be the first to know about new IDE features. It's also a good place to try fun demos and share the latest code samples. Leave a comment if you wrote a cool JavaFX sample application and want to donate it to our samples catalog!

Code Completion for Colors in CSS

We listened to your feedback and added another improvement to the NetBeans IDE for PHP: The code completion pop-up for the CSS color and background properties now includes a color chooser.

Articles

Implementing the Save Action

Tim Boudreau updated his Paint application sample and improved the "Save" action. Rather than creating your own action for saving the canvas, you can now integrate with the standard NetBeans Platform Save functionality. The code works with the NetBeans Platform 6.8 and 6.9.
