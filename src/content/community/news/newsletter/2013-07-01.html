Issue # 594 - Jul 01, 2013

Project News

Video: Preview of NetBeans UML Plugin

Led by Zoran Sevarac, the Open Source software team at the University of Belgrade is developing a NetBeans UML plugin. This screencast shows a preview of features available in the plugin.

Plugin: NBAndroid

NBAndroid provides support for the Android application development cycle. Its functionality covers a variety of aspects of Android development directly in NetBeans IDE.

Tutorials

Using WebSocket for Real-Time Communication in Java Platform, Enterpise Edition 7

This tutorial and video series shows you how to create an application that uses the WebSocket API for real-time communication between a client and a server.


Books

Newly Published: jMonkeyEngine 3.0 Beginner's Guide

Aspiring game developers can learn to build modern 3D games with Java in this new primer on 3D programming, packed with best practices, tips and tricks, and example code.


Articles

Java EE 7 Support in NetBeans 7.3.1

NetBeans IDE provides full support for Java EE 7, with tools, templates, and samples for building EE 7 applications. Arun Gupta outlines what's new in the latest NetBeans update release.

Features for the NetBeans Platform using JavaFX and CSS!

This article demonstrates several tricks to visually enhance a NetBeans Platform app with JavaFX controls embedded. It builds upon previous tutorials that demonstrate complete replacements for the default NetBeans Swing components using JavaFX. Code examples feature NetBeans Platform, JavaFX and CSS.

Released: Open Source Nautical Navigation Software on the NetBeans Platform

NaVisu is an open source nautical navigation software created on the NetBeans Platform. You can read your nautical charts, connect to a GPS, and display many beautiful JavaFX widgets on the top of the application.
