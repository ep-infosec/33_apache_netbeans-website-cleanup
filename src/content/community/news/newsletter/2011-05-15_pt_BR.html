Novidades do projeto

Plugin Ruby on Rails no NetBeans

O primeiro plugin Ruby mantido pela comunidade já está disponível a partir do Portal de Plugin do NetBean. Ruby para NetBeans fornece todos os elementos básicos esperados de uma completa IDE Ruby, e também fornece suporte para o framework Ruby on Rails.

Plugin NetBeans: Visualizador PDF e Bookmarking

Este plugin adiciona suporte ao PDF para o NetBeans IDE, oferecendo um completo visualizador PDF Java e permitindo que aos usuários criem marcadores para fácil acesso e até mesmo para abrir PDF na Internet.

Plugin Griffon para o NetBeans IDE 7.0

O plugin Griffon para o NetBeans IDE foi atualizado para o NetBeans 7.0. O IDE reconhece automaticamente os exemplos que fazem parte da distro Griffon, assim, não é necessário importação ou configuração por parte do usuário.
